　　“可是那些唐人并没有如此做，而且他们不但没有对我们斩尽杀绝，甚至连追都没有追，这样的情况难道你就没有什么特别的想法？”阿里·伊本若有所指的问道。
<br/>　　副手起初还有些迷惑，但是很快便是眼前一亮，恍然说道：“我明白了，他们，他们那种攻击性的武器一定是没有多少了，所以他们才会在突围之后直接离开。”
<br/>　　阿里·伊本见副手已经明白了自己的意思，正色点头说道：“不错，就是这样，那种威力的武器显然不是船上可以制造出来的，所以他们的武器一定有数量限制，就算是眼下他们不是武器消耗光了，只怕也是所剩不多。”
<br/>　　“那我们应该怎么办？现在就追上去？还是……”
<br/>　　“追上去，就算是不交手，也要跟着他们，不能让他们就这样从容离开。”
<br/>　　“是，我明白了。”副手重重一点头，转身离开。
<br/>　　他们的目的就是要将大唐的太上皇抓住，所以无论是因为好奇还是因为职责，都必须追上大唐的那支舰队。
<br/>　　虽然就连这次劫人行动的总指挥阿里·伊本都不知道为什么一定要进行这次任务，但命令就是命令，就算是他们再想不通，也要把任务进行下去。
<br/>　　而且事情到了眼下这个情况，他们就算是不想完成任务也不行了，大唐可不是一般的小国，被人欺负了连个屁都不敢放，如果他们不想面对大唐百万大军，他们不只能把任务继续下去，直到抓住那个可以保住他们的‘护身符’为止。
<br/>　　……
<br/>　　而另一边，暂时脱离了大食人包围的大唐舰队也在慢慢赶往这次的目的地，那一座无名的大岛。
<br/>　　舰队的远程攻击性武器已经没有多少了，贞观炮的炮弹已经告罄，火箭弹也只余下百十颗，倒是步枪弹还余下很多，细数足有不下十万发子弹。
<br/>　　这还要归功于那些大食人见到唐军炮火过于凶猛，不敢靠近，所以才省了下来。
<br/>　　至于说其它的兵器，这个很不好说，近战用的横刀什么的虽然人人都有配备，但是眼下并这个节骨眼儿只要是个人都知道，大唐的军队并不适合打近身战。
<br/>　　孤立无援的情况下，他们需要做的就是固守，能拖一天是一天，直到李承乾那边发现情况不对，派人前来接应。
<br/>　　……
<br/>　　而此时的李承乾呢，经过冷静的分析之后，他认为不管是老头子的舰队还是南海舰队，之所以会失踪，主要的原因还应该是因为有敌人拖住了他们。
<br/>　　至于说海上风暴什么的，这一点不说有没有发生，就算是发生了也不可能所有人都死光光，总会有人逃出来，然后把消息送回大唐。
<br/>　　有了这样的判断之后，随之而来的就是如何找到老头子，或是救援或是支援总要做出一个选择。
<br/>　　李震就是在这样的情况下被派出海的，整个北海舰队带上所能携带的所有物资在最短的时间里出海，沿着老头子出海的路线一路南行，必须把老头子找回来。
<br/>　　另一方面程咬金、候君集临时受命，各率二十万大军自长安出发，一路赶往天竺道与大食的边境，一路赶过西域与拜占庭的边境。
<br/>　　而在天竺和西域的驻军也被调动了起来，整整六十万的军队枕戈待旦，加强本区防御的同时，只等程咬金与候群集一到，立刻配合他们对另外两个国家发起进攻。
<br/>　　在这个时候下达这样的命令，几乎所有人都认为李承乾一定是疯了，不管是大食还是拜占庭都不是什么软杮子，两个国家加在一起的实力并不会比大唐弱上多少。
<br/>　　如果李承乾同时对这两个国开战，所引发的后果简直不堪设想。
<br/>　　褚遂良、李义府、许敬宗等人联合上奏，痛陈厉害，希望李承乾可以打消同时向两个国家开战的意图。
<br/>　　但李承乾却像是王八吃秤砣铁了心一般，不顾所有的人劝阻，执意派出兵境。
<br/>　　最后没有办法的情况下，许敬宗等人说动了孔颖达老头子，让这位在李承乾面前比较有影响力的老家伙出山。
<br/>　　孔老头儿这个时候年龄也不小了，但是为了大唐的未来考虑，老家伙不得不再次站了来，被人有轿子抬着，入了皇宫，在两仪殿见到了已经瘦了好几圈的李承乾。
<br/>　　“老臣参见陛下！”老头子的年龄有些大，说起话来都有些哆嗦。
<br/>　　“孔师怎么来了？您这身子骨现在最好是在家里静养！”李承乾自桌子后面转出来，扶住哆哆嗦嗦的孔老头儿，让他在一边的椅子上坐下。
<br/>　　孔颖达看了一眼已经而立之年的李承乾，叹了口气说道：“眼下大唐正是多事之秋，陛下理应好好保重身体才是。”
<br/>　　“孔师，朕还年轻，这点风浪还支持得住。”李承乾知道老孔一定是有其他的话要说，强撑着打笑脸打了个哈哈，然后询问道：“孔师这交何是为了大食和拜占庭而来？”
<br/>　　“陛下既然已经知道，老臣便也不想再瞒您。”孔颖达咳嗽了一声，认真看了李承乾一眼：“那大食和拜占庭都不是什么等闲之辈，若是与他们开战，陛下可想好了退路？”
<br/>　　“退路？”李承乾微微低头，喃喃说道：“朕何尝还有什么退路，这所有的一切都是他们在逼朕，除了反击朕又能如何？”
<br/>　　“陛下，万事不要太过草率啊！古语有云，兵者，国之大事，生死之地，存亡之道，不可不察也！”孔颖达盯着李承乾，语重心长的言道。
<br/>　　“孔师，龙有逆鳞，解之必死，大唐有大唐的威严，任何敢于触碰的人都要有死亡的觉悟，大唐可以灭国，但却不会忍气吞生。”李承乾并没有抬头，不过他的话却一字不漏的落入孔颖达的耳中，让这位老学究为之一振。
<br/>　　这老头儿怎么也想不到事情会严重到这个地步，双眼死死盯着李承乾问道：“陛下，能否告诉老臣，到底出了什么事情？陛下又何至于将事情说的如此严重？！”
