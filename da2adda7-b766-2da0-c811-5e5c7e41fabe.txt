　　老者话语刚落，拍卖场中就掀起一阵骚动，无数斗者巅峰的修炼者，贪婪的望着拍卖台上的玉瓶。
<br/>　　这种能够打破壁障的丹药，本就是可遇不可求，更何况是极品丹药，比起突破，价格根本就是浮云，比起清灵丹，无疑聚气散更吸引人。
<br/>　　“极品清灵丹，竞拍价十八万，极品聚气散竞拍价八十万。”
<br/>　　“十九万！”
<br/>　　“二十万...”
<br/>　　最终，极品清灵丹以每颗二十二万的价格成交，至于极品聚气散则拍出了每颗九十五万的价格。
<br/>　　即便拍出几百万的价格，慕清霄也始终面色如常，金钱对于他来说，只不过是个数字罢了。
<br/>　　在将两瓶丹药全部拍卖出去以后，接下来的几样东西在都没有卖出什么高价，甚至几样东西一次就被拍走。
<br/>　　拍卖台上，老者脸上肌肉直抽，这可关乎着他的提成啊。
<br/>　　慕清霄靠在软椅上，一直没有药鼎拍卖，让他失望透了，好歹也是黑角域最大的拍卖行，拍的尽是些废品。
<br/>　　这话若是让拍卖行背后的老板听到，非得指着他的鼻子臭骂不可。
<br/>　　然而，拍卖台上下一件物品，却是让慕清霄瞳孔微缩。
<br/>　　“接下来要拍卖的东西，倒是略有奇特，连本行都搞不清楚，到底是何物，但经过我们的分析，估计是一份藏宝地图。”
<br/>　　说罢，白发老者取来一个银盘，掀开锦布，随后一张残破不堪的古老布片就出现在所有人的视线中。
<br/>　　望着银盘中的布片，拍卖场中在寂静过后，顿时就响起一阵嘘声。
<br/>　　布片上只有一些诡异的路线，而且极为模糊，其上也没有标志性的东西，根本算不上是地图，鬼知道拍卖行是不是用破布来坑他们。
<br/>　　老者其实也有些郁闷，残图是不是藏宝图他们不知道，但是看羊皮卷上的痕迹与诡异路线又不像是普通东西。
<br/>　　既然他们搞不懂，那就让别人去摸索，至于能不能找到点什么，那就与他们无关了。
<br/>　　看到残图的第一眼，慕清霄就认了出来，因为这样的残图，他储存空间就有两张，“净莲妖火”！
<br/>　　哪怕没有药鼎，这一趟也值了。
<br/>　　小医仙也认出了残图，一只纤纤玉手自黑袍中伸了出来，在他衣袖上拉了拉，以示提醒。
<br/>　　慕清霄反将她的玉手握在手中，挠了挠她的掌心，低声道：“我清楚。”
<br/>　　拍卖台上，瞥了眼残图，老者将他拉开。
<br/>　　“如果所料不差，这地图估计有一段岁月了，虽然不清楚其中有什么，但毕竟是上古之物，总不可能是普通货，斗气大陆上的修炼者，许多都是靠着运气与机缘，最终成为强者，这地图是你们的机缘也说不定。”
<br/>　　对于老者的话，几乎所有客人都嗤之以鼻。
<br/>　　拍卖台上，老者唾沫横飞，不断介绍地图何等神秘，只是效果貌似不大，甚至有的客人都不耐烦起来。
<br/>　　最后，老者实在是没有办法，只好开口道：“按照标价，这块残缺地图，竞拍价二十万，现在开拍。”
<br/>　　话语刚落，拍卖场中一片寂静，许多人目光扫向老者，宛若是在看一位老年痴呆，谁会花二十万去买一份毫无线索的残图？
<br/>　　哪怕再有钱的人，估计也不会买，这可是关乎着名声问题。
<br/>　　正当老者准备将残图下架时，一级贵宾包厢中响起一道平淡的声音：“二十一万。”
<br/>　　竞拍声刚响起，周围所有的目光都汇聚在包厢中，至于一楼的客人则抬头望着二楼，目光中满是呆泄，真有冤大头啊！
<br/>　　不仅是普通客户，连黑角域一些强大的势力，都纷纷投来诧异的目光。
<br/>　　拍卖台上，老者终于见有人开价，总算是松了口气，这份地图简直就是他的耻辱，现在能卖出去，他比谁都开心。
<br/>　　“这位大人出价二十一万，请问还有人加价吗？”
<br/>　　听到老者的话语，几乎所有人都翻起白眼，倒是有些强大势力的人若有所思，难道真是上古的宝藏地图不成？
<br/>　　片刻后，依旧没有人加价，老者讪讪一笑，随后便是砸下拍卖锤，一锤定音：“成交，恭喜大人拍的上古藏宝图一份，请拍卖结束后，带着贵宾卡前往后台领取。”
<br/>　　拍卖过残图过后，慕清霄能察觉到，对面的二楼甚至更高层，都有投来隐晦的目光，但他都没有理会。
<br/>　　短暂的冷场过后，待女再次端来一个银盘，老者接过，掀开锦布，其中就显出一本泛黄的书籍，在场所有视线都投射过去。
<br/>　　“玄冥掌，玄阶高级斗技......竞拍底价二百万金币！”
<br/>　　玄阶高级斗技，在这黑角域中虽算不得最高等的斗技，但也是极其的强悍了，一般只有极强的势力，才能拿出玄阶高级斗技。
<br/>　　毕竟，在上一层可就是地阶斗技了，但地阶斗技可遇不可求，哪怕有可能也落不到他们头上。
<br/>　　“二百一十万！”
<br/>　　“二百四十万！”
<br/>　　“二百八十万！”
<br/>　　拍卖行中的竞争极为激烈，最后玄阶高级斗能，被顶级贵宾室中的贵宾，以四百七十万金币拍获。
<br/>　　在玄阶高级斗技拍出后，拍卖会也接近了尾声，老者接过银盘，深吸了口气，将锦布揭开，旋即一股清香席卷整个拍卖场。
<br/>　　贵宾室中，慕清霄缓缓睁开双眼，目光向拍卖台上望去。
<br/>　　此时，老者端着银盘，望着银盘中，拳头大小的彩色果子，神色激动，道：“七彩韵灵果，六品灵果，功效想必各位都听说过，吃下去有洗经伐髓的神效，价值不可估量！”
<br/>　　话语刚落，拍卖行中一片寂静，寂静过后，喧闹声就如浪潮一般席卷整个拍卖行。
<br/>　　七彩韵灵树，六品果树，生长在魔兽山脉极深处，确实有洗经伐髓，改造资质的神效，哪怕是强如斗灵，对七彩韵灵果都是梦寐以求。
<br/>　　洗经伐髓，改造资质，那便是获得了一飞冲天的机会，资质的高低，代表着修士在往后的道路上能走多远。
<br/>　　听说七彩韵灵果成熟时，会有六阶魔兽在灵树下看守，没想到拍卖行连这种灵果都弄到了，背景真是恐怖如斯。
<br/>　　想到此处，有些想打七彩韵灵果注意的修炼者，通通都安分下来，最起码在拍卖行内，他们是不敢放肆的。
