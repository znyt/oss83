<br/>				    修真界中，种植灵植也是非常害怕虫子和昆虫，还有飞鸟等动物，所以他们制作了引豸丹，然后放入在一个小小的杀阵中，引诱所有的害虫进入杀阵后被杀死。<br/>　　<br/>　　    修真界的害虫，可不是蓝海星上这些普通的害虫能相比的，可能有些虫子对于修真者来说都是头痛的，所以才要放一个杀阵，直接将引入的东西杀死，无论是虫子还是昆虫，飞鸟等等。<br/>　　<br/>　　    引豸丹因为使用不同，所以还是分了很多等级的。在这里，陈默准备弄出最最简单的引豸丹，不仅是因为高级点的没有什么用处，而且稍稍高级的就会对一些飞禽和动物起到作用，另外还有就是有些药材手头也没有，所以制作最最低级的就是他的选择了。<br/>　　<br/>　　    低级引豸丹，可以不制作成丹药也可以，直接可以制作成药液，只对一些低级虫子和小昆虫什么的有用，炼制简单，而且所有的药物陈默手头都有。<br/>　　<br/>　　    至于说炼制，呵呵，这个就是问题所在了！陈默之所以等了这么几天，就是因为在等网络上的订货，今天早上到的。<br/>　　<br/>　　    现在网络上购物这么厉害，直接上网买个铜鼎就成。而且他也怕贵一些，只要铜鼎是全铜就成。而且一次就买了二十个，花了好几万。工艺品价格就是高了些，不像是买铜锭，按公斤卖。<br/>　　<br/>　　    铜鼎买的是现代工艺品，所以没有什么好说的。之所以买铜鼎，也是有炼制药材的要求，除了炼制引豸丹外，还有其他很多的丹药或则散药可以炼制，所以一次制作好铜鼎，就能满足后面的应用。<br/>　　<br/>　　    陈默拿出包装箱内的铜鼎，都是用打磨机打磨光滑的，这是买铜鼎的时候要求店家做的，因为买的多，而且价格高，所以店家倒是没有另外加强，反正也就是个打磨而已，手头现成的工具，二十个也花费不了多少时间。打磨光滑主要是把铜鼎所有的装饰纹路，一些不平整的地方全部打磨平整光滑。<br/>　　<br/>　　    之所以买二十个，也是因为害怕一次雕刻不能成功，那么就会毁了铜鼎，所以备份一些，多几个备份，总能成功一个不是。<br/>　　<br/>　　    在大概有排球大的铜鼎上雕刻东西，可以说普通人是不能用刻刀雕刻的，以前通过模具浇注来实现，然后在打磨等等。现在通过激光雕刻等等手段来实现。不过陈默的雕刻手段很简单，就是用刻刀雕刻而已，不过刀锋上要附上自己的真元，不然能划个痕迹还是可以的，雕刻就不要想了。<br/>　　<br/>　　    为了保险起见，陈默先将纹路用笔描在铜鼎上，然后再用刻刀可是按照纹路雕刻。主要是将符文雕刻到铜鼎上去，有好几种符文，包括控温、控火、真火、引元、分离等等符文，都是炼药所用的符文。<br/>　　<br/>　　    修真界炼药，不是用普通的火来炼制，而是通过修士的真元来炼制丹药。当然高级点的丹药还需要地火，甚至是异火才能满足炼丹。而且如果有修士收服异火，那对炼制丹药非常有帮助，不仅能加快丹药的炼制，还能增加成功率。<br/>　　<br/>　　    不过在修真界，能收服异火的也是少数中的少数，能练好丹药的修士，也是极少数。好在陈默也就是炼制一些普通丹药或者散药之类的，所以要求也不高，倒是没有什么忧虑。<br/>　　<br/>　　    陈默手里有的丹方，基本都是高级丹方，只有几个丹药是低级的，看上面的药物，基本只有少部分的药材手头上有。而且这上面的名称还不一样，只能通过图样对比后才知道是那种药材。<br/>　　<br/>　　    有一部分丹方，是传功玉符中传给他的，不过夜殇师父是个修炼狂人，而且精通阵法，但是对于炼药却没有什么研究，只是会一些简单的丹药，也就是练气期和筑基期能炼制的简单丹药，其他的就不会了。好在即使是这些简单的丹方，对于陈默来说都是巨大的财富。<br/>　　<br/>　　    言归正传，陈默将纹路在铜鼎上雕刻，是非常花功夫的。并且还要专心致志的雕刻，一旦输入真元频率不对，或者手下刻刀偏差一点，就会毁掉整个的铜鼎。<br/>　　<br/>　　    此时陈默就是这样。“嘭！”的一声，手中的铜鼎直接碎裂成几块，真元爆炸所引发的冲击还是非常强烈的，要不是陈默在房子里布置了防御阵，可能房子门绝对会被冲烂。<br/>　　<br/>　　    这已经是第四个铜鼎了，都是因为手中不稳后，刻错了一个小小的纹路后，直接引起真元爆发。好在是自己的真元，虽然冲击波有些大，但是对于此刻的陈默来说，并不会造成伤害之类的，最多是灰头土脸，只要来个洁净术就一切好了。<br/>　　<br/>　　    拿起第五个继续，他有种不达目的不罢休的意志。<br/>　　<br/>　　    第五个碎裂，那么第六个继续！每一次继续，都要饮用一杯经过稀释后的灵液水，不然雕刻这么多的纹路在铜鼎上，两三个就将他自身的真元消耗殆尽了。<br/>　　<br/>　　    最终，陈默在雕刻到第十二个铜鼎的时候，终于成功了。在收笔的霎那间，就听到一小声的“当！”响声，非常的清脆悠扬，他就知道这个铜鼎成功了，可以做药鼎来用了，当然是那种最最低级的垃圾药鼎。<br/>　　<br/>　　    “哈哈！”终于成功了，此时的心情也是兴奋的，不容易啊。<br/>　　<br/>　　    整整一天的时间，都花费在了这个上面，如果要是没有灵液的辅助话，那么时间也会极大延长，那么自己还能不能这么坚持的制作一只药鼎？恐怕是不太可能了。<br/>　　<br/>　　    现在他也明白，为什么修真界那么多人，但是很多修士却专心修炼，很少涉及其他专项。最多学习个符箓的制作，方便自己对敌等等，其他什么阵法之类的，都不会去学。<br/>　　<br/>　　    精力和时间有限啊，而且很多东西都有卖的，还不如专心修炼。陈默哪里能比，自己就是一个苦逼的修士，可能还是一个星球上唯一的修士，那么很多东西只能自己动手，丰衣足食了。<br/>　　<br/>　　    等制作好铜鼎后，其他就简单了。炼制低级的引豸丹就非常方便，药材什么的，自己手头全部都有，都是常见的一些药材。<br/>　　<br/>　　    一一按照说明，将药材按照先后顺序和手法全部提炼出进化，在按照配比，引发不同符文，将药材在药鼎中炼制。<br/>　　<br/>　　    这个过程中，陈默必须不间断的输入真元，不能有丝毫的中断或者走神。炼药是个精细的过程，要对药液的混合把握极其强悍的准确才行，所以修士的神识可以说片刻不停歇的在关注着药鼎中的药液。<br/>　　<br/>　　    陈默没有人教，所以炼制丹药只能自己摸索。因此，在炼制过程中难免出错，只能一遍遍的实验，找出出错的地方，然后在下次炼制的时候避免出错，也就是说他开始炼制丹药，也是一个糟蹋浪费草药的过程。<br/>　　<br/>　　    嗯！要是有药师看到的话，绝对说糟蹋。已经失败了十几次了，但是丝毫没有在意草药的损失，还在继续。而且灵液的稀释液也是不停的喝入，怎一个豪字能说的清。<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
