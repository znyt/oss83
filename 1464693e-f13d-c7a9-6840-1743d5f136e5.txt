　　第261章
<br/>　　李婉儿难以置信的看着这一幕。
<br/>　　“这叫怎么回事，这叫怎么回事！”
<br/>　　她眼睛睁得老大，扶住被咬破的双唇，脸颊布满彩霞。
<br/>　　雨不知何时停了，被雨洗过的天空湛蓝无比，荡去尘埃的野草萌发着新绿。
<br/>　　“这个不是很好解释的，你可以理解成一种天赋，类似公主的祝福，负面buff清除，秒回血增加，敏捷速度加成。”
<br/>　　“你说我……是药，可以治病。”
<br/>　　“倒也不是能治病啦，但对我来说确实可以称作药。”
<br/>　　马车上，程大雷比手画脚向李婉儿解释着，李婉儿抱着膝盖，时不时发出一声冷哼。
<br/>　　马车继续前行，前方道上出现一个白衣和尚，他面前摆着一个硕大的木鱼，双目微闭，一下一下敲着木鱼。
<br/>　　青叶佛孟知了。
<br/>　　毛驴拉着的马车停下了，程大雷从车上下来，来到和尚对面。
<br/>　　青叶佛似乎没注意到程大雷出现，仍旧敲着木鱼，口中念念有词。
<br/>　　“大师，又见面了。”
<br/>　　青叶佛没有抬头，仍旧默默念诵。
<br/>　　程大雷微微颦眉，道：“和尚，你念什么呢？”
<br/>　　良久，青叶佛方才抬起头，道：“地藏菩萨本愿经，这片草原死了太多人，和尚为他们超渡，以望他们早日轮回。”
<br/>　　“喔，有用么？”程大雷问。
<br/>　　“呃……”青叶佛顿了顿，道：“这个我也不太清楚。”
<br/>　　程大雷笑笑，手摁在剑柄上：“打一场？”
<br/>　　青叶佛看着程大雷摁剑的手，顿了顿道：“这柄剑已经杀了太多人了，和尚不想施主再造杀业，还望施主放下屠刀，方可立地成佛。”
<br/>　　“放下屠刀，就可以立地成佛么？”程大雷盘腿在青叶佛面前坐下，将剑搁在膝上。
<br/>　　“呃……也够呛。”
<br/>　　程大雷撇撇嘴，道：“那和尚是要放我离开？”
<br/>　　“当初是我将施主放走，不可再犯错。”
<br/>　　“打也不打，放也不放，和尚你到底想干嘛？”程大雷问。
<br/>　　“今日我们不比武艺……”
<br/>　　“比想法？”程大雷追问。
<br/>　　“呃……”青叶佛挠挠头，刚才我想说什么来着，怎么突然间忘得一干二净。
<br/>　　良久，他才让胸腔中的草泥马奔过，手中的木鱼声也停住了。
<br/>　　“陛下令我抢回公主，并且除掉施主。”
<br/>　　“那你打算怎么办？”
<br/>　　“阿弥陀佛，出家人不造杀业。”青叶佛宣了一声佛号：“但陛下的命令也不能不听。”
<br/>　　“……”程大雷。
<br/>　　“请施主帮贫僧出个主意。”
<br/>　　“呃……”
<br/>　　程大雷默然，他顿了顿，忽然仰起剑，剑锋似乎要刺破空气，耳边响起蜂鸣般的颤声。等声音消失时，剑尖距离青叶佛的眉心有三指。
<br/>　　青叶佛手掌拦在剑前，两根手指被切断，从断指处汩汩流出血来。
<br/>　　程大雷一惊，强压住情绪：“大师，何必如此？”
<br/>　　“有这两指，贫僧也可以回去交差了。”青叶佛站起，双手合十，断指仍然流着血。
<br/>　　“多谢大师。”程大雷恭恭敬敬还了一礼。
<br/>　　“现在整个天下都要杀施主，因施主是一个魔头。可这样一个人人争而相食的世道，已无人诵经念佛，缺的或许正是施主这样的魔头。”
<br/>　　说罢，青叶佛捧着他的木鱼，宣着佛号，在木鱼声中一步一步远去。
<br/>　　……
<br/>　　马车继续往前行，一路却再未遇到几个拦路之人。毕竟在茫茫草原上，想寻到芝麻粒大小的二人也并不容易。
<br/>　　再远的路终究也有个终点，这一日，马车抵达戎族的圣湖女儿湖。
<br/>　　越过这片湖，就是戎族的王城，见到那位传说中的戎族王呼延跋。
<br/>　　“大概明日我们就能到了，你还有一天时间考虑，想一想究竟要不要去？”程大雷。
<br/>　　李婉儿没有回答，沉默了一会儿，却是问道。
<br/>　　“你知不知道，在草原上，有个关于女儿湖但传说？”
<br/>　　程大雷看着面前的湖水发呆，湖边长满野草，将天空白云都映在湖水中，像镶嵌在草原上的一块蓝玉。
<br/>　　“什么啊？”程大雷随口问道。
<br/>　　“据说戎族的祖先是一个牧羊女，她在放羊时产下了一个男孩，因为当时她还没有成婚，父母便将她赶出家门，她就抱着孩子在草原上走啊走的，一边走一边流泪，泪水越聚越多，她走过的地方就成了女儿湖，那个孩子，后来成了戎族的首领，他统一了草原，带领族人抵御风暴，放牧牛羊，成了草原上第一位英雄。”
<br/>　　“还蛮有史诗感的。”程大雷想了想道：“看来未婚先孕也是能做大事的。”
<br/>　　“……”李婉儿。
<br/>　　她同样有了青叶佛面对程大雷的感觉，等着心绪平静下来，方才缓缓道：“既生在帝王家，此身便不由之主，希望我去到戎族后，帝国能利用这段时间休养生息，若日后两族能不再打仗，你也多念念我的好。就像那个牧羊女一样，虽然生的不齿，但死后却能被人写进传说。”
<br/>　　程大雷看她一眼，见她又变成了明玉公主，神色漠然，白釉般的脸颊宛若神明。
<br/>　　哎，看来基因的力量还真是强大呐。
<br/>　　程大雷刮了刮她的鼻子，道：“别想那么多了，反正到了明天，一切便都会结束了。”
<br/>　　“是呐，到了明天一切就结束了。”
<br/>　　李婉儿轻轻道了一声，将肩膀歪在程大雷肩头，向远方看去，只见夕阳西下，湖水由湛蓝变成金黄，波光粼粼。
<br/>　　夕阳渐渐沉入湖底，最后一抹天光消失时，李婉儿渐渐睡去。
<br/>　　程大雷将她脑袋挪开，把她放进马车里，摇摇头，在她身上盖上了薄毯。
<br/>　　之后，程大雷离开马车，身影在湖边消失。
<br/>　　夜来，繁星满天，寒霜初降，草原上的秋夜毕竟是有些冷的，李婉儿身体下意识窝成一团，紧缩得像只小猫相仿。
<br/>　　几乎是下意识惊醒，李婉儿从马车里坐起，极恐怖的安静压过来，周围并没有程大雷的身影。
<br/>　　她没有出声，走出马车，外面依旧看不见程大雷，天空布满星辰，湖水中点点星光，野草上寒露泛起。然后铺天盖地的黑暗和寒冷席卷而来。
<br/>　　她终于意识到一个事实：程大雷已经走了。
<br/>
