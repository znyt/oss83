　　但无奈的是，两者实力相差太过于悬殊，似乎早就知道刘哔要反抗，有马贵将冷哼一声，伸手将他一把抓了下来。
<br/>　　随后，有马眼中闪过一抹暴虐，将刘哔背后的翅膀，整个撕了下来。
<br/>　　口中发出一声凄厉的惨叫声，一股剧烈的疼痛感从背后袭来，失去了翅膀的支撑，刘哔跌跌撞撞的从半空中掉落，落在了下面的钢铁地板上。
<br/>　　没给他任何反应的机会，有马贵将的身影如同鬼魅一般，瞬间出现在了他的面前，伸手抓住他的脚踝，将其狠狠丢进了囚室中。
<br/>　　“咚！”
<br/>　　头部撞在冰冷的墙壁上，似是要裂掉了一般，刘哔眼前有些模糊，挣扎着撑起了身体。
<br/>　　有马贵将站在门前，金属门缓缓开始闭合，顾不得自身的伤势，刘哔提起体内的最后一丝力气，想趁着金属门没完全关闭之前，从中逃离出去。
<br/>　　“砰！”
<br/>　　抬起自己的右脚，将眼前的喰种踹到囚室中，有马推了推眼镜，看着眼前的金属大门缓缓关闭。
<br/>　　“喰种‘修罗’，收押完毕。”掏出了腰间的联络器，他声音异常平静，没有任何起伏。
<br/>　　……
<br/>　　从黑暗中醒来，刘哔身体四处传来如同潮水一般的疼痛感，四周静悄悄的，没有一点声音。
<br/>　　他记忆中的最后一刻，是有马将他踹入了囚室，那股庞大的力道，像是和他有仇一般，直接将他踹昏迷了过去。
<br/>　　身子底下有些冰冷，混杂着剧烈的痛感，让他感觉像是身体像是要废掉了一般。
<br/>　　但他不敢使用积分修复，他很清楚的记得，这里是CCG关押喰种的囚室。
<br/>　　那房间里必然会有监视他的探头，或者其他什么东西，他的一举一动，都在别人眼中。
<br/>　　休息了一会，等到身体略微恢复了一丝气力，刘哔挣扎着从地上爬了起来，斜靠在背后的钢铁墙壁上。
<br/>　　他知道，CCG现在之所以不杀自己，是因为他先前佩戴的小丑面具，让CCG将他误认成了小丑组织的成员。
<br/>　　但是等到自己没有任何利用价值之后，CCG必定不放过自己。
<br/>　　一时间，情况似乎进入了死局。
<br/>　　但是越到这样的地步，刘哔脑海中的思路就越加清晰，末世生存了十年，多么艰难的处境他没有经历过？
<br/>　　他坚信，任何事情都会找到解决的办法，这次困境，他会熬过去的，就像以往无数次那样。
<br/>　　幽闭的空间中，没有一丝光亮，手表在之前录入档案的时候被收掉了，他也不清楚现在的时间。
<br/>　　似乎是想到了什么，刘哔有些紧张的伸手摸了摸右手小指，稍稍放下了心来。
<br/>　　戒指还在，这倒是一个好消息。
<br/>　　这里面还躺着一具SSS级喰种的尸体，这也是他此行最大的收获。
<br/>　　也不知道自己昏迷了多长时间，吞噬技能的有效期过了没有，所幸之前的时候，他将吞噬技能升了级，有效期有了一个显著的提升。
<br/>　　虽然囚室内必然有隐藏的探头，但刘哔想要吞噬店长的尸体，也只能将其从戒指中取了出来。
<br/>　　吞噬技能发动的前提，是必须接触到目标死去的尸体，空间戒指属于次元空间，要想取出什么东西，只能全部取出。
<br/>　　囚室内的空间本就狭小，再加上店长庞大的身躯，更是让室内显得有些拥挤，但这份拥挤，也没保持多长时间。
<br/>　　“叮~恭喜宿主成功吞噬SSS级喰种，获得高阶‘喰种’血脉，获得技能‘天灾’，获得特殊技能‘赫化’。”
<br/>　　当店长庞大的身躯完全消失，系统提示响了起来，感受着体内充盈的力量感，刘哔满意的笑了起来。
<br/>　　他所吞噬的目标，除了有机会额外技能之外，目标体内未消散的能量也会被他吸收到体内，而他体内突然多出的血色能量团，就是店长体内原本的能量。
<br/>　　而让他感到意外的是，店长尸体上剩余的能量，比他想象中的要强盛了许多。
<br/>　　他所不知道的是，之所以出现这样的情况，是因为空间戒指中的时间不会流逝，店长尸体内的能量，也自然不会消散。
<br/>　　也就是说，他继承了店长几乎全部的力量。
<br/>　　如果按照喰种世界实力划分，他现在的等阶已经是SSS级喰种了，加上他原本拥有的血脉和物品，甚至比店长的实力都要强横了一些。
<br/>　　虽然，相对来说刘哔对自己力量的掌控力稍微薄弱了一些。
<br/>　　“叮~恭喜宿主融合血脉成功，获得种族技能‘再生’。”
<br/>　　就在刘哔视察自己身体时，系统的提示音再度响起，随之而来的，是身体上微微有些酥麻的感觉。
<br/>　　他的伤势，在自动进行修复！
<br/>　　眼中闪过一抹惊喜，刘哔打开了技能面板，查询着新收获的几个技能。
<br/>　　——
<br/>　　名称：天灾
<br/>　　等级：S级
<br/>　　分类：主动技能
<br/>　　功效：来源于《东京喰种》位面，可消耗能量以自身为中心，向周围发射威力强劲的能量赫子，攻击范围和威力按宿主等阶变化。
<br/>　　——
<br/>　　名称：赫化
<br/>　　级别：S级
<br/>　　分类：特殊技能
<br/>　　介绍：来源于《东京喰种》位面，可消耗能量，大幅度增加宿主身体素质。

<br/>　　名称：再生
<br/>　　级别：位置
<br/>　　分类：种族技能
<br/>　　介绍：融合喰种血脉所激发的种族技能，可提升宿主身体的再生能力。

<br/>　　总共三个技能，除了未知等阶的种族技能之外，其余两个技能的评分都是S，目前为止，这也是他评分最高的两个技能。

<br/>　　能有这样的收获，也着实超乎了刘哔的预料，除此之外，还有击杀店长所获得的一万点积分，等将这些积分兑换完毕，他的实力还会提升一截。
<br/>　　就在他正考虑要给自己兑换一些什么物品的时候，囚室一颤，一股轻微的失重感传了过来。
<br/>　　关上系统面板，刘哔脸上带着自信的笑容，抱胸静静等候着，他想，应该是有人发现了他刚才的动作。
