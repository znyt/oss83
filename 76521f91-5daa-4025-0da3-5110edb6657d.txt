<br/>　　只能看着众人，淡淡的道：“这些蛇既然那么聪明，那么我就给它们一次求生的机会”

<br/>　　几位警察领命，将雄黄粉一脚脚扫散，包围圈也不复存在。
<br/>　　这时，很多围观的学子们，自动的撤后，内心还是有一点心有余悸，生怕这些蛇发疯的冲向人群怎么办。
<br/>　　很多人都退后了，因为没有雄黄粉的压制，这一千条蛇恐怕会如秋风扫落叶一样，四面八方的席卷出去。
<br/>　　“我到要看看，这些蛇到底有多灵性”
<br/>　　这句话是白楚艳看着涂小安说的。
<br/>　　涂小安神情自若，原地不动，淡淡的看了蛇群一眼。
<br/>　　霎时间，这些蛇暴动了起来，空气中都是蛇嘶嘶作响的声音，让人听了头皮发麻。

<br/>　　白楚艳看向远远围观的学子们：“你们想上课的时候与蛇为伴吗，还有人觉得应该放蛇吗，畜生就是畜生，不知好歹”
<br/>　　“白警官，稍安勿躁”
