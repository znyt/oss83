        是、是那个连少！
<br/>
<br/>        季运恒惊愕了。

<br/>        傅夜沉身着现今最时尚的黑色西装，仍掩不住他天生的贵族气韵，黝黑的短发带着一丝的狂傲不羁。

<br/>        他那如鹰般深邃的眼眸让人无法猜测他的心思，笔挺的鼻梁，微薄的嘴唇此刻紧紧地抿着，好像在隐忍着什么。
<br/>
<br/>        季运恒莫名其妙地感到一种不祥的预感漫上心头。
<br/>
<br/>        他的隐忍，对他来说，并不是一件好事。
<br/>
<br/>        傅夜沉双拳紧握，疾步走到季运恒的跟前后，令人防不甚防地直接给了季运恒一拳打在了脸上。
<br/>
<br/>        不远处随之响起了警车的鸣笛声。
<br/>
<br/>        季运恒意识到事情的严重性，立即跳车而逃。
<br/>
<br/>        傅夜沉本想去追，又见童筱颖衣衫不整，连忙脱下自己身上的外套盖在了童筱颖的身上，接着从裤兜里掏出一把精致的军用口袋刀，撬开童筱颖双腕上的手铐。
<br/>
<br/>        “对不起，筱颖，我来晚了……”傅夜沉额冒青筋，努力压抑着自己愤怒与悲哀的情绪，低沉而缓和地对童筱颖温柔地说道，“筱颖，不管刚刚发生过什么事情，都不要紧，我不会告诉任何人。筱颖……我……”

<br/>        傅夜沉欲言又止，声音有些哽咽。
<br/>
<br/>        刚刚他驰车而来，隔着老远，他便看到季运恒坐在了童筱颖的身上。
<br/>
<br/>        他气愤得差点撞上季运恒的跑车，若不是车上还坐着筱颖，他想他一定会一气之下，将季运恒连人带车一起撞入大海之中。
<br/>
<br/>        如果他动作再快点该多好！
<br/>
<br/>        筱颖就不会失身于季运恒那个混蛋！
<br/>
<br/>        傅夜沉此刻的内心痛苦不已，甚至在深深地自我责备。
<br/>
<br/>        当初，他就不应该让连华生“治好”季运恒。
<br/>
<br/>        就应该让季运恒一辈子断子绝孙！
<br/>
<br/>        童筱颖无法体会傅夜沉此刻是何种心情，更没想到傅夜沉会误会自己已经“失身”。
<br/>
<br/>        她将手伸到了傅夜沉盖在自己身上的这件西装外套下，将那个小东西拿了出来，气愤地扔到了一旁的垃圾桶里。
<br/>
<br/>        季运恒简直就是一个变态！
<br/>
<br/>        童筱颖心里恼火又恼羞，这对她来说，真的是一种耻辱。
<br/>
<br/>        “他、他拿手机……拍了……拍了照！”童筱颖脸色惨白，双唇微微颤抖地支吾道。
<br/>
<br/>        她强忍着自己的眼泪，不让自己再轻易地落泪。
<br/>
<br/>        她不想在傅夜沉的面前哭，更不想让他看到她内心的软弱。
<br/>
<br/>        因为，他傅夜沉不是她所期待与需要的男人。
<br/>
<br/>        此时此刻，童筱颖多么希望，出现在自己面前的男人是傅昼景该多好。
<br/>
<br/>        可惜，不是，傅夜沉不是傅昼景。

<br/>        那为什么是他傅夜沉呢？

<br/>        到底为什么？

<br/>        童筱颖面无表情地回答道：“他已经传出去了！”

<br/>        刚刚季运恒拍她的时候，拍一下停一下，她猜他一定是把所拍的照片，传给了另一个人。

<br/>
<br/>        “好。”傅夜沉低沉地应道。

<br/>        此时此刻，不管她提任何要求，他都会答应。

<br/>        他刚想把她从车内抱出来，却被她冷冷地拒绝。

<br/>        傅夜沉身心一怔，再次误认为她之所以不让他碰她，是因为她刚刚“失身”，给她留下了心理阴影。
