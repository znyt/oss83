　　“萧逸…”
<br/>　　“萧逸师弟。”
<br/>　　众人惊呼一声。
<br/>　　“没事。”萧逸摆摆手，冷酷的目光，紧紧地盯着冰无影。
<br/>　　刚才冰无影那一掌，显然下了重手。
<br/>　　若非自己有寒冰铠甲护身，加之肉体力量过人的话，此刻已是重伤，失去战力的下场。
<br/>　　“哼，不识好歹。”半空中，冰无影同样看着萧逸，冷哼了一声，眼眸中却闪过一丝讶异。
<br/>　　下一秒，他的目光看向了韩冥等人。
<br/>　　“你就是韩冥吧。”冰无影满意地笑道。
<br/>　　“老夫虽多年前就坐镇于东海之滨，但还是会不时回冰武王国一趟。”
<br/>　　“老夫之孙儿无疆，没少提起你。”
<br/>　　“今日一见，果真不凡。”
<br/>　　“老国主过誉了。”韩冥行了一礼，恭敬地说道。
<br/>　　“不，并非过誉，你确实不错。”冰无影认真地说道。
<br/>　　“前些天老夫回冰武王国，听到了不少流言蜚语。”
<br/>　　“更加知晓某些不自量力之辈，挑衅我冰武王国之威严。”
<br/>　　“现今一看，那些家伙，也不过如此。”
<br/>　　“倒是你，给我们冰武王国长脸了，哈哈哈哈，好，很好。”
<br/>　　冰无影口中的流言蜚语，自然是大半年前萧逸击杀冰承天，一举成名之事。
<br/>　　说着，冰无影戏谑地看向萧逸，道，“违反盛会规则，强行攻击其它队伍。”
<br/>　　“此事，两位猎妖殿裁判亲眼所见。”
<br/>　　“小子，你说此事怎么处理好呢？”
<br/>　　“冰无影统领言重了。”两位猎妖殿裁判出言说道。
<br/>　　“剑宗天骄萧逸，毕竟并未真的出手伤到冰炎圣教弟子。”
<br/>　　“双方战斗，也并未打起。”
<br/>　　“违规，还算不上吧。”
<br/>　　两位猎妖殿裁判，还算公允。
<br/>　　“怎么？”冰无影顿时冷声道，“因为老夫及时出现，拦下了他的攻击，就算不上违规了？”
<br/>　　“难不成，老夫要等到我玄冰圣教的弟子受伤了，再出来？”
<br/>　　两位猎妖殿裁判沉声道，“规则便是如此，审度，自有标准。”
<br/>　　“我们二人只是直言罢了。”
<br/>　　“若剑宗天骄萧逸，刚才的攻击真的打到冰炎圣教弟子身上，我等自会判断他违规。”
<br/>　　“并给予登记和扣除分数。”
<br/>　　“标准？”冰无影冷笑一声，“何为标准？”
<br/>　　“标准，由我们16要塞的统领共同定下。”
<br/>　　“你们猎妖殿在这一轮，只是充当裁判，凭什么与我谈标准？”
<br/>　　“这…”两位裁判迟疑了一下，随即点点头。
<br/>　　“既然冰无影统领想将标准定得更严格些，我们也无话可说。”
<br/>　　“慢着。”恰在此时，一道身影，从远方疾速飞来。
<br/>　　来人，竟是朔风前辈。
<br/>　　朔风前辈一来到，瞬间闪身到萧逸等人身旁。
<br/>　　“可有大碍？”朔风前辈问道。
<br/>　　萧逸摇了摇头。
<br/>　　朔风点点头，随即看向冰无影，“冰无影统领，规矩，是由所有统领共同定下的。”
<br/>　　“老夫亦是统领之一，岂容你说改便改。”
<br/>　　“哈哈哈哈。”冰无影大笑几声。
<br/>　　“朔风，你也好意思前来？”
<br/>　　“东海16要塞，每个要塞两位统领，合共32位统领。”
<br/>　　“每位统领，皆是极境后期修为。就你一人，只有区区极境三重，连极境中期都未达到。”
<br/>　　“你有什么资格反对我的决定？又有什么资格在老夫面前放肆？”
<br/>　　“说句难听点的，老夫手下随便一个极境中期喊你一声统领，都觉得丢人。”
<br/>　　16要塞，每个要塞，都有两位统领，合共32位。
<br/>　　统领着东海之滨这边数以百万计的人类强者士兵，可谓位高权重。
<br/>　　炎武要塞这边，两位统领，一位是朔风前辈。
<br/>　　一位是炎武王国上代国主，是位极境后期强者。
<br/>　　“若非你裂天剑宗还算武道圣地之一，你以为你在东海要塞，能当上统领？”
<br/>　　冰无影不屑一笑。
<br/>　　朔风前辈双眼一眯，道，“冰无影，你这话敢对剑姬前辈说吗？”
<br/>　　“裂天剑姬？”冰无影原本不屑的眼眸中，瞬间闪过一丝忌惮。
<br/>　　但下一秒，这丝忌惮，又顷刻消散。
<br/>　　“哼，那个女人，实力虽强，老夫自问不是对手。”
<br/>　　“但这里是东海之滨，还轮不到她乱来。”
<br/>　　“她并非东海要塞的驻守强者，在此更无任何身份。”
<br/>　　“老夫要改规矩，她无权干预。”
<br/>　　“你…”朔风前辈，脸色顿时难看至极。
<br/>　　一旁两位猎妖殿裁判出言道，“两位统领，你们这般争执下去，并无意义。”
<br/>　　“反倒只会拖延了两支队伍的时间。”
<br/>　　“第一轮比拼，斩杀妖兽，只有七天时间。”
<br/>　　“你们确定真的要继续争吵不休？”
<br/>　　“这…”冰无影皱起了眉头，“也罢，老夫也懒得与你们计较。”
<br/>　　说着，冰无影大手一挥，将远处本来被萧逸击杀的四头黑海幽蟒的内丹收入手中。
<br/>　　内丹，交到韩冥手上。
<br/>　　“这四颗内丹，便算作你们剑宗弟子的赔礼了。”
<br/>　　“两位裁判，还请记下分数。”
<br/>　　“成。”一位裁判点点头，道，“那么，此事就此作罢。”
<br/>　　“玄冰圣教的队伍，可以退去了，冰无影统领也请离开吧。”
<br/>　　“可以。”冰无影冷笑一声，看向剑宗一行人。
<br/>　　“裂天剑宗，现在只剩剑姬那女人撑着罢了。”
<br/>　　“要不了多久，你们会沦为连二流势力都不如的乌合之众。”
<br/>　　“我看朔风你到时还有何面目坐在这统领之位上。”
<br/>　　说罢，冰无影身影一闪，彻底离去。
<br/>　　“欺人太甚。”朔风前辈，气得咬牙切齿。
<br/>　　白冰雪等人，同样脸色难看无比。
<br/>　　“黑海幽蟒，可是八级妖兽，一般队伍根本杀不了。”
<br/>　　“杀一头，取得内丹的分数，比杀一百头黑海巨妖的分数还要多得多。”
<br/>　　“足足四头的分数，该死…”
<br/>　　凌羽眼中尽是愤怒之色。
<br/>　　另一边，韩冥等人，得意一笑，同样转身离去。
<br/>　　只是，韩冥离去前，一道传言，却清晰传到了萧逸等人耳中。
<br/>　　“萧逸是吧，没想到你还是位炼药师，那群杂碎的命，硬是被你给救了回来。”
<br/>　　“但，下次可没那么好运。”
<br/>　　“我弟弟韩苍因你而死。”
<br/>　　“你们裂天剑宗弟子，休想有命离开无尽黑海。”
<br/>　　“哈哈哈哈。”
<br/>　　玄冰圣教一行人，彻底离去。
<br/>　　白冰雪等人，脸色愈发难看。
<br/>　　韩冥刚才之言，其实已经承认了黑海幽蟒之事，确是他们早有预谋，引诱而来。
<br/>　　而且，他们不会就此罢手，阴谋与算计，还会陆续有来。
<br/>　　“噗。”恰在此时，叶铭以及十数剑主，同时吐出一口黑血。
<br/>　　脸色，再度发黑。
<br/>　　十数人，甚至顷刻晕了过去。
<br/>　　“叶铭。”萧逸一惊，感知了一番，脸色难看道，“该死，黑煞之毒，竟顽强如斯。”
<br/>　　叶铭等人体内的黑煞之毒，仍旧不断侵蚀着他们的身体。
<br/>　　“怎么办？”凌羽三人，面露着急。
<br/>　　“不急。”萧逸摇了摇头，“我救治一番，问题不大，只是比较费劲罢了。”
<br/>　　凌羽三人松了口气，道，“韩冥那群小人，防不胜防，麻烦了。”
<br/>　　“他们没机会来第二次。”萧逸双眼一眯，看向玄冰圣教一行人背影的目光，杀意凛然。
<br/>　　“呵，规则是吗？”
