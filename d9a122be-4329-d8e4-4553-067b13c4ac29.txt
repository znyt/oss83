　　第587章
<br/>　　昏昏沉沉，程大雷也不知道自己昏睡了多久才醒来。脑袋恢复意识后，感觉头皮还是一阵阵发麻，上眼皮和下眼皮打架，迷迷糊糊半天才睁开眼睛。
<br/>　　就看到自己在一个黑乎乎的屋子里，点着两堆火，眼前人影来回晃动。
<br/>　　“这是……地府么？”程大雷呐呐道了一声，自己此刻好像是被绑在一根柱子上。
<br/>　　“呔！”一个声音突然喝了一声：“程大雷，你可知道这是什么地方？”
<br/>　　程大雷刚刚醒来，此时既没有力气，也没有精神。周围阴沉沉的，透着一股阴森恐怖。
<br/>　　“天堂有路你不走，黄泉无门你自来投。”那声音冷冷喝着：“这里正是幽冥黄泉，程大雷你寿元到了。”
<br/>　　程大雷脑子还有些不清楚，说话声音距离他很远，也看不真切。只是在对方声音落地后，周围传来一阵桀桀冷笑，宛若鬼哭狼嚎。
<br/>　　难道我当真已死了么。心里冒出这个念头后，程大雷竟无多少求生欲，仿佛落叶归根，本该如此一般。
<br/>　　“程大雷你生前作恶多端，死后该打入十八层地狱，挖心掏肺，永世不可超生，你可服气。”
<br/>　　程大雷一激灵，死了倒也算了，无端端为何我话要受这等折磨。
<br/>　　“那个……我大概可以算一个好人吧？”
<br/>　　“呵，你杀人无数，乃是世间一等一的恶人，你算什么好人。你空活二十余年，可做过一件好事。”
<br/>　　“这个怎么没有，当初我在青牛山，力战戎族，也算是救了无数百姓吧。”
<br/>　　“哼，亏你话有脸提起此事。你挡住戎族了么，还不是任戎族大杀特杀，最后一场大水，屠杀十万戎族。现在，十万亡魂都等着找你算账呢。”
<br/>　　“这……”
<br/>　　程大雷一怔，如果这都不算好事，自己此生好像当真没做过什么好事。大闹长安城，血洗草原，凉州聚义，并州血战……再加上这次大闹万卷山，似乎每一次，都是血流成河，杀人无算。
<br/>　　李如能搞到的毒药，当也不会是什么致命之毒，可连续奔波这么久，又一次次面对生死危机，程大雷的神经撑不住这么强大压力。
<br/>　　他应该是得了重病，本来精神状态就不太好，又被这句话刺激一番，更是意志消沉。
<br/>　　忽然哈哈笑了一阵，口中长叹道：“死了好，死了好，死了一了百了。”
<br/>　　说罢脑袋一耷拉，身上再无力气。
<br/>　　一道人影急匆匆跑了过来，口中道：“怎么了，怎么了，他吓死了？”
<br/>　　程大雷迷迷糊糊抬起头，看清了此人的模样，目光中写满悲愤。
<br/>　　“原来是你呐。”
<br/>　　等程大雷再次醒来时，环境确实好了很多。说不上多么奢侈，但至少也是有床有桌，面前还有个丰腴婆子。
<br/>　　程大雷喝了半碗姜汤，热力灌入肚中，精神好了很多。他没好气看了对方一眼，不是很想搭理她。
<br/>　　“好了，好了，许久未见，我不过就是和你开个玩笑，你瞧瞧你瞧瞧，还急上了，心胸太狭窄了。”
<br/>　　“你这个玩笑开得不错，差点把我三魂七魄给吓没。”
<br/>　　对方陪着笑：“饿不饿，不如我让厨房再给你热碗粥？”
<br/>　　程大雷问道：“不开玩笑了，你怎么会在这儿？”
<br/>　　对方也叹了口气：“这，说来就话长了。”
<br/>　　此人不是旁个，正是青牛山的小白狼。当初青牛山大战之后，程大雷的人马散为几支，程大雷带着人到凉州，把从前的老兄弟又聚起来一批。高飞虎带着老弱妇孺到江南，现在也和大家一起。还有一些人，便流落各地，他们不会干别的，估计还是做山贼的老本行。
<br/>　　小白狼就是其中之一，她沦落到青州和幽州交界，依旧占山为王，如今是白马山的大当家。她和程大雷曾经在草原上碰见过一次，算起来，二人已有三五年没见过面。
<br/>　　故人相见，提起往事都是唏嘘不已。
<br/>　　程大雷振作精神，道：“下一步什么打算，随我一起去凉州？”
<br/>　　小白狼苦笑着摇摇头：“我舍不下我手底下这票兄弟。”
<br/>　　程大雷知道问了也是白问，他如今在江湖上又不是没有名气。小白狼若想去投奔他，早就去了，不用等到今日。
<br/>　　蛤蟆城的生活虽然富裕些，但终年打仗，不说小白狼，程大雷自己也是累得够呛。
<br/>　　“天下大乱，青州也不是世外桃源，未来还是要打仗的。还有戎族在关外虎视眈眈，也不知道他们在搞什么，我这次就打算去看看他们在忙活些什么。”
<br/>　　“打起仗来我就逃，实在不行劫艘船逃到海外，等着天下太平了再回来。”
<br/>　　小白狼定是已经想透了，程大雷如何说也无用。他又简单问了一下小白狼的情况，山寨人并不多，大概就二三百弟兄，不过仗着易守难攻，这几年也是安然无事。
<br/>　　程大雷就在白马山住下了，他这场病当真不轻，在床上养了两三天才下床。身上仍旧有气无力，拥着大氅窝在墙角，像半条命的痨病鬼似的。
<br/>　　也是到了小白狼的地方，程大雷可以放松下来好好歇一阵，他打算养足了精神，再出发去戎族。
<br/>　　与幽州接壤的戎族是北蛮部，程大雷委实是和对方交过手，只是不知道北蛮部的情况怎样了，程大雷这次正好过去打听打听。
<br/>　　程大雷靠在墙角打盹，小白狼提着汤盅过来，道：“来，你多吃些，我特意去山上打的野味，你身子没好，该多补补。”
<br/>　　“再补就流鼻血了。”程大雷心情有些无语，小白狼委实照顾得不错，可再这样照顾下去，自己至少得胖一圈。
<br/>　　半途，有人将小白狼唤走了。这人是白马山的二当家，名叫张三斤，使一把钢叉，算是小白狼的的得力手下。
<br/>　　“出什么事了，你着急唤我做什么？”小白狼也是一头雾水。
<br/>　　张三斤警惕的看看四周，见没有什么人，于是压低声音道：“大当家，这个人不能留啊？”
<br/>　　“什么话？”小白狼一怔。
<br/>　　“天下谁不知道这个魔头，留他在这里，是惹祸上门啊。”
<br/>(本章完)
