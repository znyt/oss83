　　　　火炼八荒大阵之外，一众强者林立。
<br/>　　望着已经缩小到方圆数千丈的阵法，望着突然消失的众人，望着阵法中那遗留的画卷，不少人面带戏虐的笑意。
<br/>　　除了阳鼎天之外，其他人不再加持阵法，因为苏莫等人不再攻击，他们无需加持了。
<br/>　　在阳鼎天的超控之下，八荒火炼大阵继续收缩，大阵的威力急速暴增。
<br/>　　因为随着大阵缩小，火焰之力更加的凝实，所谓火焰之力才会更强。
<br/>　　时间流逝，没过多久的时间，原本庞大无比的八荒火炼大阵，已经变得只有方圆十丈大小。
<br/>　　而这个时候，大阵停止了收缩。
<br/>　　整个大阵，犹如是一个巨大的火球一般，悬浮在空中。
<br/>　　天海城彻底的消失了，原地更是留下了一个巨大无比的深坑。
<br/>　　“阳鼎天，能不能将那副图卷炼化？”二魔老向阳鼎天问道。
<br/>　　“目前还不清楚！”
<br/>　　阳鼎天摇了摇头，道：“空间宝器等级极高，就算是最低级的空间宝器都很难炼化，不过，就算无法炼化，以我等的实力，只要有足够的时间，还是能破开这画卷的空间通道！”
<br/>　　“嗯！”二魔老点了点头，
<br/>　　随即，阳鼎天双手不断的掐诀，大阵中烈火熊熊，不断的煅烧着虚界神图。
<br/>　　阵法中的火焰之力极为强大，就算是皇级上品神兵，都能炼化成铁水。
<br/>　　汹涌的火焰之力，完全将虚界神图包裹，将虚界神图煅烧的滋滋作响。
<br/>　　……
<br/>　　神图空间中。
<br/>　　无数武者遍布各处，个个面色焦急，心中充满了担忧。
<br/>　　而这个时候，苏莫正在山洞之中。
<br/>　　山洞中，玄天灵柩横亘在山洞深处，苏莫则盘膝坐在灵柩边，正在静心修炼。
<br/>　　他现在时间紧迫，只能争取时间提升修为，若是能突破修为那最好不过了。
<br/>　　嗖！
<br/>　　破空声响起，海明江飞到了山洞口，望了一眼其内安静修炼的苏莫，不由得心中有些无语。
<br/>　　这都火烧屁.股了，苏莫居然还有心情修炼？
<br/>　　海明江正欲抬脚上前，和苏莫商议脱身的办法，但刚抬起脚步，他便又停下了动作。
<br/>　　哎！
<br/>　　海明江暗叹一声，摇了摇头转身离开了。
<br/>　　他知道，这次若想脱身，还是要靠苏莫，让苏莫安静修炼，说不定能突破修为。
<br/>　　毕竟，苏莫的天赋不是常人可比，修炼速度更是奇迹。
<br/>　　苏莫一边修炼，淬炼体内玄力，一边分出一缕心神，稳定住虚界神图。
<br/>　　以他现在的修为，控制力不是当初可比，阳鼎天和二魔老等人，就算实力再强，也绝不可能轻易破开虚界神图。
<br/>　　所有，目前对于苏莫来说，暂时也不用着急，安心提升修为才是王道。
<br/>　　而在外界，火炼八荒大阵依旧在不停的炼化，一切都仿佛是陷入了僵持之中。
<br/>　　足足炼化了数个时辰，虚界神图没有丝毫的变化，大阵根本对虚界神图造不成伤害。
<br/>　　“好了，阳鼎天，无需再炼化了！”
<br/>　　这时，二魔老开口了，阻止阳鼎天继续炼化。
<br/>　　“二魔老，你的意思是？”阳鼎天不解的看向二魔老。
<br/>　　“不必浪费时间了，老夫将此图封印，然而带回圣宗，自然有办法对付！”
<br/>　　二魔老声音低沉，继续道：“然后你等按照计划，收服或者诛灭其他人族大势力！”
<br/>　　“是！”阳鼎天点了点头，他们已经有了详细的计划。
<br/>　　现如今苏莫不足为惧了，接下来就是开始对付其它势力，一统苍穹世界了。
<br/>　　随即，阳鼎天停止了催动火炼八荒大阵，二魔老抬脚上前，准备封印虚界神图。
<br/>　　就在此刻，天际的尽头，突然传来轰隆隆的巨响，空间剧烈的波动了起来。
<br/>　　“嗯？”众强者见此眸光一凝，顿时面现郑重之色。
<br/>　　下一息，只见天地的尽头有一支长枪，向此地爆射而来。
<br/>　　这长枪速度极快，乍一看还在万里之外，半个眨眼间便临近了众人千里之内。
<br/>　　这时，众人才看清，这哪里是什么长枪，分明是一根巨大的水柱。
<br/>　　这水柱长约数千丈，直径足有百丈，完全是由海水凝聚而成。
<br/>　　这水柱中，蕴含惊天动地的力量，浩荡的威势铺天盖地。
<br/>　　水柱直接碾碎一切，向阳鼎天和二魔来等人暴冲而来。
<br/>　　“什么？”
<br/>　　“这是谁？”
<br/>　　“快退！”
<br/>　　阳鼎天等人身后的那些强者见此，个个面色大变，疯狂暴退。
<br/>　　因为这水柱攻击，就算是余波，都能将他们轰杀成渣。
<br/>　　唰！
<br/>　　就在此刻，一直默不作声的大魔老动了。
<br/>　　只见大魔老一步踏出，瞬息千丈，直接冲到了水柱的前方，而后一拳暴轰而出。
<br/>　　一拳出，尸气滚滚，如同怒龙临世，威势无可匹敌。
<br/>　　这尸气之强大，就算是中阶武皇境强者，也是粘之必死。
<br/>　　瞬息之间，尸拳与巨大的水柱狠狠的碰撞在了一起。
<br/>　　轰！
<br/>　　一声惊天爆响震动天地，水柱轰然炸裂，化为了漫天水珠和冲天劲力，席卷八方。
<br/>　　而大魔老，身形则是被水柱的力量瞬间震退，倒退千丈。
<br/>　　众人见此，大惊失色，大魔老居然被击退了！
<br/>　　最震惊的人，莫过于二魔老、上官昊，以及一众黄泉魔宗的强者了。
<br/>　　因为他们都清楚的知道大魔老的实力，那可是准帝中最巅峰的存在了。
<br/>　　可以说，大魔老的实力，比黄泉魔宗的宗主都丝毫不差。
<br/>　　当然，这是以前，如今的黄泉魔宗宗主，正在冲击武帝之境。
<br/>　　而现在，大魔老居然被击退了，可见来人实力之强！
<br/>　　昂！
<br/>　　就在此刻，一声惊天动地的吼声响起，吼声如龙吟虎啸，又好似天雷滚滚，将不少阴阳神宫的弟子耳膜都震碎了。

<br/>　　这蛟龙身躯呈青紫色，长约百余里，身上浩大的龙威铺天盖地，气势之强，如同真龙临世。
<br/>　　最主要的是，这头蛟龙的额头上，长有两只短小的龙角，说明此蛟龙血脉之力已经强大到极致，开始向真龙进化。
<br/>　　一般的蛟龙，是没有龙角的，只有血脉返祖，进化为真龙，才会长出龙角。
<br/>　　“东海龙王！”
<br/>　　二魔老、阳鼎天等人见此，顿时心中一惊，面色猛然大变。
<br/>　　东海龙王之名，可是威震整个苍穹世界，因为其是苍穹世界目前公认的第一强者。
