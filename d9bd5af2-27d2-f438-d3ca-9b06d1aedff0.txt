　　叶尘见状，脸上也顿时露出一抹凝重之色，
<br/>　　如今暴怒状态下的五彩神蛤，可不是闹着玩的！
<br/>　　而且这五彩神蛤的体型再次暴涨，几乎快要将洞穴给堵住了，叶尘此时也已经避无可避。
<br/>　　“八咫镜！”
<br/>　　叶尘不及多想，当即祭出八咫镜，直接迎了上去。
<br/>　　轰！
<br/>　　八咫镜瞬间变大了数十倍，犹如一面铁墙般，挡在了叶尘和苏蔓的跟前。
<br/>　　嗖嗖嗖！

<br/>　　叶尘顿时也更急，当下也来不及多想，直接一口把扶摇仙果塞进直接的嘴里，迅速嚼了几下之后，然后朝俯身朝苏蔓的娇艳红唇吻落下去。
<br/>　　这一次，苏蔓果然没有拒绝，在媚药的影响下，反而主动开始迎合起来。
<br/>　　舌头很快纠缠在了一起，叶尘也趁机将扶摇仙果，渡入了苏蔓的口中……
<br/>　　片刻之后，
<br/>　　那颗扶摇仙果，在叶尘用这种香艳的喂食方式之下，总算让苏蔓吃了下去大半。
<br/>　　而这扶摇仙果，也不愧是绝品灵果，苏蔓服下去之后，身上的炙红之色，顿时便减轻了许多。
<br/>　　渐渐地，苏蔓也终于恢复了几分理智，这才发现自己，竟然几乎全裸的趴在叶尘的怀里，而且两人竟然还在热吻……
<br/>　　蹭~~
<br/>　　苏蔓反应过来之后，俏脸顿时蹭的一下又红了起来，而且连呼吸也变得越发急促。
<br/>　　虽然一直以来，她的确对叶尘生出了异样的情感，可对方毕竟是自己的弟弟，所以苏蔓始终无法迈过那道伦理之线。
<br/>　　尽管她也知道，自己和叶尘并没有任何血缘关系，可是两人姐弟相称了这么年，想要迈出那一步，实在是太难了。
<br/>　　再加上她后来得知，叶尘和萧若曦在一起之后，就更加不敢再往那方面去想。
<br/>　　可是如今，曾经在她梦里出现过的场景，居然真实发生了，这让苏蔓有些发懵的同时，生出了一丝别样的满足感，她竟然有些舍不得从叶尘的怀里离开了~~
<br/>　　叶尘自然不知道苏蔓再想什么，他忽然发现，苏蔓刚刚明明已经有所好转，可是一转眼的功夫，浑身再次发烫，俏脸通红，心跳加快！
<br/>　　叶尘误以为，她体内的媚毒，没有完全驱除干净，于是来不及多想，当即又开始将嘴里剩下的扶摇仙果，朝苏蔓送了过去。
<br/>　　苏蔓原本还正沉溺在某种特别的甜蜜之中，猛然察觉到，叶尘居然在往她嘴里吐东西，顿时一下子意境全无，几乎下意识的将嘴巴避开一旁，
<br/>　　“呸呸！臭小子！你往我嘴里吐了什么东西？好恶心啊！”
<br/>　　苏蔓一向爱干净，纵然她对叶尘心存爱恋，也无法忍受这种事情。
<br/>　　叶尘先是一愣，继而也反应了过来，顿时大喜，
<br/>　　“蔓姐，你总算是清醒了，看来这扶摇仙果，果然名不虚传！”
<br/>　　苏蔓不明所以，
<br/>　　“什么扶，什么果？还有，我们这是在哪儿？你刚刚对我做什么？我可是你姐，你该不会是把我……你这个禽兽不如的家伙，居然连你姐都不放过！呜呜呜~”
<br/>　　叶尘顿时一阵无语，正想要开口解释，
<br/>　　“咔嚓！”
<br/>　　另外一边，
<br/>　　圣器八咫镜，在五彩神蛤的攻击之下，已经开始龟裂开来，岌岌可危了！
<br/>　　“蔓姐，你站在我身后，不要乱跑！等我先将解决了这只臭蛤蟆再说！”
<br/>　　说着，叶尘抡起紫琼剑，便朝那五彩神蛤冲了上去。

<br/>　　随着叶尘一声暴喝，
<br/>　　轰！
<br/>　　紫琼剑之上，顿时被一团赤色火焰所包裹，
<br/>　　“天地一剑！”
<br/>　　一道上千米的剑芒，陡然在整个洞穴中升起，

<br/>　　整个洞穴一阵颤动，周围的岩壁之上，顿时便出现了出现了一道深深的裂痕，整个万毒窟，似乎随时都要崩塌了一般。

<br/>　　“这尼玛！也太硬了吧？”

<br/>　　“五彩神蛤，请神容易送神难，你现在想让我走，可没有那么简单了！”
