　　李清照看到陈博忽然瘫倒在地，连忙跑过去，关心道：“陈博你怎么了”
<br/>　　“他...他...他不是人，是魔鬼”
<br/>　　那一眼，让陈博好像被一条无比可怕的毒蛇给盯上了，好像灵魂都被咬了一口，死亡笼罩全身。
<br/>　　李清照狠狠白了他一眼，都不知道说什么好。
<br/>　　这时，斗蛇真正结束了，因为不断抽搐的帝皇眼镜蛇变成了浑身干枯的尸体，毫无生命气力，已经没有反应了。
<br/>　　好像死了几个月一样。
<br/>　　紧接着，所有人撑着惊骇的目光，看着红白相间的小蛇又从干枯的尸体中轻描淡写的钻了出来。
<br/>　　“嘤嘤嘤...”
<br/>　　小蛇刚钻出来，欢乐的叫唤了几声，像是对着年轻男子在邀功。
<br/>　　如今，迷你精致的小蛇，再也没有人敢小看它了，反而大家看向它的眼神都带着一点胆寒。
<br/>　　变态的小蛇，钻进身体，吸光血液，这是无比阴邪的手段呀。
<br/>　　邪蛇，妖蛇也...
<br/>　　可这句话，没人敢说出来，今天是蛇王节，已蛇为尊。
<br/>　　白镇的人没觉得这蛇如何邪恶，只觉得这条蛇很厉害，不同凡响。
<br/>　　大蛇想吃它，它更厉害，直接钻入大蛇的身体内，吃光你的五脏六腑，吸光你的血。
<br/>　　看看谁厉害。
<br/>　　“现在我可以走了吧”
<br/>　　年轻男子忽然冲着呆若木鸡的李清照开口。
<br/>　　呃...
<br/>　　李清照半天反应不过来，待她回神之时，年轻男子已经越过了她，朝着长蛇形的大部队走去。
<br/>　　“你的蛇”
<br/>　　年轻男子走了，但是却没将地上直立，充满灵性的小蛇带走。
<br/>　　李清照的话喊着，可年轻男子身形连停顿都没有，继续朝前走，好像不要这条蛇一样。
<br/>　　但令人诧异的是，这条嘤嘤叫的小蛇，自己蹦蹦跳跳的朝着年轻男子追了过去，然后已一个诡异的弧线，高高跃起，跳到了年轻男子的肩膀上，一人一蛇，很快消失在人海中。
<br/>　　“这人到底是谁”
<br/>　　李清照发傻的看着一人一蛇消失，不仅仅是他，现场很多围观的群众都开始猜测这年轻男子的身份。
<br/>　　手中能有如此厉害的蛇种，而且能将此蛇驯服到这种听话的地步。
<br/>　　百年难得一见呀，白镇什么时候出了这样的少年英才。
<br/>　　这不过是蛇王节的一个小插曲，但是见识过的人，却终生难忘。
<br/>　　难忘男子的神情自若，难忘那条小蛇的诡异手段。
<br/>　　大家都大饱眼福，开了眼界，这场斗蛇本来以为是小孩子过家家，没什么好看，没想到既然如此的戏剧性跟精彩。
<br/>　　一条不知品种的珍稀蛇种，已残忍的方式杀死了它的对手，帝皇眼镜蛇。
<br/>　　帝皇眼镜蛇输了，也丢了小命，而他的主人陈博更是丢了魂一样的瘫坐在地上，茫然若失，跟傻子一样。
<br/>　　“陈博，你还好吧”
<br/>　　李清照将想他扶起来，但是高大威猛的陈博，如何是她一个小女子能扶的起来。
<br/>　　这时，她看了一眼自己的直播间，更是吓了一大跳，无数弹幕层出不穷，真实在线人数，一百五十万。
<br/>　　我的妈呀...
<br/>　　人海中，年轻男子闲庭信步，刚才发生的一切好像对他来说，不值一提。
<br/>　　倒是此时走在拥挤的人海中，让他隐隐很高兴，很兴奋。
<br/>　　这是人气，人的气息弥漫，而他置身其中，悄悄的融入。
<br/>　　“我涂小安终于可以在人群中倘若自若的行走了”
<br/>　　没错，这清秀而邪魅的年轻男子，不是别人，正是已购买系统商城化形术而成功化形的涂小安。
<br/>　　他不用在当一条黑暗中游走的毒蛇，光天化日下，他也一样可以行走在人类的街道上。
<br/>　　不用提醒吊胆，不用鬼鬼祟祟的怕被人发现。
<br/>　　他行走的这具身体其实是虚假的，不过是一个躯壳，没有血与肉，甚至碰到强大的外力，会立刻的溃散，恢复本体。
<br/>　　但那有怎样，如今他行走在人海中，没人会发现他不过是一条毒蛇。
<br/>　　化形之后，让涂小安有点诧异的是，他的容貌发生了翻天覆地的变化，完全是一副陌生的面孔，不是原本身生前的模样。
<br/>　　对于这一点，他是既失望既高兴。
<br/>　　失望自然是身体发肤受之父母，要是可以的话，涂小安岂能不希望恢复本来面貌。
<br/>　　但是涂小安这个人已经在白镇人民心中死掉，一个死人怎么可能活过来。
<br/>　　要是正常面孔，他也没办法现在行走在人海中，要是被熟人看到，一定会被吓的大喊鬼呀。
<br/>　　高兴也是这点，现在他这副面孔是完全陌生的，对他自己都是陌生，更不要说是别人了。
<br/>　　凡是有利就有弊，但能在脚踏实地，涂小安真的很感恩了。
<br/>　　至于遇见李清照这个小丫头，也让涂小安有点诧异，当初赵宅村，她见识了这个丫头的爱蛇如命。
<br/>　　居然对着变异鸡冠蛇说，你可以当我的家蛇吗。
<br/>　　如今在遇见，这丫头死皮赖脸的本性一点都没有变，一番纠缠下，涂小安就只能恶趣味的陪她玩一下。
<br/>　　白皙手掌前伸，肩上一抹红顺势滑落，红白相间的小蛇出现在他的手掌中，嘤嘤的怪叫。
<br/>　　“干的不错，没给我丢人”
<br/>　　涂小安看着手掌中的小蛇微微一笑，口头奖励。
<br/>　　别看这条小蛇筷子长短，但它可来历不凡，什么珍稀蛇种？
<br/>　　已涂小安现在的身份，区区珍稀蛇种岂能入眼。
<br/>　　这条可是足足有十五级的觉醒类蛇种。
<br/>　　而且还是异变而成。
<br/>　　论血脉血统，不会比赵宅村的变异鸡冠蛇逊色，甚至还比最开始的吸血蛇要强上不少。
<br/>　　此蛇也是涂小安在蛇谷准备打道回府的时候遇见。
<br/>　　他在蛇谷登基坐殿，无数蛇种顶礼膜拜，其中好蛇无数，在涂小安眼中的好蛇起码是觉醒蛇种。
<br/>　　这条叫变异太阴蛇，只所以能被涂小安眷顾走带，是因为这种蛇是太玄蛇的进化体系。
<br/>　　别看那么小，但却非常的厉害。
<br/>　　别的蛇越长越大，它是越长越小，违背大自然的生存法则。
<br/>　　但这一点完全不影响这条蛇的实力，它可以做到杀人与无形，万军之中取上将首级，如探囊取物一般。
<br/>　　涂小安很喜欢这条蛇，一眼，便直接将他收入系统带走。
<br/>　　都知道太玄蛇为蛇中宝药，而这条由太玄蛇进化的太阴蛇，更是变态，它甚至可以将一个刚死之人救活。
<br/>　　当然，这个代价，也是一命换一命。
