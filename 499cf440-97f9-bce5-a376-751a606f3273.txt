
    主动发公告，这应该是一年中的第二次了，就说两个事，一个是书，一个是网站十佳作品评选。<br/>　　<br/>　　    先谢谢昨天帮着盯榜单的朋友们，熬夜坚持，感谢哈。<br/>　　<br/>　　    先说赊刀人这本书，评论里有说写的精彩，自然也有说一般的，有说很新颖也有说不如前面两本的，反正是哪种声音都有，不过从我的角度来看呢，就是这本书没打算按照以前的路子去写，想写成生死突击那种热血沸腾的，简单啊，我照搬就可以了换汤不换药，想写古井观那种内涵感情戏的，也简单啊，继续照搬，换汤不换药。<br/>　　<br/>　　    但是真没想这么写，就想着能不能给大家带来新的新鲜感和另一种体验，所以尝试了这种写法，可能有的满意有的不满意，那我真的只能说声抱歉了，如果这本这么写不成功的话，下本我再换回来。<br/>　　<br/>　　    再写灵异我是尽量避免于古井观的情节有重合的地方，但两个作品都是我写的，又都是同一个题材，有相似或者写重了的真的挺难避免的，但你们应该看出来了，很多梗和情节都是新出的，做到了七成左右不撞车的现象，这个挺累的，真的要费尽心思去琢磨，不然写重复了就难免会有人说我偷懒了。<br/>　　<br/>　　    古井观里关于向缺和陈夏的感情我写的挺含糊，属于一笔带过之后，结尾才隆重的点了下，所以赊刀人中王惊蛰和小草的感情描写我着重突出了许多，甚至很多章节都写的特别细腻，就是想弥补下没写感情戏的缺点，生死突击中的很多情节都是各种反转，计策谋略什么的的，可能看着烧脑也累，所以这本的情节就放轻松了很多，不太那么麻烦了。<br/>　　<br/>　　    这本书的节奏是放慢了写的，但绝对是用心了的，有不满意的请海涵，有满意的我会更加努力，我的态度始终都是端正的，继续与大家同行。<br/>　　<br/>　　    说说年会十佳作品评选，这个比赛呢，第一年初出茅庐，古井观参选拿了个第十，去年生死突击荣登探花，这得感谢各位的大力支持了，名字就不提了，免得疏忽下漏了谁不好。<br/>　　<br/>　　    今年赊刀人再次挺进决赛，煽情的话啥也不说了，名次肯定是想越争越高，有能力的给个精油，守护什么的都行，觉得一般的给肥皂也可以，都不挑，如果啥也不给的也无所谓，给个好评什么的博个面子，众人拾柴火焰高，赊刀人的这些读者一起努努力，我肯定是能站在金字塔上面的，然后参加年会的话，也有面子是不是<br/>　　<br/>　　    主要是，奖金也挺丰厚的，写书是赚钱养家糊口，想赚的多也正常，所以请大家帮帮忙，基本上我一年也就参加这一次活动，平时从来不开单章拉人气拉票的，这次郑重其事的拉票，还望兄弟姐妹们能力挺老困，谢谢。<br/>　　<br/>　　    今天的更新晚点，九点半左右了，下午可能才开始写，等不及的明天看，在我这里没有断更的现象。<br/>　　<br/>　　    好叻，希望大家尽点力，送困哥名次上去，谢谢
