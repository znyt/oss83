        从红布之中拿出两尺短刀。<br/>    <br/>    　　比正常长刀略短，刀身弧线更加流畅，在光线的照射下，散发出淡淡的金色光晕。<br/>    <br/>    　　来到木桩下，凝练气势。<br/>    <br/>    　　修刀要比修剑难度更大，剑有剑势，刀有刀势。<br/>    <br/>    　　能领悟出来剑势，算是天才，领悟刀势，属于妖孽。<br/>    <br/>    　　当然，他现在这个境界，很难领悟出来刀势，将准确度提升到百分之百，才是他目前急需修炼的东西。<br/>    <br/>    　　出刀！<br/>    <br/>    　　收刀！<br/>    <br/>    　　没有任何武技可言，单纯的拔刀式，简单直接，一刀划出，空气出现一阵波动，产生强劲的气流，涌向两侧的树木，发出沙沙声。<br/>    <br/>    　　“还是不够快，刀的准头提升不少，速度依然无法达到我想要的效果。”<br/>    <br/>    　　收刀而立，拿起白色毛巾，擦拭脸上的汗水，短短一个时辰，出刀一万次，平均一分钟出刀八十三次。<br/>    <br/>    　　这种高强度修炼，没有强悍的肉身，普通人根本支撑不下来。<br/>    <br/>    　　几乎压榨了身体所有潜力，累的像是死狗一样坐在地面上，身体中的真气，消耗殆尽。<br/>    <br/>    　　艰难的站起来，剩余的三分之一淬体液全部倒入桶中，脱掉衣服钻进去。<br/>    <br/>    　　钻心的痛处，袭遍全身，肉身枯竭状态，更适合用淬体液来打熬，每一条筋脉，每一个窍穴，每一寸毛孔，都在贪婪的吸收。<br/>    <br/>    　　运转太荒吞天诀，浓郁的灵气，犹如活过来，从四面八方涌入柳无邪的院子，这几天下人一直私下议论，徐家上空的灵气，被某种神秘的力量给抽走了。<br/>    <br/>    　　筋脉传来战鼓声，这又是要突破的征兆，灵气进入身体，融入吞天神鼎，里面的液体越来越多。<br/>    <br/>    　　“肉身已经达到极限，可以突破了。”<br/>    <br/>    　　倒出一粒天灵丹，一口吞服下去，每一重境界，打磨的圆满无暇，才会选择突破下一重境界。<br/>    <br/>    　　灵液倒出，融入四肢百骸，丹田中的真气，飞速暴涨。<br/>    <br/>    　　连堂堂仙帝都搞不清楚，吞天神鼎里面的液体，到底是如何形成，随着修为的提升，液体的纯净度也在提高，凝练液体的速度在减缓，需要寻找灵气更浓郁的地方修炼。<br/>    <br/>    　　“要尽快布置一座聚灵阵才行！”<br/>    <br/>    　　炼丹师：炼制丹药。<br/>    <br/>    　　灵纹师：刻画符咒、符箓等，同样用于炼器当中。<br/>    <br/>    　　还有一种职业，阵法师。<br/>    <br/>    　　沧澜城四大家族，松家以阵法闻名，他们对阵法一途，研究颇深，排名四大家族第一位。<br/>    <br/>    　　四大家族分别是徐家，田家，松家，万家。<br/>    <br/>    　　徐家属于新晋贵族，崛起不过几十年而已，人丁稀少，这三家可是老牌家族，弟子众多，产业更是遍布大燕皇朝。<br/>    <br/>    　　田家主要经营妓院，赌坊之类的生意，最近几年涉足兵器行业，抢夺徐家的市场，一个家族长期发展下去，单一的产业，绝对不行。<br/>    <br/>    　　只有万家，在柳无邪的记忆当中最为特殊，他们是驯兽家族，掌握独特的驯兽之道，曾今驯服一头四阶妖兽，扬名大燕皇朝。<br/>    <br/>    　　四阶妖兽，等于人族洗髓境，强大的一塌糊涂，万家的产业以驯兽场、斗兽场居多。<br/>    <br/>    　　每个家族，都有自己的立足根本，相互之间，互不干扰，这次田家的做法，打乱了沧澜城几十年来的平静，一场暗波，悄然而至。<br/>    <br/>    　　真气终于不再增长，一股强横的气浪，冲击后天六重桎梏，如同一记闷锤，强行撬开这道门户。<br/>    <br/>    　　换成常人，每一次突破都要小心翼翼，柳无邪则不是，用最狂暴的方式，他的肉身，得到神秘液体改造，早已异于常人，不必担心肉身崩裂的情况发生。<br/>    <br/>    　　从夕阳到日落，天色渐渐暗下来，穿好衣服。<br/>    <br/>    　　“呼呼呼……”<br/>    <br/>    　　一拳接着一拳，施展基础拳法，脑海之中有太多的精妙拳法，却一个没有动用，他现在最重要利用基础拳法，锤炼肉身，稳固后天六重境。<br/>    <br/>    　　简单朴实，没有华丽的招式，每一拳都是虎虎生风，打得空气乱撞，吹向四周。<br/>    <br/>    　　七星连步，每走一步，地面上的树叶都会卷起来，随着他的身体而舞动，此刻要是有人经过这里，一定惊讶的张大嘴巴，这是先天之灵。<br/>    <br/>    　　领悟先天之灵，才能调动天地法则。<br/>    <br/>    　　每一拳，每一式，行如流水，浑然天成。<br/>    <br/>    　　最后一拳出击，空气突然传来一阵炸鸣，这是空气被急速压缩之后形成的爆鸣声，唯有速度加上力量凝聚到一个点的时候，才能发出这种声音。<br/>    <br/>    　　“舒坦！”<br/>    <br/>    　　收拳而立，出了一身的汗，来不及换衣服，抽出短刀，继续修炼刀法。<br/>    <br/>    　　出刀的速度，要比回来之前，快了很多，准确度也在提升，几乎每一刀都能落在印记上，接下来可以修炼血虹刀法了。<br/>    <br/>    　　“姑爷，老爷喊你去用餐了。”<br/>    <br/>    　　不知不觉，天色已经暗下来，管家的声音在远门外响起，招呼他去用餐。<br/>    <br/>    　　“好，我换完衣服就过去。”<br/>    <br/>    　　简单梳洗一遍，连续修炼，身体里面排出来许多污垢，清洗之后，换上干净的衣袍，朝内厅走去，好久没有一家人用餐了。<br/>    <br/>    　　岳父，岳母，还有徐凌雪已经等候多时。<br/>    <br/>    　　今天发生的事情，让岳母对他的态度大大改观，主动站起来，替柳无邪盛好了米饭。<br/>    <br/>    　　“谢谢岳母！”<br/>    <br/>    　　柳无邪的话很少，以前每次过来用餐，没少遭受岳母数落。<br/>    <br/>    　　徐凌雪端起碗，吃饭的姿势都是那么优雅，夹起一块肥而不腻的樱桃肉放入口中，发出轻微的咀嚼声，换成往常，每次柳无邪过来吃饭，她都会提前吃完离开，两人很少打照面。<br/>    <br/>    　　这应该还是第一次同桌用餐。<br/>    <br/>    　　“无邪，这是万家送来的请帖，我最近几天都很忙，没时间过去，你跟凌雪明天替我去一趟。”<br/>    <br/>    　　徐义林拿出一份请帖，放在柳无邪面前，最近家族发生太多的事情，炼器大师傅被田家挖走，兵器坊险些被人陷害，每天忙得焦头烂额，这种不必要的宴会，只好由柳无邪去参加。<br/>    <br/>    　　“万家的请帖？”<br/>    <br/>    　　柳无邪放下碗筷，万家跟徐家少有来往，突然送来请帖，有些错愕。<br/>    <br/>    　　“是这样的，万家最近新引进一批妖兽，搞了一个百妖会，邀请沧澜城各大家族族长跟一些青年才俊参加，我听说帝都城也有人前来，我们徐家想要发展，必须要跟万家搞好关系，你们只负责看，负责听就行了。”<br/>    <br/>    　　徐义林简单说了一遍，万家靠驯兽场跟斗兽场为生，每一次引进新的妖兽，都要搞一次规模很大的聚会，来提升斗兽场的名气。<br/>    <br/>    　　按照徐义林的意思，他们两人只是走走过场，给万家一个面子，多看，多听，多接触外面的世界。<br/>    <br/>    　　柳无邪抬起头来，正好徐凌雪也放下碗筷，四目对视，很快又分开，谁也没有反对。<br/>    <br/>    　　岳父已经做出决定，想要拒绝也来不及了，距离丹宝阁交丹药的日子还有三天，炼制天灵丹还来得及，先去百妖会，柳无邪也想见识一下。<br/>    <br/>    　　真武大陆许多事情，对于他来说，还是很陌生，想要融入这个世界，首先要去了解这个世界。<br/>    <br/>    　　“岳父，岳母，我吃完先回去了。”<br/>    <br/>    　　匆匆吃完，柳无邪站起来，朝两人鞠了一躬，转身离开。<br/>    <br/>    　　“爹，自那晚之后，他好像变了。”<br/>    <br/>    　　直到柳无邪走远，徐凌雪突然说道。<br/>    <br/>    　　说不出来，就是感觉他变了，更沉稳，眼神更睿智，尤其是眼睛，像是两颗星辰，里面包含太多的故事。<br/>    <br/>    　　“我也发现了，人总要长大，他是柳大哥的后人，不可能一直废物下去。”<br/>    <br/>    　　徐义林归结于他长大了，三人陷入短暂沉默。<br/>    <br/>    　　回到院子，来不及休息，现在每一刻对他来说，都很宝贵，点燃火种，开始炼制天灵丹。<br/
