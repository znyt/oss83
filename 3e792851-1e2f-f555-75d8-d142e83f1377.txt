剑无眼，万一我们的学员出现伤亡的话怎么办？”男老师理直气壮的说道。<br/>　　<br/>　　    “这……这……”巴特利特被男老师的话顶的有些不知所措。<br/>　　<br/>　　    这已经超出了他的认知了，怕出现伤亡就放弃比赛，这个理由虽然牵强，但是也算是说的过去，只是让人有些难以接受。难道和其他的参赛队伍比赛就不会出现伤亡吗？难道成为一名强者不需要付出代价吗？不过这话显然并不适合现在问出来。<br/>　　<br/>　　    “怎么了？主持人，难道不允许弃权吗？”男老师皱了皱眉头看向了巴特利特。<br/>　　<br/>　　    “这……比赛并没有类似的规定。”巴特利特犹豫了一下说道。教皇已经走了，他现在就是最大的负责人了。<br/>　　<br/>　　    “没有规定就是可以呗！难道打不过还不让人弃权，非要斗个你死我活才行吗？我倒是很期待你们神佑联盟的三所学院碰头的样子，有本事你们遇到的时候不要像我一样主动弃权！”男老师轻蔑的扫了一眼扫了一眼天罡学院和王者学院。<br/>　　<br/>　　    傻子都知道有圣女的光明学院可是夺冠的热门队伍，也是神圣教廷的专属学院。天罡学院和王者学院如果遇到光明学院的话，一定会毫不犹豫的选择让路的。<br/>　　<br/>　　    神佑联盟虽然不是一个国家，但是教皇陛下一言九鼎，高度的神权集中，要比卡罗帝国的皇权更有凝聚力。<br/>　　<br/>　　    别人也许不知道，但是德利莱卡很清楚育才书院为什么会选择放弃。育才书院和其他两所学院不太一样，育才书院看名字就知道，和其他两所学院有很大的不同。<br/>　　<br/>　　    育才书院主要侧重的是文，而非武，育才书院培育出来的优等生，几乎都在卡罗帝国任职，当然都是文官。育才书院的院长是卡罗帝国有名的大学士魏文星，那可是桃李遍天下的人，很多身居要职的文官见到他都会尊称一声老师。<br/>　　<br/>　　    育才学院这一次来其实主要就是为了卡罗皇家学院打个前站，他们的目标本来就是完成一场战斗，尽量将对手的实力逼迫出来，给卡罗皇家学院的人增加一些参考。<br/>　　<br/>　　    只是没有想到一上来就遇到了卡罗皇家学院，一个是为了保存卡罗皇家学院学员的体力，再一个也是防止他们暴露更多的实力，于是他们果断选择了放弃。<br/>　　<br/>　　    巴特利特犹豫了一下，最后点了点头，同意了育才书院的弃权申请。如此一来，相当于是卡罗皇家学院也变相的抽到了一张免战晋级卡。<br/>　　<br/>　　    如此一来最失望的就是观众了，一些观众已经开始叫嚣着要取消卡罗皇家学院的比赛资格。不过显然他们都要失望了，卡罗皇家学院并没有违规，所以比赛依然进行，卡罗皇家学院在众人不屑的鄙夷声中晋级了。<br/>　　<br/>　　    面对现场观众如潮水般的嘘声，张大友身为卡罗皇家学院的队长还想要反驳两句，不过被李振邦给拦下来了。<br/>　　<br/>　　    “振邦，你为什么要拦着我？”张大友有些不满的大喊道。<br/>　　<br/>　　    “你觉得你一个人能反驳的了所有人吗？这可是至少二十万的观众！”李振邦拍了拍张大友的肩膀，大声的吼道。<br/>　　<br/>　　    并不是张大友和李振邦在吵架，而是现场的嘘声太大了，不大声吼叫根本听不清他们彼此在说什么。<br/>　　<br/>　　    张大友扫视了一眼还在口水满天飞的观众们，心中升起一股无力感。这么多人，一人一口吐沫都能把他淹死几回了。<br/>　　<br/>　　    “可是我总觉得自己应该做点什么！”张大友皱着眉头吼道。<br/>　　<br/>　　    其实张大友的态度代表了很多学员的态度，这一次他们虽然赢得了胜利，但是毕竟赢的并不光彩。而且现场这么多的观众的嘘声和嘲讽声，让他们心中很不舒服。<br/>　　<br/>　　    “把你们心中的不满和怨愤，都用在下一场比赛上吧！用我们可怜的对手来封住他们的嘴就好了！”李振邦露出了一丝有些残忍的微笑……<br/>　　<br/>　　    天罡学府和暗夜学院之间可没有那么和谐了，两所学院分属于神佑联盟和暮夜联邦，彼此之间自然不会手下留情。<br/>　　<br/>　　    双方按照要求上交个人赛名单以后，分别有一名参赛选手登上了擂台。<br/>　　<br/>　　    看到有人登上了擂台，观众席上对卡罗皇家学院的嘲讽才渐渐平息下去，他们花钱就是来看比赛的。一共只有四支队伍，结果一下子少了两支队伍的比赛，他们心中有不满也是可以理解的。<br/>　　<br/>　　    如果不是今天看到了神龙见首不见尾的教皇，恐怕事情就会变得更加复杂一些了。<br/>　　<br/><br/><br/>
