　　所谓蛇上身，就跟鬼怪之说中鬼上身是差不多的一个意思。
<br/>　　让你的身体内有了另外一个自主的灵魂，然后将你的身体霸占。
<br/>　　往往在自然界，动物附体早有存在，最为知名的就属黄鼠狼了。
<br/>　　民间早有传闻，得罪什么动物，也不要轻易得罪黄鼠狼，这种动物是非常有灵性，被誉为地仙。
<br/>　　它睚眦必报，你得罪了它，它就会附你的身，做出各种古怪的事情，让你生不如死。
<br/>　　民间五大仙，黄鼠狼黄仙，蛇柳仙，都是天地间十分灵性的生物。
<br/>　　尤其觉醒类，那么更是不能轻松得罪，哪怕你是不小心得罪，也是祸从天降。
<br/>　　涂小安这时也明白了为什么自己在舒瑶的家里没有察觉到蛇类的气息，因为这条蛇精是上了陈响的身。
<br/>　　别看面对面，他依然蛇类气息浓郁，但由于不是本体释放，所以不会在环境中残留太久。
<br/>　　另外具涂小安的猜测，蛇精也不是百分百的控制了陈响，应该是阶段性的一段时间内而已。
<br/>　　不然当初舒瑶怀孕，他们去医院检查，陈响也不会被吓跑，那时候显然他是清醒的。
<br/>　　应该是晚上，阴气重，蛇精就冒出来了，霸占了陈响的身体，从而控制他。
<br/>　　因为现在就是夜晚。
<br/>　　消化了这些信息，涂小安也差不多弄清了前因后果，淡淡的看着一脸乖厉的蛇精，道：“如此说来，你是上了陈响的身，才让舒瑶怀了蛇胎，也就是说这变异双头蛇是你的后代”
<br/>　　“没错，只有本仙强大的基因，才能创造如此强的蛇种”
<br/>　　蛇精眉飞色舞，非常的得意，地板上昂首而立的双头蛇，让他怎么看，怎么满意。
<br/>　　“谢谢你”
<br/>　　金银蛇突然冒了这样一句话，将蛇精听的二丈和尚摸不着头脑，这句谢谢是几个意思呀，完全没懂。
<br/>　　涂小安笑了，妖冶的蛇眸中化过一丝玩味：“在你破门而入的时候，我刚好将这条双头蛇收为宠物，你放心，我会好生培养它的”
<br/>　　“宠物？”
<br/>　　蓦地，蛇精怔了一下，短促而痉挛地呼了一口气，我特么煞费苦心繁衍出的后代，被你收做宠物？
<br/>　　霎时间，整个套房内狂风肆意，在蛇精稍稍一愣的瞬间杀意暴涌而来，若狂风飚射。
<br/>　　“放肆，本仙的孩子，岂会当你的宠物，胡说八道”
<br/>　　涂小安一双蛇眸此时像是两汪平静无波的湖水，淡然冷漠，看了看双头蛇：“过来，跪下”
<br/>　　蛇精浑身一僵，眼睛死死盯着双头蛇，不可能的，它的孩子何等高贵，岂能当别人的宠物。
<br/>　　变异双头蛇倏然的动了，游到了金银蛇面前，直接匍匐在地，两颗蛇口趴在地上，像是对自己的王叩首膜拜，非常听话。
<br/>　　呀...呀...呀...
<br/>　　此一幕如眼，蛇精疯狂咆哮起来，简直要原地爆炸了，浑身狂暴的蛇类气息疯狂的爆涌出来，空气中炸起了阵阵爆响声。
<br/>　　“你敢对本仙的孩儿做手脚，本仙要活吞了你”
<br/>　　只见“陈响”面目狰狞，张牙舞爪的朝着金银蛇冲了过去。
<br/>　　“惊雷”
<br/>　　涂小安斜睨了蛇精一眼，浑身电光闪烁起来。
<br/>　　劈...
<br/>　　一道白引如撕裂空间，光芒一闪，霹雳一声的直接轰在了“陈响”的身上，将他一头无风自动的头发直接炸立。
<br/>　　瞬息间，他浑身抽搐，全身电光流转，整个人像得了羊癫疯一样。
<br/>　　“给我从他的身体内滚出来”
<br/>　　金银蛇悬浮起来，一双蛇眸冷淡无波却透着丝丝杀意。
<br/>　　多么可笑无稽的一个故事，所谓的蛇胎，原来只是一条母蛇被炖成蛇羹引发的凶案。
<br/>　　涂小安现在很生气，哪怕蛇精将陈响给杀了，他都不会那么的生气，舒瑶何辜，她跟整个事情毫无关系，她不过是一位向往美好生活的柔弱女子，想成为别人的妻子，想成为一位小天使的母亲。
<br/>　　蛇胎，对于一个女人是何等的残忍，比杀了她更有所不及。
<br/>　　涂小安深深的同情她的遭遇，她这是被蛇日了？
<br/>　　何等的荒诞。
<br/>　　用人身孕育蛇胎，亏蛇精想的出来，简直脑洞大开。
<br/>　　最可笑的是，还被它成功了。
<br/>　　如此还得了，世界岂不是要乱了。
<br/>　　“还不给我滚出来”
<br/>　　劈...
<br/>　　又是一道雷电，天雷滚滚带着威慑劈在了蛇精的身上，将他劈的直接双膝跪地。
<br/>　　任这蛇精在强，它附身在人的身体内，实力是发挥不出来的。
<br/>　　别看气息很强，那不过是虚有其表而已。
<br/>　　“你...你，你居然可以掌控雷电”蛇精浑身哆嗦后诧异的说出，旋即他好像想起了什么，惊讶道：“你莫非就是白镇人口中的那条掌控雷电的蛇王”
<br/>　　涂小安也不否认：“你敢在本蛇王的地界上胡作非为，找死”
<br/>　　但凡觉醒，懂得修炼，作风行事有两种，古往今来皆如此，一种是多行善事助增修行，一种是胡作非为，盗吃牲畜，食人脑髓，幻化惑人，堕落邪道。
<br/>　　人有正邪，觉醒类的生物，也是正邪之分，拿蛇来说，白镇人民之所以崇拜蛇王，是因为它是正蛇，积功德，修正果，为世人排忧就难，指点迷津，更能保一方百姓平平安安。
<br/>　　显然，这条蛇精如此的行为已然不是正蛇，邪蛇不为过。
<br/>　　“哈哈...什么蛇王，笑话，本仙还是蛇仙呢”蛇精讥笑起来：“我若出来，你将死无葬身之地，我若不出，你又拿我何”
<br/>　　“怎么，有本事继续劈呀，本仙承受的住，就怕这具身体的主人会被你劈死”
<br/>　　涂小安不屑一顾：“他死了与我何干，你以为我是为了他而来吗”
<br/>　　这次的系统任务很简单，是为了解决舒瑶的麻烦，显然，这蛇精现在就是她的麻烦。
<br/>　　涂小安也不想将自己定义为一条正蛇，因为死在他手中的人也不少了，但起码，他不会仗着自己的本事胡作非为。
<br/>　　至于舒瑶的男朋友陈响，根本不在涂小安的眼中，他或许也挺无辜，不过吃了一顿蛇羹。
<br/>　　当今世上，蛇羹早就成为了人们脍炙人口的名菜了，不知多少人吃过，也不知道多少人在此时此刻正吃着。
<br/>　　陈响得知女友因为怀上蛇胎就丢下她一人，相恋了三四年的情分，瞬间遗忘。
<br/>　　或许这也是人之常情，搁谁碰到这种事情都一时半刻承受不了。
<br/>　　但换句无情的话，他死了跟涂小安真的没什么关系。
<br/>　　他只要将这条蛇精给解决了就够。
