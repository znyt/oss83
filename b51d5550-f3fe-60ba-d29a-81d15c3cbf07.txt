　　血池罗刹站在水上，恨恨地望着叶少阳，眼神是那样的不甘和愤怒，但最后也只能命令大伙撤兵离去。
<br/>　　至于穷奇，也仿佛突然清醒过来似的，左右张望了一番，对海岛展开了狂乱的进攻。但南海大阵已成，就算秦军万马也无法破坏，更不要说穷奇还受了重伤，最后也只能在在血池罗刹和雪魔的劝解下，悻然离去。
<br/>　　他到底不知道自己遭遇了什么。
<br/>　　“没事了？”小马望着叶少阳，回想之前发生的一幕，还有点不敢相信。
<br/>　　叶少阳耸了耸肩。
<br/>　　“秋莹死了！”橙子哇的一声就哭了，扑到在叶少阳怀里，放声大哭起来。
<br/>　　美华和小白也在一旁哭泣。
<br/>　　瓜瓜等人，脸上也都现出悲戚的神色，静静地站在一边，虽然秋莹加入捉鬼联盟的时间不长，他们中大部分跟秋莹打交道的时间不多，但只要进了捉鬼联盟，那便是兄弟姐妹，彼此之间便能舍身相救，甚至为对方去死。
<br/>　　这是捉鬼联盟能在三界异军突起、打出一片天的原因。
<br/>　　叶少阳茫然地望着四周，海面上，因为太阴山撤军，显得空空荡荡，一个人都没了，他是想找秋莹的魂魄，但连魂魄也没了，已经化作精魄，进入了海眼之中，应该是去轮回去了吧，按照天地大道的规则，她轮回之后也是先从草木做起。
<br/>　　也好，她本来就是仙草，依然还是做回自己，如果精魄中保存一些灵气的话，她依然有修炼成精的机会，只是，那个他们所熟悉的秋莹，再也回不来了。
<br/>　　叶少阳摊开左掌，看着上面那一道本该属于秋莹的魂印，如今也完全消失了。
<br/>　　“岂曰无衣，与子同袍……”
<br/>　　林三生面对长空，用古曲的腔调唱了这么一句，声音充满了凄凉。
<br/>　　冲和子等人看到的悲戚的一幕，又感怀自己门人在方才一战之中多被屠戮，心中也是悲戚不已。
<br/>　　场面一时间陷入悲凉的沉默。
<br/>　　“老大，你刚才为什么不杀了穷奇，为秋莹报仇啊！”橙子突然抬起头，愤怒地冲叶少阳质问道，“跟这种邪物还讲什么道义，什么光明正大地杀死他！”
<br/>　　叶少阳没有回答，反问大伙：“你们知道，穷奇为什么会突然反水吗？”
<br/>　　一句话将所有人心底的好奇都调动了起来。慕寒等人也侧耳听着，从战斗结束到现在，他们一直在想这个问题，他们实在想不通，叶少阳对穷奇做了什么，道门的确有符咒能让邪物迷失心神，暂时听从自己的命令，但对象只能是修为浅薄的一般邪物。
<br/>　　穷奇这实力……想要控制他的心神，只怕酆都大帝都未必做到（比杀死它要困难十倍），所以，大家都觉得，一定还有别的原因，但实在想不到来由。
<br/>　　叶少阳打开背包，把包子化身的缩小版的通灵佛子花取出来，手掌在上面抹了一下，那个胖嘟嘟的小孩又出现了。
<br/>　　包子坠落在地上，四仰八叉，睁开疲惫的眼睛，探寻地望着叶少阳。
<br/>　　“成功了，你居功至伟。”叶少阳伸出手，像对待孩子似的轻轻捏了捏他的脸。
<br/>　　“包子！啊我知道了，是他用幻术迷惑了穷奇么？”美华恍然大悟地叫起来。
<br/>　　叶少阳点头，“他给穷奇编织了一个梦，一个颠倒敌我的梦，让他以为那些邪物是我们的人，反而以为我们是他的同伴。”
<br/>　　“这……这么简单？”大伙惊叫起来。
<br/>　　“就是这么简单。”
<br/>　　包子的幻术一旦成功，就能摄取对方的神识，为他编织出特殊的梦境，当初叶少阳等人都中招过，这个造梦的过程，就只有包子自己知道了。
<br/>　　大伙看包子的样子十分疲惫，于是上次查看他的情况。包子的回答是用脑过度，现在大脑一片空白。
<br/>　　“不对啊，老大，穷奇实力那么强，怎么会随便中招？”小青不解地问。
<br/>　　修为越强大，对神识的保护也越强，更难被各种幻术迷惑，否则包子早就三界无敌了。
<br/>　　“所以，之前我才要你们全力进攻穷奇，争取重伤他，就是要分散他的注意力……”
<br/>　　大伙略一思索，恍然大悟，之前叶少阳命令大家围攻穷奇，所有人——包括血池罗刹等人，都以为他是报仇心切，杀红了眼，没想到……这都是假象，是叶少阳自救计划中的一个步骤。
<br/>　　“你既然能带人接应我，是不是早就看穿了我的计划。”叶少阳抬头看着林三生。
<br/>　　林三生没正面回答，说道：“杀红了眼的叶少阳，我见过，但就算是为了报仇，你也不可能把大伙的命都赌上，那不是你。我看到你一再带领大伙舍身攻击穷奇，起初也迷惑，再一想就明白了。”
<br/>　　“为什么不突围？”慕寒想到这里，也不顾两人的关系，脱口而出，“当时你若突围，至少有一大半人能走掉。”
<br/>　　叶少阳淡淡地看了他一眼，道：“我要的是所有人一起回来，若要牺牲个别人，那还不如所有人一起死。”
<br/>　　慕寒目瞪口呆，一句话也说不出。
<br/>　　因此，在极端危难之际，叶少阳反而冷静下来，想到了包子，当时包子就在自己身边，于是用神识告诉了他自己的计划，然后发动大伙围攻穷奇，穷奇重伤、尤其是眼睛被刺瞎时，穷奇本能地分神了一瞬间，包子不负众望，抓住这一瞬间的机会，控制了穷奇的神识……
<br/>　　明白了这一切，慕寒等人心中大为震动，叶少阳这计划虽然绝妙，但风险也是极大，当时如果血池罗刹不是为了求稳妥，直接上前围攻，内外夹击，他们可能都得战死，一个都跑不掉，但叶少阳竟然抓住了机会，创造了奇迹。
<br/>　　最重要的是，在面对整个一支大军的包围、看不到一线生机的情况下，换成别的人，要么想到突围，要么就一起战死，就算给他们包子，在那种时候也几乎不可能想到这条妙计。
<br/>　　而叶少阳恰恰想到了，然后就创造了奇迹，将自己一干门人一个不少地带了回来……
<br/>(本章完)
