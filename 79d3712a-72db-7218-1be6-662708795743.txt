　　空气中弥漫着一股恶臭味，像是某种东西腐烂掉了一般，让刘哔感觉有些熟悉。
<br/>　　他并没有着急动手，刚才的攻击消耗了太多能量，他需要一些时间，来缓冲一下。
<br/>　　似乎是因为过于剧烈的运动，刘哔身体已经隐隐开始出现了一些丧尸的征兆，此时的他，几颗獠牙从口中露出，手上的指甲，也开始缓缓生长了起来。
<br/>　　最明显的，是他的眼睛，变成了如同鲜血一般猩红，在漆黑的夜色中很是明显。
<br/>　　“你不是人？”女鬼语气有些惊异，她注意到了刘哔的变化。
<br/>　　挑了挑眉，一股凶戾的气息从刘哔身上散发出来，他语气冷漠的开口，道：“你猜对了，所以请你去死吧。”
<br/>　　随着话语落音，刘哔身体猛然从原地消失，女鬼顿时一惊，感受到脑后传来一股凌厉至极的气息。
<br/>　　比常人快了几倍的速度，根本让女鬼没有一点反应的时间，再加上她本身就已经受了很严重的伤势，此刻更是来不及躲避。
<br/>　　下一瞬间，一道剧烈的疼痛感传进了女鬼脑海，伴随着一声惨叫，一根粗糙的树枝，深深插进了女鬼头颅之中。
<br/>　　按照常理来说，寻常的物品根本不可能对鬼魂造成任何伤害，但是这根树杈不同，因为上面，带着刘哔体内已经所剩无几的天地灵气。
<br/>　　因为灵体的侵蚀，褐色的树杈上，一缕缕猩红色的丝线缠绕游走，将女鬼的小半个脑袋，都搅的稀烂。
<br/>　　这一式不可谓不强大，但是却消耗掉了刘哔体内所有剩余的天地灵气，随着这一击，一股无以伦比的虚弱感从他体内涌了上来。
<br/>　　事实上，这已经是极为冒险了，女鬼的实力要比他想象中的强大了许多，如果不这样做，他迟早要落败在女鬼手上。
<br/>　　这可不是开玩笑的事情，如果落败，下场可想而知。
<br/>　　故此，刘哔便也没有任何保留，直接将所有的灵气都灌注在了这次攻击上，要么击杀女鬼，要么女鬼杀了他。
<br/>　　一阵凄厉的惨叫声响起，像是在忍受着莫大的痛苦一般，成片的肉块从女鬼头颅上脱落了下来，在半空中化为一片片灰白色的雾气。
<br/>　　虽然在拼命挣扎，但女鬼根本挣不开那根小小的树杈，只能眼睁睁看着自己辛苦凝成的血肉，化成了一片片本源雾气。
<br/>　　意识逐渐开始有些模糊，女鬼眼中露出一抹绝望，她知道，用不了多长时间，自己就会彻底死亡，连鬼都做不成了。
<br/>　　天空中微弱的星光洒下，照射在这片荒芜的土路上，成片的灰白色雾气，更是让这里变得鬼气森森。
<br/>　　突然，女鬼面色一喜，她察觉到脑袋上的这根树枝，上面残存的力量越来越弱，她知道，刘哔现在体内的能量也不是很多了，不由得挣扎更加剧烈了起来。
<br/>　　后方，刘哔脸色沉了下来，他紧紧握着手上的树枝，身体止不住的颤抖，和女鬼猜想的一样，他现在确实快要力竭了。
<br/>　　看着眼前不断挣扎的小半颗头颅，一抹阴狠从刘哔目光中闪过，张了张嘴，一口咬了上去。
<br/>　　紧接着，‘咔擦’一声，那是头颅破碎的声音，他感觉自己咬上了一个柔软的物体，像是在吃棉花糖一般。
<br/>　　味道有些奇怪，但出奇的有些好吃，刘哔手中拿着用树枝串起的头颅，粗鲁的撕咬着，眼中猩红色的光芒，变得愈加旺盛。
<br/>　　伴随着一阵凄厉无比的鬼叫，女鬼的挣扎逐渐变的微弱，几个呼吸的时间，她就彻底失去了意识。
<br/>　　几个呼吸的时间，女鬼的头颅就被刘哔尽数吞进了腹中，扔掉手中的树杈，刘哔长吸一口，将附近的鬼气也吸了进去，随后抹了抹嘴巴。
<br/>　　味道不错，很是鲜美。
<br/>　　“叮~恭喜宿主击杀初级鬼魂一只，获得积分一百。”
<br/>　　“叮~恭喜宿主成功吞噬初级鬼魂一只，获得技能‘鬼手’，请宿主自行进入技能面板查看。”
<br/>　　脑海中传来系统的提示，刘哔胃里冰凉一片，像是有一块寒冰在自己胃中一般。
<br/>　　第一次吃鬼，没什么经验，他也不知道自己的食用方法正不正确，不过按照系统的提示来看，这次吞噬算是成功了。
<br/>　　只是有些可惜，他之前准备的调料却是一点都没用上。
<br/>　　叹了口气，刘哔撑着有些疲惫的身体，找了一颗树木缓缓坐下，进入系统查看着新获得的技能。
<br/>　　名称：鬼手
<br/>　　级别：F级
<br/>　　分类：主动技能
<br/>　　介绍：来源于《僵尸先生》位面，可消耗能量，在宿主五米的范围的地面上，召唤出鬼手束缚敌人，召唤数量由宿主等级而定。
<br/>　　关上系统面板，刘哔对这个技能有些满意，他现在缺少的正是主动技能，虽然这个等级是有些低，但还是可以拿来勉强使用的。
<br/>　　再说，击杀这么一头低等级的鬼物，他也不奢望能得到什么逆天的物品，只要还算实用，那他就非常满意了。
<br/>　　有了‘鬼手’技能的帮助，明天对付任家的那只丧尸，他也要多增加了一些胜算。
<br/>　　虽然有心想要试验一下新技能的威力，但苦于体内的能量不足，他也只能作罢，在树下休息了片刻，等身体恢复了一些行动力之后，刘哔站起身来，向义庄的方向走去。
<br/>　　这场战斗花费了太久的时间，现在已经很晚了，如果再不回去，九叔他们难免会有些担心。
<br/>　　随着刘哔的离开，这片荒芜的土路又恢复了先前的宁静，只有地面上那个巨大的坑洞，证实了先前这里发生过一场惨烈无比的战斗。
<br/>　　也是从这里开始，刘哔彻底改变了原著剧情，至于以后剧情要怎么发展，他不知道，但也不关心。
<br/>　　这个世界里的鬼怪这么多，就算剧情被改变了，他也有的是机会赚取积分，等任家那头丧尸被自己解决之后，熟知剧情对他起不到什么作用了。
<br/>　　如果他的猜想正确的话，等所有的剧情全部提前完成之后，他也就会找到离开这个世界的办法了。
