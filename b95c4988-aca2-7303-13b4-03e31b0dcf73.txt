        &lt;h3&gt;第三卷第一百一十四章  受伤的凌辰&lt;/h3&gt;
<br/>
<br/>        凌辰的手机掉到了地上，手机屏幕摔碎了，过了好久一个人摇摇晃晃的走过来，一脚踏在手机上，“凌....凌总，你特么的太....太厉.....厉害，哥.....哥们都喝，喝爬下了，咯......你还站着。”
<br/>
<br/>        凌辰的耳朵里只有宁雅手机里传出来的那种娇滴滴的，喘息声。
<br/>
<br/>        凌辰的心似乎被冻住了。
<br/>
<br/>        凌辰不知道是怎么回到了酒店，混混浩浩中他看到了宁雅的笑，那笑像晨曦的初阳温暖和煦，又像是正午的烈阳刺的他眼睛疼。
<br/>
<br/>        凌辰双手捂住眼睛，低声的呢喃着：“宁雅，你怎么能这样？宁雅！”
<br/>
<br/>        “醒了，醒了。”病床前站着的人欣喜的喊。
<br/>
<br/>        “哎呀，终于醒了，我还以为他很能喝，谁知道，特么的，我用手一推他就倒了，吓死我了。”那个特么大哥絮絮叨叨的，他的心脏都受不了了。
<br/>
<br/>        不过这小子够义气，够拼。不偷奸耍滑，这几年的代理就不变了。
<br/>
<br/>        “醒了就好，醒了就好。”旁边的人也随声附和着。
<br/>
<br/>        凌辰睁开眼周围一片白，他挣扎着坐起来：“我这是在哪里？”
<br/>
<br/>        “你可醒了，你特么的都睡了一天一夜。吓死我了，你说你小子要是过去了，我这全国总代不是要换人了吗？”特么大哥笑着说。
<br/>
<br/>        “董总？”凌辰一脸懵圈，他怎么就在病房了？不是就接了宁雅的电话了吗？电话呢？
<br/>
<br/>        “你这啊，要学会和他们一样偷奸耍滑，别实打实的的喝。”董总说。
<br/>
<br/>        旁边的人尴尬的笑了笑。
<br/>
<br/>        他们都有望能争到全国总代的资格，董总招呼大家在一起，大家都藏了小心思，将别人喝倒，好送董总回家。
<br/>
<br/>        这倒好，将凌辰给喝倒了，大家到医院陪了凌辰一天一夜，董总不走谁敢走。
<br/>
<br/>        “我睡了一天一夜？”凌辰问。
<br/>
<br/>        “我们都陪了你一天一夜。”旁边的人说。
<br/>
<br/>        “哎呀，对不起，你看我这酒量，真丢人。”凌辰懊恼的说：“不但丢人还给大家添麻烦了。”
<br/>
<br/>        “丢什么人，你喝得比他们都多。”董总笑着说。
<br/>
<br/>        “主要是咱的酒太好了，而且见了董总，心里一高兴，就喝多了。”凌辰那冷冰冰的脸上裂开一条缝。
<br/>
<br/>        耳边还有宁雅的声音，但是面前的情景并不是他伤春悲秋的时候，旁边的人还虎视眈眈的盯着全国总代的位置，他也知道哪些人的小心思，能把他凌辰挤下去是他们的梦想。
<br/>
<br/>        在这里的人并不缺钱，缺的是资源。
<br/>
<br/>        “你小子，这么拼，那一天喝过去了，我就将全国总代的位置让给你的儿子。哈哈哈”董总笑着说。
<br/>
<br/>        “那怕你要失望了，我过不去，儿子也还没有着落，你还是将就着用着我吧。”凌辰翻身起床，这身体睡了一天一夜还行，缓过来了，只是心窝处还是隐隐有些疼。
<br/>
<br/>        他追了这么多年的宁雅，追到别人的怀里了，他情何以堪？
<br/>
<br/>        “还是年轻好啊，要是我这条命，怕是几天都缓不过来，你看看人家，精神百倍。”另外的一个代理商说。
<br/>
<br/>        “是啊，年轻有年轻的好处，年老有年老的好处，有些人年轻有闯劲不顾后果，年老沉稳老练，各有千秋啊。”另外的一个戴眼镜的代理商也接话。
<br/>
<br/>        凌辰冷冰冰的看了一眼那个经销商，这是给他上眼药呢？
<br/>
<br/>        那个经销商只感觉背后冷飕飕的，他不由自主的打了个寒颤。
<br/>
<br/>        “你醒过来了就好，今天大家休息一天，改天带你们参观参观，安排你们去附近逛一逛，大家也辛苦这一年了，元旦之后我们再商议工作的事情。”董总发话了。
<br/>
<br/>        “哈哈，那就感谢董总的盛情了。也感谢凌辰睡了一天一夜啊，要不然我们还没有着殊荣呢。董总破费了啊。”还是哪个眼镜男。
<br/>
<br/>        凌辰看了一眼眼镜男，何少华，甸城人，据说是甸城一霸，黑白通吃，非常有钱，是他最大的竞争对手之一。
<br/>
<br/>        “小子，老子记住你了！”凌辰心里暗暗的说。
<br/>
<br/>        “董总，我这睡了一天一夜，耽误了大家的时间，而且我又是全国总代，这几天大家的吃喝用度就归我了，不能让厂里出一分钱。成晓，记得安排好大家啊。”凌辰对着董总和大家说，并且安排了和他一起来的工作人员。
<br/>
<br/>        “行！那你就来安排！”董总看着凌辰心里说：“嘿，这小子，还有两把刷子。”
<br/>
<br/>        何少华偷鸡不成蚀把米，没想到又给凌辰创造了一把装逼的机会，不过，没事，总有机会让你倒下去。
<br/>
<br/>        后面的路还长呢！
<br/>
<br/>        凌辰安排了成晓带着大家配合厂家进行了参观，后来又安排大家吃喝玩，在此期间倒是收拢了一些人心。
<br/>
<br/>        但是在强大的利益面前，一切的堡垒都有可能被攻破。
<br/>
<br/>        有几个佩服凌辰的豪爽和仗义，平时关系也不错，几个人自然就形成了两派，一派以何少华为首，一派以凌辰为首，开始了暗暗的争夺总代的位置。
<br/>
<br/>        这几天凌辰都没有给宁雅发信息，也没有打电话，他将自己关的酒店里，想着这几年和宁雅少得可怜的来往，因为工作忙，他天南地北的跑着，平时最多的就是给宁雅打打电话，发qq。
<br/>
<br/>        元旦的早晨，董总打电话给凌辰，让凌辰去他家一趟。
<br/>
<br/>        董总看着凌辰，对凌辰说：“何少华昨天找我了，对于代理的位置他很想拿到，而且给出的条件很是丰厚，销售额也翻了一倍。”
<br/>
<br/>        凌辰一直冷着他的那张脸听着董总的话，他知道现在并不是他说话的时候，如果董总想把代理给何少华的话是不会叫他来的。
<br/>
<br/>        “我女儿从香港回来了，我安排你和何少华一起，你们都是年轻人，沟通起来很方便，你们就一起过过元旦吧。”董总看着凌辰。
<br/>
<br/>        “好，您安排，我一切都听你的。亚子小姐是什么时候回来的？”凌辰见过董亚子一面，印象还是比较深刻，亚子长相一般，有点像她的老爹，富态。
<br/>
<br/>        “毕业了，回来有一段时间了。”正说着，董亚子从外面回来了。
<br/>
<br/>        董亚子看着凌辰眼睛一亮：“凌辰哥哥来了？爸爸说你来了好多天，我都没有见到你。”
<br/>
<br/>        “亚子好，你是大忙人，我那能见到你呢。”凌辰冷冷的打着招呼。
<br/>
<br/>        “哎呀，我来迟了，董总，亚子妹妹，真不好意思，这个是刚刚从甸城空运过来的土特产，我去机场取回来的，有点迟到了，不好意思啊。”何少华也提着几个包装紧致的礼盒回来了。
<br/>
<br/>        “舅舅让我代他像您问好呢。”何少华又接着说，并且看了一眼凌辰。
<br/>
<br/>        “啊哈哈哈，凌总也在啊，凌总的身体好些了吧？”何少华像才发现凌辰一眼打着招呼。
<br/>
<br/>        “不劳何总挂心，我已经好了。”凌辰的语气淡淡的。
<br/>
<br/>        “凌辰哥哥，你怎么了？哪里不舒服吗？”董亚子关切的问。
<br/>
<br/>        何少华的脸都绿了。
<br/>
<br/>        “你们三个就出去逛吧，我年龄大了，就不陪你们了啊。”董总看着眼前的三个年轻人说。
