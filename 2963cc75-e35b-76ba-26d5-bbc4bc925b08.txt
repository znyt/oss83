　　这刚刚，这些荒古门弟子前来，嚣张无比，封锁空间。
<br/>　　如今现在，局面却是完全颠倒了过来。
<br/>　　那尊荒古门弟子，看向楚风眠，一脸气愤愤怒吼道。
<br/>　　“小子，你还想怎么样？”
<br/>　　“我想怎么样？”
<br/>　　楚风眠撇了撇嘴，冷冷的开口道。
<br/>　　“我刚刚不是就已经说过吗？怎么，你们难道是耳朵聋，听不清？”
<br/>　　“那我就在告诉你们一遍，你们几个，没有一个人，今天可以活着离开！”
<br/>　　楚风眠一句话中，充满这无尽杀意。
<br/>　　他要杀，当然不杀一个，这些荒古门弟子，今日都要死。
<br/>　　剩下他们三人，难道是等着让他们三人去通风报信，来对付楚风眠？
<br/>　　楚风眠的话一出，全场哗然。
<br/>　　楚风眠已经是杀了王智，荣北了。
<br/>　　这还不够。
<br/>　　今日楚风眠话中的意思，是要将这荒古门五人，全部斩杀，这话可所谓的嚣张到了极致。
<br/>　　“小畜生，你找死！”
<br/>　　荒古门三人听见楚风眠的话，一个个的怒意，再也无法忍耐下去。
<br/>　　他们三人，也都是荒古门弟子，绝对的天才人物，如今在楚风眠的口中，却仿佛如同豆腐块一样，想要被楚风眠怎么捏就怎么捏。
<br/>　　“今日我们就算是拼上荒古神丹，也要将你，彻底灭杀！”
<br/>　　荒古门三人，同时各自都拿出了一枚灵丹，这灵丹上，铭刻有着极为复杂的纹路。
<br/>　　“荒古神丹！这是荒古门独特炼制而成的荒古神丹！”
<br/>　　一些北域武者，看到这荒古神丹的一刻，眼神中都是深深的忌惮。
<br/>　　荒古神丹，是荒古门独特炼制的一种灵丹，只有荒古门才可以炼制的独特丹药，在北域，都是极负盛名。
<br/>　　因为是一旦服下荒古神丹的话，便是就足以令荒古门弟子的所有潜力，都给瞬间激发出来，令实力暴涨数倍，在对拼之时，服用一枚荒古神丹，绝对是最强手段。
<br/>　　“小畜生，我们本来没有打算浪费荒古神丹来对付你的！你现在激怒了我们！那我们今日就成全了你！去死吧！”
<br/>　　那三尊荒古门弟子，同时怒喝一声，一口将荒古神丹，全部吞了下去。
<br/>　　在服下这荒古神丹的一刻，他们的心中都是心疼无比。
<br/>　　这荒古神丹，就算是在荒古门中，也都是价值极高之物，他们花费了大价钱才得到的，用来在颜古龙帝洞府中探索遇到危险所用的。
<br/>　　荒古神丹吞噬炼化，这荒古门三人的实力，都在瞬息之间开始疯狂暴涨起来，他们的境界，也一跃之间，冲击到了生死境一重的地步。
<br/>　　单单是身上的气息，只怕是足以跟生死境四重的强者所抗衡！
<br/>　　“没有人可以跟我们荒古门，跟七大宗门对抗！小畜生，今日你必死！”
<br/>　　那三尊荒古门弟子，一身实力暴涨之后的，顿时疯狂无数灵力，这一道道灵力，开始互相凝聚起来。
<br/>　　在空中，化为了一条巨大无比的大道，他们三人，踏入这荒古大道之上，身上的气息，一下子更是变的更加恐怖了许多。
<br/>　　“荒古神丹？为了对付我，这倒真是大手笔。”
<br/>　　楚风眠心中冷笑连连，看向那三尊荒古门弟子。
<br/>　　荒古神丹，楚风眠自然清楚，这是荒古门的独特丹药，足以激发出荒古门弟子身上的潜力来，获取力量。

<br/>　　可一旦是药力失效，他们本身的潜力，都会受到极大的损伤，很难再进一步。
<br/>　　可以说是玉石俱焚的手段。
<br/>　　“小畜生，你身上的血脉，绝非是寻常龙族血脉，甚至可能是龙王，龙皇的血脉，杀了你夺取了你的血脉，我们的一切损失，都可以弥补！”
<br/>　　一尊荒古门弟子，看着楚风眠大声喝道。
<br/>　　“准备受死吧！”
<br/>　　“荒古大道！彻底凝聚！”
<br/>　　三尊荒古门弟子身上的灵力，全然被催动起来，一股股灵力开始凝聚，周围的虚空，都开始发生着变化。
<br/>　　他们脚下的荒古大道，也逐渐开始由虚变实。
<br/>　　最终化为了一条古铜色的大道，在这荒古大道之上，刻画有着无数古老的铭文。

<br/>　　“轰隆！”
<br/>　　荒古门弟子凝聚灵力，猛然席卷过去，这灵力，看起来只是平凡无比，但是在荒古大道之中打出，其中却是突然蕴含了无数玄妙无比的力量。
