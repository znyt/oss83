<br/>				陈默看着世兴文被推上救护车，然后在两个护工的帮助下固定好移动病床。<br/>　　<br/>　　也许是世兴文在最后走的时候想通了吧，也许是其他原因。在车门关闭的一霎那间，陈默发现世兴文对自己点了点头，算是告别？<br/>　　<br/>　　人要找死，谁也不能阻拦。本来呢陈默还是好心，即使将自己的脸打的啪啪想，还是又劝说了一次他，但是结果还是一样被拒绝。而且他自己的脸也被再一次啪啪打了一遍。<br/>　　<br/>　　“陈医生，你还好吧？”李普河看到陈默一直在注视着远去的救护车，有些弱弱的问道。对于世兴文的行为，他也是有些无语。但是既然是他自己的选择，那么对于他来说也无所谓。正所谓，见的多了也就没有什么担忧的。<br/>　　<br/>　　每天都在接触这些伤者，所以有的时候不可能满足每一个伤者的要求，只能按照大多数人的意愿来做事情。<br/>　　<br/>　　“没有关系，就是有些可惜！”陈默说道。<br/>　　<br/>　　“呵呵！陈医生，每个人都有自己选择道路的权利。我们将这些事情全部摆到了他的面前，这就是他的选择！至于前面是好是坏，那就看他的运气了！”李普河化身成心理专家，给陈默说了一通。<br/>　　<br/>　　陈默倒是没有什么感觉，就是知道明明能将这个人的伤治疗好来着，却不让自己治疗，真他么的操蛋！要不是看在为国受伤的份上，谁他么的想啪啪被打脸两次。<br/>　　<br/>　　“好吧，我们去看其他的病人吧！”陈默对李普河说道。<br/>　　<br/>　　“陈医生，这个龙血藤已经送过来，你看看合适不合适？”李普河将早上和病人一通送过来的龙血藤递给了陈默。<br/>　　<br/>　　这个龙血藤是从上市特管局的药库中找出来的，而且是活株。所以送来的时候用一种陶土花盆装着，真个株体大概有个一米多高，整体成血红色。盘绕在花盆中间的木柱上，蜿蜒犹如虬龙，本身又是血红色，才得名龙血藤。<br/>　　<br/>　　顿时让陈默双眼放光，要不是李普河和常文斌还在旁边，他绝对将这株龙血藤给收进乾坤珠内。<br/>　　<br/>　　“很好，至少有五十年了东西了，不错！等会送到我的别墅哪里吧，我就可以用它来配置一些药丸。”陈默说道。<br/>　　<br/>　　“没有问题，等会就给你送过去。”李普河说道。对于这个东西，既然拿过来，那么就是让陈默随便处置的，只要能将伤者治好，话多少钱都是愿意的。<br/>　　<br/>　　“另外，陈医生，昨天你看的药方，也已经按照你说的配置好了，你看……？”李普河接着询问道。<br/>　　<br/>　　“都是根据我昨天开的药方配置的？”<br/>　　<br/>　　“是的！”<br/>　　<br/>　　“那就好。你们就按照普通中药熬制程序，将中药按照疗程一副三煎就成，然后按照我的医嘱服用。”陈默说道。<br/>　　<br/>　　“好的，我明白了！”李普河说着。<br/>　　<br/>　　然后常文斌去熬制中药，李普河陪着陈默去看病人。<br/>　　<br/>　　“那个，常哥！龙血藤先不要动，等我看完病人后一起送过去吧。主要是院子里有大黄和大灰，它们不会让你进去的。”陈默叮嘱了一下。<br/>　　<br/>　　“好！”<br/>　　<br/>　　常文斌本来想先将龙血藤送过去的，都要弯腰抱起来了，却听到陈默这么说，也就点点头。对于大黄和大灰，他还是知道的，自己就是陌生人，抱着龙血藤送过去，就是挨咬去了。<br/>　　<br/>　　剩下的三个人，其中一个是今天早上送过来的。并且也是身体中有异种能量，已经被折磨的形容枯槁，都说不出话来。<br/>　　<br/>　　这三个人，两个人是这种症状，另外一个就是疑难杂症，进过很多医生都没有办法查找出原因。人自从受伤后，就一直昏迷不醒。<br/>　　<br/>　　陈默三个人都检查了一边后，就先治理这个伤者。主要是异种能量需要耗费心神，等治疗完后，可能就没有太多的精力治疗这个病人了。<br/>　　<br/>　　进过陈默的神识的检查，发现这个病人的脑子中枢神经下方，有一个细细的小虫子，一动不动的待在中枢神经的地盘。<br/>　　<br/>　　要不是他的神识比较清晰，也要比一些精密仪器要观察入微的多，还真不容易发现这个虫子。<br/>　　<br/>　　因为这个虫子身体颜色趋于白色，和人体神经颜色相近，很容易让仪器默认为神经系统。<br/>　　<br/>　　要不是陈默的神识特殊，能够察觉到生命体征的话，他也会忽略过去。<br/>　　<br/>　　没有想到能够遇到如此神奇的东西，心中也是万分的稀奇。怪不得看病看多了，就经验丰富。这都是见多识广啊！<br/>　　<br/>　　不过对于头部中枢神经位置的交叉地带，要是一不小心，就会导致伤者成为白痴，要不就是全身瘫痪的角色。所以陈默发现这个特殊的小东西后，就小心翼翼的制定治疗方案。<br/>　　<br/>　　由于不了解这个小东西，所以他就先将所有的通道封闭。此刻伤者的头部插满了银针，然后陈默这才渐渐缩小封闭区域。<br/>　　<br/>　　没有想到的是就要将中枢神经附近的经脉等封闭，就发现这个小东西终于动了起来，似乎是发现了危险的到来，开始吞噬起伤者的神经。<br/>　　<br/>　　这就让陈默大吃一惊，没有想到这个小东西能够吞噬神经，就凭借这么小的身体，却依然能够将伤者的神经啃食出一个小缺口。<br/>　　<br/>　　陈默立刻拔出了最后的一根银针，就是靠近中枢神经附近的银针。小东西也就随着不动了。<br/>　　<br/>　　这个过程，虽然也就几秒钟，但是却让陈默满头大汗。没有想到刚才这么凶险。<br/>　　<br/>　　中枢神经系统是神经系统的主要部分，包括位于椎管内的脊髓和位于颅腔内的脑;其位置常在动物体的中轴，由明显的脑神经节、神经索或脑和脊髓以及它们之间的连接成分组成。<br/>　　<br/>　　此刻小虫子就趴在这个连接处，如果按照它的这个啃食速度，再怎么粗的神经也不够它啃食多久。<br/>　　<br/>　　还有就是，这个小家伙就如同头发丝粗细，大概有个一厘米的长度，但是却啃食起来这么迅速，那么组成神经的白质都去哪里呢？<br/>　　<br/>　　看来是个棘手的东西，但是陈默从他吃东西上，就想到了一个办法。那就是使用引豸丹！<br/>　　<br/>　　当然，陈默想到的引豸丹，可不是外边菜地中现在用的丹药，而且他配置出来后一直保留的丹药。这种丹药可是比外边菜地用的，要高上一个等级。<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
