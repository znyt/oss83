　　“你……！”拓跋青云闻言，顿时面色一滞，好像的确是这么回事。
<br/>　　无论对方说什么，他都不会完全相信。
<br/>　　看来只有擒拿住对方，才能彻底弄明白秦陨的底细了。
<br/>　　“既然如此，那你就跟我回地王宫吧！”
<br/>　　拓跋青云冷喝一声，随即大手一张，猛然向天尘宇凌空盖压而去。
<br/>　　霎时之间，天空风云变色，整个天穹，虚空陡然旋转了起来。
<br/>　　虚空旋转，形成了一个巨大的漩涡，仿佛末日黑洞，向天尘宇笼罩而下。
<br/>　　一股恐怖的威压，降临而下，仿佛要凝固一方世界，无比的可怕。
<br/>　　天尘宇见此，面色猛然一变，他周身源力一震，粉碎千里虚空，身形急速暴退。
<br/>　　不过，天尘宇的速度虽快，那巨大的漩涡却是速度更快，仿佛裹挟着整个世界的力量，狠狠的笼罩到了天尘宇的头顶上方。
<br/>　　“给我碎！”
<br/>　　天尘宇怒喝出声，他的手中出现了一柄雪亮的银色长剑，剑芒骇人，如同一根万丈神柱。
<br/>　　轰！
<br/>　　天尘宇一剑斩出，重重的斩向笼罩而下的虚空漩涡。
<br/>　　这一剑极为不凡，剑光之中，仿佛裹挟着一个宇宙，有万千星辰闪烁，又仿佛有无数个小世界，为这一剑助长声威。
<br/>　　恐怖的剑威，让得虚空不断的崩塌，大地不断的崩裂。
<br/>　　轰隆隆！
<br/>　　璀璨的剑光，重重的斩击在虚空漩涡之上，一阵恐怖的爆响，震动方圆十万里。
<br/>　　虚空漩涡剧震，边缘有些溃散，但威力并没有多少减弱。
<br/>　　而剑光瞬间炸裂，化为无数的星光，迸射八方。
<br/>　　“你复录帝一魂的绝学也无用！”拓跋青云面色冷漠，对方施展的剑招，乃是来自于帝一魂。
<br/>　　天尘宇的元神，极为特殊。名为‘天录’，在太古武魂榜上排名第十一。
<br/>　　这种天录元神，本身并没有多大的威力，但是，却有一种特殊的本领。
<br/>　　这种本领，称之位复录，或者叫刻录，它能复制别人的元神之力，或者说复制别人的绝学，可谓是极为逆天。
<br/>　　当然，复制的他人绝学，和拥有这种绝学本身之人，肯定有一定的差距。
<br/>　　毕竟，两个人修炼的道统、法则等等，都完全不同。
<br/>　　剑光崩碎，巨大的虚空漩涡，裹挟磅礴的威势，瞬间就笼罩在了天尘宇的身上。
<br/>　　轰隆隆！！
<br/>　　天尘宇的身影消失无踪，但是，巨大的虚空漩涡，却是不断的震动，仿佛其内有一只洪荒猛兽，在疯狂的挣扎。
<br/>　　显然，天尘宇在漩涡之中，正在极力的挣扎，想要挣脱漩涡的束缚。
<br/>　　不过，天尘宇虽然强大，但是，还是远不如拓跋青云。
<br/>　　仅仅是呼吸之间，巨大的虚空漩涡，便平静了下来，随即缓缓缩小。
<br/>　　很快，虚空漩涡凝聚成了直径约莫一丈的球体，被拓跋青云摄取到了手中。
<br/>　　镇压了天尘宇之后，拓跋青云没有多留，身形一闪，化为一抹黑色的光芒，向轮回神朝返回。
<br/>　　不过，他刚刚飞出十几个呼吸的时间，却是猛然停了下来。
<br/>　　因为此时此刻，天际的尽头，万丈高空之上，正有一人踏步而来。
<br/>　　此人一声蟒袍，体型高大魁梧，脸庞宽大刚毅，面色淡漠如霜。
<br/>　　此人不是别人，正是帝一魂。
<br/>　　见到踏步而来的帝一魂，拓跋青云的眸中，射出骇人的精光，如同两道利箭，粉碎层层虚空。
<br/>　　“拓跋，别来无恙！”
<br/>　　帝一魂背负着双手，脚踏虚空，一步万丈，如同闲庭信步一般，大步而来，停在了拓跋青云前方数十里处。
<br/>　　他的目光，也在打量着一身黑袍的拓跋青云，不过，并不能看出对方的虚实。
<br/>　　“呵呵，你终于还是来了！”拓跋青云面色不变，嘴角勾起了一丝轻笑。
<br/>　　他知道，对方很可能会出现，所以，并没有任何的意外。
<br/>　　同时，他也能猜到，暗处肯定还有天王宫的强者，最起码天君河肯定在。
<br/>　　同理，想必帝一魂也能猜到，地王宫的强者也会隐藏在暗处。
<br/>　　任何一方的人，都不会轻举妄动，这一次，是他拓跋青云和帝一魂两人，一次互相的试探。
<br/>　　“我来了，所以，放了天尘宇吧！”帝一魂声音淡漠，略显沙哑的说道。
<br/>　　他知道，既然拓跋青云返回，那肯定是镇压了天尘宇。
<br/>　　天尘宇虽然厉害，但肯定不是拓跋青云的对手。
<br/>　　“放也可以，你告诉我，秦陨是不是分魂？”拓跋青云的目光，紧紧的盯着帝一魂的眼眸，仿佛要看出帝一魂的想法。
<br/>　　“是与不是，对你来说，又有什么关系呢？”帝一魂面色平静，古井无波的说道。
<br/>　　拓跋青云闻言，眼眸微眯，稍稍沉默了片刻，便哑然失笑。
<br/>　　此言倒是在理，秦陨杀了他侄儿拓跋流明，早已是地王宫的死地。
<br/>　　就算秦陨不是分魂，他们地王宫，也会想尽办法找到秦陨，加以诛杀。
<br/>　　所以，无论秦陨是不是分魂，他们都要找到秦陨。
<br/>　　“所以，你放了天尘宇吧！”帝一魂沉声说道，他知道对方明白他的意思。
<br/>　　拓跋青云沉默不语，天尘宇对于他来说，的确是没有多大的用了，放是肯定会放的。
<br/>　　相比于天尘宇，摸摸帝一魂的底细，才更加的重要。
<br/>　　“帝一魂，我们之间的胜负，今日或许可以揭晓了！”
<br/>　　拓跋青云沉声说道，言罢，他大袖一挥，天尘宇的身形显现而出，直接被丢到了数千里之外。
<br/>　　天尘宇恢复了自由，面色略显阴沉，目光望着帝一魂和拓跋青云，有些惊疑不定。
<br/>　　“天尘宇，你退下！”帝一魂向天尘宇摆了摆手，今日既然来了，他自然不会放弃机会。
<br/>　　只要能镇压拓跋青云，他将一飞冲天，无人可挡。
<br/>　　当然，前提是，能在地王宫强者出手救援之下，以雷霆之势镇压对方。
<br/>　　而此刻，拓跋青云和帝一魂的想法一样，今日，如果能镇压帝一魂，一切都将结束了。
<br/>　　无论是拓跋青云，还是帝一魂，对于自身的实力，均是自信无比。
<br/>　　这一战，是宿命之战，或许不会彻底的分出胜负，但也会探查到彼此的虚实。
<br/>　　天尘宇默默的点了点头，身形缓缓飘退，退到了三万余里之外。
