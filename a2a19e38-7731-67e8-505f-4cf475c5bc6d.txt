　　步入正轨的这五万人很快进入了正规训练，这批新兵也终于开始了他们为期两个月的噩梦时间，刘玉定更是在这处军营将他不祥之兆的称号贯彻到底。
<br/>　　段氏基地此时则是完全成为了狼牙各部门的大本营，整个基地的中心都被狼牙的人占领，只要是最好的屋子都空了出来，连设施最好的战前会议场地都被改为了狼牙部门的地盘，鲁初雪等人不得不退居外圈，进入了条件简陋的空屋。
<br/>　　但是面对狼牙如此霸道的行为，却没有任何段氏基地的人抱怨，因为狼牙各部门来此的目的，在他们到达的第二天就完全呈现在外。
<br/>　　首先是卢宏盛的锻造部，各种仪器在最快的时间内被重新组合，放在了基地中心处最大的车间内，而这批锻造部的人则是只休息了一天，就立即开工赶制武器。
<br/>　　从热武器到冷兵器，从流水线中不断生产而出！
<br/>　　眼下的段氏基地虽然集合了很多新兵，但也有军需不够的难题，哪怕有着很多基地的资源共享，也难以一次性解决这么多人的用度，并且南部联盟军的北上战争还在进行中，那批队伍同样需要物资。
<br/>　　于是狼牙战团的锻造部强势迁移而来，就地生产！
<br/>　　这期间还不断有新人类进入考核石壁参加总考核，又不断有新人类的等阶提升，亦或是更多的普通人觉醒成为新人类，这些人从总考核中带出了大批大批的将领，而后一股脑的上交。
<br/>　　成批的新武器就这么制造而出！
<br/>　　至于研究部，他们的工作就更直接明了，就是不断配出觉醒剂和等阶提升剂，让这些新兵在最快的时间内提升战力。
<br/>　　这时候的鲁初雪等人才知道，狼牙的研究部竟然已经研究到了如此深远的阶段，早已将完全解决了当下新人类不足的难题。
<br/>　　只是当下狼牙所研发而出的三种药剂，除了丧尸引诱剂之外，其余的两种都需要用丧尸晶体作为原材料，这些晶体早在末世纪元爆发之出就被各基地大量收购，狼牙自然存了不少，更多的还是他们一路屠丧尸所得。
<br/>　　但哪怕如此，丧尸晶体也远远不够，是最为稀缺的材料！
<br/>　　关于这一点，鲁初雪当机立断的就联系了段江伟，而后又以文奇胜发声，第二天各大基地就送来了大批大批的丧尸晶体。
<br/>　　丧尸晶体近乎是每一个基地研究部最为珍贵的东西，是他们研究和探索的重要材料，稀缺又珍贵，更别说那些高阶丧尸的晶体，向来都是有价无市，毕竟一旦开始研究就代表着报废，因为目前为止除了狼牙，还没有任何一家基地研究出这些晶体的用途到底在那里，只是知道这些晶体内有着无比强大的能量，等阶越高能量越足。
<br/>　　于是当狼牙研究部的成果曝光后，便再一次让南部联盟军中的众位军官无言以对，深深觉得狼牙隐藏太深，楚涵更是个心机帝！
<br/>　　但无论内心怎么吐槽，他们也很清楚这时候不得马虎，各基地送往段氏基地的丧尸晶体数量，也绝对不在少数。
<br/>　　和上的科技部一切照旧，一边研究着那些特殊金属用途，一边不断与锻造部和研究部合作，时不时的还会给那百来架战斗直升机添加些其他东西。
<br/>　　但是这些部门再如何快速又有条不紊的赶工，也不及苏行的工程队。
<br/>　　当初两百人的工程队当下早已扩充了好几百人，这些人来到段氏基地只是休息了一夜，第二天就集体乘坐直升机前往南署基地，还带着大量的物资。
<br/>　　面对这一大批人的行为，段氏基地的每个人都无法理解，但当十天后他们有人去南署基地给工程队送物资的时候，才被震惊到怀疑人生！
<br/>　　苏行一行人一到南署基地，就第一时间将这一整座基地都考察了一个遍，紧接着就一刻不停的开始改造。
<br/>　　没错，就是改造！
<br/>　　整个基地的改造！
<br/>　　这是一项极其浩大的工程，更是狼牙这么多部门中，任务最艰巨的一个！
<br/>　　第一个遭受大尸潮冲击的就是南署基地，没有人在这紧急的情况下问过楚涵如何得知，而是以最快的速度准备起来。
<br/>　　南署基地可不小，是末世纪元里的十大基地之一，苏行等人想要如何改造，以及改造成什么样子，则是早在很久之前就已经和楚涵商量过。
<br/>　　依旧是楚涵提供想法和雏形，苏行将其合理化并实现！
<br/>　　改造的第一个项目就是基地城墙，南署基地的原本城墙一面只能抵挡十万数量的尸潮，当二十万数量的尸潮一股脑的朝同一面墙冲击的时候，那墙就会立即倾倒。
<br/>　　当下的末世纪元很多基地都是如此，并没有将城墙建造的有多坚实，这还是因为狼牙基地早已曝光了丧尸引诱剂的原因，基地在遇到尸潮的时候可以很快将大量丧尸分开，让其余城墙同时承受冲击，更让基地内的士兵有足够的时间将危险解决。
<br/>　　当时不久后的大尸潮，显然力度不是之前他们所认为的尸潮可比，区区十万尸潮所能承受的极限，根本不够看！
<br/>　　于是苏行的工程队一上来就开始对南署基地的城墙大改，改动的阵势极大，其变化也骇人听闻，以至于让得知这一消息的所有人都大吃一惊。
<br/>　　远在北部的南部联盟军，当文奇胜得知这一事件的细节后，先是沉默了许久，紧接着才看向段江伟道：“那工程队不是楚涵问你借的人么？”
<br/>　　段江伟也还没从震撼中回过神，他点头道：“是，但他有借无回……”
<br/>　　上官荣这时候接话：“那两百人在狼牙是不是被楚涵装过外挂？”
<br/>　　没有人答得上来，因为南署基地的改造虽没完成，但无论是速度还是变化程度，都实在是太令人震惊。
<br/>　　仅仅十天，苏行等人就将南署基地的一面城墙改造完毕，其样貌则是维持了原本那面墙的高度不变，但实际上则是早已将整面墙都推倒重砌，此外在这面墙之外的三十米，又造了一面墙，这面墙又高又厚，内部有着只可供一人行走的道路，上上下下曲折环绕。
<br/>　　除此之外在这两座墙之间，还修了地下隧道！
<br/>　　简直改的翻天地覆！
