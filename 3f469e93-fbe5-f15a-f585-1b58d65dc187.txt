　　叶少阳对茅山北宗不甚了解（凤兮除外），但是跟这个什么凌波子，还真的是有一面之缘：当初自己为了救后卿的时候，委托青云子帮忙，把这个凌波子从阴司找来，破了后卿身上的道化虫……要不是云春生提起这个名字，叶少阳都快忘了这个人。
<br/>　　当下冲云春生笑了笑，道：“祖师，反正我是茅山一脉，你就不要打听我的来历了。”
<br/>　　云春生听他这么说，自以为自己猜的没错，北宗弟子行事一向诡秘，不愿意暴露自己的身世，也是可以理解，当下心中释然，对叶少阳说道：“南宗北宗，本是一家，法无善恶，贫道看你颇有天赋，还望你恪守道心，好生修炼，不要误入歧途。”
<br/>　　“祖师教训的是。”叶少阳拱了拱手，敷衍道。
<br/>　　这时候，鬼姨那边已经结束了祷告，带着一干弟子朝山门走去。
<br/>　　山门大开，没有一个桃花山的弟子在。梧桐等人径直越过山门，朝山上走去。
<br/>　　“开战了开战了！”
<br/>　　这些小门派的弟子十分兴奋，紧随其后，闹哄哄地跟上去，叶少阳看了一下，人数加在一起至少有数百人之多。
<br/>　　真是法术界的盛世。
<br/>　　“我们也上去吧！”叶少阳说完，也跟上了大部队。
<br/>　　他的内心也充满了期待，倒不是为了见识什么大场面——当日悬空观之战和星宿海之战，场面不知道比这要宏大多少倍，强者也要更多，更不要说自己还是作为主角参加的。就大场面来说，叶少阳觉得在场的这些人，应该真没有谁比自己见识的更多。
<br/>　　他所期待的，是看一看桃花山上的那个神秘人，到底是谁，法术公会出动了这么多精英来围剿他，他是不是能够扛得住。
<br/>　　“梧桐，你小心点！”叶少阳摸到梧桐身后不远的地方，出声提醒道。
<br/>　　梧桐回头看了他一眼，点点头。
<br/>　　张晓寒在一边走着，脸色很是不好看，却自恃身份，不好跟叶少阳斗嘴，身旁的陈晓宇却是毫不犹豫地嘲讽额道：“你这个癞蛤蟆就不要多想了，还是关心你自己吧！”
<br/>　　叶少阳脸上仍然带着笑，没有理他。
<br/>　　现在不是对付他的时候，自己可以忍，反正也就这一天了。
<br/>　　虽然对自己的实力毫不怀疑，但法术公会一众人等，还是十分小心，步步为营地上山，但是一路之上，并没有遇到伏击之类的，也没有看到一个人影，身后那些法师们满心怀疑，开始嘀咕怎么回事，甚至有人认为桃花山的弟子已经偷偷逃走了。
<br/>　　一直到爬上山顶，穿过正殿，来到最高的那座山峰下面，一切怀疑都得到了解释：在这座桃花山最高峰上，有一座凉亭，在凉亭的一角，盘膝坐着一个身穿蓝衣的男子。
<br/>　　不用说，这就是那个让法术界猜测许久的那个神秘人物了。一时之间，众法师们目光都落在这个蓝衣人的身上，对他的身份展开了各种讨论，是人，是鬼，还是别的什么邪物？
<br/>　　因为距离太远，尽管叶少阳很努力地张望，还是看不到男子的脸。在凉亭的下方，山路两边长满了桃树，桃花簇簇，每一株树下，都站着一个人。
<br/>　　桃花山的弟子，都在这里，叶少阳粗略的数了一下，人数差不多也有上百，他们这样站着，一动不动，自然是布置出了某种阵法。
<br/>　　在山路的尽头、小路下方一点的地方，站着一个身穿水波道袍、头戴法冠的老道士，双手在身前虚怀若谷，目视下方，冷冷说道：“贫道乃是桃花山之主山河真人，尔等闯我桃花山，不知所为何事？”
<br/>　　鬼姨走到峰下，抬头望着上方，说道：“你桃花山暗藏妖孽，荼毒法术界同仁，三日之前，我以代法术公会对你下过通牒，令你等下山，却是执迷不悟，今日我等一同上山，前来诛杀妖孽，再给你们最后一个机会，你若率领弟子就此下山，便是既往不咎，若再负隅顽抗，便是背叛法术界，我等屠杀上去，到时泥沙俱下，难有活口，山河你考虑清楚！”
<br/>　　山河真人冷哼一声，朗声说道：“有眼无珠，认不得真神。不过贫道已懒得分辨，既然你们想要杀我师祖，我桃花山虽小，却也不是任人欺凌，你们而今欺上我山门，那便来吧，有死而已！”
<br/>　　“螳臂挡车，螳臂挡车啊！”叶少阳身边充满了这样的言论。
<br/>　　“你怎么看？”妙心问叶少阳。
<br/>　　叶少阳摇摇头，目光一直锁定在那个蓝衣人的身上，尽管看不见容貌，但是给叶少阳的感觉，却是十分地熟悉，似乎在哪里见过……
<br/>　　叶少阳心中，隐约想到了一个人……心头顿时一寒，不可能……是那个人吧？
<br/>　　“先看看再说，要开打了。”叶少阳说话的工夫，山脚下，响起了鬼姨“咯咯”的笑声，“既然你们死不悔改，那我等也没什么好顾忌的了，大家动手，格杀勿论！”
<br/>　　那二十来个第二梯队的弟子，听见命令，立刻就朝山路上冲了过去。
<br/>　　一踏上山路，几乎同一时间，狂风乍起，吹得桃树花枝乱颤，无数桃花纷纷飞落，在狂风中飞舞，却是井然有序，从山顶兜下来，将那二十来个法术公会的弟子尽数裹在中间……
<br/>　　果然是阵法，而且范围仅限于山路上面，在山下完全感觉不到一点风，也没有丝毫的威力，但是光是看这场面，就证明了这阵法的强大。
<br/>　　近二十个人的影子，在花海之中若隐若现，都各自作法，抵抗着风暴。
<br/>　　“爬上去，击杀守阵的人，阵法威力自会减弱！”鬼姨大声指挥道。
<br/>　　这些闯入阵法中的弟子，在山路上艰难地行进，接近那些桃花树下的弟子。
<br/>　　一声惨叫。
<br/>　　是离山脚下最近的一个桃花山弟子，被斩杀。
<br/>　　众人继续上山，又杀了第二个，结果去杀第三个的时候，其中有一人被风暴吹走，立刻被那些看似柔弱的花瓣绞碎了身躯，连魂魄也被绞碎……
<br/>(本章完)
