　　“你自己小心！”合并的藤蔓网之中，已经看不见妙心的身影，只是传来了她的声音。
<br/>　　“放心吧。我是谁啊。”
<br/>　　叶少阳几乎是对自己说，转过身，望着那朵十几米高的巨大的花，现在，只有自己一个了。
<br/>　　叶少阳双手叉腰，深吸了一口气。
<br/>　　“老子什么大场面没见过，怕你不成！”
<br/>　　说这句话的时候，叶少阳心中有些豪气，但也尽量保持了冷静。
<br/>　　叶少阳右手解下了勾魂索，左手把太乙拂尘拿在手中，朝着这株巨花冲了过去，快到鲜血蔓延的地方时，叶少阳用力蹬地，一个箭步跳起来，踏上了被藤蔓包围的左侧栏杆上，左手摸出了太乙拂尘，朝围剿过来的藤蔓横扫过去。
<br/>　　太乙拂尘是何等神器，被它扫中的藤蔓立刻枯萎，从栏杆上掉下去。叶少阳三两下打出了一个缺口，作势要往外跳。
<br/>　　结果这些藤蔓为了防止他跳河，立刻快速地聚拢过来，再度布置成了一张网，凌空兜住，如果叶少阳就这么跳下去，可能太乙拂尘还没有扫清障碍，叶少阳自己就被这些藤蔓抓住了，这是叶少阳唯一的机会，也是一条死路。
<br/>　　不过，这一次连巨花也估算失误了：叶少**本没有跳桥，而是借力一跃而起，朝着巨花过来了。
<br/>　　在送妙心出去之后，没有了后顾之忧，叶少阳压根就没想过逃走，他要与这巨花一战！
<br/>　　纵然自己现在只有天师不到的法力，纵然看上去是完全的死局，但叶少阳仍然想要试一试。
<br/>　　在这个前后夹攻、完全被包围的局面之下，叶少阳冷静地找到了其中最大的软肋：巨花本身！
<br/>　　有句话叫做最危险的地方也是最安全的地方，反过来也是一样，这巨花的本体，看上去是最为安全的所在，实际上却也是最大的弱点。
<br/>　　叶少阳心如明镜，明白不管自己怎么对付眼前的藤蔓，都是治标不治本，不光白白浪费了时机，也消耗了法力，擒贼先擒王！只有干掉巨花本身，才能一劳永逸地解决眼前的局面。
<br/>　　要么逃走，要么拖下去，坚持到妙心等人来救自己，这是眼下看似仅有的两条路，但叶少阳却选择了第三条，并且制造了一个跳桥的假象，吸引了巨花的注意，自己再猛然折身，奔向巨花本体……连巨花自己也被他骗过，等调集藤蔓前来围剿的时候，终究是晚了一步，叶少阳已经飞身踏上了一朵花叶，花叶看上去厚实，实际上并不承重，不过叶少阳也是早有准备，右手一探，勾魂索已经勾住了花茎，用力一拉，人纵身上去，朝着花蕊方向飞过去。
<br/>　　之前花蕊打开之后，将曹雨兴的尸体直接吸收了，中间虽然长出了长长的茎干，但是还留下了一道膜，尸体被消化之后，留下一团好像液体的所在，鼓鼓的看上去像是一个水袋，还发出淡红色的光。叶少阳断定，这个东西多半就是这株巨花的命门所在。
<br/>　　巨花仿佛也看出了他的意图，几只叶片一起移动，前来围剿叶少阳，叶少阳用太乙拂尘凌空刷过去，借助太乙拂尘强大的灵力，将这些叶片一一刷开，一只手抓着勾魂索，荡到了花蕊的附近，左手一翻，将太乙拂尘暂时收起，从袖子里摸出一张早就画好的暗金神符，只是差了符头上的一笔，当下尾指弹出朱砂，用太乙拂尘接住，在恶上面涂抹下了这最后的画龙点睛的一笔。
<br/>　　焚天符，以自己目前的法力，根本施展不出来，叶少阳只能借助太乙拂尘的灵力。
<br/>　　焚天符画好之后，叶少阳立刻就要念咒，突然感觉头顶上空阴风乍起，抬头一看，是那个好像蛇颈龙一样的长颈弯了下来，五片花瓣一起张开，如果一只怪物的嘴。
<br/>　　试试谁快吧！
<br/>　　这个时候如果退缩，自己靠着意外突袭争取来的机会就会泡汤，再想找机会就难了，叶少阳一咬牙，开始念咒：
<br/>　　“太上三清，四方大帝，神威通天，焚天……”咒语还没念完，突然一声撕心裂肺的惨叫，从身后传来。
<br/>　　叶少阳猛然一惊，回头看去，因为自己现在悬在了半空中，能够俯瞰到桥上的全貌，叶少阳清晰地看见，在一片藤蔓的海洋之中，妙心被抓住了，四肢都被藤蔓紧紧困住。
<br/>　　有藤蔓从她的嘴巴、鼻孔和耳朵里钻了进去。妙心连叫都叫不出来了，只能用绝望的眼神望着叶少阳。
<br/>　　她被抓住了……
<br/>　　现在过去救她是来不及的，就算自己不要命，也只会一起死，叶少阳一咬牙，继续念咒，就在这时，两团藤蔓刺进了妙心的眼睛。
<br/>　　噗的一声，眼球爆开。然后藤蔓钻了进去，一起扭动，脑袋咔嚓一声被打开了，红色的血，白色的脑浆子，一起流淌出来……
<br/>　　妙心，地师家族传人，就这么惨死在了自己面前。
<br/>　　“妙心！！！”叶少阳发出一声沉痛的呐喊，妙心对他来说，不过是一个认识才几天的朋友，但是……她在叶少阳心中的印象非常好，觉得是一个可以做朋友的人，或者说，他已经把妙心当成了朋友。跟内心邪恶的曹雨兴当然不是一个地位。
<br/>　　眼看着妙心就在自己面前惨死，而且她的死，还跟自己有关，一贯冷静的叶少阳，这时候心态也有点崩了，心中百感交集，哪里还顾得上念咒。
<br/>　　突然间，一道黑影从天而降，叶少阳猛然抬头，看到是那只弯下来的茎干，没等他回过神来，前端那个打开的花骨朵一下子咬住了他。
<br/>　　叶少阳顿时感到身边一片漆黑，有一股强大的力量，正在试图吸收他体内的罡气，罡气吸完，那就是血肉了。叶少阳心神一凛，强行作法，抵挡这股吸力，但是自己身体四肢也都被定住，完全动弹不得，感受着这股吸力不断增强，叶少阳心中也是明白，自己陷入绝境了。
<br/>　　“太乙无极，堪破虚境，急急如律令！”
