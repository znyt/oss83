　　天地悠悠，岁月如水，缓缓流淌，在三界那平静的气氛之下，劫气，正缓缓升起。
<br/>　　夏桀的昏庸无道，终究导致了大夏朝的覆灭，人族历史上第一个奴隶制王朝，终究是走上了末路，湮灭在历史的浪花之下。
<br/>　　夏灭商兴，商汤建立大商朝，金乌十太子谋划成功，商汤的确是成了人族共主，建立新王朝，建都朝歌，分封八百诸侯，共尊商朝。
<br/>　　九州人族境八百诸侯国，敬服于商朝的强大，尊其为主，商朝之主为天子。
<br/>　　在天乙成为人族共主之后，十太子便回到了娲皇宫，因为他怕人族知道此事后，定会捉拿他，只有回到娲皇宫，女娲才能庇佑他。
<br/>　　实际上这一步他走对了，在汤成为共主后，人族便知道了此事，但此时大事已定，众人也无可奈何，但却也导致了商汤只有六百年的国运可享。而当众人在寻找始作俑者，他却回到了娲皇宫，人族对女娲还是保持着几分敬意的，虽然在心中对此不满，但也不会多说。
<br/>　　而同时女娲也知道了此事，她想起了妖帝临终前对她的托付，她当天便进十太子叫过去训斥一通，最后直接将其禁足，不得出娲皇宫。明面上是惩罚十太子，暗地里却是帮助他。
<br/>　　历史的车轮滚滚向前驶去，无可阻挡。
<br/>　　八百诸侯，代表着八百条异心，以东南西北四大诸侯为首，一代代传承下去，对商朝的认可度也逐渐低了，一种叫做野心的东西在逐渐形成，说不得哪天便爆发了。
<br/>　　东伯候姜文龙率先作乱，商天子派西伯候前去剿灭，双方大战，姜文龙后来却被自己的亲人给暗算了，东伯候大败被抓，叛军被绞杀。而东伯候的儿子姜尚，只有八九岁却侥幸躲在了尸体中借此机会逃了出来，之后被自家的四代家仆姜铉救下，因家逢大难，姜尚悲伤过度，大病起来。
<br/>　　两人想要去朝歌见东伯候最后一面，因为是逆臣之后，两人都不敢走官道，而是走了山路。姜铉因为年老关系，经过风餐露宿，后来得了重病，不久撒手人寰。
<br/>　　最后，只剩下姜尚一人磕磕碰碰，经过千辛万苦，最终走到了商都朝歌。然而，到了朝歌后，见到的却是自己父亲的人头，姜尚大悲，晕倒在一家酒馆门前。
<br/>　　酒馆的老板姓宋，名异人，却是个热心的主儿，将姜尚倒在自家门前，便叫人抬到后堂，有唤了郎中给他治病。
<br/>　　姜尚醒来后，却只说自己姓姜，名子牙，双目无亲，贫苦无依，来朝歌乞讨为生。宋异人觉得其中另有隐情，但也不好多问，便留下了姜子牙。
<br/>　　姜子牙浑浑噩噩，平时除了帮宋异人干一些杂货，就是喜好读兵书战策，宋异人一概不问，任由他自便。
<br/>　　时光易逝，商天子帝文丁在位，凡事有条理，殷商表面上有振兴之势，实际上天下诸侯之心已散，只是畏惧闻太师的威严、崇候虎的军马，暂时不敢有动作罢了。帝文丁在位二十九年便死了，之后太子帝乙继位，帝乙在位，也是战战兢兢，凡事不敢逾越规矩，成汤的天下还算安定。
<br/>　　姜子牙在宋异人家中却是有三十二岁了，宋异人为他说亲，他却是不同意。这日，他正在读书，却听到有人在叫唤他。
<br/>　　姜子牙一惊，回头看去，却看到一老翁，这老翁穿着淡黄道袍，长眉如雪，手持桃木杖，颇有种仙风道骨的模样。
<br/>　　姜子牙见此，却心知这人不是普通人，他这辈子虽然生活艰辛，但也养得一双看人的眼睛，一眼便知普通人与贵人的差别。他上前行了一礼，问道：“道长唤我前来所谓何事？”
<br/>　　那老翁呵呵一笑道：“子牙，我知你心中愁苦，无以排遣，特来与你说上一二？你想一生都如此度过吗？”
<br/>　　姜子牙闻言，回答道：“小子身世飘零，一生孤苦无依，但却有大仇在身，又怎会甘心如此度过余生？”
<br/>　　那老翁笑道：“红尘轮转，人生多苦，苦海无边，众生沉沦，这世上又有哪一人不是红尘人，子牙，你可愿随我入山修道，求其清净，却好过沉沦红尘？”
<br/>　　“子牙愿意！”姜子牙闻言，大喜道。
<br/>　　“好，你这便随我去吧。”说完，转身便走，姜子牙快步跟上。
<br/>　　这老翁便是南极仙翁，奉元始之意下山前来朝歌度姜子牙上山，入昆仑拜入元始座前。元始早已知晓这量劫的应劫之人，是故早早做了安排。
<br/>　　到了山前，却见拜师的还有一人，南极仙翁本来是不想多管，来昆仑山拜师之人虽然不常见，但多少还是有的，如今日这般并不少见。但就在此时，山上元始天尊却传来话音，让此人跟着一同上去。仙翁只好带着两人上山去。
<br/>　　上了山门，拜见元始天尊，两人便等着天尊的命令。
<br/>　　上首元始看着下方两人心中却极为惊讶，因为他看到应劫之人竟然并不是一人，而是两人，这又是怎么回事？
<br/>　　“下方两人，各为何名？”天尊问道。
<br/>　　“小子姜子牙，拜见天尊！”
<br/>　　姜子牙闻言，躬身拜道。刚开始在知道自己拜的是天罚圣主元始天尊的时候，姜子牙还是激动了好一会儿，没想到鼎鼎大名的圣主就在自己面前，一想到以后就要拜在圣人门下，他更是激动不已。
<br/>　　“散仙申公豹，拜见天尊！”
<br/>　　另一人也在此时开口道。不过，他却是有些修为在身，不似姜子牙一般是一个凡人，但他的修为也不高，只是刚过了凡人境，为真仙境界。
<br/>　　“姜子牙，申公豹？”
<br/>　　元始嘴角露出一个诡异的弧度，好似想到了什么好玩的主意。
<br/>　　“尔等可愿拜入本座门下，成为吾之记名弟子。”元始天尊对着两人道。
<br/>　　“吾等愿意。”
<br/>　　两人自无不可，虽然是记名弟子，但好歹也是圣人的记名弟子，好过其他数倍。
<br/>　　“嗯。”元始点了点头，道：“这是你俩的修炼仙经，在我门下，记住需遵守教规，不得违背，好好修炼。”
<br/>　　说完，元始打了两道流光飞出，飞入两人手中，变成普普通通的两本书籍。
<br/>　　“多谢老师！”两人大喜，拜道。
<br/>　　“嗯，去吧！”
<br/>　　两人再次拜谢，之后随着南极仙翁出了玉虚宫，在昆仑山上找了一地，住了下来。
<br/>　　姜子牙回到屋中，翻开秘籍，却见上面写到《玉清元始经》，之后秘籍上的字如同活了过来，变成白色流光飞入他脑海之中。
<br/>　　与此同时，申公豹以掏出秘籍，带着激动的心情准备打开查看，之前他修炼的都是普通仙法，并不出众，如今拜入圣人门下，想必仙法一定是顶级的。
<br/>　　打开一看，他却立刻傻眼了，因为上面写着《道友，请留步》几个神秘大字，同时一股神秘的波动在上面流转，顿时吸引住他的目光。
