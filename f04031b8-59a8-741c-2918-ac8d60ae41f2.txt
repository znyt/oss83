　　这一群不速之客，足有三四十人，个个都是武皇境以上修为的武者，其中还有五名准帝。
<br/>　　而为首的一人，是一名留着短须的褐袍老者，身形微胖，脸色阴沉。
<br/>　　一众人大步而来，很快便走到了苍穹神宫正在建造的府邸之前。
<br/>　　褐袍老者瞥了一眼府邸门前，正在谈笑的苏莫和夕儿，眸中闪过惊讶之色。
<br/>　　因为苏莫和夕儿年纪轻轻，居然达到了武皇境九重的境界，非常的惊人。
<br/>　　“你们是什么人？”老者朗声喝问道。
<br/>　　但是，没有人回答老者的话，苏莫和夕儿依旧在闲聊，府邸中那些苍穹神宫的强者，亦在各自的忙碌。
<br/>　　一座座石殿拔地而起，一座座简易的亭台修建完毕。
<br/>　　老者的脸色阴沉了下来，老者身后的一众强者，亦是脸色难看。
<br/>　　一群不知来历的人，居然不经过他们途家的同意，在他们途家的地盘上大兴土木，简直不将他们途家放在眼里。
<br/>　　“这里是途山镇，你们在此新建如此规模的府邸，问过我途家了吗？”老者骤然一声暴喝，声浪滚滚，震动四野。
<br/>　　言罢，老者陡然一掌击出，霎时之间，一道浩大的掌印，瞬间便轰击在了刚刚竖起的府邸院墙之上。
<br/>　　轰隆隆！
<br/>　　一阵剧烈的爆响，一大片延绵十数里的院墙，直接被一掌拍的粉碎，无数的碎石激.射八方。
<br/>　　这一下，所有苍穹神宫的强者都楞住了，全部都停下了手中的动作，脸色铁青了起来。
<br/>　　府邸的轮廓刚刚建成，居然有人来搞破坏！
<br/>　　府邸大门前，苏莫和夕儿从大石上站了起来，两人也是让这突如起来的变故，搞的有些愣神。
<br/>　　随即，苏莫瞥了一眼褐袍老者等人，脸色阴沉了下来。
<br/>　　唰！
<br/>　　一步踏出，瞬息数千丈，苏莫来到了褐袍老者等人的前方。
<br/>　　其他苍穹神宫的人，也是纷纷前来，飞到了苏莫的身后，和一众不速之客对峙了起来。
<br/>　　“你们这是什么意思？”苏莫目光如刀，直视褐袍老者，冷漠的问道。
<br/>　　“什么意思？”
<br/>　　褐袍老者闻言，顿时冷笑一声，道：“这里是我途家的地盘，没有我途家的同意，你们居然如此大张旗鼓的新建府邸，简直是不将我途家放在眼里！”
<br/>　　“途家是什么势力？”苏莫再次问道，他心中的杀机，已经缓缓升腾了起来。
<br/>　　“哼！我途家是途山镇第一家族，途山镇的霸主！”
<br/>　　褐袍老者满脸傲气，眼眸斜视着苏莫，继续道：“给你们一个忠告，要么滚出途山镇，要么臣服与我途家！”
<br/>　　老者的语气霸气无比，蕴含不容拒绝的威严，这是作为途山镇霸主的威严。
<br/>　　他也是看苏莫等人，修为都不弱，才有招揽之意，不容就直接轰走了。
<br/>　　一众苍穹神宫之人闻言，个个面带怒色，不过，他们也没动手，毕竟他们初来乍到，还不清楚情况。
<br/>　　“很好，你很有勇气！”苏莫笑了起来，嘴角勾起了一丝森然的笑容。
<br/>　　“什么意思？”
<br/>　　老者闻言一怔，但他还没有反应过来，一道黑色的剑光陡然暴起，向他当头斩来。
<br/>　　这道剑光速度奇快无比，杀机凌厉森然，瞬间撕裂空间，降临到了他的头顶。
<br/>　　是苏莫出手了，手持十亿魔剑，狠狠的斩向褐袍老者。
<br/>　　“什么？”老者大惊失色，因剑光速度太快，他根本来不及躲避，电光火石之间，他只能极力催动护体玄力，在头顶形成了一道防御。
<br/>　　嗤！
<br/>　　噗嗤！
<br/>　　黑色的剑光斩过，护体玄力被撕裂，剑光一斩而下，褐袍老者被一劈两半。
<br/>　　没有惨叫声响起，更没有鲜血喷涌，褐袍老者的血液，几乎是瞬间便被十亿魔剑吸干了。
<br/>　　嘭！
<br/>　　褐袍老者的身躯倒了下去，被一剑秒杀！
<br/>　　静！
<br/>　　四周死一般的寂静！
<br/>　　除了苍穹神宫之人，没有太多惊讶之外，那些跟着褐袍老者到来的途家强者，以及远处围观的途山镇武者，全部都吓傻了！
<br/>　　众人望着倒下的褐袍老者，纷纷脑中剧震。
<br/>　　这名老者不是别人，乃是途山镇途家大长老，准帝级强者，途山镇第二高手。
<br/>　　如此强者，居然被一剑秒杀了！
<br/>　　而出手之人，只是一名武皇境九重的青年！
<br/>　　所有途山镇的人，都是满脸呆滞之色，怔怔的望着变成两半的褐袍老者。
<br/>　　“大长老！”
<br/>　　“大长老！”
<br/>　　“大长老！”
<br/>　　少倾之后，一众跟随褐袍老者而来的途家强者，纷纷惊呼出声，各个满脸悲戚之色。
<br/>　　他们仿佛不敢相信自己看到的一切，大长老居然死了，还死的如此凄惨。
<br/>　　“你……你杀死大长老，你们必死无疑！”一名途家强者怒视着苏莫，厉声大喝
<br/>　　“你们都要死！”
<br/>　　“敢杀我途家人，你们逃不了！”
<br/>　　“你们等着！”
<br/>　　一众途家强者纷纷怒喝，但是他们连褐袍老者的尸体都不敢收取，个个转身就逃，仿佛极怕苏莫对他们出手一般。

<br/>　　本来，他们见苍穹神宫如此多强者，在这里大兴土木，便趾高气昂的前来，准备赶走或者收服。
<br/>　　没想到一个照面，大长老便惨死，他们哪里还敢多待。
<br/>　　苏莫望着一众途家之人逃走，他也没有阻拦，这些人识趣最好，不识趣的话他也不介意大开杀戒。
<br/>　　“这些人是什么人？居然敢杀途家大长老？”
<br/>　　“不清楚，如此多的强者，会不会是哪个大势力的人？”

<br/>　　远处围观的途山镇武者，议论纷纷，声音吵杂不已。
<br/>　　所有人都是感到震撼，这群外来者居然如此的强势。

<br/>　　就在此刻，一声惊天暴喝，突然在远处响起，如同怒雷炸开，震动四野。
