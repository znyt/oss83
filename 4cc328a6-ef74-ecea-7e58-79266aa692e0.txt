　　天空上，瞬间响起男子竭斯底里的惨叫声，他已经后悔，若是知道会碰到这种恐怖的怪物，给再多报酬他也不敢来。
<br/>　　但现在说什么都晚了，随着九霄琉璃焱的焚烧，他的灵魂也化作虚无，彻彻底底的死亡。
<br/>　　实力到了斗皇级别，灵魂已经能离体，若是有媒介，甚至能够借助媒介活下去，就像药尘那样，最后重生也说不定。
<br/>　　但慕清霄连他的灵魂一起焚化为虚无，直接断绝了这种可能。
<br/>　　站在远处，美杜莎美眸中满是凝重，后者的强大实在是将她惊到了，仅仅三秒的时间，就秒杀一位五星斗皇，简直恐怖如斯！
<br/>　　古河更是如此，眼中满是惊惧，望着自高空坠落的五星斗皇尸体，身体控制不住的颤抖起来。
<br/>　　他虽是斗皇，但一心钻研炼药术，战斗能力本就不强，更别说只有三星斗皇，对上慕清霄，估计就是炮灰。
<br/>　　回过神来，古河不顾远处的云韵，独自转身倾尽全力向沙漠外逃去。
<br/>　　然而，刚离开原地没几秒，耳边就传来嘹亮的凤鸣，使得他浑身上下都被恐惧所充斥，紧接着背部就传来一阵锥心刺骨的疼痛。
<br/>　　不知何时，慕清霄已经来到他的身后，利爪直接刺进他的背部，贯穿至胸膛。
<br/>　　“啊啊...！”
<br/>　　伴随着献血四射，天际回荡起古河凄厉的惨叫声，被爪子直接贯穿，剧烈的疼痛冲击着他的灵魂，差点使他晕厥过去。
<br/>　　慕清霄将利爪抽出，火焰将之蒸发，古河在空中踉跄几步，鲜血自口中狂喷而出，摇摇欲坠，最后向下方的沙漠坠去。
<br/>　　身体坠落，顿时将黄沙地面砸出一个人形，鲜血渗入黄沙之中，已经是出气多进气少。
<br/>　　但斗皇毕竟是斗皇，哪怕身体被贯穿，也绝对不可能轻易的死去，慕清霄更不可能放过他，以免放虎归山。
<br/>　　不是你死就是我活，古河为了异火要杀自己，慕清霄自然不会手下留情，而且还会比他们更狠。
<br/>　　弱肉强食，便是本世界的生存法则。
<br/>　　黑色羽翼一振，身躯已经降落在古河的身旁，居高临下的望着他。
<br/>　　此时，古河心中除了恐惧还是恐惧，灵魂都在颤栗，抬头看向天空中的黑袍人，语气虚弱，道：“咳咳，云韵，救......”
<br/>　　“蓬！”
<br/>　　然而，他的话语还未说出口，慕清霄的利爪就踩在他的脑袋上，直接将他按进黄沙中，狭长的眼眸中，杀意凛然。
<br/>　　此人真是伪君子，他可不相信，古河与云韵相处如此长的时间，会不知道对方的实力，现在求救，根本就是要拖她下水。
<br/>　　云韵现在不过三阶斗皇巅峰，上来还不够自己一抓呢！
<br/>　　眼眸中闪过一丝戏谑，同时，慕清霄也清楚了自己的力量，他现在是化神二重中期，但化身黑凤后，却能越级秒杀五星斗皇。
<br/>　　最重要的是，他根本没用全力，按照他的预测，抗衡七星斗皇估计没问题，拼命的话，战九星斗皇也不是不可以！
<br/>　　要知道，他可是凤凰，不管是肉身，或者是速度都占据绝对的优势，更何况还有九霄琉璃焱，恐怖的威力，触碰到一缕就能将对方烧成渣！
<br/>　　而且，他还有底牌，那就是永恒万花筒写轮眼的极强幻术，哪怕打不过，施展幻术，拖延时间逃跑就行了。
<br/>　　瞥了眼上空的云韵，慕清霄能感觉到，她也紧张到了极点，却没有立刻离开，真是傻的可以。
<br/>　　九霄琉璃焱自凤爪上蔓延而出，伴随着古河绝望的惨叫声，肉身和灵魂皆是被焚烧的一干二净。
<br/>　　待古河的尸体消失，慕清霄妖异的瞳孔盯着云韵，这小妮子吓得后退几步，险些让他笑出声来。
<br/>　　随后，慕清霄又瞥了一眼美杜莎女王，这女人颇为识趣，没有上来参合，否则，就算是九星斗皇，也要将她按在地上狠狠的修理一顿，管你是不是女王！
<br/>　　见慕清霄视线投向她，美杜莎柳眉紧皱。
<br/>　　后者下手虽毫不留情，但她却不反感，刚才施展出的火焰，恐怖至极，而且非常诡异，不像别的火焰，根本察觉不到一丝的温度，不知是何种品级的兽火。
<br/>　　她身怀蛇人阴寒血脉，与他战斗本就不利，但异火对她来说却太过重要，于是开口道：“你将青莲地心火给我，有什么要求你尽管提，只要本王能办到都会满足你，你意下如何？”
<br/>　　声音中没有先前的高傲，但清冷依旧，慕清霄打量着她那妖艳的脸蛋，丰满玲珑的娇躯，脑海中不禁浮现将女王压在身下的画面。
<br/>　　最后想了想，慕清霄还是打消了这种念头，强扭的瓜不甜，真要提出这种要求，自己岂不是与小人无异，青莲地心火虽不能借她，但他却能用别的办法帮她。
<br/>　　美杜莎进化的要求，并不是吞噬异火，异火蕴含的毁灭性力量可不是开玩笑的，直接吞下去，估计能将她肉身连带灵魂都焚为虚无。
<br/>　　她想要进化，需要用极强的火焰焚烧自己的身躯，使她在焚烧中蜕变，蜕变中寻求突破......
<br/>　　异火虽强，但跟九霄琉璃焱比起来，简直是小儿科，宛若萤火于皓月，完全不可同日而语。
<br/>　　最重要的是，慕清霄能随意控制九霄琉璃焱，能强能弱，简直就是她进化最好的选择。
<br/>　　回头看了一眼，云韵依旧站在高空，警惕的视线紧紧盯着他，慕清霄微微摇头，看来自己不走，这妮子是不敢擅自离开了。
<br/>　　最后看了一眼云韵，慕清霄朝美杜莎挥了挥翅膀，让她跟上来，双翼一振，便化作一道黑色流光，转眼就消失在她们的眼中。
<br/>　　美杜莎诧异的看了眼云韵，空间一阵扭曲，身影也自原地消失，追寻着慕清霄而去。
<br/>　　等慕清霄与美杜莎离开，云韵深吸了口气，紧绷的心弦终于是松懈下来，想到自己刚才在鬼门关走了一遭，心中后怕不已。
<br/>　　但心中却觉得怪怪的，刚才的那只凤凰看自己的眼神不对劲，却没有下杀手，难道是因为自己没有向他动手？
<br/>　　回过神来，脑海中浮现一张神骏的面孔，云韵美眸中闪过一丝温柔，低喃道：“你现在身在何处，有没有好好修炼？”
