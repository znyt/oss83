　　　　变异银蚊，十二级。
<br/>　　涂小安得到了这只大蚊子的数据，到也不意外了。
<br/>　　唯一让他有点惊讶的是，这种变异蚊子，居然能够吸食人的精血。
<br/>　　这可了不得。
<br/>　　人的精血等于生命力，少一滴，寿命就会大减，身体会变的异常的虚弱。
<br/>　　再看这只大蚊子，通体银白，它的口器长而尖锐，它的双翅既然寒光凛然，刀片般锋利，可见血封喉。
<br/>　　涂小安万万没想到一只蚊子能达到这种地步，就跟系统进化出来似的。
<br/>　　此时，涂小安也好像明白了，为什么庙里的中毒者，有些毒深，有些毒浅，现在看来毒浅的应该是被那些白色蚊子叮咬的。
<br/>　　而毒深的，想必就是被这只银蚊所害。
<br/>　　毕竟一只觉醒类的蚊子叮咬人，把人叮死，好像也没什么可意外的。
<br/>　　涂小安伸出手，将这只银蚊的双翅给掐住，彻底控制，几个念头闪过，这种觉醒类的蚊子，百年难得一见啊，杀了到有点可惜了。
<br/>　　不如收服了留着玩玩也好。
<br/>　　“臣服于我”
<br/>　　涂小安的意识力重新投放到这种银蚊身上，意识威压迸发。
<br/>　　霎时间，这只银蚊不断的开始怪叫挣扎，而大青石上，鹰钩鼻男子更是脸色骤变，怒吼：“你在干什么”
<br/>　　一句话刚吼出，他没来由的一口血喷了出来，脸色猛然苍白起来。
<br/>　　反而，那只被涂小安掐在手中的银蚊已经消失不见。
<br/>　　“我的蚊子呢，你到底对它做了什么”
<br/>　　鹰钩鼻男子好似晴天霹雳当头一击，他既然发现自己跟银蚊的联系断了。
<br/>　　这怎么可能啊，虽然是只蚊子，但确是他的宠物，跟他是有着灵魂契约的。
<br/>　　他现在完全没了银蚊的心灵感应，那口血喷出，等于银蚊已经死了一样，让他受到了反噬。
<br/>　　“他现在是我的了”
<br/>　　涂小安微勾唇角，妖冶的眸子中化过一丝玩味，意念一动，消失的银蚊凭空出现，缭绕着他在飞。
<br/>　　鹰钩鼻男子看到这一幕，不敢相信自己的眼睛，自己的宠物反叛了，认对方为主了？
<br/>　　他到底是怎么做到的。
<br/>　　怪物，这是一个彻头彻尾的怪物。
<br/>　　涂小安想强制性收服这只级别的生物，何等简单，他现在可是二十三级呀。
<br/>　　不理会鹰钩鼻男子的大惊小怪，淡淡的看着他道：“说说吧，为什么用这种蚊子来害人”
<br/>　　目前，涂小安想不到对方这样做的目的是什么。
<br/>　　莫不成只是为了报复社会？对方可是人类觉醒者，不会做这种傻逼事吧。
<br/>　　“这与你无关，我们井水不犯河水，要是在下无疑冒犯到阁下你，我向你赔罪，你将我的蚊子还给我，我立刻离开这里”
<br/>　　鹰钩鼻男子心在滴血，极为忌惮的开口。
<br/>　　“井水不犯河水？”涂小安像是听到了什么天大的笑话，微微斜睨了他一眼：“知道今天为什么街道上那么人满为患吗，今天是白镇的蛇王节，是我的节日，你破坏了我的节日，还想全须全尾的离开白镇这个地界吗”
<br/>　　此话一出，鹰钩鼻男子如雷贯耳，难以置信。
<br/>　　今天是蛇王节，我的节日？
<br/>　　霎时间，鹰钩鼻男子撑着一双睁大的眼睛看着涂小安，对方身上的气息很强大，但气息却透着无比阴冷，不像是人类的气息。
<br/>　　现在他的话又如此说，鹰钩鼻男子惊讶万分，莫非对方是一条蛇精？
<br/>　　不，是蛇妖？幻化人形。
<br/>　　可现在在人类的地界上，怎么可能出现会妖精级别的生灵存在。
<br/>　　短短几个呼吸，鹰钩鼻男子无数个念头划过，不过他好像真的踢到了铁板上。
<br/>　　对方之变态，他现在也算领教过两回了，第一回无形中杀死了他一百只白色蚊子，这一回，悄无声息的夺走了他的觉醒类宠物。
<br/>　　退意，他现在皆是退意。
<br/>　　“你要不说，我会让你死的很惨的”
<br/>　　涂小安冷淡无比的看着大青石上的鹰钩鼻男子。
<br/>　　“哼...人妖不两立，你不要做的太过分了”鹰钩鼻男子周身能量在涌动，头发迎风而动。
<br/>　　涂小安笑了：“狗急了要咬人？”
<br/>　　“我不管你是什么东西，将我的蚊子还给我，否则我会不惜一切代价的出手”鹰钩鼻男子一脸的凝重，手中不知何时出现了一根雕刻精巧的玉笛，这玉笛一拿出来就散发出一股冰寒的气息，一看就不是凡物。
<br/>　　涂小安还真没跟人类觉醒者真正战斗过。
<br/>　　只是第一次在蛇博士陈默的手中吃了一个大亏。
<br/>　　这次又是人类觉醒者，他还真想看看对方有什么手段，以后估计他会遇到越来越多的人类觉醒者。
<br/>　　这次蛇王节事情闹的很大，他也不断的装逼显化，肯定会引发一些特殊人群的关注。
<br/>　　不过涂小安并不怕，实力决定一切。
<br/>　　“我给你出手的机会”涂小安霸气凛然，浑然不惧。
<br/>　　“拼了”鹰钩鼻男子眉头一紧，咬了咬牙，将精巧的玉笛放到了嘴边，旋即，便是一阵悠悠笛音便缓缓扬起，笛音婉转缥缈，不绝如缕，宛若朱雀般轻鸣，另带着一股摄人勾魂之意。
<br/>　　霎时间，空气像是煮沸了一般，四周的生物都躁动不安起来。
<br/>　　笛声就似一股无形的能量波动，朝着上方的林中幽然飘去。
<br/>　　涂小安原地不动，冷冷看着，不知道对方到底想做什么，但对方手中的玉笛貌似不错，是个宝贝。
<br/>　　隐晦间，涂小安想起了蛇博士陈默的口哨声，两者好像有异曲同工之妙。
<br/>　　不过蛇博士的口哨声一响起，涂小安就会非常痛苦，对方这笛声，让他听了到一点感觉都没有。
<br/>　　就跟欣赏一段音乐似的。
<br/>　　可随着笛声的悠扬，这时诡异的事情发生了，山脚下，树林中，很多隐藏在暗处的野生蚊子嗡嗡嗡的飞了出来，一批又一批的朝着大青石上方络绎不绝的凝聚而去，蚊子的数量随着笛声的传播越来越多，多到最后恐怖的形成了一片可怕的乌云，乌云遮天蔽日的笼罩在了大青石上方的鹰钩鼻男子身上。
<br/>　　一块蚊子形成的乌云将上方的星光，月光彻底的隔绝了。
