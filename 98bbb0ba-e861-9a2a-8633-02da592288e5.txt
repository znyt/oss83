　　“剑祖荒？”
<br/>　　寒月漓看向那出声之人，一眼便是看得出来，眼前这剑祖荒的实力，只怕是比起这天羽神域中大部分的武者，都要强大的多。
<br/>　　在这天羽神域中，真正能够跟剑祖荒抗衡的武者，几乎都找不到多少。
<br/>　　寒月漓来到这天羽神域中，也是听说过剑祖荒的名字，深深的知道剑祖荒的厉害。
<br/>　　“不错，正是我，寒月漓，我念在你跟我们天剑宗无冤无仇的份上，今日我不想杀你，只要你选择臣服与我，我可以饶你一命条性命。”
<br/>　　剑祖荒看着寒月漓，冷笑连连开口道，语气霸气无比，高高在上。
<br/>　　“臣服于你？我还从来没有臣服过谁，还请诸位请回吧。”
<br/>　　寒月漓冰冷的开口道。
<br/>　　“如今你以为你除了有臣服这一条道路外，还有其他可能吗？”
<br/>　　听到寒月漓的话，剑祖荒突然冷笑连连起来道，看着寒月漓开口道。
<br/>　　“我知道你的身份，你是北邙学院的弟子，北邙学院，在整个九域来说只能算得上是二流宗门，在我们天剑宗的面前，更是如同蝼蚁一样，我杀了你，北邙学院也不敢说什么。”
<br/>　　“现在跪下，臣服于我，才是你唯一的出路，只要你做了我的女人，专心服侍我，我可以让你进入到天剑宗去，比起在北邙学院要强上一百倍。”
<br/>　　“休想。”
<br/>　　寒月漓听完剑祖荒的话，只是淡然吐出这两个字。
<br/>　　“是吗？既然如此，那我就擒下你，在将你降服！”
<br/>　　剑祖荒猛然冷喝一声。
<br/>　　在他身边的无数天剑宗弟子，便是纷纷领命，突然之间无数剑光，便是向着寒月漓斩了过去。
<br/>　　这漫天剑气，顿时将这周围冰川，团团覆盖。
<br/>　　这剑祖荒此番前来，将所有的天剑宗弟子都给带了过来，已经是将这冰川周围，全部团团困住。
<br/>　　如今寒月漓就算是插翅也难飞，一点逃走的机会都没有。
<br/>　　“绝地冰恋”
<br/>　　寒月漓眼神冰冷，手持灵剑，猛然一剑斩出，漫天冰花，纷纷展现出来。
<br/>　　今日她自己宁死，也不会选择臣服。
<br/>　　来到这归墟天中，寒月漓也有着极大的区域，她的境界已经是达到了生死境七重的地步，一身实力隐约之间已经是超越了当初的剑苍穹等人。
<br/>　　如今面对围攻，她的脸上也是没有丝毫惧怕，漫天冰剑，纷纷起舞，跟无数天剑宗弟子，战成一团。
<br/>　　“冰莲剑术，据说这是曾经剑道之主座下的剑术，就连我们天剑宗都没有记载过这种剑术，如今擒下这女子，也可以为我天剑宗来收复一道剑术。”
<br/>　　剑祖荒冷哼一声道。
<br/>　　“不用担心伤她，只要是不死，我天剑宗都有丹药将她救活，只要将她速速擒下！”
<br/>　　“是！”
<br/>　　周围的天剑宗弟子，听到剑祖荒的命令，都是纷纷脸上露出几分狠色，再度出手。
<br/>　　他们生怕伤到寒月漓，再回一直以来压制力量，现在听到剑祖荒的命令，一个个实力全开。
<br/>　　其中五名天剑宗弟子，便是突然合力之下，凝聚出一道足以千丈剑气，突然从天而降，向着寒月漓斩杀而去。
<br/>　　“冰冻千城！”
<br/>　　寒月漓看到这千丈剑气落下，脸上也是露出了几分凝重，急忙凝聚灵力，斩出一剑，顿时在她的面前，天际都被冰冻起来。
<br/>　　在这冰裂谷中，寒月漓足以借助周围寒意，爆发出极强的力量，才足以是跟七大宗门的围攻周旋。
<br/>　　“真是麻烦，此女子的剑术，在这冰裂谷中倒是如虎添翼，不好对付。”
<br/>　　剑祖荒看到这一幕，心中冷声想道。
<br/>　　“现在便是不应该在拖延时间下去，不然的话，圣王宗，荒古门的那些人也都该赶来了，免得夜长梦多！”
<br/>　　想到这里，剑祖荒身躯突然一动，他手中剑锋突然出窍，一股剑气便是向着寒月漓斩杀过去。
<br/>　　剑祖荒终于也是忍耐不住，亲自出手。
<br/>　　他的剑气，足有万丈之长，横扫而下，在寒月漓面前凝聚的寒冰，顿时全部破碎，寒月漓的身躯都是连连退后，来躲避着剑锋的锋芒。
<br/>　　“好！”
<br/>　　“剑师兄无敌！”
<br/>　　周围天剑宗弟子，看到剑祖荒出手，一个个纷纷大声笑道。
<br/>　　“寒月漓，既然你敬酒不吃吃罚酒，那就休怪我不怜香惜玉了，不过放心，我不会杀你，我还要你来服侍我呢！”
<br/>　　剑祖荒看着寒月漓，阴冷一笑，再度一剑，向着寒月漓斩了过去。
<br/>　　铺天盖地的剑气，轰然落下，恐怖至极的剑气，将周围的冰川都给硬生生的打裂。
<br/>　　“剑祖荒？你算是什么东西，也敢来打我女人的注意，看来我斩杀你一个化身，还不够给你一个教训是吗？”
<br/>　　突然之间，凭空之中，一股庞大无比的力量降临，只见一座宝车，从天而降，战龙拉车，一时间涌现无尽龙威。
<br/>　　随着宝车上的八尊战龙突然怒喝一声，这周围的所有剑气，尽数破碎，就连那剑祖荒，都后退数步，只能避其锋芒。
<br/>　　在这天魔宝车上，这坐着一个人影，这人影正是楚风眠，他坐在宝车上，高高在上，俯视众生。
<br/>　　“楚风眠？”
<br/>　　寒月漓看到坐在宝车上的楚风眠，脸色大喜，一直以来在她脸上绷劲的紧张表情，如今都是轻松了许多，如同冰川融化一般，脸上展现出几分笑容来。
<br/>　　“来的有些晚了，让你受苦了。”
<br/>　　楚风眠看着寒月漓，柔声说道，他手一挥，寒月漓便是落入到了战龙宝车之上。
<br/>　　“不过不用担心，这些人，今日一个个的都要死。”
<br/>　　楚风眠的眼神中，杀意决断，现在这剑祖荒等人，居然想要打寒月漓的注意，便是触犯了楚风眠的逆鳞。
<br/>　　所有人，今日都要死，一个不留。
<br/>　　“哪里来的野小子，你算是什么东西，区区生死境四重的境界，也敢来跟我们天剑宗作对？找死！”
<br/>　　一名天剑宗弟子，打量了一眼楚风眠的境界，脸上露出几分不屑，猛然之间身躯一动，便是向着楚风眠一剑斩杀而来。
<br/>　　“恩？”
<br/>　　楚风眠双眼看见这天剑宗弟子出手，突然冷哼一声，他坐下的八尊远古战龙，猛然咆哮一声，突然一道巨大无比的龙爪轰出。
<br/>　　这天剑宗弟子的胸口被龙爪直接拍碎，直接化为了一团血肉，当场陨落。
