　　闻听苏莫之言，老者的脸色不断的变换，无比的难看。
<br/>　　苏莫将他击成重伤，杀了玄羽宗不少弟子，还将宗门毁坏成这幅模样，他恨不得将苏莫挫骨扬灰。
<br/>　　但是，他知道，自己现在很难是苏莫的对手。
<br/>　　仅凭刚才一剑，他已经看出来了，苏莫的战力在他之上，虽然没有强过他太多，但他现在已经身受重伤，很难抗衡苏莫了。
<br/>　　而且这里是玄羽宗，一旦与苏莫撕破脸皮，怕是整个玄羽宗都要陪葬了。
<br/>　　“好，星空传送大阵给你使用！”少倾之后，老者黑着脸，默然的点了点头。
<br/>　　他不得不同意，实力不如苏莫，再强自反抗的话会有大祸。
<br/>　　“嗯！”苏莫点了点头，心中不禁感慨，这人就是贱啊！
<br/>　　之前跟你好说好讲，还赠与灵晶你不同意使用传送阵，非要强势把你打痛了之后才同意！
<br/>　　这不是贱是什么？
<br/>　　“随我来！”
<br/>　　老者招呼一声，便向不远处的传送大殿飞了过去，苏莫立刻跟了上去。
<br/>　　“上官小友，不知是何门何派的弟子，武帝境七重修为，居然就有武尊境的实力，真是不凡！”
<br/>　　飞行之中，老者旁敲侧击，想要询问苏莫的身份。
<br/>　　“小门小派，无足道哉！”苏莫摇了摇头，他当然不会说出自己的身份。
<br/>　　此时，他不禁暗想，他的修为玄力，实在是太特别了，以后看来还要尽量避免使用。
<br/>　　比如，他融合后的玄力，乃是三彩之色，极其特殊，与一般的武者完全不同，很容易引起其他人的注意。
<br/>　　这方面还是要谨慎一些，不然一旦消息被巫族之人知晓，他又要危险了。
<br/>　　老者闻言脸皮抽了抽，他可不相信这种话，小门小派能培养出这样的天才？
<br/>　　这可能吗？
<br/>　　估计就是传闻之中，那真圣榜第一的苏莫，也不过如此吧！
<br/>　　很快，老者带着苏莫来到了传送大殿之中。
<br/>　　大殿中一片空旷，只有中央的位置，布置了一个大型的星空传送大阵。
<br/>　　苏莫二话没说，便走到了传送大阵之中，而后，老者也没有再问他索要丝毫的灵晶，便自己拿出灵晶，随后催动了传送大阵。
<br/>　　“前辈，告辞了！”
<br/>　　阵法之光闪耀而起，苏莫向老者微微抱拳，身形缓缓的消失在了阵法中。
<br/>　　就这样，苏莫没有出一块灵晶，只用了一剑，便得以使用了星空传送大阵。
<br/>　　老者看到苏莫离去，脸色一片阴沉，他甚至想要中断传送大阵，让苏莫流落到星空之中。
<br/>　　但想要中断传送大阵，就必须破坏此阵，他真是有些舍不得。
<br/>　　这星空传送大阵，可是他耗费重金，找高深的阵法大师布置，不能轻易损坏。
<br/>　　“哎！”
<br/>　　良久之后，老者暗叹一声，心中后悔不已。
<br/>　　早知道这名青年这么强大，他同意给予对方使用传送阵的话，不仅能收取一万下品灵晶，也不会有任何麻烦。
<br/>　　现在倒好，不仅灵晶没有赚到，自己还身受重伤，宗内损失更是不小。
<br/>　　……
<br/>　　一阵轻微的晕眩感散去，苏莫出现在了一座更广阔的大殿之中。
<br/>　　他眼眸打量四方，此殿辽阔无比，殿高十丈，占地百丈，这同样是一个传送大殿。
<br/>　　殿中星空传送大阵，足有数十个之多，还有几名武尊境的武者，专门在此操控传送大阵。
<br/>　　殿内人来人往，大量的武者在此使用传送阵往来。
<br/>　　苏莫询问了一番，了解了每个传送大阵的目的地，便再次开始了传送。
<br/>　　接下来的时间中，他不断的使用星空传送大阵，辗转一颗又一颗星辰。
<br/>　　有的时候，到达之地不是传送中转站，没有传送大阵，还需要他自行的寻找。
<br/>　　就这样一个月又一个月，苏莫一路不停，没有丝毫的休息，足足耗费了九个多月的时间，他辗转了四百多颗星辰，终于回到了古周星之上。
<br/>　　苍茫的白云之间，苏莫身形如电，向着苍穹神宫的方向飞去。
<br/>　　他肯定不能再回神武学府了，准备先回苍穹神宫，再考虑接下来的事情。
<br/>　　现在，他的修为已经达到了武帝境八重境界，一切水到渠成，他只是在赶路之时，自行修炼，并没有刻意的冲击境界瓶颈，便成功的突破了修为境界。
<br/>　　这得益于神灵液，将他的身体天赋提升了一大截，还让得他的修为境界无比的扎实和稳固。
<br/>　　所以，自真圣界出来之后，他才能轻轻松松的接连提升了两重修为境界。
<br/>　　不过，目前接连晋升了两重境界之后，他感觉根基再次有些虚浮了起来，玄力不再纯净凝实，怕是以后再想提升，不会这么水到渠成了。
<br/>　　“现在在所有人看来，我已经死了，不知道古天意会不会将聂乾和韩盈从镇魂之地弄出来？”飞行之中，苏莫低声自喃。
<br/>　　事实上，若不是因为韩盈和聂乾，他都不会回古周星了，将会直接去明虚星河，寻找宏青璇去了。
<br/>　　聂乾和韩盈两人，要在镇魂之地受五年的镇魂之苦，苏莫实在说服不了自己就这样离开，不管两人的痛苦。
<br/>　　他虽然和两人相处的时间不长，但对于两人重情重义的为人还是非常欣赏的，他早已将两人当成了朋友。
<br/>　　他心中已经决定了，待解决韩盈和聂乾之事，他就立刻去明虚星河。
<br/>　　现在，苏莫唯一担心的事，就是古天意知道他死了以后，根本不会兑现承诺，将聂乾和韩盈弄出来，这样的话就麻烦了！
<br/>　　他只有通过古天意，才能让两人脱困，但是他又不能暴露身份，此事还要仔细的斟酌一番。
<br/>　　又过了数日之后，苏莫风尘仆仆的回到了苍穹神宫。
<br/>　　此时的苍穹神宫，又经过近一年的发展，再次大变了模样，府邸再次扩建一倍。
<br/>　　而整个途山镇也已经变成了一座小城，苍穹神宫的所在，则是一座城中之城。
<br/>　　途山镇虽然偏僻，但随着不断的发展壮大，也吸引了不少武者来此，渐渐变的繁华了起来。
<br/>　　苏莫回来并没有大张旗鼓，只是通知了父亲苏洪和宫内的几位高层人物，没有引起任何人的注意。
