　　对于小鬼子，李承乾从来就没有什么好感，这是从骨子里带过来的，不管经历多长时间也不会有所改变。
<br/>　　满朝文武都很奇怪，为什么这位皇帝陛下对倭国如此‘情有独钟’，就算这个国家的人表现的再恭敬，也依旧得不到他的任务。而且最让他们奇怪的是，既然皇帝陛下如此厌恶这个国家的一切，那为什么又不直接将他们亡国灭种，反而只打了一半便让军队停了下来，甚至从那个岛国里面撤出了全部的兵力。
<br/>　　但是李承乾却有他自己的想法，直接把倭国灭了固然可以，但是倭人是杀不光的，不管战争打到什么样子，总会有一些倭人活下来。
<br/>　　既然这样，那就不如让倭国彻底的乱起来，控制着他们永无休止的杀戮下去，让他们自相残杀，杀他们杀出世仇，让那个小岛变成人间地狱。
<br/>　　秦吉胜并不知道李承乾的想法竟然如此凶残，现在的他还在做着皇帝梦，想要从大唐向到更多的武器装备，只等将来有一天彻底将那个什么山背大兄打败，到时候就可以高枕无忧，再也不用听从大唐的摆布。
<br/>　　在他看来，他是在曲线救国，倭国天皇一脉已经把大唐彻底惹毛了，那么就按照大唐的意志，把他们全都消灭掉，这样至少可以保证倭国至少会有一半人活下来。
<br/>　　不得不说，秦吉胜的确是打的好盘算，只可惜，事实上李承乾对所有的倭人都没有好感，归化人也是一样。
<br/>　　“王大人，他年一别，别来无恙呼？”鸿胪寺中，王玄策奉命接待了秦吉胜一行，才刚一见面，那秦吉胜便开始给老王拽文。
<br/>　　王玄策看着面前卑躬屈膝的倭国伪政权首脑，心中升起一丝怜悯，不过这个念头只是在他脑中一闪而逝，脸上并没有露出任何异色，只是微笑着说道：“秦大人何时登基啊？到时候可别忘了通知王某一声，说不定王玄策还有机会亲自去恭喜你一下。”
<br/>　　之所以王玄策会称呼秦吉胜为秦大人，那是因为倭国伪政权眼下是属于于大唐领导之下，是属国，那秦吉胜还有东倭路大都督的职衔在身。
<br/>　　秦吉胜这次过来就是抱着溜须拍马的想法，王玄策很给面子的顺势捧了他一下，立刻让他感觉到了春天般的温暖，赔着笑说道：“登基可不敢啊，没有陛下的口喻，下官不敢冒犯呢。”
<br/>　　“哈哈哈，秦大人的客气了，大人是否登基其实与我大唐并没有什么太大的关系，就像其他属国一样，秦大人何时见大唐会插手属国内政了？”
<br/>　　面对眼着眼睛说瞎话人王玄策，秦吉胜对此只能报以讪笑。
<br/>　　大唐的确是没有干涉过属国的内政，可是特么大唐现在还有属国么？全都是大唐的国土了好吧！不过这话秦吉胜可不敢说，因为他现在还需要大唐的武器装备，万万不敢真的把大唐得罪了。
<br/>　　想到这里，秦吉胜果断的转移了话题，四下看了看，然后故作神秘的说道：“王大人，能不能借一步说话？”
<br/>　　“哦？何事啊？”王玄策坐在自己的位置上并没有动，只是对身边几个属官摆了摆手，示意他们先出去。
<br/>　　秦吉胜定定看着房间中所有人都出去了，这才一脸谦卑的说道：“王大人，实不相瞒，这次下官之所以提前过来，实在是有事相求，希望大人能在皇帝陛下那里为我们归化人美言几句。”
<br/>　　王玄策隐隐已经猜到秦吉胜这家伙想要说什么，但却没有表现出来，只是皱了皱眉反问道：“哦？美言什么？难道你那里出了什么事情了？”
<br/>　　“唉，一言难尽啊。”秦吉胜长长的叹了口气，然后愁容满面的说道：“您想必知道，我们归化人在岛上的人口基数其实并不大，这些年连场大战打下来，早已经师老兵疲，后继乏力，如果再得不到支援，只怕很快就要被山背大兄他们给吞并了。”
<br/>　　“哦？竟然打的如此艰难？”王玄策的面色终于起了变化，严肃的问道。
<br/>　　“何止艰难，归化人已经快要山穷水尽了。”秦吉胜苦涩的说道。
<br/>　　“这样啊……”王玄策皱着眉头，打着官腔说道：“如果这样的话，我需要找机会向皇帝陛下请示一下，看看陛下是什么意思。”
<br/>　　“谢谢大人，若是归化人能够得到大唐的支援，收复全部国土之后，必将永世尊大唐为主。”秦吉胜一听王玄策答应，也不管他是不是在打官腔，立刻兴奋的说道，末了还从袖子里抽出一张纸，放到了老王身边的茶几之上。
<br/>　　“秦大人这是何意？”王玄策看着那价值万贯的不记名存单，有些不悦的说道。
<br/>　　“王大人，你我相交多年，本来应该从倭国给您带些特产过来，不过这一路山高水远实在是不方便，所以这小小心意您一定要收下！”秦吉胜很会说话，赔着笑脸让人一时有些无语。
<br/>　　王玄策低头看着放在身边的那一万贯的存单，心中很清楚这家伙是什么意思，犹豫了良久之后缓缓伸手将之拿了起来，抖了抖说道：“小小心意？秦大人真是大手笔啊，让在下很难找到理由拒绝。”
<br/>　　秦吉胜等的就是王玄策这句话，闻言之后喜出望外，连连摆手说道：“王大人说的哪里话来，咱们这么多年的交情，说这些岂不见外！”
<br/>　　“也罢，既然你都这样说了，此事我也不好不管，你且说说你都需要些什么东西吧，我看看能不能跟陛下商量商量。”
<br/>　　“如此，那就太感谢王大人了。”秦吉胜先是拱了拱手，表示了感谢，然后才缓缓说道：“其实现在我们真的是什么东西都缺，粮食，武器，衣物，只要能用上的我们都需要。”
<br/>　　“这么多？”王玄策略微犹豫了一下，低头看了看手中的存单。
<br/>　　秦吉胜的心再次提了起来，他很怕王玄策拒绝他，毕竟如果这次被拒绝，他将很难再开口。
