　　看着狼狈逃窜的旧多二福，刘哔嘴角露出一抹冷笑，欺身上去，挥出手中的长刀。
<br/>　　“跑？跑得掉么？”
<br/>　　右眼的伤势让旧多二福的身体反应大幅度下降，面对他这样凌厉的攻击，根本没有反抗的余地。
<br/>　　瞬间，一颗清秀的头颅被刘哔砍下，旧多二福的身体惯性的后退了几步，摔在了洒满鲜血的地板上。
<br/>　　“叮~恭喜宿主击杀独眼喰种旧多二福，获得积分一万点。”
<br/>　　“叮~恭喜宿主成功吞噬独眼蚕种旧多二福，喰种血脉提升，获得技能残影。”
<br/>　　“叮~宿主已成功击杀四头SSS级喰种，完成位面目标，系统将在一个月之后进行传送，请宿主提前做好准备。”
<br/>　　一连串的系统提示音响起，当听到系统最后一句的时候，刘哔脸色突然僵住。
<br/>　　击杀四名SSS级喰种就可以回归吗，但上个位面离开的要求，却是完成所有主线剧情呢。
<br/>　　难道说，所有位面离开的要求都不相同？
<br/>　　还真是让人始料未及呢。
<br/>　　脸上带着沉思的表情，刘哔对接下来的计划做了一个改变。
<br/>　　在他原本的计划中，他是不想这么早离开的。
<br/>　　喰种位面的危险程度并不高，况且还有这么多可以击杀的能量体，这对他来说是一个非常合适的位面。
<br/>　　如果提前知道离开的标准，他大可晚些完成任务，趁着这段时间多赚取一些积分。
<br/>　　反正这里的时间和他本来的世界流逝速度不同，上一个位面他呆了一个多月，外面也只是过去一星期而已。
<br/>　　离开之前，他想将实力提高一些，那样回到主世界中也能轻松一点。
<br/>　　可惜系统并没有给他这样的机会。
<br/>　　不过还好，还有一个月的缓冲时间，如果利用好了，他也可以做很多事情。
<br/>　　对接下来的行程慢慢规划着，刘哔收起了翅膀，离开了美林大厦。
<br/>　　……
<br/>　　清晨，二十区的一栋居民楼里，刘哔满身血迹的站在一扇防盗门前。
<br/>　　这是一个老式小区，虽然装有摄像头，但是大部分已经失效了，之所以安装，也只是做做形式上的样子罢了。
<br/>　　“咚咚咚。”
<br/>　　沉闷的敲门声响起，一抹初阳，透过昏暗的走廊窗照射了进来，洒在刘哔身上。
<br/>　　昨天经历了太多战斗，他身上的血迹，一时半会却是很难清理干净了。
<br/>　　“吱呀。”
<br/>　　防盗门缓缓打开，露出一个朝气蓬勃的身影。
<br/>　　开门的是一个学生模样的男人，带着眼睛，一头淡金色的半长头发，模样倒还算英俊。
<br/>　　“是西尾啊，什造呢？”
<br/>　　看了一眼站在门前的青年，刘哔笑了笑，从门口走了进去。
<br/>　　似乎是在担忧什么，西尾锦查看了一下楼道，紧张兮兮的关上防盗门。
<br/>　　“大人，铃屋小姐还未起床，要不要我叫她起来。”
<br/>　　站在刘哔身后，西尾锦脸色微微有些发苦，这个两个家伙，昨天突然闯入他家，霸占了他们的房间，还一副和他们很熟悉的样子。
<br/>　　可之前他明明没有见过这两个人。
<br/>　　不过，他们好像也确实知道他的一些事情，而且，这两人的实力，也根本不是它能够对抗的。
<br/>　　无奈之下，西尾锦也只能乖乖的让出了房间，还按照刘哔的吩咐，在清晨五点十五分的时候烧上了洗澡水。
<br/>　　没办法，实力差距太大，他如果惹的这个男人不开心了，可真是会死的。
<br/>　　喰种世界就是这样，实力，就是唯一的规则。
<br/>　　“唔，不用，你回头帮我把衣服清理一下，对了，洗澡水烧好了么。”
<br/>　　轻轻点了点头，刘哔来到了洗澡间门口，脱掉了自己染血的上衣，和拳套一起丢在了地上。
<br/>　　“已经烧好了，大人随时可以享用。”
<br/>　　微微躬了躬身，西尾锦脸上带着恭敬的神色，对刘哔过分的要求不敢有丝毫抗议。
<br/>　　洗衣服就洗衣服吧，只要不是洗屁股就好。
<br/>　　自嘲般的苦笑了一声，西尾锦苦中作乐的想到。
<br/>　　洗澡间内传来‘哗哗’的水声，看着地上沾满鲜血的衣物，西尾眼中有些吃惊，这是得杀了多少人，才能将衣服搞成这样。
<br/>　　就连穿在最里面的背心，也都被浸染上了大片的鲜血。
<br/>　　这些血液，有喰种的，有人类的，甚至还有气味十分特殊的。
<br/>　　他曾经在金木研身上嗅到过这种气味，虽然稍微有些区别，但大体上也还是相同。
<br/>　　这不免让他心中有些吐槽，这个家伙，昨天难道是屠了一座城么。
<br/>　　“昨日，CCG成功捣毁一个名为月山集团的喰种据点，击毙大量高阶喰种，其家族直系成员‘美食家’月山习被成功抓捕，据悉，青铜树组织也有参与这次活动，在CCG局长和修政的带领下，已全部击杀。”
<br/>　　“沉痛哀悼，CCG特等指挥官有马贵将，以及大量在此次行动中牺牲的优秀精英，他们都是帝国的英雄。”
<br/>　　“此次行动中，唯有一名喰种负伤逃脱，据现场视频显示，此名喰种代号‘修罗’，危险程度极高，万请大家多加注意，如有发现，立即报警……”
<br/>　　客厅的中，传来了早间新闻主持人甜美的声音，西尾锦抱着手中的衣物，呆愣的站在原地。
<br/>　　电视机里的画面，正是刘哔和有马贵将交战时的后半部分，他脸上原本戴着的月山家面具，已经不知道掉落在了什么地方。
<br/>　　西尾揉了揉眼睛，有些惊惧的转过头来，望向洗澡间的方向。
<br/>　　刘哔就是修罗，这店长和他说过，不过，他并没怎么关注。
<br/>　　只是知道，修罗是一个嗜杀成性的喰种，好像也是一名共喰者，曾一度在20区掀起过一阵腥风血雨。
<br/>　　而有马贵将他再也熟悉不过了，或者说，几乎所有的喰种都知道这个名字。
<br/>　　白色死神，CCG的最高战力，曾无数威名赫赫的高阶喰种都败在他的手上，然后被抓捕。
<br/>　　他想象不到，居然真的有喰种，能将有马贵将击败。
<br/>　　更加扯淡的是，这个击败有马贵将的男人，和他只有一墙之隔，甚至都没穿衣服。
<br/>　　直到此时，西尾心中对于刘哔强占他房屋的不满，也瞬间烟消云散了。
<br/>　　开玩笑，这么粗的大腿，讨好还来不及，怎么可能再去得罪。
