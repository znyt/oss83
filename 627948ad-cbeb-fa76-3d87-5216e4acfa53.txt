<br/>    陈明挥动袖子，白色的袖子挥动，无数的亡魂朝着他的袖子窜去，狂风阵阵，震得陈明的浑身衣裳狂舞。
<br/>    
<br/>        陈明心说这里的灵魂，恐怕不下百万斤，在自己念经之前，估摸着应该是在十多万斤，念经之后，便是已经变成了百万斤。
<br/>    
<br/>        所以陈明便动了一些心思，这《燃罪书》，可以使灵魂变得更加纯粹，变重，还有着超脱灵魂的效果，绝对不能被发现了。
<br/>    
<br/>        所以在陈明的袖子之中，实际上有着两颗魂珠，一颗魂珠是自己的，收纳了大半的灵魂，另外一颗则是刚刚二太子给他的那一颗。
<br/>    
<br/>        这一颗魂珠之中，陈明收纳了二十万斤的灵魂。
<br/>    
<br/>        待得一时三刻过去，陈明挥动自己的袖子，抓住袖口便是一收，所有的灵魂，已经全部收到了他的袖中魂珠中。
<br/>    
<br/>        二太子飞身而下，落在陈明的身旁，却见陈明从袖子中取出二太子之前给的那一颗魂珠，递到二太子的身前，“小僧幸不辱命，二太子请收下。”
<br/>    
<br/>        对于陈明拿得动魂珠，二太子没有什么奇怪的地方，对于普通人来说，灵魂是有重量的，但是修心僧，这种和尚，却是什么都拿不动，唯独拿得动灵魂这一类奇怪的东西。
<br/>    
<br/>        就在这时，太阳升起，一缕阳光穿破无尽的黑夜，照耀在陈明的背上，二太子看向陈明，只觉得陈明是下凡的佛陀。
<br/>    
<br/>        二太子接在手中，少说有二十万斤！
<br/>    
<br/>        此地的灵魂，一般只要十万斤，经过这和尚度化之后，竟然有二十万斤！
<br/>    
<br/>        二太子当即拱手道：“大师乃是得道高僧，还请不要见怪之前我的冒犯。”
<br/>    
<br/>        陈明微笑，“何曾冒犯？”
<br/>    
<br/>        二太子也是尴尬一笑，没有想到，前来帮一次小七十九，居然能够让他遇到一位得道高人！
<br/>    
<br/>        二太子问道：“不知道大师的法号？”
<br/>    
<br/>        陈明道：“小僧，燃罪。”
<br/>    
<br/>        二太子注意到了，自己管陈明叫大师，陈明并没有反驳，那就是说明
<br/>    
<br/>        这真的是一位大师。
<br/>    
<br/>        至于陈明自称小僧，这肯定是自谦啦。
<br/>    
<br/>        二太子和陈明一同回到城中，二太子询问道：“燃罪大师，不知道大师接下来准备去哪儿，在下若是有什么能够帮到大师的，必定助大师一臂之力。大师帮助在下解决了这么大的麻烦，在力所能及之处，必定不会推辞！“
<br/>    
<br/>        陈明笑道：“小僧自东土大唐而来，前往西天拜佛求经，听闻佛门金顶有着众多佛门经文，心向往之，想要观看参悟一番。”
<br/>    
<br/>        二太子皱眉道：“这可有些难办啊，佛门金顶一向超然物外，即便是四大仙国，也不可能对他们下达命令，都是尊重有加，敬仰不已。”
<br/>    
<br/>        陈明知道二太子什么意思，这天庭的人，都一个尿性，首先，他会告诉你这件事情很难办，然后再帮你去做。“若是如此，小僧不敢劳烦施主。”
<br/>    
<br/>        二太子挥手道：“非也，在下不是这个意思，虽然很难，但是既然是为了帮助大师，在下会想想办法。”
<br/>    
<br/>        你看，本来是二太子为了偿还陈明的恩情，现在经过他这么一说，又变成了他帮陈明了，所以说啊，天庭的，没一个好东西！
<br/>    
<br/>        陈明微笑，“那就多谢施主了。”
<br/>    
<br/>        二太子看向陈明微笑的样子，只觉得自己什么手段都没有用，不过也对，不然，怎么算得上是大师呢？
<br/>    
<br/>        二太子道：“在一个月之后，佛门金顶将会有一场辩经，决定金顶修心院的主持之位，大师既然对于佛门金顶的经文心向往之，不如就由在下陪大师一同前去吧。”
<br/>    
<br/>        陈明自然是点头称是，“善哉善哉！”
<br/>    
<br/>        嗯，好像还可以顺路把佛门的经文都给抄个底啊！自己好像差点忘了，佛门也是有无字碑的啊！
<br/>    
<br/>        一个月之后。
<br/>    
<br/>        陈明和二太子一路登山，登上佛门金顶，陈明通晓万道，时不时点出一两句，让二太子敬佩不已，只觉得陈明乃是大师之中的大师。
<br/>    
<br/>        这一日，前来观礼的各方势力都有着不少，二太子
<br/>    
<br/>        既然来了，便是亮出了身份，代表天庭前来观礼。
<br/>    
<br/>        二太子来到佛门金顶的山门之前，二太子看向陈明，笑问道：“大师，不知道我此次登上佛门金顶，应该以什么身份？”
<br/>    
<br/>        陈明道：“施主既然已经来了，那便以天庭的身份前去吧，这样一来，也可显得天庭对佛门金顶的尊重。”
<br/>    
<br/>        二太子微微一愣，陈明是如何看出他的身份的，二太子道：“大师，可否一算在下的身份？
<br/>    
<br/>        陈明却是微微一笑，手指微微掐动，高声道：“天庭二太子，前来观礼！”
<br/>    
<br/>        二太子惊了惊，这个燃罪大师，好生厉害！居然一语道破了自己的身份，而且对方更是自信之极，没有丝毫犹豫，直接开口朝着佛门金顶的和尚说出这一句！
<br/>    
<br/>        他都没有想过自己会算错，没有问自己，直接便说了。
<br/>    
<br/>        二太子朝着陈明双手合十，道：“大师果然神机妙算。”
<br/>    
<br/>        陈明心说小样，你陈仙师的《皇极经天道》可不是白修的！
<br/>    
<br/>        二太子忽然动了一些别的心思，父亲不是在寻找拥有大神通的戏命师吗？这燃罪大师，可谓是真材实料啊！
<br/>    
<br/>        不知道，这燃罪大师，和那无冕者议会的祭祀，孰高孰低？
<br/>    
<br/>        不过二太子转瞬晒然一笑，不对，他此行前来，乃是准备看看能不能请到这两位争夺主持之位的大师，这燃罪大师，虽说修心的功力极高，也能揣测天机，想必比上无冕者议会的这一位戏命师，还是差了很多的。
<br/>    
<br/>        和尚听到是天庭的二太子，当即道：“诸位施主，还请随我上山。”
<br/>    
<br/>        就在这时，二太子忽然侧目一望，看到了天庭大太子麾下的一员骁将也是来到了此处，“该死的，大哥居然也派人来了！”
<br/>    
<br/>        看来，大哥这一次，也是对这两位大师志在必得啊！
<br/>    
<br/>        两人争夺的目标，都将会是获胜的那位大师，天庭有九位太子，其中，最有机会得到帝位的，便是大太子，二太子，以及九太子。
