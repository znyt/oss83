　　昆仑山玉虚宫中，元始对着白鹤童子道：“童儿，你去太乙道场一趟，告知他去陈塘关李府收李靖三子为徒，灵珠子可封神量劫主角之一，资质极高，可为我阐教再添一员护法。”
<br/>　　“哦？对了，还有这剑，让太乙交给李靖吧。”似乎是想到了什么，元始又取出一柄银色宝剑，递给童子，宝剑样子普通至极，与一般长剑无二，并没有什么出彩的地方。
<br/>　　“弟子谨遵老师吩咐！”
<br/>　　一旁的白鹤童子顿时接过宝剑，躬身应道，之后转身退出大殿，他可不认为老爷为拿出一柄普通宝剑，虽然这看起来是普通宝剑，但那时因为他眼光不好，看不出来而已。
<br/>　　“女娲，她不是一直喜欢在两岸观风么，这次怎么会下来淌这趟浑水，本座可不信你仅仅只是被纣王的几句诗句便给气糊涂了。”元始看着娲皇宫方向，默默思索道。
<br/>　　“也罢，既然你都出手了，那此次量劫若是不将你拉下水，那恐怕就不太好玩。”元始突然神秘一笑道。
<br/>　　“龙族最近好像也有些不太安分了，是想要参与此次大劫吗，也罢，就让灵珠子去试探一番。”
<br/>　　其余诸圣对于女娲投放灵珠一事反应不一，老子面色不变，继续炼丹，好像什么事都不放在心上似的，接引与准提目露精光，不知道在打什么注意，通天双手捧剑，全身剑意爆发，坐在一艘小木舟上，随波逐流。
<br/>　　……
<br/>　　陈塘关，乃是殷商王朝下统治的重要关隘之一，城中人口众多，驻兵十万，毗邻东海，是一出险峻雄关。大城如一头庞然大物盘踞在那里，让人望而生畏，而且这还是一处战争要塞，它的存在，在一定程度上防范着西边各诸侯国的叛乱。
<br/>　　陈塘关总兵李靖，武艺高强，对殷商也是忠心耿耿，为人有些愚忠，而且李家世代都效忠于商朝，深得君王信任。
<br/>　　李靖这生活过得倒是挺好的，有一个美丽贤淑的妻子，又有两个儿子，还被仙人收作徒弟，他的家庭还算幸福美满。可是近两年，有一件事情总是让李靖烦心不已。因为他的妻子怀孕了，按理来说，这本来也是一件好事，没什么不好的。可是，这事情怪就怪在怀孕的时期太长了。有听说过怀孕一怀孕就怀了三年多的吗？这件事总让他心中不安，以为是怀了一个妖怪，他平时没少听说妖怪害人的现象。
<br/>　　今日，是个特别的日子，那就是他那怀了三年多的孩子，终于要出世了，这让他既激动，又有些害怕，害怕生了一个怪物儿子。
<br/>　　“啊！”
<br/>　　“哎呀——”
<br/>　　房屋内传来凄惨的叫声，带着难忍的痛楚，仿佛声嘶力竭一般，听得屋外的李靖心中焦急不已，就算是有过两次父亲经历的他，此时也像是一个无助的人，变得坐立不安起来，在门外焦急的胡乱转悠起来，口中还不断嘟囔着：“希望上天保佑母子平安，千万不要出事啊。”
<br/>　　“啊！”
<br/>　　“妖怪！这是什么怪物啊！”
<br/>　　“……”
<br/>　　陡然，房间内传出惊骇地叫唤声，声音中带着浓浓的惊恐，紧接着，产婆推开房门，带着几名妇人慌乱逃了出来，面上还得这浓浓的惊恐之色，神色间满是惊骇与恐惧，满眼都是不可思议，好似见了妖魔鬼怪似的。
<br/>　　“不好……”
<br/>　　李靖心中暗叫一声不好，他最担忧的事情还是发生了，他的儿子可能是个妖怪。如今产婆都出来，可他的妻子还在里面，这要是出事了该如何是好？来不及多想，李靖连忙带着宝剑冲入房间。
<br/>　　“夫人，夫人，你没事吧？”
<br/>　　李靖冲入房间，便大声喊道，声音之中带着急躁与不安，在看到躺在床上脸色苍白的妻子后，李靖轻松了一口气。没事就好，虽然极为虚弱，但还不至于丢了性命。
<br/>　　接着，李靖转头看去，却见到桌子上有一个跳跃的肉球状的东西，他不由得瞳孔一缩，惊叫道：“什么？我儿子呢？是不是被妖怪给吃了？”
<br/>　　“嘭！”“嘭！”“嘭！”
<br/>　　只见桌子上那丈许大小的肉球状东西，散发出妖异的红光，还在不断弹跳着，一会儿高，一会儿低，诡异而妖艳。
<br/>　　李靖瞳孔一缩，紧接着便是目眦尽裂，目中带着浓浓地惊恐与悲伤，他猜测自己的儿子是被这怪物给吃了，顿时心中对那死去的儿子悲伤不已。
<br/>　　“锵——”
<br/>　　一声剑吟，李靖立刻拔出身上的宝剑，猛地对着那肉球劈去，口中还大叫道：“我劈了你这妖孽！还我儿命来！”
<br/>　　“不要啊……”
<br/>　　见此一幕，他的妻子在床上叫道，想要扑上前阻止，可惜哪里来得及，不由心生绝望，声音凄惨而哀伤，那可是她的亲身骨肉啊。
<br/>　　“噗！”
<br/>　　那弹动的肉球此次竟然没有弹跳开来，被李靖的利剑劈个正着！
<br/>　　“嗡嗡嗡……”
<br/>　　一道神秘的声音响起，同时一股妖异的红芒爆射而出，染红了整个房屋，紧接着投射而出，直冲天际，染红半边天际，霎时间笼罩半个陈塘关，瞬间引动所有百姓以及大小官员的注视。
<br/>　　“什么？这是什么，从总兵府冲射出来的东西！”
<br/>　　“红芒耀天，这难道是预示着什么吗？是吉？还是凶？”
<br/>　　“那是什么，哇！好漂亮啊！”有小孩不懂事，惊叫道。
<br/>　　“杀星降临，乱世将启！”一些有点道行的人见到这一幕却面色大变。
<br/>　　“……”
<br/>　　城中无数百姓、官员、士兵们，都一阵哗然，带着一丝好奇，一丝震惊，向着总兵府赶去，想要知道发生了何事。
<br/>　　都成，朝歌，观星台上。
<br/>　　“不好！妖星降世，天下大乱不远矣。”一人慌张道。接着，那人带着一丝惊慌，向宫殿内跑去。
<br/>　　……
<br/>　　“父亲，母亲！”
<br/>　　李靖府中，无尽光芒退去，原先的肉球早已消失不见，只留下一个三岁大小的小孩从桌上跳了下来，一双灵动的眼睛眨呀眨呀的，有些好奇地看着这个全新的世界。
<br/>　　突然看到对面持剑的男子与床上虚弱的女子，叫道：“父亲，母亲！”
<br/>　　“你就是我的儿子？”李靖带着狐疑的目光打量着眼前的小孩，心中却惊疑不定，难道这不是妖怪变成的？
<br/>　　“父亲，我就是你儿子啊！难道父亲不要我了吗？”小孩大叫道，边叫还想要哭出来，眼中带着一股天大的委屈。
<br/>　　“你真是我的孩子？”李靖上前一步，抱起小孩，体会到那种血脉相连的亲近感，他终于确定了，这就是自己的儿子，他心中不由大喜。
<br/>　　“好孩子，我还以为你被妖怪吃了呢，没事就好！”李靖激动道。一旁的李靖妻子也跟着兴奋起来。
<br/>　　“报告，老爷，外面有一位道长求见！”
<br/>　　突然，一位管家似的仆人跑了过来，对李靖喊道。
<br/>　　“哦？快快有请！”
<br/>　　李靖一挑眉，看着自己怀中的小孩，好似明白了什么，他知道自己这个儿子不简单，便连忙对着管家说道。同时他也想知道自己这个儿子到底是怎么回事。
<br/>　　“见过这位道长，不知道长尊号，在哪里修行？”出门一看，便见到一位仙风道骨，手中挽着白丝拂尘，身穿青色道袍的道人，李靖第一时间开口问道。
<br/>　　“贫道乃阐教门下乾元山金光洞太乙真人是也，见到李大人，想必这位便是李大人今日才出生的儿子吧。”
<br/>　　道人首先介绍一下自己，说出自己的背景，之后再向李靖行了一礼，最后将目光投向李靖怀中的小孩。
<br/>　　“太2真人？”李靖怀中，小孩眼珠子乱转，奶声奶气地疑声道。
<br/>　　……
