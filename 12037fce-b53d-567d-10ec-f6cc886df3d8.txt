　　　　“简直就是蝗虫过境寸草不生。”高少辉看着外面的场景出声，而后有些担忧的对楚涵道：“如果，我是说如果，要是最后情况不敌，我就先带你走？”
<br/>　　连高少辉这名战力恐怖的神秘家族嫡系少爷都觉得眼下的情况难以招架，可见此时此刻终于失去控制爆发的尸潮，有多恐怖！
<br/>　　楚涵没有出声，只是专注的调试着和上新发明的大音响。
<br/>　　在最后这几百万的丧尸一下子涌·入基地之后，那些丧尸引诱剂便不再有作用，它们挤满了这个基地的所有地方，但凡有一点空地都被占满，哪怕它们踩踏着死去同伴的尸体，也难以阻挡它们不断拥挤往基地中心靠近的脚步。
<br/>　　不仅第一战区沦陷，一下子分不清丧尸和人类，高墙上的三战队更是被大量的高阶丧尸包围，那些开始涌进迷宫战区的丧尸群，也逐渐开始吞并着这个基地最后的净土。
<br/>　　甚至越来越多的低矮房屋顶都消失不见，彻底沉入了这片尸海。
<br/>　　站在四百处高台上的射击队成员在拼命开枪，但根本无济于事，停在了各处屋顶的直升机不得已腾空，机舱内的人员望着下方场景，心中都涌·出了一股绝望。
<br/>　　整个基地都是黑色，一点一点不停的蠕动，在这黑色的海洋之中，根本分不清哪些是狼牙战团的同伴。
<br/>　　楚涵一直注意着丧尸群的动向，也清楚这个基地的每一处地方都在爆发着无视场绝地逢生的战斗，不是大批的丧尸群倒下，就是狼牙战团的成员被吞噬。
<br/>　　每一个人都在经历此生最接近死亡的一刻！
<br/>　　几百万丧尸与九万人类，完全不对等的数量对比，在流动失去控制的一瞬间，所有的秩序都将崩塌。
<br/>　　团队作战变成了一个笑话，被无处不在的丧尸群包围，还存在什么团队？
<br/>　　恐怕连队友在哪都不知道！
<br/>　　尸潮如同黑色的洪流，淹没了这整座基地，只剩下最中心唯一的房屋尚存，那是他们狼牙战团堆放物资的地方。
<br/>　　何峰等一干狼牙军官早已经冷汗浸·湿全身，骇然的望着外面的场景久久不能出声，这时候他们这才发现以往遇上的尸潮，有多小儿科！
<br/>　　眼下这个在短短几分钟内就将一整座基地吞噬的洪流，才是真正的尸潮！
<br/>　　南署基地不远处的高空，有几架并没有离开的段氏基地直升机，他们奉命停留在附近时刻注意战况，一旦有任何动向都要第一时间回去禀报。
<br/>　　此时好几名飞行员才刚刚从南署基地内那恐怖的场景内缓过劲来，这会儿还没几分钟一抬头，就看到了前方那突然爆发的场面，顿时尖叫惊呼连连。
<br/>　　“发生了什么事？！”
<br/>　　“这些丧尸群之前不还在基地外徘徊的么？怎么一下子全部涌进基地里面去了！”
<br/>　　那些本将南署基地包围的像个孤岛一般的尸潮，当下就像是收缩了一样进入了基地内部，只存留一小部分在外，那是南署基地的地理面积已经饱和，它们挤也挤不进去。
<br/>　　“一号，你回去汇报，快！”
<br/>　　一架直升机立即调头离开，其余人则是继续观察，他们的心情很沉重，望着远处那密密麻麻的黑点，好几人都有种想要昏厥的冲动。
<br/>　　“完了！南署基地被攻陷了！”
<br/>　　“狼牙战团失败了……”
<br/>　　“准备了这么久，大动干戈的调动了九万部队前来，竟然还是失败了？”
<br/>　　“已经很不容易，哦不，应该说是奇迹。”一人持不同意见开口：“尸潮的真正数量是我们看到的两倍之多，也就是说在这两天的时间里，狼牙战团的九万人就已经歼灭了能塞满整座南署基地的丧尸！”
<br/>　　“那为什么？”
<br/>　　“以九万人在两天之内杀了好几百万的丧尸群，甚至还包括数量不明的飞行丧尸，这尸潮内丧尸的等阶不同于我们之前所见，这已经是狼牙战团的最大极限了吧！”
<br/>　　一瞬间不禁有人试想，如果不是狼牙战团提前到达这里，如果不是苏行改造了这个基地，或许早在两天前这个南署基地内所有的幸存者，都已经被丧尸啃食的只剩下碎骨！
<br/>　　两天的时间，最多只扩张到了九万人的狼牙战团，却也只能坚挺两天，虽然这两天他们解决掉了这场大尸潮的一半丧尸。
<br/>　　但依然结束了，依然失败……虽然他们做的很好，并且也没人能比他们做的更好。
<br/>　　接下来等待的就是尸潮离去，它们不会在这个基地内停留太久，当发现已经没有任何鲜活生灵可以吞噬的时候，尸潮就会转移方向，朝着下一个人类的聚集点而去。
<br/>　　尸潮与逗留在城市内的那些漫无目的的丧尸不同，单个丧尸的热感应距离很短，如果那处范围内恰好没有生物，那它就会像个无头苍蝇一样乱窜，但一群丧尸聚在一起，热感应的范围则会成倍的放大。
<br/>　　尸潮形成的原因不外乎两个，小型尸潮就是一群又一群的丧尸因为同一个地点的热感应聚集，从而应发的群体行动。
<br/>　　五十万数量以上的尸潮，除却以上原因之外，可能还会有特殊丧尸的有意识组织。
<br/>　　多过五百万数量以上，到达了眼下一千五百万之多的大型尸潮，则就一定有超级丧尸的小头领在控制。
<br/>　　而此时这些七百多万的丧尸群，完全能够感应到热感应最强的方向，也就是有大批人类所在的居住地，加之有超级丧尸的小头领指挥，它们很快就会朝着下一个基地进发。
<br/>　　或许那个基地，就是段氏基地！
<br/>　　因为有着楚涵在两个月前的动作，这片边境地带，段氏和南署基地之间的那些中小型基地，所有的幸存者都已经转移离开。
<br/>　　这是一片空白的区域，唯独有着一万作为后勤幸存者的段氏基地除外。
<br/>　　在高空观察的那几名飞行员都能想到这一点，他们之中有人闭上了眼睛，开始对几天后段氏基地的沦陷绝望悲痛。
<br/>　　当这些人的心情绝望低落，前去汇报的那名飞行员心急如焚之时，待在南署基地指挥间的楚涵……
<br/>　　终于抬起了头！
