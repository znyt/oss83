　　十三位半步天人境强者，已经是占据整个大罗天星域的半步天人境强者半数以上了，现在全然陨落，不止是大罗天星域，周围的无数星域，都会掀起震动。
<br/>　　这大震动，如今却只是一位楚风眠一个人掀起的。
<br/>　　无数围观武者的目光，看向楚风眠，眼神之中都是无比的尊敬了。
<br/>　　以一人之力，灭杀了十三位半步天人境强者，从此之后，大罗天星域内，无人可能在跟楚风眠争锋了。
<br/>　　楚风眠便是绝对的大罗天星域的第一强者了。
<br/>　　青梦在一旁，看到这一幕，心中都无比感慨，她曾经邀请楚风眠进入圣龙宗的时候，楚风眠才只是一个资质不错的小家伙。
<br/>　　短短的时间之内，谁能想到，楚风眠已经是成为了可以称霸大罗天星域的人物。
<br/>　　这一切的大震动，其实也都在楚风眠的意料之外了，楚风眠本来只是想要先对付圣龙秘境的老古董，先占据圣龙秘境。
<br/>　　没想到机缘巧合之下，楚风眠得到了始祖天龙之爪，更是引起了逆天大劫，引来了无数强者的注意，最终血战。
<br/>　　在这之中，楚风眠经历了无数次的危机，只要是楚风眠稍有不慎，便是万劫不复了。
<br/>　　可楚风眠依然是度过了，斩杀十三位半步天人境武者，楚风眠在大罗天星域的威严，已经立下。
<br/>　　“三大势力的半步天人境武者，大半都被斩杀了，现在正好，先将这三大势力降伏再说！从此大罗天星域，就落入到我的手中了！”
<br/>　　楚风眠的目光，看向了五行天的方向。
<br/>　　他要拿三大势力开刀，第一个就是五行天了。
<br/>　　“师姐，我们走。”
<br/>　　楚风眠现在也不着急去圣地了，反正楚风眠已经是将圣龙秘境的老古董全部斩杀了。
<br/>　　就算是还有几只漏网之鱼，在知晓现在楚风眠的强横无敌之后，也不会在敢跟楚风眠作对了。
<br/>　　这三大势力，既然是想要在楚风眠度逆天大劫的时候来趁火打劫，那也该付出代价了。
<br/>　　“恩。”
<br/>　　青梦点了点头，化为了一道遁光，带着楚风眠向着五行天宗门所在的方向飞去。
<br/>　　五行天宗门所在，距离圣龙秘境足有上亿公里遥远，对寻常古帝来说，飞过去至少需要数日的时间才行。
<br/>　　可现在楚风眠带着青梦，刹那间便是足以跨越百万公里，不足半个时辰的时间，楚风眠就来到了一处巨大的大陆上空。
<br/>　　在楚风眠的脚下，乃是一片大陆，一片由无数绝地组成的大陆，其中有着蕴含无数火山的绝地，也有着常年冰封，足以冻死古帝的极寒之地。
<br/>　　各色绝地，足有数十个，组合在了一起，化为了一座大陆。
<br/>　　百绝大陆。
<br/>　　这里就是五行天的宗门所在，这百绝大陆之上，五行天是绝对的霸主，不过除此之外，还有一些二流，三流的宗门，隐藏在一个个绝地之中。
<br/>　　倒是要比起圣龙秘境繁华的多。
<br/>　　不过圣龙秘境毕竟是一位大人物一手建立而起的一处区域，而这百绝大陆，却就是在无尽星空中诞生的一处独特的大陆。
<br/>　　楚风眠跟青梦的身形，来到了这百绝大陆的中心，这里，就是五行天宗门所在。
<br/>　　当楚风眠跟青梦来到了这百绝大陆中心天空之上的一刻，周围的大阵，突然浮现而出，五行之力凝聚，化为了一道道剑光，向着楚风眠斩杀而来。
<br/>　　这一道道剑光，几乎都媲美九劫古帝的轰击，一齐爆发而出，足以逼退半步天人境武者。
<br/>　　这就是五行天的护山大阵。
<br/>　　只是这护山大阵，跟天星河相比的话，倒是还差一些，毕竟天星河可是天地诞生的独特屏障，拥有灵性，一般武者都难以找出天星河的破绽。
<br/>　　可是这五行天的护山大阵，却是人力建立的，楚风眠一眼便是足以看出无数破绽，想破就破。
<br/>　　不过现在楚风眠当然不用那么麻烦，楚风眠现在的实力，九劫古帝都轰杀了不少，又怎么在乎这一道小小的护山大阵。
<br/>　　看着数百道五行剑锋突然斩杀过来，楚风眠只是大手一挥，一掌拍下。
<br/>　　轰隆！
<br/>　　楚风眠掌风落下之处，一道道五行剑锋，被震得粉碎。
<br/>　　接下来楚风眠的一掌，更是轰击在了这五行天的护山大阵之上，一股巨大的灵力碰撞。
<br/>　　咔嚓！
<br/>　　这五行天的护山大阵，就在楚风眠的一掌之下，被打出了一道道的裂痕，轰然破碎了。
<br/>　　这就算是碰到半步天人境强者都可以抗衡一阵的护山大阵，却是就在楚风眠的面前，随手就可以打破。
<br/>　　这护山大阵，对楚风眠来说，简直是视如无物。
<br/>　　五行天宗门之内，本是有着不少的五行天弟子，还在潜修，其中的一座座宫殿，一位位殿主，都还在讲课受到。
<br/>　　可突然之间，随着护山大阵破碎，整个五行天都是变的地动山摇了起来，如同末日一般。

<br/>　　无数五行天弟子都是愣住了，看着五行天护山大阵的破碎，惊慌无比。

<br/>　　什么时候有人可以来攻打五行天。
<br/>　　可现在这五行天的护山大阵，可都被人打破了。

<br/>　　可当他们仔细看清楚其中身影的一刻，更是惊讶无比了。
<br/>　　在这五行天的天空之上，一共只有两个人。
<br/>　　他们本以为，敢来攻打五行天，至少也是像是圣龙秘境，齐林山这样，同样为大罗天星域的四大势力之一，才敢来攻打。
<br/>　　可现在，只有两个人，简直是超出了所有人的意料之外。
<br/>　　“什么人？敢擅闯五行天！”
<br/>　　突然间，就在这五行天之内的一座宫殿之中，一道身影猛然飞了出来，化为了一道遁光，直冲天空之上。
<br/>　　五行天之内的闭关不出的一尊殿主，都被惊动了，出关了。
