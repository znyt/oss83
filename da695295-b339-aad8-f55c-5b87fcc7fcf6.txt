　　何培源安然无恙的坐在沙发上，他的对面还有一个衣着白裙的年轻女子，角落里还有个面带一丝苍白的少年。
<br/>　　而在这间屋子的门旁，也就是江凌轩等人的正对面，一个被斗篷遮挡的严严实实的人，好似脑袋要掉下来一般的歪着脖子，正在朝旁边倒下。
<br/>　　江凌轩看到了此人的脖颈间有着几道抓痕，血淋淋的伤口上溢出带有酸臭味的血液。
<br/>　　啪！
<br/>　　衣着斗篷的人倒下发出的声音，惊醒了在场发愣了众人。
<br/>　　一片哗然间，众猎者都不明所以，一个个都慌了神。
<br/>　　发生了什么？
<br/>　　唯有江凌轩用最快的速度镇定下来，抬眼看向何培源问道：“您没事吧长老？”
<br/>　　“一切安好。”何培源笑着回答。
<br/>　　江凌轩点着头，而后一把掀开地上已死去之人的黑色面罩，道：“这人就是王之暴君，果然是个异种！”
<br/>　　这一瞬间很多人都看到了地上死去之人的样貌，其还未来得及闭上的双目一片腥红，加之血液里带有的酸臭味，其异种的身份已经确定。
<br/>　　“王之暴君竟然是个异种？！”有猎者惊恐出声，好似三观都崩塌。
<br/>　　“幸好发现的不晚，否则定要出大事！”也有猎者庆幸不已。
<br/>　　“对了，王之暴君的战力可是极强，怎么死了？是谁杀的？”更有带着崇拜的声音响起。
<br/>　　江凌轩抬眼望向了那位坐在何培源对面的白裙女子，眼神带着一丝意味不明的开口道：“还未请问这位恩人大名？多谢出手相助，否则我们长老……”
<br/>　　“我也只是恰好路过。”白允儿清冷的声音响起，好似一股冷泉，浇灭了在场众人情绪起伏的内心。
<br/>　　四下顿时一片安静，很多人都被这个清冷的声音所吸引，一不小心都失了神。
<br/>　　江凌轩却在此刻腾起了另一股心思，他刚刚只是试探一问，毕竟在他们冲进来之前这间屋子里只有四个人，死去的王之暴君，毫无战力的何培源，角落里的十一岁少年，以及眼前这个让人看不出深浅的年轻女子。
<br/>　　果然不出江凌轩的所料，杀死王之暴君的就是这个白衣女子，只是他并不知道此人是谁，又究竟是如何动的手？
<br/>　　看这王之暴君的死亡状况，以及屋内没有一丝一毫的破坏场面，根本不像有过激烈的打斗。
<br/>　　想到这里的江凌轩内心一凉，忍不住再次深深看了白允儿一眼，此女的战力……
<br/>　　这时候的白允儿忽然微微偏头，露出了美绝人寰的侧面，也一下子让屋内的气氛一凝，很多人都屏住了呼吸，不可思议的望着她令人惊艳的侧脸。
<br/>　　白允儿的目光看向了地上的王之暴君，被掀开后暴露而出的面容，毫无疑问是一名异种，只是白允儿的眼神却微不可查的一顿，而后便没有再关注。
<br/>　　江凌轩顺着白允儿的目光望去，他皱眉看着地上死去的异种，叹气道：“没想到这位在猎者联盟名声极高，身价不菲的高级猎者，竟然不是人类，也不知道他当初是怎么混进来的，整个猎者联盟上下竟然都瞒了过去。”
<br/>　　白允儿轻笑：“你们急急忙忙的跑来，不是已经有所察觉了么？”
<br/>　　江凌轩点头：“那是多亏了猎王提醒，否则差点酿成大错，早知道在他一进来的时候，就应该拦下。”
<br/>　　白允儿轻轻摇头：“猎王早有安排，所以我才会在这里，再者这个异种已经九阶巅峰，你们拦不住。”
<br/>　　此话一出，在场的猎者们都不禁发出阵阵惊呼声，望向地上这名死的不能再死的异种目露不可思议。
<br/>　　九阶巅峰的异种！
<br/>　　江凌轩也心脏猛然漏跳了半拍，紧接着后怕道：“原来您是猎王请来相助的，多谢！”
<br/>　　何培源环顾了一圈，笑道：“把异种的尸体处理一下吧，毕竟他们体内也带着病毒，另外向外公布一下王之暴君的身份，免得被其他异种利用。”
<br/>　　“明白。”
<br/>　　江凌轩将异种的尸体抬走，一群猎者也纷纷离开此处，不多时屋内便安静了下来。
<br/>　　何培源看着白允儿：“你发现了什么？刚刚眼神不对啊！”
<br/>　　白允儿捧起桌上已经凉掉的茶，唇角清勾：“那个王之暴君是假的。”
<br/>　　说着白允儿又一顿，忽然又摇了摇头，双眸带着一丝思索神色：“或者应该说，真正的王之暴君，死了一个分身？”
<br/>　　何培源眨了眨眼睛，浑浊的双眼里突显一抹睿智：“王之暴君本来是谁呢？”
<br/>　　白允儿放下茶杯，声音清丽：“异种王。”
<br/>　　何培源好奇问道：“异种王的相貌你见过？”
<br/>　　白允儿的双目眯了眯，回忆道：“只要是完全蜕变的异种，都可以伪装成异种王，也都可以伪装成王之暴君，不是么？”
<br/>　　一直在旁边听着谈话的白羽猛然一惊，乍然抬头开口道：“我们在银市遇到的那个，不会也是假的吧？”
<br/>　　白允儿伸出一根手指，轻轻叩着桌面：“你见到的可能是假的，但真的当时也一定在场，而白忧见到的……说不定也是假的。”
<br/>　　随着白允儿的话音落下，某个巨大的阴谋好似初露锋芒，让白羽浑身发寒，也让何培源露出一抹惊色。
<br/>　　“那个异种王……”何培源的语气带着一抹迟疑：“莫非在下一盘大棋，想把白家当枪使？”
<br/>　　“谁知道呢？”白允儿不甚在意的轻抚衣袖：“总之我所知道的异种王，没有这么弱，没有这么蠢，更不可能甘心给白家办事。”
<br/>　　何培源的身子向后靠了靠，有些好奇：“这么说来，他这次假死，反而助他摆脱了白家？那接下来呢？他野心这么大，不可能永远躲着。”
<br/>　　白允儿的冷眸中寒光一闪而过：“说不定早在银市之前，他就有了另一处异种大本营呢？”
<br/>　　白羽则想到了另一点：“白忧还会派其他人来暗杀制裁者么？”
<br/>　　白允儿点头，语气平淡：“会啊，没有了异种，不是还有丧尸么？”
<br/>　　白羽错愕的望着白允儿，不能明白当下说出这番话的她，内心在想什么。
<br/>　　何培源也差点被白允儿绕了进去：“你这是希望老朽死呢？还是希望老朽活？”
<br/>　　看其毫不在意的态度，似乎也没打算在这里保护何培源。
<br/>　　白允儿抬眼，直视何培源：“那要看楚涵的意思。”
