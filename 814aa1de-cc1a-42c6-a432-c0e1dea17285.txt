　　　　转眼便是一天一夜，由于舒瑶被涂小安刻意的控制沉睡，她恬静的躺在沙发上就没起来过。
<br/>　　不过随着一天一夜过去，涂小安发现舒瑶的气息渐渐的转弱，毕竟是肉体凡胎，这不吃不喝身体难免会受不了。
<br/>　　正常来说，一个人真正不吃不喝，应该可以存活五天到七天不等。
<br/>　　一天下来，虽然让舒瑶的气息转弱，但并不会有什么大碍。
<br/>　　只是她腹部慢慢的又开始颇动起来，恐怕里面的小东西有造反的征兆。
<br/>　　“先试试”
<br/>　　涂小安意念一动，一片偌大的蛇皮出现，就如毛毯一样的盖在了舒瑶的肚子上。
<br/>　　蛇皮散发浓郁的能量，还带着一股异香，对生物有着致命的吸引力。
<br/>　　就用蛇皮慢慢的引诱她腹中挨饿的蛇出来。
<br/>　　这招能不能行，其实涂小安是一点底气没有。
<br/>　　他做完这一切，继续耐心的等待，等待事情能出现转机。
<br/>　　这也是目前他唯一想到了办法，真的实在不行，他恐怕要破釜沉舟，亲自去舒瑶的体内走一趟。
<br/>　　不到万不得已，涂小安不愿意那么做，风险太大，而且也太匪夷所思。
<br/>　　时间一分一秒的流逝，一个小时过去。
<br/>　　只见沉睡中的舒瑶睫毛颤抖起来，面容带痛苦之色。
<br/>　　紧接着，她的身体开始抽搐起来，像是得了羊癫疯，抽的越来越激烈。
<br/>　　那娇美的面容慢慢狰狞起来，显得非常的痛苦。
<br/>　　舒瑶醒了，被硬生生的痛醒了，头昏目眩，刚醒便脸色骤变，一个猛然转身，朝着地板开始剧烈的呕吐起来。
<br/>　　呃...呃...呃...
<br/>　　一声声呕吐，腹中翻江倒海，一股不可压制的力量由下往上冲涌，她的腹中再次收缩像是有什么东西冲破一切封锁。
<br/>　　就在这时。
<br/>　　“哇...”
<br/>　　一团黑色的物体被呕吐了出来。
<br/>　　那是什么，一条三十厘米的小黑蛇全身黏糊糊的，全身包裹液体，画面非常的恶心。
<br/>　　“哇...”
<br/>　　舒瑶昏昏沉沉的再次呕吐一声，又是一团黑色物体。
<br/>　　同样，还是一条全身液体的黑蛇。
<br/>　　恶心，触目惊心。
<br/>　　纵然是涂小安早有心理准备，也看的毛骨悚然，渐渐的，一条条小黑蛇不断的被舒瑶呕吐出来。
<br/>　　她吐的像是把身体给掏空。
<br/>　　十分钟后，舒瑶晕死过去，地板上，十二条小黑蛇触目惊心的在蠕动。
<br/>　　特么太恶心了。
<br/>　　涂小安纵然自己现在也是一条蛇，看到这种画面弄的直犯恶心。
<br/>　　“给我消失”
<br/>　　这尼玛太辣眼睛了。
<br/>　　涂小安浑身电光流转，准备直接将这些蛇给毁尸灭迹。
<br/>　　可忽然，他僵硬住了，这些小黑蛇朝着自己的同胞游去，旋即张开蛇口，咬了过去。
<br/>　　霎那间，整整十二条小黑蛇相互的吞噬起来，凶煞可怖，十二条转眼变成了六条。
<br/>　　渐渐的，六条变成了四条，四条变成二条。
<br/>　　在看最后存活的两条黑色，体型直接达到100公分，一米长度，浑身散发蛇类独特的阴冷气息。
<br/>　　看呆了，这完全把涂小安给看呆了。
<br/>　　这些是什么蛇，刚出世就如此的凶煞，直接吞吃自己的同胞成长。
<br/>　　最后两条达到一米的黑蛇，眼冒凶光的决一胜负，好像这就是它们的宿命。
<br/>　　虽然有十二条出世，但能真正活下来的，只有一条。
<br/>　　它们刚出世，就知道弱肉强食，只有强者才配活在这个世上。
<br/>　　又或许说，将其它十一条血脉的基因成全一条。
<br/>　　这最后活下来的那一条蛇，血脉无比正统。
<br/>　　邪蛇，彻彻底底的邪蛇。
<br/>　　两条黑蛇激烈的厮杀，它咬住它的蛇尾开始吞噬，它也咬住它的蛇尾吞噬。
<br/>　　相互吞噬。
<br/>　　涂小安也算开了眼界，还真是大千世界无奇不有，原本从人的身体出来，就已经惊世骇俗，这一出世，就相互厮杀，彼此吞噬。
<br/>　　看不下去了。
<br/>　　金银蛇周身雷电充斥，好像有人在擂鼓助威。
<br/>　　劈...
<br/>　　一道耀眼的电光将这间套房照得通亮，粗大的雷电直接将两条相互吞噬的黑蛇笼罩其中。
<br/>　　嘶嘶...嘶嘶...
<br/>　　顿时间，无比凄惨的蛇吼声响彻起来，宛如深渊厉鬼的叫，让人头皮发麻。
<br/>　　好在舒瑶是晕死过去，哪怕她在整个呕吐的过程中，都是昏昏沉沉不清醒的。
<br/>　　否则这一幕幕被她看到，被吓成傻子都不过分。
<br/>　　这蛇胎的由来，涂小安还没弄明白，但最少现在他将蛇胎给解决了。
<br/>　　呕吐出来的十二条蛇，自己吞噬，剩下最后两条，相互吞噬，被一道雷电终结。
<br/>　　这件事情舒瑶是不会知道，等她醒来，便是万里晴空。
<br/>　　套房内白光消失，恢复平静，但却又一股无比阴冷的气息不知何处的席卷出来，将整个空间都激荡的波动起来。
<br/>　　涂小安蛇眸一凝，露出了难以置信的表情。
<br/>　　只见地板上居然还有一条蛇的身影，它昂首而立，空中既然摇曳着两颗蛇头，张牙舞爪，浑身散发着可怖的蛇类气息。
<br/>　　这种蛇类气息非同小可，一般的毒蛇，都要诚惶诚恐，俯首称臣。
<br/>　　这...
<br/>　　“给我洞察”
<br/>　　“叮...变异双头蛇，一阶觉醒生物，等级十级，一身双头，超强变异进化体蛇种”
<br/>　　我特么一道雷电，还将两条相互吞噬的黑蛇给整成变异进化，融为一体了？
