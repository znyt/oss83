　　第761章
<br/>　　断壁残垣，几丛枯草搭着矮墙，米粒大小的野花随便绽放，也无人在意。
<br/>　　李行哉一瘸一拐的从矮墙下走过，伸出手摘下一朵野花，用手指碾碎，随意丢在地上。
<br/>　　他嗅了嗅指尖残余的花香，笑了笑，推开面前一扇尘封的木门。
<br/>　　立刻便有兵器响动声，房间内的人如临大敌。他们呈扇形将李行哉围住，情况一有不对，便立刻准备动手。
<br/>　　李行哉挥挥手，在一张凳子上坐了：“好啦好啦，都收起兵器吧。搜查的官兵已经走了，我这个逍遥王虽然没多少面子，但他们还不至于完全不将我当回事。”
<br/>　　几人互相看了看，这才收起身上的兵器，稍稍放松警惕。
<br/>　　“多谢。”为首一蒙面人道。
<br/>　　李行哉继续道：“我说你们胆子也真够大的，凭你们几个人就敢刺杀百里胜。还真被你们做成了。想起来，这件事也有些匪夷所思。”
<br/>　　“杀父灭族之仇，总是要报的。”
<br/>　　说着话，他摘下了脸上的黑布。不是旁人，正是宋伯康之子宋游渠。
<br/>　　李行哉看着他的模样，叹了一声：“话说蛤蟆寨还真是一个历练人的好地方，凭你们十几人真就做成了一件惊天动地的大事。”
<br/>　　崔相被扳倒，宋伯康即将被处死。这件事被牵连的，并不仅仅是崔宋两家，许多与相府有牵扯的家族都遭了灭顶之灾。
<br/>　　但就算是天罗地网，也同样会有漏网之鱼。每个家族或多或少都会逃出一二人，这些人聚集在一起，便是一股不容忽视的势力。当然，朝廷对他们是赶尽杀绝的态度，可在巨石之下也同样有野草偷生。
<br/>　　“多谢逍遥王出手，按照事先说好的，崔宋残族愿唯逍遥王马首是瞻，誓死追随左右。”
<br/>　　李行哉点了点头，然后又皱起眉头：“将你们带出长安城，可不是一件容易的事情呐。”
<br/>　　宋游渠早就和李行哉勾搭在一起，不过中途发生许多事，兵荒马乱，李行哉沦落到东海，双方的联系早已断了。但通过这件事，阴差阳错，双方又搭上了线。
<br/>　　李行哉并没有杀百里胜的心，他的死活对李行哉关系不大。他肯出手遮掩，也并非是因为和宋游渠过往的交情，他们之间几乎没有交情。他看重的是崔宋两家的残存势力。
<br/>　　崔家幸存一个崔白玉，宋家有一个宋游渠，除此之外还有不少漏网之鱼。这些人凝聚在一起，或者能成为自己手中一股重要力量。
<br/>　　没办法，落凤城缺人，李行哉要想方设法帮自己拉拢人才。
<br/>　　……
<br/>　　满庭芳别院，看着眼前的狼藉，程大雷也是唏嘘长叹。
<br/>　　乱世人命若浮萍。张天赐这句话未必没有道理，就连百里胜这样的人物，都说死就死了，何况是那些普通人。
<br/>　　“那玉雀的尸体呢，需好生安葬。”程大雷道。
<br/>　　卢俊义顿了顿：“那个女人……没有死。”
<br/>　　“咦？”程大雷一楞，别院除了百里胜被砍掉脑袋，其他的丫鬟仆役都死的干干净净，怎么偏偏玉雀活了下来。
<br/>　　“那个女人有重大嫌隙，如今被关了起来，一定要撬开她的嘴，令她说出实情。”卢俊义恶狠狠道。
<br/>　　程大雷白了他一眼：“那姑娘明显是个普通人，杀鸡的力气都没有，更不用说杀人？何况，如果这件事果真和她有关系，她早就随刺客一起走了，还会等在这里任你审？”
<br/>　　“这……”卢俊义无言以对。
<br/>　　“好男儿的本事要用在做大事，别为难一个女人，那算不得好汉子大丈夫。”程大雷看着卢俊义道：“被关在哪里了，带我去，那个姑娘我要了。”
<br/>　　“这……万万不可。无论如何，那女人都有嫌疑，还没审问清楚，怎就可以交给程当家？”
<br/>　　“我是在与你商量么？我是要你立刻放人。”程大雷板起脸：“百里胜死了，你很难过。这我可以理解，但不要想凭着这点难过胡乱做事，本当家可不是一个吃你脸色的人。”
<br/>　　卢俊义双拳攥紧，脸色青一阵红一阵，面对盛气凌人的程大雷，他显然很愤怒。可他虽然是个武夫，但并不是莽夫，心中也清楚，程大雷是自己招惹不起的人。
<br/>　　自己不是莽夫，程大雷可是天字头一号莽夫。自己与他讲规矩，他可未必同自己守规矩。
<br/>　　程大雷也有他的担心，其实玉雀若是活着，程大雷心中多少还轻松些，答应过她的事情就可以不做数了。但不幸的是她还活着，在一个美丽便是原罪的世道，她落入一群兵痞子手中，下场如何可想而知。
<br/>　　既然她还活着，程大雷便必须担起责任。
<br/>　　玉雀被丢在城防军大营中，有专人看守。直到现在，她还没有回过神来。她虽然是七窍玲珑心，但专长是在揣摩男人的心思。但今日是见了血，也死了人。这种残酷的血淋淋，并不是一个养在温室中的娇花可以想象的。
<br/>　　如今她被关在一间单独的屋子中，俏脸生白，身体不停颤抖。而等待自己的命运如何，她现在完全无法想象，或者说是不敢去想象。但或多或少，心中意识到一点：男人之间的游戏，女人最好不要卷入其中，那种残酷和血腥，只是站在权力场的边缘，也会很轻易被卷入其中，然后不得好死。
<br/>　　门吱呀呀打开一条缝，一个满脸横肉的男人挤进来，随之房门被关上。
<br/>　　男人姓龙，唤作龙胖子，是随着百里胜一路从凉州杀出来的，算是铁杆亲信。
<br/>　　他没有别的爱好，最是好色，平常见着大姑娘便流口水。这次铲除相府余孽，不知道多少大姑娘被他一身肥肉压死。
<br/>　　关于这玉雀，他自然也听说过，只是没有见过。如今，这旁人见一面都困难的小娘子落在自己手中，他又怎肯放过。
<br/>　　只听说他是如何如何美貌，可真正见过了才知道，所有传言还需要加一个更字。
<br/>　　梨花带雨，我见犹怜。
<br/>　　粗糙的龙胖子也升起一股柔情。
<br/>　　“小娘子，不用怕，有胖哥哥在，没人可以伤害你。”
<br/>(本章完)
