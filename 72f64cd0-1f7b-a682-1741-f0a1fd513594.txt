　　空间一阵扭曲，慕清霄的身影已经来到高楼上，其中没有护卫，倒也省事，最后追着气息来到其中一处偌大客厅中。
<br/>　　此时，韩枫正坐在客厅中，与一名中年男子交谈。
<br/>　　韩枫穿着一身绿色锦袍，脸上持着沐浴春风的微笑，气质儒雅，看上去给人一种亲切的感觉。
<br/>　　慕清霄只是瞥了一眼，并没有特地去观察，韩枫现在可是七品炼药师，强大的灵魂对感知就极其敏锐，一旦暴露，就只有强行击杀。
<br/>　　“黑印城的事情，有头绪了吗？”
<br/>　　“属下无能，并没有追查到凶手的痕迹，阴阳玄龙丹也不知去向。”
<br/>　　说着，大斗师巅峰的中年继续道：“但属下猜测，此事估计与黑骷墓无关，范凌在两名斗灵护卫下，毫无反抗的被击杀，凶手最有可能是斗王级别。”
<br/>　　韩枫面无表情，沉吟片刻，道：“黑印城的事情你就别管了，现在有一件事情，需要你去一趟迦南学院，将信息传给里面的管事。”
<br/>　　“是，属下这就去办。”大斗师男子闻言，恭敬的退了出去。
<br/>　　等男子离开高楼，韩风手指在桌上轻轻敲响，望着窗外的景象，陷入沉思之中。
<br/>　　“嗡嗡！”
<br/>　　正当韩枫沉思之际，耳边忽然传来一阵凌厉的破风声，回过神来，一股刺痛自喉咙处席卷心头。
<br/>　　“噗嗤！”
<br/>　　伴随着炙热的鲜血喷洒而出，韩枫的头颅高高抛起，身躯无力的倒在地上。
<br/>　　旋即，一团深蓝色的火焰，携带着一缕虚幻的灵魂，自他的身躯上冒出，只见韩枫脸色狰狞的望着面前的黑袍身影。
<br/>　　深蓝色的火焰不是别物，正是异火榜排名第十五的海心焰。
<br/>　　望着韩枫的灵魂，慕清霄道：“真的已经迈入七品炼药师了啊。”
<br/>　　“混账，你是谁，为何要杀我！”
<br/>　　慕清霄手持魂剑，瞥了一眼院落深处，真是多亏了院落中的斗宗，才使得韩枫没有丝毫警惕，他才能一击斩首。
<br/>　　此时，韩枫望着慕清霄，呲牙欲裂，显然不明白，慕清霄是如何瞒过暗中强者，来到此处的。
<br/>　　“当然是有人让我来取你狗命，不过你身上也有我感兴趣的东西。”
<br/>　　“是谁，究竟是谁？”
<br/>　　慕清霄不屑一笑，院落中的斗宗已经来袭，九霄琉璃焱乍现，形成包围网，直接将韩枫的灵魂与海心焰包裹住。
<br/>　　“是一个你做梦都想不到的人，对了，谢谢你的阴阳玄龙丹。”
<br/>　　话语刚落，韩枫瞳孔一缩，没想到黑印城的事情，也是面前的黑袍人搞的鬼。
<br/>　　随后，他就感觉灵魂都要撕裂开来，发出一阵凄厉的惨叫声：“这是什么火焰，求你放过我，要什么我都给你。”
<br/>　　慕清霄不屑一笑，韩枫这种欺师灭祖的人，他可不敢放过，这种人为达目的不择手段，放了简直就是超级祸害。
<br/>　　更何况，他可是承诺过药尘，哪怕没有承诺，韩枫也必须死，九霄琉璃焱涌上他的灵魂，眨眼间，韩枫就带着各种复杂的情绪，消失于天地间。
<br/>　　“你的性命，异火，我都需要！”
<br/>　　韩枫的灵魂消失，手指一卷，一枚纳戒就出现在他的手中，旋即周围空间一阵扭曲，慕清霄就消失不见。
<br/>　　慕清霄前脚刚消失，高楼客厅中的空间就一阵扭曲，旋即一道黑色身影凭空而现，此人浑身笼罩在黑袍中，身上散发出阴森无比的气息。
<br/>　　“居然敢在我眼皮子底下杀人，并且夺走了海心焰，真是狂妄！”
<br/>　　瞥了眼韩枫的尸体，空间一阵扭曲，黑袍人的身影也凭空消失，朝着慕清霄消失的方向追去。
<br/>　　离开韩枫的府邸，几分钟后，慕清霄剑眉一皱，旋即落在高楼顶端，望着西边方向，低喃道：“本不想现在与魂殿接触，看来不想接触都不行了。”
<br/>　　十几秒过后，上空空间一阵扭曲，紧接着一名手持黑色锁链，隐藏在黑袍中的身影就出现在了视线中。
<br/>　　魂殿，原是中州的最大势力，乃是远古种族魂族的附属势力，古称魂族，远古八族之一。
<br/>　　魂族拥有媲美古族古界的自成空间，名为魂界，魂殿嚣张跋扈，满大陆搜捕各种灵魂体，而他们本身就是由灵魂体组成的组织，是斗气大陆上最强的势力。
<br/>　　魂殿中强者无数，慕清霄可不想杀了一个最弱的，引得一身骚。
<br/>　　“八星斗皇，你是用什么方法，躲过我视线的？而且，你似乎领悟了空间奥妙，真是惊才艳艳。”
<br/>　　黑袍人盯着笼罩在黑袍中的慕清霄，声音森冷道：“不说也没关系，本宗待会自己找，将你的灵魂留下吧。”
<br/>　　能在他的眼皮子底下，将韩枫击杀，而且在斗皇境界就领悟空间奥妙，此人绝对不能留下，否则将来绝对是一危害。
<br/>　　慕清霄不屑一笑，道：“凭你四星斗宗就想杀我，你是在做梦吗？”
<br/>　　“嗯？竟能看穿我修为，看来你身上隐藏着不少秘密啊，更加留不得你！”魂殿斗宗森冷一笑，黑色阴森的锁链，闪电般向慕清霄席卷而去。
<br/>　　黑色铁链席卷而来，仿佛能够无限延伸，慕清霄心念一动，魂剑已经出现在手中，直接向锁链斩去。
<br/>　　“铿锵！”
<br/>　　斗宗不愧是斗宗，锁链上的强横力量，使得慕清霄倒飞而出，最后停在了五米开外。
<br/>　　黑色锁链上刻着一些神秘符文，接触的瞬间，灵魂都感到一丝冰寒，这锁链估计对灵魂有着显著的效果。
<br/>　　此时，魂殿斗宗愣愣的站在上空，抓着锁链，望着上面的一丝剑痕，仿佛陷入莫大的惊骇当中。
<br/>　　片刻后，森冷的声音再次响起：“你手中的长剑真是怪异，竟能损伤魂器，明明只有八星斗皇的修为，但爆发出的力量却相当于二星斗宗，真是嘀咕了你的天赋。”
<br/>　　“不过，本宗最喜欢的就是囚禁天才灵魂，看到他们恐惧的模样，简直比吃了仙丹还要爽快，将你的灵魂交出来吧。”
<br/>　　说话的同时，黑色锁链已经席卷而出，力量与速度都是刚才的数倍，见慕清霄棘手，魂殿斗宗当下也不敢怠慢。
