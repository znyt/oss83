　　随着战斗的持续，魔神死得越来越多，到如今，仅仅只剩元释、魔祖、时辰、扬眉四人，但他们依旧在与盘古战斗，就算拼了命也要将盘古打杀，因为他们都知道，魔神与盘古之间，只有两种结果，不是你死，就是我亡！
<br/>　　魔神们在嘶吼，目露凶光，各自使出自己的那名绝招，杀向盘古，煞气弥漫混沌，狂暴而汹涌。
<br/>　　“逆五行印！五行混乱，给我杀！”到了最后，元释也不在保留，他需要痛痛快快地与盘古大战一场。
<br/>　　“魔道屠天，杀！”
<br/>　　“时间长河，现！”
<br/>　　“虚空破碎，寰宇震荡！！”
<br/>　　显然，到了最后一步，谁也没有放弃。
<br/>　　“哈哈哈！好，这是本尊打得最痛快的一战了！哈哈哈！盘古斧！给我——聚！”虽然盘古浑身浴血，但战意却越来越强，面对四大魔神联手杀招，一点也不显得畏惧。只见他大手一招，那原先已经分裂成三大先天至宝的混沌开天斧又再次在盘古手中凝聚成形！
<br/>　　“七斧——起苍茫！”
<br/>　　一斧既出，斧芒迸射八方，让天地更广的同时，竟然再度将四大魔神联手攻击直接崩灭了。盘古脸色苍白，一脸阴沉，好似刚才消耗很大。
<br/>　　“怎么可能？！”
<br/>　　“这是怎么可能，盘古斧已经分裂，还能重聚？！”众人心里异常震惊，接受不了这个事实。实际上，就连元释也一脸懵逼，这怎么可能，都分裂了，还能重聚？还要不要脸啦，这让我们怎么打啊？还能不能愉快地大战了？
<br/>　　“哼！有什么不可能的！虽然这只是暂时凝聚，但用来对付你们几个，已经足够了。”盘古冷哼一声，道。
<br/>　　元释等四大魔神心中恍然，原来如此，还真吓人一大跳，这样才正常。要是真的能分开又重聚，那才是真恐怖！
<br/>　　“战！”元释不再废话，再次与盘古大战起来。
<br/>　　“因果绞杀！因果业杀！杀！”
<br/>　　“来的好！给我斩！”
<br/>　　……
<br/>　　就这样，元释等四大魔神再次与盘古大战起来。当然，大多数时只是盘古与元释在大战，其余三魔神虽然也在大战，但还是相差元释两人不止一筹。要是没有元释，三魔神早已被盘古劈了。这就是混元圆满与混元后期的差距。
<br/>　　当然，元释也是差了盘古一筹，毕竟盘古已经半步不朽了，元释的战力只是堪比混元十二重天，境界更是差，只是混元十一重天。
<br/>　　但，盘古终究是开天时所耗的精力太大了，没有了巅峰时期的战力，更何况，他还一边大战，一边开天。不然，元释也差盘古不止一筹。能跟不朽沾边的，就没有简单的，就算是半步不朽也一样，这已经是本质的差距了。
<br/>　　元释与盘古交手多次，可惜，终究是差了一筹，好几次都差点被盘古的斧头劈了，身死道消。但，终究是挺了下来，不过，离死也不远了，魔神真身已经残缺不全了。
<br/>　　当然，这也不是没有好处，至少元释的因果大道法则距离突破混元十二重天，达到了第三步的巅峰只差一步之遥，相信只要有时间好好沉淀一番此次大战的感悟，突破只是水到渠成而已。而且，其他几种法则也通过大战消化的差不多了，若是有时间，他的战力将再次翻倍。
<br/>　　不过，这些终究是空话，因为他真的不行了，不过，这也不是空话，因为——
<br/>　　“强结——因果术！”元释利用自己最后一点力量，将一条隐匿在虚空之中，谁也没有发现的因果线悄悄延伸至盘古身体之中，至于因果线的另一端，则延伸至谁也发现不了的鸿蒙元始印之中。
<br/>　　“哈哈，因果，因果，诸果之因！我明白了！哈哈哈！”
<br/>　　元释大笑三声，笑声戛然而止。魔神真身也随之没了声息。大战中谁也没发现，元释仅剩的点点精血随着真灵遁入一个虚无之中，而那里，正是鸿蒙元始印之中。
<br/>　　……
<br/>　　元释一死，剩下的三大魔神也就不足为虑了。不过，事情终究会有意外——
<br/>　　“盘古，你毁了混沌，我也要让你不好过！”时辰嘶吼，全身上下气势如虹，整个魔神变得胀大起来。
<br/>　　“不好，快闪——”
<br/>　　轰隆！轰隆！
<br/>　　“咔擦！”随着一声爆炸声响，混沌青莲终于不堪重负，出现一道裂缝，仿佛就要碎裂开来，可见时辰魔神自爆真身的威力之大。
<br/>　　“咳咳！”盘古此时也显得有些疲劳了，开天时所需消耗的能量已经够大了，现在又经历一番厮杀，虽然大多数魔神对于他来说很弱，但架不住人数多啊。现如今，明显是精力所剩不多了，况且强行挣断因果，反噬的叠加也够他受的。正因为如此，元释悄悄施展强结因果术时，盘古才没有发现，要是平时，盘古发现不了他就不是盘古了。
<br/>　　这一场自爆，扬眉与魔祖被劫煞之气所影响的神志，顿时一清，但他们也仅仅只剩下点残破的元神了，两人不敢再与盘古战斗，化作两道流光遁入新世界消失不见了。
<br/>　　盘古有心去追，可是想到自己只需要证道成功，还怕找不到吗？现在最重要的是证道，况且这还只是人劫，还需度过天劫与地劫，他才能证道成功，以他如今所剩的实力，地劫无惧，天劫恐怕不妙啊。
<br/>　　天发杀机，斗转星移；地发杀机，龙蛇起陆；人发杀机，天翻地覆！
<br/>　　人劫已过，龙蛇已殁，地劫当无，只剩天劫，不知这天劫，又当如何？
<br/>　　盘古欣慰的看着自己开辟的新天地，等完善这方天地之后，度过天劫，他就能证道成功了。
<br/>　　“轰隆隆！”
<br/>　　“什么？”
<br/>　　盘古听到响动，抬头一看，顿时发现新开的天地开始下降了，有着想要再度融合的趋势。盘古大怒，证道就这么难吗，还是自己开辟的世界不够完善，他看了看自己座下的混沌青莲，眼中闪过一丝不舍，不过瞬间就恢复了过来。
<br/>　　“混沌青莲出，顶天立地！”
<br/>　　盘古直接将混沌青莲抛了出去，青莲瞬间变大，稳稳地接住了下落的天地，见此，盘古不但没有放松反而变得更加凝重了，因为他知道，这只是权宜之计，这座青莲撑不了多久，他必须尽快找到顶替青莲的天柱。
<br/>　　果然，没过多久，虚空就传来阵阵咔擦声响，不一会儿，四大混沌至宝之一三十六品混沌青莲顿时化为无数碎片，洒向盘古新开辟出来的洪荒天地。这些碎片就是后世无数的先天灵宝，其中最出名的就有四大莲子所化的莲台与莲茎所化的弑神枪、五色五行旗、乾坤鼎、天书、地书、冥书等。
<br/>　　看着那缓缓下降的天空，盘古无奈，怎么证道如此之难，都毁了本尊两大混沌至宝了，还不能成功。
<br/>　　“盘古真身！”盘古无法，但又不可能眼睁睁地看着他新开的这方天地给毁了，于是，他显出法相真身，两手托天，脚踏大地，顶天立地！
<br/>　　就这样，盘古有多高，天就有多高。随着盘古顶天的时间越来越长，他那已经肉身不朽的强悍神力也越来越少了。而，天也越来越稳固了。
<br/>　　但，就在这时，天劫降临了。
