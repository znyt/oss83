        只是当童筱颖回过神来后，才意识到，自己这下“死”定了。
<br/>
<br/>        而接下来的事情，也果然如她所料，傅夜沉对他们开始了“魔鬼训练”。
<br/>
<br/>        在去吃早餐前，还得围着两千米的操场跑三圈。
<br/>
<br/>        三圈是什么概念啊？
<br/>
<br/>        男生还好，对于班里那些不爱运动的女生来说，一圈都是能“要命”的节奏，更何况是三圈。
<br/>
<br/>        童筱颖就想不通了，傅夜沉不是傅大少爷吗？他不是接手了傅氏集团吗？上次她回来时，就听赵斌说，公司里忙得不可开交，那么他傅大少爷怎么有空来给他们当教官呢？
<br/>
<br/>        看傅夜沉那样，也不是吃饱了撑着没事找事做的人啊！
<br/>
<br/>        三圈下来，童筱颖和严恩恩跑完了，腿都发软了，舍友蓝筱蝶和恭如兰两人搀扶着还有一圈未跑完。
<br/>
<br/>        “体力不错！”
<br/>
<br/>        当童筱颖双手撑腰，弯着身子，大口大口喘气的时候，耳畔忽然响起了这个恶魔般的声音。
<br/>
<br/>        她不用抬头，就知道这个声音来自于谁。
<br/>
<br/>        “为什么将我的手机号码拉入黑名单？”傅夜沉淡定地问。
<br/>
<br/>        童筱颖直起身子，喘着粗气，回答道：“大哥，我……”
<br/>
<br/>        “你是觉得我在‘骚扰’你？”傅夜沉微微挑了挑眉。
<br/>
<br/>        童筱颖无奈地笑了笑，不吭声。
<br/>
<br/>        “筱颖，有史以来，你可是第一个将本少拉入黑名单的人！”傅夜沉意味深长地说，可语气里似乎是有一丝不悦。
<br/>
<br/>        童筱颖仍旧不吭声。
<br/>
<br/>        她香汗淋漓，胸前起伏比较大。看着她这个样子，傅夜沉突然心疼了，转而非常温和地说：“回宿舍去洗个澡，我会安排晚点集合。”
<br/>
<br/>        “大哥……”
<br/>
<br/>        走之前，童筱颖又转过身来唤了声。
<br/>
<br/>        傅夜沉看向童筱颖：“嗯？”
<br/>
<br/>        “我回宿舍后，会将你从黑名单里放出来。”童筱颖说完，转身既走。她可以说，她真真是“怕”了他吗？
<br/>
<br/>        她都怀疑，这次转移军训基地并延长了半个月，会不会跟这位傅大少的命令有关。
<br/>
<br/>        傅夜沉看着童筱颖渐渐走远的背影，不禁会心一笑。
<br/>
<br/>        然而，背对着傅夜沉的童筱颖却在心底暗自叫苦。
<br/>
<br/>        惹不起的人，她躲都不能躲，该怎么办是好？
<br/>
<br/>        不过，回到宿舍后，蓝筱蝶和恭如兰仍旧对这位傅大教官充满了无限好感，一个个纷纷嚷着：“筱颖，我们的傅大教官，也就是你的大表哥，重新介绍给我们呗！”
<br/>
<br/>        “我都说过了，他有未婚妻，你们别想了！”童筱颖裹着湿漉漉的长发，从浴室里走了出来。
<br/>
<br/>        而且，他的未婚妻，可是苏漫雪啊！
<br/>
<br/>        想到苏漫雪，童筱颖心里不禁疑惑起来。
<br/>
<br/>        距离上次她和苏漫雪被冷爷爷和连华生算计的那件事情也有一段日子了，苏漫雪不仅没来参加军训，就连人影都没见着过。
<br/>
<br/>        难道，真像傅夜沉所说的，被迫留在傅家大庄园里学习傅家的规矩吗？
<br/>
<br/>        苏漫雪可不是那种能熬得住礼节束缚的女人。
<br/>
<br/>        白天傅夜沉得陪童筱颖军训，晚上他还得处理公司里那一大堆的文件。
<br/>
<br/>        而他为了童筱颖所做的这件事情，也就只有助理郑忠威知道。
<br/>
<br/>        在军校军训的这些天里，童筱颖没再收到表白卡、玫瑰花、水果和零食了。果然是军校，“守卫森严”，就连手机都不让带，不过她们全都偷偷带了手机，就连打个电话也得偷偷摸摸的。

<br/>        军训的最后一天，学校里组织去靶场打靶。
<br/>
<br/>        然而，当大家都进入靶场后，傅夜沉却接到了一个陌生人的电话。

<br/>        “四少也有为了一个女人而动真格的一天了！”
<br/>
<br/>        手机听筒里传来嘶哑的男音，对方的声音很显然是电脑合成后的变音。

<br/>        自从他从部队里退出来后筱颍，他的档案便消掉了，所以，除了自己的那屈指可数的几个战友以外，没有人知道他傅夜沉就是“四少”。

<br/>        “你是谁？”傅夜沉皱眉问道。
<br/>
<br/>        对方笑呵呵地回答：“冒险者游戏，四少忘了吗？”

<br/>        “上次商场的炸弹事件，是你做的？”
<br/>
<br/>        “时隔这么久，四少还记得，真是好记性了！这次，我们又来玩一个游戏怎样？”对方洋洋得意地说。
<br/>
<br/>        傅夜沉不屑地回答：“我没空。”
<br/>
<br/>        “四少的未婚妻是叫苏漫雪吧？她就在这群军训的学生们中。你说，我派去的狙击手，有没有可能在这些全部身着迷彩服的学生们中，一枪便击中四少的女人呢？”手机里的男音阴险狡诈。

<br/>        傅夜沉凝眉的同时，在怀疑他们会不会把筱颖当成了苏漫雪。
