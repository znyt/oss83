<br/>				  再说李瑞，将蔬菜拉回去后，由于是在自己所管辖的旗舰店使用，还有一些其他店面，范围有些小，但是却足够消耗了。<br/>　　<br/>　　  并且因为是初次使用陈默他那里的蔬菜，所以前面去拉蔬菜的数量还是比较小的。<br/>　　<br/>　　  几家店面分了分，也就将一车蔬菜全部消化掉了。然后和港区酒店不一样的是，他推出了赠予服务。在店里吃饭，消费满500元的，送一盘精品蔬菜。<br/>　　<br/>　　  大家都知道，在饭店吃饭，菜点不了几个，但是价格却就成百上千。尤其是李瑞他们家，都属于中档餐饮，所以价格都有些小贵。好在能去这里吃饭的人，也就不在乎那三瓜两枣的，只要好吃小贵也不算什么。<br/>　　<br/>　　  基本只要去消费的，每个餐桌都已经是超过500元的，包厢就更不用说了。所以送菜等于是每个餐桌都有，但是却不一定每个餐桌的人都吃。<br/>　　<br/>　　  送的菜，一般都是快吃完饭了，才端上来的，所以大家也没有在意什么。送的菜而已，有什么好吃的！<br/>　　<br/>　　  有的餐桌就不去动筷子，然后大家挺着吃饱的肚子，结账走人。但是有些餐桌上，总有那么些吃货，见到有新菜上来，直接就挑筷子吃。<br/>　　<br/>　　  这一吃，好吧，就止不住了，没有想到这个送的菜这么好吃。顿时好心的就呼朋唤友，大家一起吃，也就是几筷子的量。<br/>　　<br/>　　  还有一些更狠，就埋头开吃，还将菜端到自己面前，直接大口大口的吃下去。同伴询问味道如何，却是回答不好吃，一般般。<br/>　　<br/>　　  但是看到这家伙的吃相，还真就不信了，然后抢一筷子后，顿时不满意了，好家伙，这么好吃的才竟然让一个人吃嗨皮了，绝对的不愿意，但是看到菜盘子里也已经寥寥无几，顿时欲哭无泪。<br/>　　<br/>　　  当然，还有些人，尝过这个送的菜以后，直接就抬手点餐，就点这个送菜。却得到价格是普通蔬菜的几倍，一盘炒青菜都要200元，这是精品来着，而且今天只是赠送，持续三天，等三天过后才会在餐单上体现。<br/>　　<br/>　　  服务员根据李瑞编制的话语，直接传达给顾客。这些菜都是最新联系到了货源，还不是本地的。<br/>　　<br/>　　  而且所有的蔬菜都是有机蔬菜，从种子开始到收获为止，没有使用工业化肥料，都是采用有机肥料。<br/>　　<br/>　　  蔬菜所含有的有害物质指标远远低于国家标准不说，一些营养等等还超过市面上同类蔬菜很多，而且数量不多，运输成本高等原因，造成达到餐桌上的价格就偏高。<br/>　　<br/>　　  听了这些后，只要是尝过蔬菜的顾客，都纷纷表示三天后再来，一定要尝尝。还有的一些没有吃过的，也就无所谓了，反正也没有吃到，倒也没有什么。<br/>　　<br/>　　  就这样，三天的宣传期一过，李瑞手下所有的餐厅都增加了一种精品蔬菜，不仅有主菜，还有精品蔬菜做的配菜，都在精品栏中，并且价格上也是提高不少。<br/>　　<br/>　　  李瑞也不是乱开价，在将蔬菜进回来后，他就让厨师做了几个拿手菜，有主菜也有配菜。<br/>　　<br/>　　  结果就是，在做主菜的时候，那味道简直是吃过还想吃。做配菜，也能将所有的菜的味道提高一个档次，就拿西红柿炖牛腩来说，当用陈默哪里的西红柿做配菜的时候，炖出来的牛腩不仅鲜嫩，还有着一股淡淡的清香味，不仅吃起来好吃，而且还入味，将牛腩的味道直接提高一个等级。<br/>　　<br/>　　  等他吃过所有的餐点后，就将这种蔬菜制作的菜点价格提高，做主菜一般都开开到好几倍的价格。用来做配菜的，价格也开到了一倍以上。<br/>　　<br/>　　  一盘青菜，以前也就是38元，现在直接200元，一盘西兰花，以前42元，直接就是260元等等。当然，普通蔬菜李瑞他也没有去掉，依然是作为餐厅主要原料，两个菜单，两种价格，任由顾客选择。<br/>　　<br/>　　  来李氏餐厅吃饭的顾客，有很多都是老顾客，所以在点餐的时候，服务员就会推荐一下，这也让老顾客惊讶了，怎么推荐的餐单这么贵？倒也没有说什么，直接点几个菜尝尝，要是不好吃直接就掀桌子的算。<br/>　　<br/>　　  最后，顾客还是掀了桌子，不是不好吃，而是太好吃了，但是还想点的时候，却没有货了，听了解释，有的也就算了，有的暴脾气的就要掀桌子，要不是店长又是赔礼又是道歉的，每个店绝对有好几个人会掀桌子的。<br/>　　<br/>　　  还有一些人，是前几天尝过的，直接订餐后就要上次的精品蔬菜。至于说价格，对于吃货来说，还是在容忍的范围内。结果就是吃了还想吃，要不是吃不下，都要将这些再点一遍的说。<br/>　　<br/>　　  也就是因为太受欢迎了，所以各个餐厅因为备货不足，导致了很多顾客没有吃好，后面来的很多顾客想在点，餐厅却表示没有货了。<br/>　　<br/>　　  这要是不掀桌子，那个时候掀？所以也闹的有些纷纷扰扰。李瑞听到这个消息之后，也有些哭笑不得！除了赶紧派车到陈默哪里进货以外，只能先采取优惠、送菜等等措施，安抚住顾客。<br/>　　<br/>　　  这一次精品菜点开局，真的很不错。由于有前面三天的试吃活动，也造成这次开局的圆满成功，当然掀桌子的就不算了。李瑞手头有相邻几个市区近十家连锁店的资源，这一晚上的收入，就超过以前一晚上收入的一倍多。这还是精品蔬菜种类少，配菜开发的少而已，要是等以后多了，那收入绝对杠杠的。<br/>　　<br/>　　  李瑞这一计算下来，顿时喜上眉头，没有想到老四种的菜这么给力，后面一定要加大进货量，另外要将这个渠道控制好。不过他也不担心，这是老四的渠道，自家兄弟。<br/>　　<br/>　　  想到家族里其他和自己竞争的家伙们，顿时感觉神清气爽的。只要有了老四这里的进货渠道以后，自己已经立在不败的地位上了。做餐饮的，最主要的是什么？味道啊！只要好吃，才能引来众多的顾客。<br/>　　<br/>　　  至于这些人以后去陈默哪里进货，呵呵！只要自己不开口，老四绝对不会将蔬菜买给他们。所以李瑞非常放心，不过为了以后的合作和关系，李瑞绝定将蔬菜进价在提高一些，每斤蔬菜定在20元好了，自己赚钱，兄弟也要有汤喝不是，这样也就更加照顾到兄弟伙里的感情。<br/>　　<br/>　　  如果感觉不错，请推荐和收藏一下，谢谢了！<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
