　　灵气屏障外，一众猎妖师心有余悸地看着前方一片空白，他们知道，那里隐藏着极危险的洞府。
<br/>　　他们也在暗暗庆幸自己的死里逃生。
<br/>　　铁达大叔一行人，除了庆幸自己没死外，便是不停的叹息。
<br/>　　“如此剧烈的爆炸，怕是达到洞玄境的威力了。”
<br/>　　“易霄兄弟就是之前没死，这次也肯定死透了。还有那个凭空出现的强者，应该和天星雷鳄同归于尽了吧。”
<br/>　　林飞和黄源你一言我一语，却万万想不到他们的易霄兄弟就是那救他们性命的强者。
<br/>　　“闭嘴。”铁达大叔喝了一声，而后看了眼浅沫。
<br/>　　林飞黄源两人注意到浅沫的神色，立刻闭上了嘴巴。
<br/>　　此刻的浅沫，脸色苍白，神情恍惚，也不知是因为爆炸的余波让她受了比较严重的伤势，还是在难过地想着些什么。
<br/>　　铁达大叔拍了拍浅沫的肩膀，道，“我们走吧，这次出任务，能遇到易霄兄弟这么好的队友，是我们的运气，只可惜，唉...”
<br/>　　.....
<br/>　　洞府内，爆炸的威力，让得这里一片狼藉。
<br/>　　就连囚禁雷鸣荒鳄的房间中的禁制，也被炸得一阵激荡。
<br/>　　幸好，狂血玄君的禁制确实强大，在一阵激荡后，又恢复了正常。
<br/>　　只不过，倒引起了无数雷鸣荒鳄的嘶吼声，令得洞府内兽吼一片，让人闻之胆寒。
<br/>　　当然了，这些兽吼声再大，隔着外头的灵气屏障也传不出去，只是令得洞府内某道狼狈的身影耳朵发疼。
<br/>　　“咳咳。”萧逸灰头土脸地咳嗽了两声，心有余悸地说道，“好险。”
<br/>　　他万万没想到，烈焰手套增幅后的噬火百刃，再经过武技融合，爆炸的威力竟然这么强。
<br/>　　秒杀了天星雷鳄不说，整个洞府都被席卷。
<br/>　　怕是普通的洞玄境一、二重武者都受不了，要不然不可能令得狂血玄君留在洞府的禁制都产生激荡。
<br/>　　“要不是有这幽魂面具，我必死无疑。”萧逸说着，取下了面具。
<br/>　　幽魂面具，有两大特效。
<br/>　　其一，佩戴者能掩盖自身气息，如幽魂之影，来去无息。
<br/>　　只有高自己一个大境界的以上武者才能看穿这层效果。
<br/>　　其二，能注入真气，储存一层防御罩，为佩戴者抵挡一次攻击。
<br/>　　防御罩的强度，只与注入真气者的强弱有关。
<br/>　　破玄九重修为的武者注入真气，防御罩便能阻挡一次破玄九重全力一击。
<br/>　　后天九重武者注入真气，防御罩便能抵挡一次后天九重全力一击。
<br/>　　这与实力无关，只与自身修为有关。
<br/>　　所以，当年狂血玄君斩杀掉一位地元境武者，声名大噪后，便再也没有用过这幽魂面具。
<br/>　　毕竟，当时他虽然是破玄九重的修为，但实力已经远超破玄九重了。
<br/>　　他最后一次注入的真气，一直没有使用过，多年后的今天，倒救了萧逸一命。
<br/>　　这也是萧逸选择戴上面具的原因，就是为了以防万一，能保自己一命。
<br/>　　当然了，这个机会只有一次。
<br/>　　防御罩只要使用一次，里头的真气就会全失。必须再次注入真气，才能再次使用。
<br/>　　萧逸不禁摇摇头，自语道，“狂血玄君留着的真气已经散尽，以我的修为，就算注入了真气，防御罩也强不到哪里去，聊胜于无罢了。”
<br/>　　倒是那隐藏气息的效果，对现在的自己更加有用。
<br/>　　收敛心神，甩去杂念，暗道，“铁达大叔他们应该已经安全离开了吧。”
<br/>　　“这里灵气如此浓郁，修炼一番再离去也不迟。”
<br/>　　想罢，萧逸回到了下层那古朴房间中。
<br/>　　顺带一提，狂血玄君在幽魂面具上留下了一道特殊的灵气屏障，与洞府内的禁制气息相连。
<br/>　　萧逸凭此可到达洞府内的任一地方。
<br/>　　当然，若是出了洞府，这道屏障就会失效。
<br/>　　回到蒲团下盘膝坐下，两大武魂齐开，加上蒲团的增幅，萧逸快速修炼起来。
<br/>　　在见识过洞府内各种灵气屏障的诸多神奇后，他也渴望自己有一天能像狂血玄君那般，对天地灵气拥有恐怖的掌控力。
<br/>　　这也许就是武者漫长而枯燥修炼生涯中的唯一乐趣了，享受各个境界中带来的好处与手段。
<br/>　　那种绝对的力量掌控，会让任何一个热血的人为之着迷。
<br/>　　本来自己才刚刚突破到后天九重，要突破到先天境，应苦修一段时间，储存足够的真气才能再去突破。
<br/>　　但，这里足足溢散了百多年的浓郁灵气，却可以让自己省掉这些时间。
<br/>　　加之自己的武魂吸收速度又如此之快。
<br/>　　强大的武魂，本身就是打破修炼常规的底气。
<br/>　　后天突破先天的过程，称之为合脉。
<br/>　　当九大主脉合为一条真脉时，武者便算成功晋升到先天一重。
<br/>　　主脉，乃武者天生拥有，只不过是堵塞的。
<br/>　　所以才需要后天武者去炼化杂质。
<br/>　　真脉，则是由武者融合主脉创造出来。
<br/>　　大概思索了一下，做好准备，萧逸便开始了合脉。
<br/>　　两大武魂快速吸收的灵气，早已转化为真气，供他随时做突破之用。
<br/>　　“合。”萧逸心中暗喝一声。
<br/>　　体内第二、第三....第九条主脉开始强行从气丹之上分离，准备往第一条主脉上靠拢。
<br/>　　这个过程相当痛苦，不亚于分筋拆骨，很多武者便是在这开头就忍不住疼痛晕厥过去，导致合脉失败。
<br/>　　萧逸感受到体内剧烈的疼痛，却没有哼一声，脸色仍旧平淡，只不过额头上已经布满了水柱似的汗水。
<br/>　　半晌，待得八条主脉全部分离，来到第一条主脉旁边，萧逸立马开始了融合。
<br/>　　这个过程，讲究控制力和天赋。
<br/>　　九条主脉必须分毫不差的重叠，主脉内的真气也要一丝不漏。
<br/>　　很多资质不佳的武者，则是失败在这个过程。要么是主脉有那么一丁点儿的没重叠好，要么是真气在融合过程溢散了一丁点儿。
<br/>　　半个时辰后，当萧逸停下修炼，体内九条主脉已经消失，取而代之的是一条更强大的真脉。
<br/>　　第一次合脉便成功，这是属于绝世天才的范畴。
<br/>　　“这就是先天境武者的真脉吗。”萧逸微微一笑，试着调动体内真气。
<br/>　　轰的一声，一股澎湃的真气瞬间在手中涌动。
<br/>　　“好快，好强。”萧逸微微吃惊。
<br/>　　真脉的坚韧度，宽阔度，还有调动真气的速度，都远超主脉，不愧是九条主脉重叠而成。
<br/>　　真脉真脉，事实上，这条由武者自己创造出来的经脉，就是因为最适合储存真气、调动真气，才被称之为真脉。
<br/>　　而且别忘了，其他武者是九条主脉融合成真脉；而萧逸是九条被金脉丹增幅过的主脉融合成真脉。
<br/>　　他的这条真脉，比其他武者强了数十倍都不止。
<br/>　　真脉中蕴含的真气数量，调动真气的速度，也同样是其他武者的数十倍。
<br/>　　单靠自身真气，他的实力就足以媲美先天六重。
<br/>　　若是加上灵器和其他底牌，他的实力将比之前对战天星雷鳄时更为强横。
<br/>　　言归正传，先天境的后续修炼，便是依靠这第一条真脉，积累真气，衍生出其它的真脉。
<br/>　　一条真脉等于一重实力。
<br/>　　拥有九条真脉，便是先天九重修为。
<br/>　　“嘭”，忽然，萧逸将一身真气凝聚于自己全身，一件犹如衣服般的薄薄真气层出现。
<br/>　　此乃真气外衣，拥有极强的防御力。
<br/>　　以往，萧逸也曾凝聚过‘火焰外衣’，但那不过是利用武魂释放真气而成，顶多算是加一层火焰在自己身上。
<br/>　　可真气外衣，乃是借由真脉，连接全身真气而成，更浑厚，更凝实，防御力也更惊人。
<br/>　　一个是释放真气而成。
<br/>　　一个是连接全身真气而成。
<br/>　　两者的防御力自然不可同日而语。
<br/>　　真气外衣，乃是先天武者的标志，但又不是所有先天武者都拥有。
<br/>　　这同样需要天赋和控制力。
<br/>　　很多先天武者都尝试过凝聚真气外衣，但其结果，却是外衣不稳，真气溃散。往往需要达到先天七重以上才真正有那个能力去凝聚真气外衣。
<br/>　　之前在紫云城，萧家也不过大长老二长老和三长老才拥有。慕容家也不过慕容山等寥寥几人拥有，诸如慕容槐慕容蝎之流则未曾凝聚成功。
<br/>　　毫无疑问，先天外衣，同样是属于天才武者才拥有的东西。
<br/>　　化去真气外衣，感受到自己充满力量的身躯，萧逸脸上写满了自信。
<br/>　　该离去了，但在走前，萧逸微微跪在蒲团前，对着卧榻跪拜了一下。
<br/>　　“虽说与你非亲非故，但此番你的幽魂面具救我一命，又是前辈，倒也值得我这一跪拜。”
<br/>　　萧逸扣了一头，轻声说罢，准备离开。
<br/>　　恰在此时，卧榻上‘噗’的一声，如同某道禁制破碎。
<br/>　　三样东西凭空出现。
<br/>　　一个玉瓶，一把剑，一本书。
