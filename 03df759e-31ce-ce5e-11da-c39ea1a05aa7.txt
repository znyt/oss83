　　一声暴喝，在身后响起。
<br/>　　与此同时，身后飞来一枚舍利子，打在了道风后背上。
<br/>　　道风本已就是强弩之末，元神既然涣散，重重地挨了这一下之后，再也抵挡不住，元神加速涣散，甚至无力控制自己，在空中漂浮起来，身影越来越淡。
<br/>　　原来死亡的感觉……是这样的。
<br/>　　道风仅存的一丝残念，低头看着自己的元神躯体，一点点分解、融化。
<br/>　　他死过两次，第一次是斩却肉身，是人类意义上的死亡，第二次是在死寂迷林，斩去灵根魂魄，两次死亡，都给他带去了不一样的体验，让他开了悟。
<br/>　　然而，这两次都不是真正的死亡，简单说就是，在两次死亡的瞬间，他虽然有濒死体验，但也清楚这只是证道的过程，并不是真正的死亡。
<br/>　　而这一次，则完全不同。
<br/>　　纵然坚强如他，在元神寂灭的最后一刻，他依然感到了恐惧和迷茫，甚至有点绝望。
<br/>　　我这么做，到底是对是错，到底值不值得？
<br/>　　他感到自己在向上飞，飞出了修所，一直飞到半空中去，他看到了那几个大佬，站在北斗观的院子里，仰头看着他。
<br/>　　（他甚至不知道是哪一个出手给了自己最后一下，不过这一点也不重要了）
<br/>　　他看到了飞奔而来的清长风，清长风也发现了他，在半山腰猛然站住，抬头凝视着自己。他的手中，拿着打神鞭和番天印。
<br/>　　曾经属于自己的专属法器。
<br/>　　曾经的荣耀，风之谷，叶少阳，杨宫梓，陈露……这一切，都逐渐远去了。
<br/>　　什么都没带走，什么也都没留下。
<br/>　　一生之中，兜兜转转，纵横三界，横扫天下，又如何呢？
<br/>　　他看到了酆都大帝，端坐朝堂，虚怀若谷，法度森严。
<br/>　　他看到地藏菩萨，在炼狱之中，手持经卷，拈花不语，悲天悯人。
<br/>　　他看到了少阳，看到青云子，师徒三人，大冬天在房间里吃着火锅唱着歌。
<br/>　　他看到了自己前世，仿佛就在面前，又仿佛很遥远，终究还是走马观花地一幕幕翻过去了。
<br/>　　他看到了前世的记忆、穿梭而成的历史的演变。
<br/>　　他看到了芸芸众生，吃喝拉撒，生老病死。
<br/>　　（日光之下，永无新事）
<br/>　　最后这两幕又融合在一起，他神识深处，仿佛突然开悟一般，明白了一些道理。
<br/>　　一切都在规则之下，时间，空间，生灵。亘古不变的规则。所谓的天地大道，原来就是这样。
<br/>　　“风儿……”
<br/>　　耳边突然传来青云子的声音，把他带到了一个新的天地。
<br/>　　十几岁的他，站在茅山一处悬崖边上，肃杀的秋日夜晚，星光斑斓。
<br/>　　道风一直站在悬崖边上，望着远处的群山、那掩映在群山中间的星星点点的灯光和鬼火，让年少的他内心充满了向往。
<br/>　　青云子和当时还只有几岁的叶少阳，这两个二逼正在草丛里抓蛇，打算烤了吃。
<br/>　　“道风你站那装什么呢，赶紧过来帮忙！”青云子骂。
<br/>　　道风不理他，继续淡定地望着远处。
<br/>　　这一老一小忙活半天没抓到蛇，抓了一些秋后的蝗虫，青云子也凑合烤着吃了。
<br/>　　“你一直在这站着干啥呢？”青云子问。
<br/>　　道风手里提着一串蚂蚱，指着远处，说道：“师父，我什么时候才能下山？”
<br/>　　青云子背着手走过来，站到他边上，小屁孩少阳也跟过来，好奇地站在一边。
<br/>　　“从这看过去，能看到什么？”青云子问。
<br/>　　“山，到处都是山，灯光啊就这些了。”少阳啃着烤焦的蝗虫，吃得满嘴都是黑灰。
<br/>　　“黑夜。”道风说道。无尽的黑夜。
<br/>　　青云子点点头，摸着叶少阳的脑袋（把手上的黑灰全蹭到他脑袋上），道：“你看到山，那就去爬山，山顶的风景，总是要比山下的好，但山外有山，你永远爬不到最高的那一座山上，就算你爬上去，比你高的还有天，你会当凌绝顶，天下人仰视，到那一步，你该怎样？”
<br/>　　叶少阳忽闪忽闪地眨着眼睛，随口说道：“爬山是为了好玩啊，爬到最高的山上，就行了啊，为什么要上天去？”
<br/>　　青云子微微一笑，又问道风：“黑夜便是天，无穷无尽，不受任何约束，但黑夜也代表着寒冷和孤独……少阳这孩子，将来贵不可言，能够站到最高那座山上，但他心性天真，率性而为，但世上之事，哪里什么都能由着你的性子？想让他突破天际，还得他自己开悟，自己找到理由，到那时，没人能拦得住他。
<br/>　　而你，道风，你如同黑夜，无边无际，不受约束，但你终究是黑暗的孤独的，你，不害怕？”
<br/>　　这差不多是青云子一生中为数不多的说话最有水平的一次。
<br/>　　道风转头俯瞰悬崖，微微笑道：“师父，我从来都不害怕什么，也不期望什么，不过，十年之后，天下人皆知我名，纵然不齿，亦对我无可奈何。‘道风’这两个名字，必将震动天下，搅翻三界！”
<br/>　　…………
<br/>　　当年的往事，依然历历在目。
<br/>　　“师父，而今十年不止，我当年所说，如今可都做到了。”
<br/>　　十三岁，持三尺法剑，独斩尸王；
<br/>　　十五岁，游历天下，降妖除魔；
<br/>　　十八岁，战遍天下同辈弟子，无一败绩。
<br/>　　二十二岁，单剑斩杀十二阴神、三大巫魔、闯六道轮回……名动天下，受封“人间道神”。
<br/>　　入魔十年，归来之后，开宗立派，三分鬼域，以一己之力对抗阴司和太阴山。
<br/>　　我这一生，何其荣耀，何其孤独，何其不被人理解，但是……这不就是我想要的吗？
<br/>　　终归没有白走一场。
<br/>　　只是……师父，在这一刻，我多想再见你一面，还有少阳，
<br/>　　只是，我得到了我要的一切，却感觉不到幸福。
<br/>　　我怀念茅山的那段日子。
<br/>　　我在不断地得到也在不断地失去。
<br/>　　我一生作为，只是为了不当别人的棋子。
<br/>　　终归劫历尽，尘世也成空。
<br/>　　在元神泯灭的最后一刻，道风想到了酆都城北门外那块石碑上亘古不变的那行字：
<br/>　　（今天有比赛，大家可以看看，我是没这个福分，最近颈椎有问题，头晕想吐，坚持写了一章，大家不要急，不可能说大道风死了就死了，继续往后看……）
