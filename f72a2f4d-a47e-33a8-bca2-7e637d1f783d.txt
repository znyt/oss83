　　对，即便这一次暂时杀不了莫忧。
<br/>　　但，起码萧逸还留了后手，让得之后的古帝之墓一行，不必再有后顾之忧。
<br/>　　天藏学宫一行，便就此告一段落了。
<br/>　　虽说对于古帝之墓一行，萧逸也不敢说有百分百把握。
<br/>　　不，准确来说，是他也不知道自己有多少把握。
<br/>　　但，即便得不到古帝之墓真正的机缘，萧逸也自信多少会有不错的收获。
<br/>　　若是能得些许不错的机缘，或许得到足够多的修炼资源，让他修为大涨的话，那也便算不虚此行了。
<br/>　　修为，终究才是武者质的变化。
<br/>　　以他的庞大气泉和小世界，修为上来了，真正的实力，必能大幅度飙升。
<br/>　　数个时辰后。
<br/>　　“呼。”萧逸轻呼出一口浊气。
<br/>　　体内伤势，已然痊愈。
<br/>　　在他的完美丹药下，外加金焱圣火这一疗伤圣火，还有修罗之力的帮助，轻松恢复了这些伤势。
<br/>　　接下来，便要开始修炼了。
<br/>　　火焰一道，暂时不必管。
<br/>　　能够提升的方面，也不多。
<br/>　　而去古帝之墓，他打算以自己的真实身份前去，自然的，现在该修习一番剑道。
<br/>　　而说到剑道，能够让他短时间内得到大量实力提升的，也就…
<br/>　　“破晓无极。”萧逸眯了眯眼。
<br/>　　他的修为，现今在武道大能层次。
<br/>　　体内气泉元力，已然填满；而小世界元力，则达到五成高度。
<br/>　　这般庞大的精纯元力下，足够他爆发极强的剑道水准了。
<br/>　　修为方面，短时间内，是无法提升了。
<br/>　　那么，能提升剑道实力的，便只剩下两样。
<br/>　　一是星幻剑道，所能掌控被降下的星光之力。
<br/>　　二则是破晓无极。
<br/>　　“不对。”萧逸似是想起了什么。
<br/>　　“东离剑宫的无上剑诀。”
<br/>　　说起来，他得了无上剑诀的最后一成后，还未看过。
<br/>　　萧逸手中光芒一闪，取出了无上剑诀。
<br/>　　自东离剑宫大比后，无上剑诀的9成副本，他就已经得到。
<br/>　　后来傲东楼，又给了他最后一成。
<br/>　　故整本无上剑诀，已然完整。
<br/>　　萧逸观看着最后一成的剑诀。
<br/>　　半晌，当他看完后，眼眸中，已写满了惊色。
<br/>　　“原来如此，难怪这门剑诀敢称‘无上’二字。”
<br/>　　单单之前9成副本，萧逸还看不出什么。
<br/>　　只觉这门无上剑诀，其玄奥程度，甚至还有星幻前辈的传承之上。
<br/>　　现在看到这最后一成，萧逸彻底恍然。
<br/>　　这门无上剑诀，并没有所谓的什么极其强悍的手段，也并没有想象中的无比厉害。
<br/>　　但它最厉害的一点，在于它修炼出来的剑气，能够融合。
<br/>　　不错，融合。
<br/>　　这门无上剑诀，单论前面9成，可谓博大精深。
<br/>　　修炼出的剑气，又浩然壮阔。
<br/>　　而最重要的，便是这些剑气，能够与别的剑道直接融合。
<br/>　　无论是什么剑道，都能融合其中，从而增幅自己的剑道实力。
<br/>　　单论这一点，果真是‘无上’之名。
<br/>　　因为无论武者修炼到哪一个地步，只要是剑修，这门剑诀都能发挥出它的功效。
<br/>　　数天后。
<br/>　　嗖…
<br/>　　萧逸指尖一道剑气打出。
<br/>　　剑气，呈白色，无任何属性可言。
<br/>　　但小小一抹剑气，出手之后，却瞬间化作庞大剑气，直直冲出山洞，随后在天际划出一道巨大剑痕才算渐渐消散。
<br/>　　“厉害。”萧逸脸色一惊。
<br/>　　“单论无上剑诀自己本身的威力，就是一门相当了不起的剑道功法。”
<br/>　　“若再加上我的星幻剑气…”
<br/>　　萧逸眼中，闪过一丝期待之色。
<br/>　　……
<br/>　　时间，渐渐过去。
<br/>　　十天后。
<br/>　　萧逸结束了闭关。
<br/>　　虽只是短短时间，但他的收获，并不算小。
<br/>　　“这一次的古帝之墓一行，把握大了不少。”萧逸笑笑。
<br/>　　哗…
<br/>　　萧逸手中光芒一闪，取出了自己的公子服。
<br/>　　摘下面具，脱下一身劲装，换回了自己的一身公子服。
<br/>　　本来之前离开东离剑宫，他就已换回自己的身份，打算以自己萧逸的身份行走。
<br/>　　但后来遇到剑姬前辈一行人。
<br/>　　既剑姬前辈要‘易霄’前往，他便只能再度换回易霄的身份。
<br/>　　现今，天藏学宫之事已告一段落。
<br/>　　学宫所有长老执事，尽皆受伤，偌大个天藏学宫，被毁了大半。
<br/>　　通天峰长老，天叠峰长老，身死。
<br/>　　莫忧这个当事人虽未死，但也已修为尽废，日日遭受火焰灼烧折磨。
<br/>　　公道，暂时算是取回了。
<br/>　　故他现今自然换回自己的身份，准备前往古帝之墓。
<br/>　　嗖…
<br/>　　萧逸手中紫电神剑祭出。
<br/>　　散了周遭屏障，毁了周遭修炼痕迹，萧逸直接御空离去。
<br/>　　算算时间，距离古帝之墓开启，还有6、7天左右。
<br/>　　现在赶去，时间上差不多。
<br/>　　对于古帝之墓一行，他并无准确的把握。
<br/>　　但，如果能真正得里头机缘，那自然是最好的。
<br/>　　古帝，这位上古一帝，成就帝位时间更早于剑帝的传说人物，会留下何等机缘呢？
<br/>　　萧逸期待着。
<br/>　　说起来，他自身的底牌不少。
<br/>　　但一直以来，他始终相信着，冰鸾剑方是他最大的底牌。
<br/>　　可是，冰鸾剑要么不用里头力量，一用，便是反噬惊人。
<br/>　　而另一点，他相信这世间，没有什么是绝对的。
<br/>　　冰鸾剑虽强，却绝对不代表着无敌。
<br/>　　以前，在天元境、地极境那些层次时，冰鸾剑里头力量，堪称可怕。
<br/>　　可现今，早已是圣境之后的境界。
<br/>　　自己更已是武道大能修为，对上的，起码都是绝世以后的强者，甚至是传奇。
<br/>　　冰鸾剑，能否拉平这些差距，还未可知。
<br/>　　故，萧逸已经很久没用过冰鸾剑里头的力量了。
<br/>　　这世间，没有什么是绝对无敌的。
<br/>　　而这一次的古帝之墓一行，或者便是一个契机。
<br/>　　若能真正得到里头机缘，自己，或许便有一次真正一飞冲天的机会。
<br/>　　“只能尽力了。”萧逸自语一声。
<br/>　　……
<br/>　　另一边。
<br/>　　远方，风刹地域之内。
<br/>　　风刹、修罗两位总殿主，看着手中情报，不禁一笑。
<br/>　　“啧啧，这紫炎易霄，竟这般大闹天藏学宫。”
<br/>　　承风殿主躬身笑道，“此事，早已传遍中域，那小子，这次可谓再次声名大噪。”
<br/>　　“而且也如之前两位总殿主猜错的那般，猎妖殿、炎殿、药尊殿三位总殿主，果真前去帮那紫炎易霄了。”
<br/>　　“听说，姓洛的差点没气得七窍生烟。”
<br/>　　“霍老院长倒是挺平静。”
<br/>　　风刹总殿主笑笑，“姓霍的这老小子，只能吃这哑巴亏，他当然只能平静。”
<br/>　　“说不定，他看了紫炎易霄那般厉害的本事和手段，还念着双生子的另一人呢。”
<br/>　　“可惜，当年他天藏学宫没眼光，萧逸小子来了我风刹殿。”
<br/>　　“对了。”承风殿主忽然脸色一变，“总殿主，萧逸那小子，算算时间也差不多要去古帝之墓了。”
<br/>　　“那姓洛的，向来不要脸，这次…”
<br/>　　“不慌。”修罗总殿主摆摆手，看了眼手中情报，笑道，“这一次，姓洛的，没机会的。”
<br/>　　“我稍后去黑云学教一趟便是。”
<br/>　　“猎妖殿那三个老家伙，都为了自家接班人出手了；我风刹、修罗二殿，自然也不能斋看着自家接班人被欺负。“
<br/>　　“你和横天二人，去古帝之墓一趟。”
<br/>　　“姓洛的没法出手，其他人，便无须担心，你二人的实力，足以护萧逸小子周全了。”
<br/>　　“记住。”风刹总殿主沉声道，“这一次，古帝之墓盛事一结束，你便把人逮回来。”
<br/>　　“若再让萧逸小子跑得无影无踪…承风，你看老夫如何罚你。”
<br/>　　承风殿主苦笑一声，“是，总殿主。”
