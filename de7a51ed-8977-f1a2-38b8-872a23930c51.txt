　　酆都，偌大中州里的偏安一偶，远在北方的一座城市，通过此地，便可进入传说中的鬼界。
<br/>　　酆都城外是一片荒地，除了一些山峦湖泊，一眼望不到尽头，树木杂草丛生，没有规律整洁可言。
<br/>　　当然，道路是有的，只是被草木遮掩了而已。
<br/>　　酆都城内，呈现出一副安详和谐的景象。
<br/>　　传说中的鬼城，表面看上去与其他城市柄没有任何的不同，也没有任何的诡异，居民安居乐业，其乐融融。
<br/>　　就在这时，酆都城外，上空空间轻微一阵波动，黑色裂缝浮现，仿佛天空都被撕裂一般。
<br/>　　“不愧是鬼城，好浓郁的阴气。”
<br/>　　清朗温和的声音响起，慕清霄从空间门中跨出，望着前方脚下的城市，眼中闪过一抹好奇之色。
<br/>　　哪怕是站在千米的高空上，都能感受到四周扑面而来的阴寒气息，也只有在这种阴寒地域，鬼物才敢肆无忌惮的在外生活。
<br/>　　自从踏入修仙路以来，他见过各种千奇百怪的东西，鬼怪也见过不少，但鬼魂生活的城市，他是第一次见到。
<br/>　　在别的位面，鬼怪是恶的象征，在大部分生灵眼中，鬼族就是一种异类。
<br/>　　以至于，某些势力根本不允许这种东西存在，也唯独仙剑位面比较特殊。
<br/>　　六界平等！
<br/>　　望着下方的和谐的城市，慕清霄目光微抬，视线尽头，城市的最后面，是一座金碧辉煌的宫殿。
<br/>　　宫殿的金色匾额上，刻着龙飞凤舞的三个字‘鬼皇宫’。
<br/>　　“有意思。”
<br/>　　收回目光，慕清霄嘴角牵起一抹笑意，空间微微扭曲，身影就出现在城门口。
<br/>　　望着城门上‘酆都’两个黑色字体，慕清霄视线放在紧闭的城门上，下一秒视线转换，身影就出现在了城内。
<br/>　　霎时间，耳边就被喧嚣声所充斥。
<br/>　　虽远离人族，但鬼族也有鬼族自己的生活环境，与常人无疑。
<br/>　　站在街道上，慕清霄所看到的和睦景象全都是真实的，只是他们的身份有所转变而已，人族换成了鬼族。
<br/>　　走在街道上，慕清霄并没有隐匿身形，一袭白衫，腰带玉佩，黑发随风飘荡，因此吸引来一道道复杂的目光。
<br/>　　当然，复杂归复杂，鬼族自然能察觉出他是人族，当即收回目光，并没有搞事，而是坐起各自的事情。
<br/>　　没有想象中，露出一副副狰狞可怖的嘴脸，甚至各种厌恶目光，有的只是平静与复杂。
<br/>　　似乎，他们也向往平静的生活，对于人族修士进入鬼城，也见怪不怪。
<br/>　　如果鬼族闯入人族城市，情况可能就非常的糟糕了，修士不群起而攻之都是好的。
<br/>　　虽说六界平等，但暗地里的摩擦却不断，各族也时常会引发一些小规模战斗。
<br/>　　和睦相处，完全是不可能的。
<br/>　　走在街道上，随意的看了看，街边摊位也有不少的好东西。
<br/>　　别看鬼族呈现出一副和睦画面，但这些摆摊的可都不是善茬，或者说他们都是鬼修，有的实力也很强。
<br/>　　不像人族，普通人注定就是普通人。
<br/>　　而鬼族不同，哪怕是鬼界最边缘，居住在鬼城中的鬼魂都拥有一身不弱的修为。
<br/>　　目光在摊位上扫了一圈，慕清霄发现其中拥有不少的好东西，甚至许多种高阶灵药。
<br/>　　拥有高阶灵药的摊主，基本都是一些颇有实力的鬼修。
<br/>　　在中州地域中，各种福地都被仙门占领，哪怕不是仙门范围，灵药也被采摘一空，根本不可能留着等你去摘。
<br/>　　但鬼族地域不同，他们修炼靠的是阴气，各种特殊灵药，人族修仙灵药，他们九成都用不上。
<br/>　　所以摊位上的灵药也是拿来贩卖给人族的。
<br/>　　“公子，看上那一株灵药，都是最近摘过来的，价格便宜。”
<br/>　　摊主是一名老者，实力堪比元婴修士，见慕清霄气质不凡，当下微笑着说道，但语气中并没有讨好的意思。
<br/>　　酆都是鬼族的地盘，他们并不惧怕人族修士。
<br/>　　而且，人族修士也不敢在此地乱来。
<br/>　　慕清霄目光落在摊位上的紫色鲜花上，道：“养魂花，八品灵药，在中州内地可看不到这种灵药。”
<br/>　　“公子好眼光呐，在人族仙门中地位不小吧？”
<br/>　　“我并非仙门中人，只是一介散修，养魂花怎么卖？”
<br/>　　老者浑浊的双眼中闪过一抹诧异，但也没有升起任何不好的念头，敢孤身一人来鬼城，绝对是有着底蕴。
<br/>　　至于慕清霄是不是仙门中人，地位如何与他完全没有任何关系。
<br/>　　“养魂花是八品灵药，我也不宰公子，五十颗鬼丹，或者一本高阶鬼修心法，在整个鬼城中，绝对找不到第二株养魂花。”
<br/>　　养魂花，在八品灵药中也属于比较高级的灵药。
<br/>　　这种灵药极其稀少，炼制成养魂丹，能够滋养灵魂，对于魂魄受损的修士而言，是不可多得的好东西。
<br/>　　鬼丹，慕清霄到是知道，由极为阴寒的灵气凝聚而成的丹类，利于鬼族修炼。
<br/>　　至于鬼族心法，这种东西也是非常稀少的存在，一般都只有强大的鬼修才拥有，或者鬼族势力拥有。
<br/>　　由此可见，老者与他是同一类人，不属于任何一个实力，当然，这都是慕清霄自己的猜测。
<br/>　　“鬼丹我暂时没有，至于鬼修士心法，我是人族，并没有那种东西。”
<br/>　　说罢，慕清霄思考了一阵，道：“隔日我在来此，对了，在鬼城中，人族的钱币不能使用吗？”
<br/>　　闻言，老者怪异的瞥了他一眼，道：“冥币或者鬼丹是通用货币。”
<br/>　　若有所思考的点了点头，慕清霄转身离去，他身上并没有两种东西，好比如在人族城市没有钱。
<br/>　　没有钱简直是寸步难行，总不能仗着自己修为高就出手抢吧，到时候绝对惹来一身骚。
<br/>　　虽说慕清霄并不怕，但低调行事总有低调的好处。
<br/>　　而且，他有自己的目的，灵药可以晚一点在来买，但火灵珠必须尽快拿到手。
<br/>　　思来想去，慕清霄抬头看向酆都城尽头金碧辉煌的宫殿，‘鬼皇宫’也只有鬼王级别的鬼物能够居住。
<br/>　　鬼王在鬼界，相当于人族的帝皇。
