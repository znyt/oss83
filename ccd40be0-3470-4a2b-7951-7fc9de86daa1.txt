　　只见帝一魂再次击出一剑，重重的斩向拓跋青云。
<br/>　　这一剑，剑气分开了整个时间，浩大无边，威力无法用言语来形容，只能用恐怖二字，能概括其威势。
<br/>　　剑气磅礴无比，剑气之中，凝聚了万界之力，强大到无可匹敌。
<br/>　　所谓的万界圣域，其实是依靠万界圣魂，施展的一种能力。
<br/>　　万界圣魂，能称之为万界，便是因为其拥有万界之力，此元神就如同是一方浩瀚宇宙，其内拥有大量的星辰，每一颗星辰，便是一个世界。
<br/>　　十几万颗星辰，就是十几万个世界，换言之，帝一魂的万界圣魂拥有十几万个世界的力量。
<br/>　　当然，每一个世界的力量，能有多大的威力，还要看帝一魂的修炼造诣。
<br/>　　这一剑的强大，若是在下方宇宙之中，没有神域空间的桎梏，足以摧毁整个古灵星河，湮灭无数星辰。
<br/>　　天地仿佛失声，天地之间，只剩下了一道浩大无比的剑气，纵横无匹。
<br/>　　另一边，拓跋青云面色沉重如铁，他自然感觉到了此招的恐怖，让他都很是震惊。
<br/>　　但是，他却是不会有丝毫的退避，面对帝一魂，他必须有舍命一战的准备。
<br/>　　只见他手中的黑色小鼎，再次膨胀，瞬间化为了万丈巨鼎，而他的身形也进入了鼎中。
<br/>　　巨鼎没有去硬挡这一剑，而是鼎口朝向剑气，急速旋转了起来。
<br/>　　霎时之间，巨鼎化为了一个巨大的漩涡，这漩涡之大，直径足有数十万丈，仿佛是吞噬天地的大口，足以吞噬一切，容纳一切，骇人无比。
<br/>　　漩涡之中，有九条巨.龙飞舞，扬天长啸，震动诸天环宇。
<br/>　　这一切说来极慢，实则都发生在电光火石之间，快速无比。
<br/>　　轰！
<br/>　　一声恐怖的爆响，天地齐震，万界轰鸣，凝聚了万界之力的恐怖剑气，重重的斩在了漩涡之中。
<br/>　　轰隆隆！！
<br/>　　如同世界毁灭一般，恐怖的爆响，充斥在天地之间，剑气完全没入了巨大的漩涡之中。
<br/>　　巨大的漩涡通体剧震，不断的溃散，逸散的力量，毁灭方圆数万里的虚空和大地，仿佛是世界末日一般，让人心生恐惧。
<br/>　　昂！昂！昂！！
<br/>　　漩涡之中，巨龙的咆哮声更加的剧烈，庞大的龙影翻腾不朽。
<br/>　　就在此刻，异变陡生，远处天际之上，一道硕大的掌印，和一道万丈雷霆，裹挟无限的杀机，向漩涡中袭杀而去。
<br/>　　掌印和雷霆的威力，恐怖无比，不仅不比帝一魂的攻击弱，反而威势更强一分。
<br/>　　是天君河和天王宫大长老天罡雷一起出手了，两人想要以雷霆之势，镇压乃至灭杀拓跋青云。
<br/>　　他们选择的时机非常的巧妙，正是拓跋青云极力抵挡帝一魂的攻击之时。
<br/>　　这两道攻击，足以重创或者灭杀拓跋青云。
<br/>　　而几乎就是同一时刻，另一方的大地之下，同样有两道可怕的攻击，向帝一魂袭杀而去。
<br/>　　这两道攻击，一道是浩大的斧芒，一道是一个巨大的拳印。
<br/>　　两道攻击之强，同样惊人无比，气势铺天盖地，丝毫不比天君河和天罡雷的攻击弱。
<br/>　　这是地渊神王，和另一名地王宫的高级神王出手了，想要镇杀帝一魂。
<br/>　　两人选择的时机，也是最佳的时机，因为，帝一魂凝消耗了大量的万界之力，正是最虚弱的时候。
<br/>　　这个时候出手，他们有五成的把握镇杀帝一魂，有七成的把握重创帝一魂。
<br/>　　轰隆隆！！
<br/>　　巨响震天，两道攻击袭向拓跋青云，两道攻击袭向帝一魂，滔天的气息横扫诸天。
<br/>　　“不好！”
<br/>　　惊呼声响起，不过，发出惊呼之人，却并不是帝一魂和拓跋青云，而是天君河和地渊神王等人。
<br/>　　他们是找到了时机，纷纷出手了，但是，却是来不及阻拦住对方的攻击了。
<br/>　　不过，帝一魂和拓跋青云，面对这种局面，并没有措手不及。
<br/>　　因为，两人早有提防之心。
<br/>　　只见帝一魂，在攻击袭来的刹那，身形便急速的闪烁，快速的避开了。
<br/>　　而拓跋青云所在的巨大漩涡，吞没了帝一魂的剑气之后，正在急速的溃散。
<br/>　　这一刻，漩涡通体一震，彻底炸裂开来，无尽的劲力横扫九天。
<br/>　　漩涡消散，再次化为了巨鼎，急速缩小，化为了磨盘大小，并飞速远去。
<br/>　　轰隆隆！
<br/>　　天君河和地渊神王等人的攻击，纷纷落空，在一阵虚空爆响之中，消失在了天地的尽头。
<br/>　　帝一魂和黑色大鼎，纷纷停在了远处，拓跋青云的身形，出现在了大鼎之上。
<br/>　　嗖嗖嗖！！
<br/>　　这一刻，天君河和地渊神王等人，纷纷现身而出。
<br/>　　天君河和罡雷大长老飞向帝一魂，地渊神王和另一位地王宫的高级神王，飞向拓跋青云。
<br/>　　几人均是纷纷松了口气，现在的局面，算是平分秋色，没有发生不愿看到的情况。
<br/>　　帝一魂和拓跋青云两人，没有望向众人，而是目光紧紧的相视。
<br/>　　不过，两人的表情，却是完全不同。
<br/>　　帝一魂的眼眸之中，露出一丝不可差觉的笑意，而拓跋青云，却是面色凝重无比。
<br/>　　因为，这一战，看似平分秋色，实际上却是分出了胜负。
<br/>　　拓跋青云败了！
<br/>　　只不过，其败的并不明显，一般人根本差距不出。
<br/>　　“天君河，你真是老奸巨猾！”地渊神王来到了拓跋青云身边，立刻转头，看向飞到了帝一魂身边的天君河，怒目而视。
<br/>　　“地渊，我们彼此彼此！”天君河冷笑一声，朗声说道。
<br/>　　“哼，今日只是个开端，希望以后，你这个老家伙还能笑的起来！”
<br/>　　地渊神王冷喝一声，随即对拓跋青云道：“青云，我们走！”
<br/>　　地渊神王并没有和天君河交手的意思，也不愿意多留，立刻招呼拓跋青云离开。
<br/>　　因为，他看出来了，拓跋青云受伤了，必须尽快回去。
<br/>　　拓跋青云点了点头，随即，三人破空而去，转眼间便消失在了长空之中。
<br/>　　见拓跋青云等人离去，帝一魂和天君河等人，也没有阻拦，因为他们知道自己拦不住。
<br/>　　“尘宇，你没事吧？”
<br/>　　罡雷大长老问向飞来的天尘宇，对方是他的儿子，他自然格外的关心。
<br/>　　罡雷大长老，面容较为年轻，身形高大魁梧，浓眉大眼，一副莽汉的模样。

<br/>　　“帝一魂，这一次交手，你有什么看法？”天君河的目光，紧紧的盯着帝一魂。
<br/>　　“拓跋青云，不过如此！”
<br/>　　帝一魂微微一笑，沉声道：“他已经被我重创，千年之内，是绝对不可能晋升高级神王了！”
<br/>　　“哦？”
<br/>　　闻听此言，无论是天君河还是天尘宇，亦或是罡雷大长老，均是眼眸一亮。
<br/>　　“他受伤了？”天尘宇惊讶的问道，他并未看出拓跋青云受伤了。
<br/>　　“拓跋青云的战力，和一部分的天赋，来自于九龙神鼎元神，适才一击，此元神的九龙之灵已经被我重创，千年之内难以恢复！”帝一魂声音低沉的说道。
<br/>　　正因为此，他断定拓跋青云，千年之内绝对不可能突破了。
<br/>　　“好！”天君河闻言，顿时大喝一声，面带喜色。
<br/>　　如果真是这样的话，那么，帝一魂就赢定了，或者说他们就赢定了。
<br/>　　因为，帝一魂在千年之内，是有把握晋升高级神王的，到时候，一切都该结束了。
