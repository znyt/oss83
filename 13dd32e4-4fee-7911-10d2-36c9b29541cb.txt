　　“呼。”
<br/>　　萧逸盘膝坐下，稳下体内虚弱的气息。
<br/>　　半晌，才呼出一口浊气，站了起来。
<br/>　　刚才施展天火印，不仅将自身元力耗尽，连怒炎戒内的力量，也被消耗九成以上。
<br/>　　不过，这并非大问题。
<br/>　　怒炎戒作为极品元器，一段时间后，会自动吸收天地灵气，慢慢恢复回来。
<br/>　　体内元力也一样。
<br/>　　自身修为境界在，会自动慢慢恢复的。
<br/>　　“领域？”此时，萧逸皱眉看着寒冰殿主的尸体。
<br/>　　萧逸记得，天火圣者的传承中，对这招武技的定义，便是如此。
<br/>　　大致有提到过领域，但具体怎么回事，却很是模糊。
<br/>　　刚才火焰充斥于周身百米范围内时。
<br/>　　那无穷无尽的火焰力量，让萧逸为之震撼。
<br/>　　萧逸甚至觉得，在那个范围内，自己能主宰一切。
<br/>　　这门超越天阶巅峰的武技，实在是让人不可思议。
<br/>　　摇了摇头，萧逸没有再多想。
<br/>　　目光，看向寒冰殿主尸体上，手中握着的风雪罗盘。
<br/>　　大手一挥，将其吸摄而来。
<br/>　　当年，风雪剑主得到此物，却是祸非福，惹来杀身之祸。
<br/>　　寒冰殿主，身居高位，却不惜杀人夺宝。
<br/>　　萧逸认真地打量着此物。
<br/>　　圆盘，散发着无尽的古老气息。
<br/>　　最上方，刻着‘天机’二字。
<br/>　　“难道是天机圣盘？”萧逸瞳孔一缩。
<br/>　　萧逸摸了摸圆盘上的花纹，又看了看背面。
<br/>　　“绝对是天机圣盘。”萧逸的声音中，带着惊骇。
<br/>　　萧逸曾在猎妖殿主殿的古老卷宗上，看到过此物的描述。
<br/>　　当然，毕竟是上古之物，年月相隔太悠长了，记载中的描述，也并不详细。
<br/>　　只知，此物乃是阵法师的至宝。
<br/>　　阵法师、炼药师、魂师，在上古时期，乃是三大职业。
<br/>　　沧海桑田，岁月流转，至今只剩下炼药师一门尚有传承留下。
<br/>　　其它两门，不知何故，早已失传，消失在历史长河中。
<br/>　　魂师，以诡异闻名，擅长无形手段。
<br/>　　阵法师，以阵法闻名，擅长以玄妙阵法，操控天地之力。
<br/>　　日月星辰、山河江海，皆可被他们的阵法所借用。
<br/>　　以微弱之力，发挥出远超自身的惊天之能。
<br/>　　不过，阵法师有着一个致命的缺点。
<br/>　　那便是需布阵，随后引敌入阵，之后才能发动大阵杀敌。
<br/>　　布阵，需要耗费大量时间，大量珍贵宝物。
<br/>　　这在正面厮杀中，几乎不可能做到，所以才要提前布阵，引敌入阵。
<br/>　　这让阵法师，有了许多限制。
<br/>　　当然了，根据记载，在上古时期，阵法师，让人闻风丧胆。
<br/>　　一些成名的阵法师，更是无人敢惹。
<br/>　　“可惜了。”萧逸摇了摇头。
<br/>　　上古时期，必是比现在精彩得多吧。
<br/>　　萧逸不禁透漏出一丝向往。
<br/>　　真正的强者，都会有一颗强者之心，渴望那种危险却精彩的生活。
<br/>　　言归正传。
<br/>　　天机圣盘，之所以是阵法师至宝。
<br/>　　便是因为罗盘上，能刻下阵法，随时可触发大阵。
<br/>　　阵法师携之，等同随身携带自己布下的大阵。
<br/>　　可想而知，这是多么逆天之物，又被称为阵法师至宝。
<br/>　　寒冰殿主，之所以将此物命名为风雪罗盘，估计也是怕真正有见识之人，听到‘天机圣盘’四字，找来不必要的麻烦。
<br/>　　而且，此时，天机圣盘上刻下的阵法，正是风雪大阵。
<br/>　　命名为风雪罗盘，倒也贴切。
<br/>　　阵法师这个职业虽然失传了，但还是有少数一些简单而粗糙的阵法，被一些大势力保留遗传至今。
<br/>　　风雪大阵，算是比较低级的一种阵法。
<br/>　　如果没猜错的话，这门阵法，应该是风雪剑主得到天机圣盘后，自己烙印下的。
<br/>　　据萧逸所指，裂天剑宗，就保存着几门低级阵法。
<br/>　　风雪大阵，便是其中一门。
<br/>　　当然了，虽只是低级阵法，却是宗门中视若珍宝的存在。
<br/>　　风雪大阵，本身并不怎么强，但在天机圣盘的振幅下，威力却极强。
<br/>　　加之，这里乃是极寒之地，终年风雪不断。
<br/>　　风雪大阵，在这个地方，简直是如虎添翼。
<br/>　　甚至于，若时间足够的话，借助天机圣盘，这个风雪大阵真的能掌控整个极寒之地的风雪力量也不一定。
<br/>　　另外一点，极寒之地这边，上古遗迹无数。
<br/>　　当然，这些遗迹，掩藏于各种不毛之地，无人知晓。
<br/>　　风雪大阵，却能通过天机圣盘，指引并感知出这些遗迹的所在之地。
<br/>　　且，遗迹中虽有着强大屏障，或洞府禁制。
<br/>　　但风雪大阵足够强大的情况下，却能轻松破去、乃至无视这些禁制。
<br/>　　要得到遗迹内的宝物、传承，轻而易举。
<br/>　　难怪寒冰殿主，如此野心勃勃。
<br/>　　若给他足够的时间，或许这一切真的都能实现也不一定。
<br/>　　“呼。”萧逸深呼吸一口气。
<br/>　　事情，大概是弄清楚了。
<br/>　　不过，对于极寒之地中的遗迹，他并无半分兴趣。
<br/>　　也并不愿意浪费时间。
<br/>　　收起天机圣盘，萧逸将目光看向周遭的尸体。
<br/>　　寒冰殿主、不夜宫主、雪刀门主、冰武教主……
<br/>　　猎妖殿五殿强者，一宫一门二教数千武者，已悉数身死。
<br/>　　之前不夜宫主等人想跑，却被萧逸的‘天火流星’，悉数击杀。
<br/>　　虽然刚才他体内元力耗尽。
<br/>　　但怒炎戒内，多少还残余一些力量，足够他施展一次‘天火流星’了。
<br/>　　天火流星，作为从天火圣者传承中得到的天阶巅峰武技。
<br/>　　其威力，远在怒炎斩之上。
<br/>　　不夜宫主等人，无法逃脱，也在情理之中。
<br/>　　顺带一提，寒冰殿主等人，原本设下重重埋伏，是要对付秦飞扬三人。
<br/>　　不过，他们并不想节外生枝。
<br/>　　故，早在之前，就已经将千里范围封锁，驱赶了不干事的武者。
<br/>　　所以，此次大战，并未波及其他武者。
<br/>　　周遭，虽狼藉一片，却并无半个不干事的武者敢来看热闹。
<br/>　　当然，谨慎起见，萧逸还是大手一挥，不下了一层禁制。
<br/>　　布好禁制，萧逸的眼睛，猛地一变。
<br/>　　太阴太阳之眼，凝聚而出。
<br/>　　地面上，数千具尸体的武魂力量，悉数涌来。
<br/>　　寒冰殿主、不夜宫主两人的武魂，乃是蓝色巅峰阶品。
<br/>　　雪刀门主、冰炎教主、冰武教主三人，乃是蓝色阶品。
<br/>　　而其他尸体中，竟也有十数人，拥有蓝色阶品。
<br/>　　剩下的，几乎清一色在青色阶品以上，最弱都是绿色巅峰。
<br/>　　不得不说，极寒之地这边的武者素质，确实惊人。
<br/>　　待得所有武魂力量吸收殆尽。
<br/>　　萧逸小世界中的控火兽武魂，终于突破，达到蓝色阶品。
<br/>　　冰鸾剑武魂，本就深紫色的颜色，微微深了一些。
<br/>　　不过，这些许变化，几乎微不足道。
<br/>　　做好一切，萧逸大手一挥，将寒冰殿主等人身上的乾坤戒、乾坤袋，吸摄一空。
<br/>　　随后身影一闪，彻底离去。
<br/>　　此次，极寒之地这边涉事的猎妖殿，共5个。
<br/>　　寒冰主殿，以及四座分殿。
<br/>　　其中，寒冰主殿方面，强者尽出。
<br/>　　四座分殿方面，则仅仅是执事以上猎妖师涉事，执事以下普通工作人员，不曾出现在此。
<br/>　　还有一座分殿，未曾参与此次战斗。
<br/>　　他们是否涉事，有否漏网之鱼。
<br/>　　个中情况，以及调查，只需让殿主派遣别人过来处理即可。
<br/>　　萧逸无需多管，也懒得多管了。
<br/>　　毕竟剩下的，都只是些小事、杂事，自不必他多费心。
