　　叶少阳把法结最上方一缕枝条调整到固定位置，指向自己居住的客栈，这样那个什么妙心真人一定会打听到自己住的地方，毕竟小镇就一个客栈，也只有他们几个外地人，很容易找得到。
<br/>　　“那个什么妙心真人，既然是紫狂真人的传承人，对于那个古墓的事，肯定也知道一二，有些事情，我们需要问她。”叶少阳这番话，得到了道渊真人和毛小方的认同。
<br/>　　三人回到客栈，要了一桌饭菜，在房间里餐桌前刚坐下，美华赶来了，告诉他们事情已经办妥。
<br/>　　“对了，五通神那些手下，怎么样了？”叶少阳猛然想起这茬，问美华。
<br/>　　美华笑道：“已经都被我收编了，成了我的手下。”
<br/>　　“那就好，不过……话说，你也不要邪修，不要伤人，不然我不知道拿你怎么办。”
<br/>　　美华有点不好意思地点了点头，“我只是在水中修炼，并不伤人。”
<br/>　　“那就好。”叶少阳说完，请大伙开吃。他是几个人中最有钱的，这顿饭自然是他请，要了几个好菜，一壶酒，也算是庆祝一下搞定了五通神。毛小方和道渊真人也很开心，三人开怀痛饮。美华站在叶少阳身边，像个婢女一样，给他们倒酒倒茶，叶少阳把她当成百年后的美华，倒也受用，没把她当外人。
<br/>　　酒足饭饱之后，叶少阳招呼小二把饭桌撤了，然后说起正事。
<br/>　　“之前说起炼尸缸，你们两个脸色都变了，那到底是什么鬼东西？”叶少阳靠在床上，嘴里叼着牙签，好奇地问道。
<br/>　　道渊真人跟毛小方互相看了看，毛小方道：“我对这个估计知道的多一点，我来说吧。少阳子，你听说过长生不老药吗？”
<br/>　　叶少阳嘴角抽了抽，道：“这个你应该去问徐福，传说他为秦始皇找过，后来去了岛国，繁衍出了打膏药旗的那些后代。”
<br/>　　毛小方摆了摆手说道：“不开玩笑。道门的炼丹术，在古代一直有很多人崇信，就连那些皇帝都深信不疑。这个你应该知道。”
<br/>　　叶少阳点了点头，这个他当然知道，炼丹，一向都是道门法术中重要的一环，炼丹有内外之分，可以增进修为，延年益寿，还有一些特殊的丹药，有各种特殊的功效。
<br/>　　只不过当炼丹术流行开之后，很多外行的一些道士之类的也跟着炼丹，却不得其法，炼丹的时候大量运用铅汞等有害物质，最后把自己吃坏了吃死了的都有，历史上有不少皇帝都是吃丹药吃出问题的，淮南王刘安更是搞笑，炼丹不成，居然炼出了豆腐……
<br/>　　不过炼丹术却是真实存在的，只是长生不老那是扯淡，这是道门的共识，叶少阳也不知道毛小方为什么提起这个。
<br/>　　毛小方也不卖关子，往下说道：“古代的皇帝，其实都是信道教为主的，不过更多的是寻求养身和壮阳的法门，也都希望自己能长生不老。有些道士为了迎合皇上，就想方设法用各种法子炼丹，皇帝为了能活得久，也是各种支持，要什么给什么。
<br/>　　大部分道士虽然坑一点，但也是为了荣华富贵，骗点钱之类的，但有一些别有用心的道士，利用皇帝的这种信任，干起了邪修的勾当。
<br/>　　炼尸缸的传说，就来自于西汉末年王莽建立的新朝政权。跟之前那些皇帝一样，王莽也信道教，十分迷恋长生不老药，当时各种宠信道士，其中最被他宠幸的是一个道号叫“天卓”的道士，自称来自终南山，去过蓬莱仙岛，有仙人给的长生不老药的药方。
<br/>　　这长生不老药十分难配，其中就有一味药，需要人油……而且需要数百个活人榨出来的人油。
<br/>　　皇帝当时一心只想长生不老，正好赶上当时有一些小股的起义被镇压，于是让天卓道长从这些被关押的战俘里挑选合适的去炼药，对王莽来说，这不过是一次“废物利用”，因为这战俘多半也是要杀掉的。
<br/>　　天卓道长却拿着鸡毛当令箭，挑选了足有近千人，分批杀死，办法当时十分简单也十分残酷，就是特意打造了十口特殊的铜鼎，把人押送到鼎中，用柳木在下面烧，把人活活烤死。铜鼎之上刻有符文，能把人在死亡中产生的怨气，以及死者的魂魄，都吸附在鼎壁上。
<br/>　　“因为需要杀的人太多，所以传说这个天卓道长用了三口鼎，一起杀人，每一口鼎中至少都杀了数百人……就算没有特殊符文的加持，光是行刑数百人，这些鼎也是绝对的大凶之器。更不要说死者的怨气和魂魄都在鼎中……少阳子，你想一下这是什么概念？”
<br/>　　听完毛小方的讲述，叶少阳深深吸了口气，古代有刽子手所用的行刑刀，就算是家传，也不能超过杀死百人，不然传说就会生出邪灵。更不要说，这每一口鼎中都杀过几百人……
<br/>　　最重要的是，刀一类的刑具，都是开放的形状，怨气和魂魄难以依附，而按照法术界的说法，鼎为缶，中间可以盛放任何有形和无形的东西。
<br/>　　毛小方也说了，这几口鼎还是特制的，上面刻着道纹，又是慢慢将人烤死，所产生的怨气，和这些人的魂魄都被鼎壁吸收了……这样一种邪性到极致的东西，叶少阳简直闻所未闻。
<br/>　　“那个什么天卓道长，杀这些人是干什么的？”叶少阳很想知道下面的事情，他当然不相信是真的为了炼制什么长生不老药。
<br/>　　“什么长生不老药当然是幌子，他这么做，是为了获取那些鬼魂和产生的怨气，留着自己慢慢使用……”
<br/>　　怨气和鬼魂，都是邪修法师最喜欢的两样东西，可以做各种用途，可以用来炼制法器，也可以直接炼化提升修为，这是法术界的共识，因此毛小方也没有特意解释。
<br/>　　“之后呢？”
<br/>　　“之后他刚炼成没多久，起/义军就打进长安了，新朝灭亡，这些道士当然也跑不了，据说当时天卓道长倒是跑了，不过这三口鼎却不可能带的走，留在了长安城一处地方藏了起来，之后就失踪了。”
