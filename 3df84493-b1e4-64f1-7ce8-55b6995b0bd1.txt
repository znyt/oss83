　　木长老，继续说道。
<br/>　　“药楼之巅的禁制，每年都在强行摄取大量星辰之力。”
<br/>　　“以作温养楼内天材地宝之用。”
<br/>　　“每过数年，星辰之力便会暴增，导致过剩。”
<br/>　　“星辰之力，乃天地之力。”
<br/>　　“既能温养天材地宝，亦能温养人体。”
<br/>　　“有淬炼肉体，增加修为，提升感悟以及武道瓶颈之神效。”
<br/>　　萧逸闻言，看了眼四周的星辰之力。
<br/>　　星辰之力，如有灵性般，四处乱窜。
<br/>　　散发着骇人高温，狂暴无比。
<br/>　　很显然，他是一种火焰，星辰之火的别称，也是因此而来。
<br/>　　同时，由于这是天地之力，故必定精纯无比。
<br/>　　无论是淬炼肉体，还是增加修为，都必定效果极大。
<br/>　　用神效来形容，一点儿不为过。
<br/>　　这有点像裂天剑宗剑阁考核内的那些剑气，同样是击碎或捕获后，能增加自身修为。
<br/>　　但，与那些剑气相比。
<br/>　　这里的星辰之力，效果惊人得多。
<br/>　　而且，那些剑气只能增加修为。
<br/>　　而星辰之力，能增加修为、淬炼肉体、提升感悟，以及武道瓶颈，自是珍贵百倍不止。
<br/>　　这时，萧逸饶有深意地看了眼月纷舞。
<br/>　　踏入33层时。
<br/>　　自己几人都面露惊色，唯独她，只是微笑。
<br/>　　很明显，她以前来过这里。
<br/>　　而且，她那股青白相间的强悍火焰，气息与这里的星辰之火，一模一样。
<br/>　　显然，她操控的那股火焰，便是星辰之火。
<br/>　　星轮羽衣，本不具备火焰属性和能力。
<br/>　　但羽衣上的星轮，却与星辰的气息相契合。
<br/>　　若没有猜错的话，应该是某位修为滔天的强者，强行将这些星辰之火烙印到她的羽衣武魂之上。
<br/>　　并凝聚出火焰种子。
<br/>　　这才让她有了操控星辰之火的能力。
<br/>　　结合她的身份，作为谷主高徒，以前来过这里倒也正常。
<br/>　　而那位修为滔天的强者，应该便是药王谷谷主了。
<br/>　　当然，由于她的武魂本身不具备这种能力。
<br/>　　这般强行强加进去。
<br/>　　会让得本就狂暴的星辰之力，变得极不稳定，难以操控。
<br/>　　萧逸现在才恍然，难怪之前会感知到她体内有一股极不稳定的火焰。
<br/>　　这时，木长老继续说道，“现今，这周遭的333道星辰之力，便是这数年来过剩的。”
<br/>　　“你们的比拼，便是捕获星辰之力的数量。”
<br/>　　“数量最多者，便为这次王都大比的第一。”
<br/>　　“第一名的奖励，乃是任选两样天材地宝。”
<br/>　　说着，木长老看了眼最中央那堆堆积如山的天材地宝。
<br/>　　里头，连八品天材地宝都存在，且效果堪称极品。
<br/>　　“这次大比，无论是星辰之力，抑或我药王谷的奖励，皆是不可多得之重宝。”
<br/>　　“能得到多少，能收获多少，全凭你们自己的本事。”
<br/>　　“如无疑问，这次的大比，正式开始。”
<br/>　　木长老话音刚落下。
<br/>　　萧逸、叶铭、月纷舞、血焰、白墨寒，5人瞬间身影一闪。
<br/>　　朝四周飞跃的星辰之火而去。
<br/>　　“禁。”白墨寒率先大喝一声。
<br/>　　一道禁制，瞬间落下，禁锢了一道星辰之火。
<br/>　　但，那道青白相间的流光，显然不愿束手就擒。
<br/>　　竟在禁制内四处冲撞，半晌，直接将禁止烧出了一个小洞，飞跃而出。
<br/>　　萧逸看了眼，微微心惊，“这些星辰之火，已有微弱灵智。”
<br/>　　“且，这位列世间强悍火焰之一，温度极高。”
<br/>　　“想捉住它们，绝非易事。”
<br/>　　时间渐渐过去数分钟。
<br/>　　五人的身影，在这33层的巨大空间屏障内，肆意飞跃。
<br/>　　除了白墨寒的身影显得比较游刃有余，渐渐加大了禁制强度，捕捉了两道星辰之火外。
<br/>　　其余四人，显得略微有些束手无策。
<br/>　　但渐渐的，几人也非常人，开始凭借自身实力，捕捉了一道。
<br/>　　“休想跑。”萧逸低喝一声。
<br/>　　大手一挥，漫天紫炎，倾泻而出。
<br/>　　瞬间包裹了数道星辰之火。
<br/>　　同为世间强悍火焰，星辰之火胜在足够狂暴，且速度极快。
<br/>　　而紫晶灵炎，则是焚毁万物之效，包括焚烧这些星辰之火。
<br/>　　星辰之火，已有些许灵智，当即在紫炎之内不敢动弹，生怕被烧成灰烬。
<br/>　　萧逸冷笑一声，低喝道，“收。”
<br/>　　一道星辰之火被他握于手中，而后，瞬间被吸收。
<br/>　　精纯的能量进入体内，萧逸顿时打了个激灵，浑身神清气爽。
<br/>　　“原来如此。”萧逸瞬间恍然。
<br/>　　星辰之火，那奇怪的火焰高温，能焚烧肉体杂质，淬炼肉体。
<br/>　　达到增强肉体强度之神效。
<br/>　　内头蕴含的精纯力量，可转化为体内元力，增加修为。
<br/>　　且由于这是天地之力，内头蕴含一丝天地法则之力。
<br/>　　能达到提升感悟，增加武道瓶颈之效。
<br/>　　得知其神效，萧逸再不迟疑，当即在紫炎内禁锢的另外几道星辰之火擒获。
<br/>　　化作自身力量。
<br/>　　另一边的木青云以及木长老，则在那堆天材地宝中观看着，便作着记录。
<br/>　　萧逸等人，都陆陆续续开始捕获到了星辰之火。
<br/>　　唯独叶铭，还在手忙脚乱地追逐着一道星辰之力。
<br/>　　果然如木长老所说，这次比拼，若无实力，来了也是白来。
<br/>　　得不到什么好处。
<br/>　　但，或许是皇天不负有心人，最终还是被叶铭辛苦地擒下了一道。
<br/>　　正当他准备吸收这道星辰之火时。
<br/>　　血焰凭空而现，一把抢过了这道星辰之火，率先将其吸收。
<br/>　　“你做什么？”萧逸脸色一冷，瞬间出现在叶铭身旁。
<br/>　　血焰冷笑一声，“紫炎，你刚才是不是没有认真听规则？”
<br/>　　“这次比拼，比的是捕获星辰之火的数量。”
<br/>　　“除了我们自身擒获的之外，亦可以抢夺他人的。”
<br/>　　“这是一场真正实力上的比斗。”
<br/>　　萧逸眉头一皱，“照这样说来，我也可以抢夺你的了。”
<br/>　　“不错。”血焰冷声道，“但，恐怕你没那个闲工夫对付我。”
<br/>　　萧逸眉头皱了皱，转过头。
<br/>　　那里，白墨寒正手握神锋剑，冷冷地盯着自己。
<br/>　　“难怪之前在药楼外，我扇了你一巴掌，你尚肯罢手。”萧逸冷笑一声。
<br/>　　“原来你想将战场放在这里。”
<br/>　　白墨寒作为药王谷的座上宾，似乎早就知道了这药楼之巅的比赛规则。
<br/>　　所以之前药楼外，大庭广众下，并未急着出手。
<br/>　　木青云的喝止，他也立刻罢手。
<br/>　　“今日你一道星辰之火都休想得到。”白墨寒脸色阴沉。
<br/>　　“紫炎易霄，此次炼药大比之后，只会沦为笑话。”
<br/>　　话音刚落，白墨寒瞬间出手。
<br/>　　萧逸怡然不惧，两人的战斗，瞬间打响。
