　　朦胧之间，刘哔仿佛听见了一阵奇怪的声音，像是某种古老的音乐，但他竖起耳朵去听的时候，却又什么都没听见。
<br/>　　刘哔心中猜测，他等的那只女鬼，应该马上就要出现了。
<br/>　　秋生此时依旧没察觉到什么，晃晃悠悠的向前骑行着，四周的雾气越来越浓，刘哔敏感的发现，四周的天地灵气一阵变动。
<br/>　　突然，像是察觉到了什么，刘哔转了一下头，紧紧注视着后方的土路。
<br/>　　或许是因为雾气的影响，土路稍微有些泥泞，从道路的尽头，缓缓走出来几道抬着花轿的人影，刘哔伸出舌头，舔了舔有些干裂的嘴唇。
<br/>　　花轿逐渐露出了全貌，是那种古代常用于婚礼的轿子，轿子的四周，由几名脸色惨白的男童抬着，在这些男童身上，感受不到一丝生人的气息。
<br/>　　看到目标已经出现，刘哔便不再等候，径直从藏身的树冠上跳了下来，挡在了花轿前方，负手而立。
<br/>　　“砰！”
<br/>　　花轿落地，溅起了一阵混杂着落叶的尘土，轿子前方红色的帘布缓缓掀开，露出了一个面色红润的娇媚女子。
<br/>　　虽然看起来和正常女子没有什么区别，但是刘哔很清楚的知道，这个千娇百媚的女子，是一头不知死了多少年的鬼魂。
<br/>　　“道长深夜挡路，不知所为何事？”女鬼挑了挑眉，面带不悦的向刘哔问道。
<br/>　　如果鬼魂不主动显露的话，常人是无法看到她们的，能看见她的，都是凝结了灵种的修炼者，对于这类人，这个世界都统称为道长。
<br/>　　“捉鬼。”刘哔微微一笑，言语中的杀意，却是怎样都遮掩不住。
<br/>　　听见这句话后，女鬼面色微寒，在四周弥漫的雾气之中，她的面孔有些模糊，给人一种颇为阴森的感觉。
<br/>　　不过很快，她脸上露出灿烂的笑容，看着刘哔道：“道长未免有些不讲道理了吧，二十年来，我隐于墓中，从不害人，也从未和任何人结过仇，道长为何咄咄相逼。”
<br/>　　“多说无益，动手吧，此地无人，动手最为合适。”舔了舔口中的獠牙，刘哔对此却没过多解释，有些事情，他是没有办法和女鬼解释清楚的。
<br/>　　见事态无法挽回，女鬼脸上的笑意逐渐消失，紧接着，随着一声清脆的声响，女鬼的脑袋，猛然向下沉了一截，像是快要断掉了一般。
<br/>　　刘哔站在原地，脸上露出一副饶有兴致的表情，他还是第一次和鬼魂交手，不知道对方有何种手段。
<br/>　　随着女鬼的动作，四周的武器逐渐开始消散，四周的场景也变的越来越清晰，随之消失的，还有那台花轿和抬轿人。
<br/>　　这些东西，都是她体内的鬼气所化，如今即将要和刘哔交手，她自然不会继续浪费自己体内的鬼气，用来维持这些东西。
<br/>　　眼看着女鬼手上长出长长的指甲，脸上的皮肤都存存裂开，露出了下面鲜红的嫩肉，刘哔胃里涌上一股不太舒服的感觉，女鬼的这幅模样，影响到他的食欲了。
<br/>　　此时，女鬼一大半的脸庞，都变成了一层烂肉，看起来让人感觉有些恶心。
<br/>　　刘哔这才叹了口气，缓缓伸出了一根手指。
<br/>　　“道长，只身前来捉拿老朽，连法器都不肯拿出来，未免有些看不起人了吧。”脸上露出一抹阴冷的笑容，女鬼的头发如同刺猬一般，尽数炸起。
<br/>　　这个时候，她的声音已经变的苍老无比，就像经历了很久的岁月一般，很是沙哑难听。
<br/>　　耸了耸肩，刘哔心中有些无奈，他不是不想要带武器，而是他根本就没有女鬼口中所说的法器。
<br/>　　至于菜刀铁锹之类的，他也不认为会对女鬼产生什么作用。
<br/>　　下一瞬间，女鬼的身影一动，脑后的万千长发，如同一根根钢针般呼啸而出，直袭刘哔的面门。
<br/>　　刘哔依旧保持着先前的动作，一根手指在半空中缓缓凝聚出一道巨大无比的虚影，向着女鬼站立的地方碾压过去。
<br/>　　就在先前交谈的时候，他已经做好了充足的准备，将全身的灵气都汇聚在了手指当中，就是为了尽快解决战斗。
<br/>　　他很清楚自己如今的实力，虽然依靠增强过的身体素质以及这两天修炼出来的灵气，拥有了和一些鬼怪战斗的资格，但是缺少相应的道法和武器，他的持续作战能力并不是很强。
<br/>　　就像是一个浑身蛮力却不懂得运用的现代人，和人打架的时候最好的方式就是靠着一身蛮力，上来就进行碾压式的打击，如果给敌人缓过气来，发现了他的弱点，那么战斗的胜负，就犹未可知了。
<br/>　　而女鬼对于刘哔的心思，却是丝毫不知，按照这个世界正常的打法，两方都是先要相互试探一番的，但感受到那根手指上散发出来的强大威势之后，她的脸色陡然一变。
<br/>　　但是仓促之下，她也只能堪堪偏了一下头，但还是没有完全躲过刘哔的攻击，一阵轰鸣过后，她的整个身体，都化作了一片灰色的雾气。
<br/>　　这是她用本源鬼气凝聚而成的身体，不知道花费了多少年的岁月方才凝练出来，如今在刘哔一击之下，却全都化为虚无了。
<br/>　　看着空中那片灰色的武器，女鬼脸色有些阴狠，一时大意之下，她居然遭受到了这样的重创。
<br/>　　因为先前躲避的动作，女鬼先前的进攻也只能被迫停止，但没有了躯体，并不代表她没有了战斗的力量。
<br/>　　只见那片被刘哔攻击砸出来的坑洞里面，女鬼的头颅从中飞了出来，甩了甩脸上的泥土，面部的皮肤已经完全破碎，脸上的烂肉中，长出一片像是藤条一般的东西。
<br/>　　“你敢伤我！”口中发出一声尖叫，女鬼愤怒的盯着刘哔，随着她的话语，脸上藤条般的东西一阵蠕动，看起来格外恶心。
<br/>　　平静的和女鬼对望着，刘哔胃里那股不舒服的感觉又涌了上来，见惯了那些令人恶心的丧尸，但女鬼的这幅尊容，依旧让他感觉有些辣眼睛。
<br/>　　女鬼脸上那些蠕动的像是藤条的东西，给人带来一种柔软的感觉，就如同，那是血肉铸成的一般。
