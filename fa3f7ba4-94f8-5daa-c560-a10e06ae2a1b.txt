　　第41章
<br/>　　“打劫！”
<br/>　　这两个字不轻不重，随风飘散，传到山坡下每个人耳中。高飞虎心里一激灵，怀疑自己是不是听错了。
<br/>　　“喂，打劫啦，打劫啦！男的站左边，女的站右边，不男不女站中间！”程大雷骑在马上，喊着胡编乱造的山歌。
<br/>　　高飞虎激灵一下，这下确认自己没有听错，他也终于明白，程大雷紧闭寨门，闭门不出，并不是因为幼稚的生气，他是要——打劫！
<br/>　　可是为什么呀！
<br/>　　高飞虎想不明白这个问题。
<br/>　　难道他没有看见朱家寨血流成河，上下三十几口惨死如猪狗！
<br/>　　难道他没有看见陆哼精兵过百，枪挑人头招摇而过！
<br/>　　难道他没有看见青牛山大小山寨的头目都耸头耷尾，缀在人家的马屁股后面！
<br/>　　他当时看见了吧，可他还是骑着一匹瘦马晃了出来，面对精兵如从，人头云集说着两个字。
<br/>　　打劫。
<br/>　　陆哼微微眯眼，随即嘴角咧起，抬起手掌。
<br/>　　“数五之内，过来跪在我的马下，我可以选择原谅你刚才说的话。”
<br/>　　程大雷轱辘下了马，将斧头搁在一边，卷起衣服的下摆挽在腰上。
<br/>　　高飞虎看到这一幕轻轻点头：一时抽疯不要紧，现在向陆哼磕头求饶纵然屈辱些，但能活下来，这些屈辱又算什么。
<br/>　　正这个时候，黄三元和徐神机推着一架重弩到大门前，然后飞速躲开。
<br/>　　程大雷来到重弩前，转动绞盘，拉开弩弓，将一支特制的箭安置在弩身上，说是箭，或者叫做矛更合适些，长有一丈，手腕粗细，黑铁枪尖，在日光下闪闪发光。
<br/>　　高飞虎睁大眼睛，恍然觉得自己想错了，这不是磕头求饶的意思，他是要——进攻！
<br/>　　这有没有搞错，以一敌百竟然要率先进攻，高飞虎觉得自己的世界观正在被颠覆。
<br/>　　陆哼刚数到三，竖起三根手指，手尴尬的杵在空中，有些数不下去了。
<br/>　　程大雷双手举起斧头，望山下望了一眼，似乎对陆哼还没有下马表达不满。
<br/>　　“说了打劫啦！”
<br/>　　声音落地，斧锤狠狠砸在扳机上，扣簧被敲开，绞盘飞速旋转，上面架着的重箭被推动，愤怒的向车队冲去。
<br/>　　这种重弩，或者称作床弩更加合适，算是大型杀伤武器。程大雷使用的这架重弩，射程达八百步，杀伤范围在五百步。
<br/>　　而从蛤蟆寨的寨门到山坡下距离四百步。
<br/>　　一名骑兵被掀翻，重箭扎在马车上。
<br/>　　“骑兵散开，步兵以马车为防御。”
<br/>　　陆哼第一时间下达命令，他知道这种重弩威力很大，但精度不够。
<br/>　　所以陆哼没有闪避，只要他运气不是太糟糕，就不会被这重箭射中。他卸下马上铁弓，弓开如满月，射向山坡上的程大雷。
<br/>　　程大雷没有躲避，他重复着刚才的动作，转动绞盘，搭箭，用斧锤砸动扳机，激发重箭射向山下。
<br/>　　一人手提长枪骑马从蛤蟆寨走出，甩枪挑开陆哼射来的箭。
<br/>　　也有骑兵从山下向上射击，首先是距离太远，超出攻击范围，而且由山坡下向上射，即便到达程大雷身边，也无多少杀伤力。
<br/>　　在秦蛮的护持下，程大雷重复着刚才的动作，张开弩弓，搭箭，敲下扳机，动作笨拙却坚定，对于山下纷纷射来的箭视作无物，只是不断重复着刚才的动作，对着山下的队伍，干净杀戮。
<br/>　　惨叫声响起，有尸体倒地，烈马失控狂奔，荡起烟尘。
<br/>　　等系统奖励的十根重箭射完之后，程大雷长身而起，拍落尘埃。
<br/>　　陆哼眼中的怒火已经肉眼可见，他握紧手中铁棍，口中爆喝一个字。
<br/>　　“杀！”
<br/>　　一声令下，骑兵为先，步兵在后，一窝蜂向山上冲来。
<br/>　　“秦蛮何在！”程大雷大喝。
<br/>　　“属下在！”秦蛮提枪。
<br/>　　程大雷抬起手臂，指着冲锋的陆哼：“我要他项上人头。”
<br/>　　“属下为大当家取来！”
<br/>　　秦蛮一勒马缰，一手持枪，一手控马向山下冲去，那纷纷射来的羽箭根本无法近秦蛮的身，而秦蛮的目标只有一个，就是马上的陆哼，更确切来说，是陆哼头顶的人头。
<br/>　　杀人者人恒杀之。
<br/>　　陆哼能斩人头，就该想到有朝一日他也会被人砍去脑袋。
<br/>　　程大雷翻身上马，注视着山下冲锋的队伍。
<br/>　　向上进攻的队伍在刚深入山坡就遇到问题，骑兵轰然倒地，兽夹发动，藏在土中暗箭被触发……
<br/>　　蛤蟆寨重建之后，李行哉见到山寨的生活设施后便大呼小叫，可是他并不知道，山寨重建，那些生活设施只是捎带手的，程大雷的重心在山寨的防御上。
<br/>　　秦蛮当初是一个优秀猎人，如今是顶级山贼，在一马平川的山坡上究竟埋着多少陷阱，怕只有蛤蟆寨的几个人知道。
<br/>　　所以从山上到山下只有一条安全道路，就是秦蛮所走的那一条，除了这条路外，其他道路都埋满陷阱。
<br/>　　这四百步的距离对于这些士兵来说，成了此生走过最漫长的路。
<br/>　　程大雷翻身上马，却并未发动进攻，只是平静的注视着山下。
<br/>　　蛤蟆寨三个能打的，秦蛮是顶级，林少羽是优秀，而程大雷只是普通，所以他最不行。也无非能仗着斧头沉重，杀伤范围大乱撞一通。
<br/>　　林少羽也催马走出山寨，立在程大雷身后。
<br/>　　再远的距离也有走完的一刻，在付出惨重代价后，攻击的敌人终于快接近程大雷。每个人眼里都恨不得喷出火来，如果他们能靠近程大雷，丝毫不怀疑他们会生吃了程大雷。
<br/>　　有鼓声响起，墙头上徐灵儿奋力捶鼓，这鼓是系统奖励，附带鼓舞技能的战鼓。
<br/>　　鼓声像是敲在众人心脏，咚咚作响，而敌人与程大雷的距离已越来越短。
<br/>　　程大雷和林少羽几乎是同时用薄纱遮脸，程大雷口中高喝一声：“勿急，我有神法，能请天雷！”
<br/>　　程大雷会五雷法这件事传开的范围不小，有人信自然也有人不信，可今日他在众目睽睽之下，如此做派，难道他真的掌握五雷法。
<br/>　　程大雷昂首挺胸，双臂伸展，似乎是在催动仙法。
<br/>　　“又来了，又来了！”高飞豹大喊：“快看天雷啊！”
