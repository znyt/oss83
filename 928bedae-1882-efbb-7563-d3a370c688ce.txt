夏夕绾心里像抹了一层蜜般甜丝丝的，她发现这个男人还真会哄人。
<br/>
<br/>        这时夏妍妍已经推门而入了，她看到了屏风，想抬脚走过来，“陆总，你怎么在这里，你在干什么？”
<br/>
<br/>        夏妍妍要过来了，夏夕绾吓得迅速掐了陆寒霆一下。
<br/>
<br/>        陆寒霆觉得她在给自己挠痒痒，相比女孩的紧张，他一张俊脸波澜不惊，只淡淡的开腔道，“站住！”
<br/>
<br/>        夏妍妍迅速停下了脚步，不敢再往里面走了。
<br/>
<br/>        “我来这里有一份紧急文件要处理，你找我有事？”
<br/>
<br/>        夏妍妍见陆寒霆一面都难，更别说有这种独处的机会了，所以现在跟陆寒霆在一个房间里她迅速心跳加速，“陆总，我有个问题想问你，你真的跟绾绾分手了么？”
<br/>
<br/>        里面的陆寒霆看着夏夕绾，只见夏夕绾用力的点头，那意思是快说，快说我们已经分手了。
<br/>
<br/>        陆寒霆默了一下，然后用手指指了指自己的俊颊，让她亲。
<br/>
<br/>        夏夕绾现在有求于人，所以迅速踮起脚尖往陆寒霆的脸上亲了一口。
<br/>
<br/>        陆寒霆这才“恩”了一声。
<br/>
<br/>        听到他肯定的回答，夏妍妍心花怒放，她迅速道，“陆总，其实我有些话一直想跟你说，你跟绾绾不合适。”
<br/>
<br/>        “绾绾从小就有男生缘，哪怕她已经跟苏希订了婚，但还是有很多男孩子喜欢她，她很会在男人之间玩暧昧，这些年在乡下她也没有好好学习，没什么学历，我爸妈对她疏忽了管教，她就想从男人这里入手，她跟你在一起完全是贪图你的荣华富贵，她不是真心喜欢你的。”
<br/>
<br/>        “陆总，而我从见你的第一面起，我就对你一见钟情了，我深深的喜欢上了你。”
<br/>
<br/>        “但是我知道我们之间的差距，我还不够好，配不上你，所以这些年我一直追随着你的脚步，努力的让自己变得优秀，我也在洁身自好，从没有交往过一个男朋友，我的心里装的都是你，我渴望有一天可以站在你的身边跟你匹敌，跟你共同面临风雨。”
<br/>
<br/>        “前段时间你跟绾绾在一起了，我只能将自己的爱意深深掩藏，不过现在你跟绾绾分手，我就不想再压抑自己的感情了，陆总，我要向你表白，我喜欢你，你可以给我一个机会么？”
<br/>
<br/>        夏夕绾真的没有想到自己会来见证夏妍妍对陆寒霆的表白，这段表白从头到尾都在踩着她上位，她都快被夏妍妍的绿茶给感动到了。
<br/>
<br/>        夏夕绾仰着黑漉的眸子看着陆寒霆，还挑了一下精致的柳叶眉，想看看他怎么回答。
<br/>
<br/>        但是陆寒霆什么都没有说，甚至也不知道他有没有在听夏妍妍的这段表白，他只是看着她，深邃的眸底满满都是她的倒影。
<br/>
<br/>        “陆总，你在听么？”
<br/>
<br/>        陆寒霆一直不吭声，外面的夏妍妍就有些怀疑了。
<br/>
<br/>        夏夕绾迅速捏拳锤了他一下你哑巴了？
<br/>
<br/>        陆寒霆掀了掀薄唇，无声道，“叫老公。”
<br/>
<br/>        “…”
<br/>
<br/>        他是在趁火打劫？
<br/>
<br/>        外面的夏妍妍更急了，“陆总，你怎么了，你在里面么，我现在进去了哦。”
<br/>
<br/>        夏妍妍抬脚就往里面走。
<br/>
<br/>        夏夕绾的心一下子就紧绷了起来，她紧张的看着陆寒霆喂！
<br/>
<br/>        陆寒霆可一点都不急，只重复了一句，“叫老公。”
<br/>
<br/>        “陆总。”这时夏妍妍真的走近了。
