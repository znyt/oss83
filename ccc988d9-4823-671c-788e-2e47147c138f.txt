        “我只希望你回到我和尚轩的身边，我只希望你能够尽一个做丈夫的责任，我有错吗？”季思妍忍不住嗷嚎大哭起来，“我知道你恨我拆散了你和童筱颖，可你有没有想过，如果童筱颖真心爱你，就算我再怎么做，都拆散不了你们，不是吗？明明是你们俩爱得不够深，为什么你要把所有的过错都推到我的头上来？”
<br/>
<br/>        “你把苏漫雪放了，我跟你去把结婚证领了。”傅昼景凝眉道。
<br/>
<br/>        季思妍却冷哼：“她怀了你的孩子，你让我把她给放了？以后，留着她和你的私生子，来跟我家尚轩争父爱争继承权吗？”
<br/>
<br/>        “苏漫雪是怀过我的孩子，但是被我亲手打掉了！”傅昼景冷冷地呵斥道。
<br/>
<br/>        季思妍身心一怔。
<br/>
<br/>        傅昼景深吸了口气，冷静地接着说道：“那都是半年前的事情了。”
<br/>
<br/>        “我不信！是她苏漫雪亲口跟我说的，她还从我这里拿了一百万，说什么会带着孩子从我面前消失！”季思妍愤愤不平地指向一旁被五花大绑坐在椅子上的童筱颖。
<br/>
<br/>        傅昼景的目光，也顺着季思妍的指向，落在了童筱颖的身上。
<br/>
<br/>        第一眼，他以为她是苏漫雪，所以目光并未在童筱颖的身上多做停留，而是耐着性子，安慰季思妍：“阿妍，我已经半年没碰过苏漫雪了。你相信我！就算她怀了孩子，也绝对不是我的。”
<br/>
<br/>        “真的吗？”季思妍慢慢地安静了下来。
<br/>
<br/>        傅昼景微微点了点头。
<br/>
<br/>        “我扶你回房间去休息，好吗？”傅昼景接着说道。
<br/>
<br/>        季思妍的神情仍旧有些恍惚，但眼神一直透着杀意，恶狠狠地瞪着童筱颖。
<br/>
<br/>        童筱颖双手搓了许久，甚至把手腕上的皮肤擦破了，终于挣脱掉了手腕上的麻绳，甩了甩手腕后才用力撕掉自己嘴上的胶布。
<br/>
<br/>        一阵疼痛，撕扯得她挤出了眼泪。
<br/>
<br/>        童筱颖随之替自己的双脚解绑。
<br/>
<br/>        “我不是苏漫雪！”
<br/>
<br/>        当她走到傅昼景的面前，喘着气大声说这句话的时候，傅昼景立马就信了。
<br/>
<br/>        季思妍顿时也难以置信地看着童筱颖。
<br/>
<br/>        “我现在得去海边，你把车钥匙借给我，好不好？”童筱颖看着傅昼景拜托道。
<br/>
<br/>        傅昼景和季思妍面面相觑后，才想起今天是大哥和筱颍的婚礼。
<br/>
<br/>        “筱颍，你怎么会……”傅昼景欲言又止，顿时明白了苏漫雪的用意。
<br/>
<br/>        看着身边几乎为自己发狂的季思妍，又看着面前这个已经不再爱自己的筱颍，傅昼景苦笑着选择了释怀。
<br/>
<br/>        “我带你去海边，但是你不能吭声。”傅昼景接着说道。
<br/>
<br/>        童筱颖一脸茫然地看着傅昼景：“为什么？”
<br/>
<br/>        “因为，我想看看大哥能不能认出你来。”
<br/>
<br/>        “我不同意！”童筱颖当即反驳。
<br/>
<br/>        傅昼景却得意地笑了笑：“原来，你对大哥这么没信心啊！”
<br/>
<br/>        “……”童筱颖顿时无言以对了。
<br/>
<br/>        其实，并不是她对傅夜沉没信心，而是她太忌讳苏漫雪了。
<br/>
<br/>        她依稀记得，苏漫雪也是用同样的方式，将她困在了临海城的依山别苑里。
<br/>
<br/>        “阿妍也一起去看看吧！毕竟，我们眼前的这位，是我们的真大嫂。”傅昼景搂住季思妍，会心一笑。
<br/>
<br/>        一入傅昼景的怀中，季思妍就变得安静乖巧起来。
<br/>
<br/>        傅昼景说什么便是什么，季思妍怔愣地仰着头，好像十分崇拜地看着傅昼景应了声：“好。”
<br/>
<br/>        童筱颖看着他俩，一脸的无奈。
<br/>
<br/>        为了不被人认出来，季思妍让童筱颖戴上了头巾和墨镜，遮住了整张脸。
<br/>
<br/>        傅昼景拿着请柬，带着季思妍和童筱颖如约来到了海边。
<br/>
<br/>        婚礼还未开始，但是来参加婚礼的人却已经全部到位。
<br/>
<br/>        除了傅家这边来了几个有头有脸的亲戚朋友以外，就连夏海东也在场，包括连中凯他自己把自己的朋友们也带了过来。
<br/>
<br/>        现场人不多，但是都是一些商政大人物。
<br/>
<br/>        傅昼景带着季思妍，季思妍则拉着童筱颖坐在了人群里的最后一排。
<br/>
<br/>        现场类似于教堂婚礼那样，两边是方正的贵宾席，中间是一条通往司仪宣誓台的红毯，还有漂亮的玫瑰花心形拱门。
<br/>
<br/>        “下面用热烈的掌声，有请我们的新娘！”台上的司仪，扯着高调的嗓音呐喊。
<br/>
<br/>        在一阵雷鸣般的掌声里，苏漫雪一袭白纱，挽着连中凯的手臂，缓缓地走向等候在红毯终点的傅夜沉。
<br/>
<br/>        当大家的目光都在新娘苏漫雪的身上时，童筱颖的目光，却一直都在傅夜沉的身上。
<br/>
<br/>        他的新郎装穿身上，不像往日一袭西装那般冷酷严肃。
