　　子弹击中了维尔纳特的右小臂，直接是将他的小臂自手腕上方三寸的位置打断。
<br/>　　握着黄金M92F手枪的右手飞了出去，在地上撒落下一串血珠。
<br/>　　砰！
<br/>　　被子弹打断的右手，因为反射弧的关系，还是扣动了手枪扳机，手枪中那颗弹头用金棒切削而成的子弹被击发，子弹好巧不巧的擦过了绑着林晓阳的绳子，击中了旁边一名“黑箭”队员的大腿。
<br/>　　大腿中弹的“黑箭”队员捂着自己大腿上血流不止的弹孔，靠在墙壁上嚎叫着，“我中弹了！我中弹了！”
<br/>　　只不过还不等旁边的“黑箭”队员上前帮助他，又是一颗竞赛级狙击步枪子弹飞来，直接命中了“黑箭”队员的脑袋。
<br/>　　子弹直接把那名“黑箭”队员的半个脑袋给打飞，血浆喷薄而出，原本雪白的墙壁上也是沾上了星星点点，带着骨头碎渣的血点。
<br/>　　博物馆大厅一时间陷入了混乱之中，林晓阳用力挣脱开自己身上的绳索，一个翻滚来到那个失去半个脑袋的“黑箭”队员身边，抓起他挂在身上的M4A1战术步枪，对着周围其他的“黑箭”队员扫出一个长点射，逼退那些“黑箭”队员后，顺手拔出尸体上的军刀和手枪，握着军刀翻滚到其他B组队员身后，用手枪掩护着射击，拿着军刀割断他们身上的绳索。
<br/>　　博物馆外，布谷对着博物馆内的“黑箭”队员接连点名，任何想要对B组队员造成威胁的“黑箭”队员全都是布谷点名的对象，压制地对方不敢随意露头，掩护着林晓阳救援其他B组队员。
<br/>　　被一枪打断右手的维尔纳特呆呆的在原地站了有好几秒钟时间，任凭断腕不断地向外喷涌着鲜血，直到一名“黑箭”队员把他扑倒在地，维尔纳特才惊醒过来，左手用力掐住右手腕，减缓失血的速度，嘴里发出撕心裂肺的嚎叫声。
<br/>　　把维尔纳特扑倒的“黑箭”队员拖着维尔纳特来到一个角落处，用止血绷带在他的右前臂上收紧，将他的断腕进行包扎。
<br/>　　“黑箭”小队被称为“force-recon中的‘海报六队’”，执行过很多高危险级别的任务从未有过失手，但这次却是遭遇到了滑铁卢。
<br/>　　原本被控制的齐凡、莱昂和孙涛三个人也早就已经摆脱了控制，孙涛抬脚对着身边的一个“黑箭”队员就是一个弹踢。
<br/>　　孙涛的弹踢是有着致命杀伤力的，一脚就是将他身边的“黑箭”队员踢得倒飞出去，接连撞倒、撞碎几个展示柜和展示柜上的玻璃罩，倒在地上，口鼻中不断地涌出鲜血，气息已经是非常微弱。
<br/>　　维尔纳特看着博物馆外狙击步枪子弹不断地飞入，自己小队队员一个接一个的倒下，龇目欲裂，左手握着对讲机大声吼道：“给我炸死那个狙击手！！”
<br/>　　博物馆外，一架挂着两个火箭弹发射巢的“小鸟”直升机快速朝着布谷埋伏的大楼抵近而来。
<br/>　　狙击镜中，距离楼房还有四五百米距离的“小鸟”直升机被放大，透过狙击镜，布谷能看到直升机驾驶舱中两个戴着飞行头盔的飞行员。
<br/>　　布谷把枪口对准直升机驾驶舱左侧那名飞行员，心中完成计算，在直升机快速靠近的过程中，扣动了扳机。
<br/>　　一枚纯铜弹头用超过四倍音速的枪口初速飞出枪膛，旋转着飞向直升机驾驶舱中左侧的飞行员。
<br/>　　啪！
<br/>　　子弹击中了直升机驾驶舱，但是却没有能穿透驾驶舱的防弹玻璃，对驾驶舱中的飞行员造成有效杀伤。
<br/>　　“妈的！”
<br/>　　布谷看着驾驶舱玻璃窗上受到子弹攻击留下的白色印痕，嘴里咒骂一句，连放在窗台边的狙击步枪都没有来得及收起，丢下狙击步枪转身就朝着自己身后的窗户跑去，把自己身上的登山扣扣在早就固定好的绳子上，一跃而下。
<br/>　　咻咻咻咻咻……
<br/>　　轰轰轰轰轰……
<br/>　　“小鸟”直升机上两个火箭弹发射巢同时开火，火光闪烁间，火箭弹飞入楼房，引发剧烈的爆炸，火焰从楼房窗户中翻滚而出，被气浪震碎的玻璃碎片，夹杂在翻滚而出的火焰中向四周迸射以及朝着地面坠落下去。
<br/>　　原本用来挡风遮雨的玻璃，这一刻却成了致命的武器！布谷抓着攀登绳，并没有立刻滑下楼房，而是紧贴着楼房外墙，躲避从自己头顶上坠落下来的玻璃碎片，同时抽出了一直别在后腰的“海盗枪”（截短了枪管和枪托的M79榴弹发射器）。
<br/>　　这架“小鸟直升机”必须要解决掉，不然它能造成的威胁实在太大，会严重威胁到随后的突围撤离。
<br/>　　那架“小鸟”直升机悬停在布谷之前潜伏的楼层前，在火箭弹的洗礼下，整个楼层都被炸得一片狼藉，地上全是被炸碎的地砖碎片和枝叶残破的盆景。
<br/>　　没有看到布谷的尸体，“小鸟”直升机中的两个飞行员驾驶着直升机绕过大楼，想要确认布谷是否已经死亡。
<br/>　　刚绕过大楼，直升机中两个飞行员就看到了抓着攀登绳挂在大楼外墙的布谷，还有他握在手中的“海盗枪”！
<br/>　　“Fuck！”
<br/>　　直升机中的飞行员眼睛中闪过惊恐之色，立刻控制着直升机想要掉头离开；可是直升机离开的速度，又怎么比得上布谷扣动扳机的速度？
<br/>　　嘣！
<br/>　　随着布谷扣动扳机的动作，“海盗枪”中那枚40mm的高爆榴弹在空中划出一道漂亮的抛物线，朝着匆忙掉头离开的“小鸟”直升机坠落下去。
<br/>　　轰！
<br/>　　榴弹击中了“小鸟”直升机的螺旋桨，将一片螺旋桨炸断下来。
<br/>　　被炸断地螺旋桨旋转着朝着布谷所在的那栋大楼飞来。
<br/>　　布谷脸色剧变，把“海盗枪”插在后腰，握着攀登绳的手一松，整个人朝着地面快速滑落下去。
<br/>　　铿！
<br/>　　嘎吱——！
<br/>　　断裂的直升机旋翼一头扎在大楼外墙上，在刺耳的金属疲劳声中，旋翼在外墙上犁出一道足有半年长的切痕，这道切痕距离布谷的索降绳只有不到二十公分的距离。
