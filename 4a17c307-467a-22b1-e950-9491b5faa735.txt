　　“巫尘，你是如何闯过第三层的？”巫穹走到了苏莫身边，居高临下的问道。
<br/>　　他对于巫尘有些了解，据说是杀了巫魔，在黑耀城有同阶无敌的实力，但仅此而已。
<br/>　　盘膝调息的苏莫闻言睁开了眼眸，淡淡的瞥了一眼巫穹，便再次闭上了眼眸。
<br/>　　简直是笑话！
<br/>　　他如何过关的，还需要向别人解说吗？
<br/>　　他根本懒得理会对方，自顾自的继续闭目调息。
<br/>　　“你……！”巫穹见此，顿时心中怒火升腾，巫尘居然如此的无视他，让他感到受到了巨大的侮辱。
<br/>　　不过，此刻正是少司选拔之时，还有大祭司等族中高层在观看，他只能压下了心中的怒气。
<br/>　　“待少司选拔结束，再找你算账！”冷冷的留下一句话，巫穹转身走到了另一边。
<br/>　　巫穹并没有将苏莫放在眼里，首先他的天赋高于苏莫，其次他可是武帝境四重的武者。
<br/>　　当然，天赋高于苏莫，这只是巫穹自己认为的而已。
<br/>　　时间缓缓流逝，片刻之后，巫墨羽、巫罡天等排名在少司候选榜前列的天才，纷纷来到了塔顶，不多时便聚集了二、三十人。
<br/>　　所有登上塔顶的人，见到巫尘之后，都是不免惊讶了一番。
<br/>　　只不过，并没有人再像巫穹那般，霸道的询问苏莫是如何过关的。
<br/>　　依旧不时的有人来到塔顶，对于巫族的这些超级天才而言，排名少司候选榜三十名或四十名，六十名或七十名，其中的差距非常的小。
<br/>　　毕竟，大家比的是天赋，或者是越级战斗的实力，与修为关系不大。
<br/>　　随着时间的推移，塔顶上聚集的人越来越多，巫天御也成功的闯过三层宝塔，来到了塔顶之上。
<br/>　　只不过，巫天御的胸前，有着斑斑血迹，仿佛是受了一点伤。
<br/>　　看到端坐在塔顶的巫尘，巫天御眼眸微眯了下来，心中不禁微微一沉。
<br/>　　巫尘比他先到达塔顶，这说明什么？
<br/>　　说明巫尘的越级战斗力，可能比他还要强大。
<br/>　　呼！
<br/>　　巫天御深吸了口气，比他先到达塔顶也说明不了什么，不战一场，谁更强大尚未可知。
<br/>　　第一关依旧在持续，虽然苏莫等人没用多长时间便过关了，但更多的人却是异常耗时。
<br/>　　足足过去了半个多时辰，依旧偶尔有人登上塔顶，而此时塔顶上已经聚集了一百六十余人。
<br/>　　当然，也依旧不时的有人被淘汰。
<br/>　　塔顶上过关之人，并不是只有巫天御受了伤，很多人都受了或轻或重的伤，没有受伤之人，不足五十人。
<br/>　　足足过去一个时辰之后，第一关终于彻底结束了，三层宝塔内再无一人。
<br/>　　登上塔顶之人，只有一百八十余人。
<br/>　　这时，大祭司从座位上长身而起，目光望着塔顶上的一百八十余人，朗声道：“恭喜你们通过第一关的考验！”
<br/>　　言罢，大祭司一挥袖袍，苏莫等一百八十余人，便被一股无形之力托住，缓缓飞出了塔顶，落在了广场上。
<br/>　　而后，三层宝塔急速缩小，化为一道流光落入了大祭司的掌中。
<br/>　　“接下来就是第二关的考验！”
<br/>　　说话间，大祭司再次一挥袖袍，广场的上空突兀的出现了一座暗红色的山峰。
<br/>　　山峰不大，高约千丈，其上拥有密密麻麻大量的山洞，通向山峰的内部。
<br/>　　此山之上，散发着惊人的血煞之气，仿佛是有无数精血凝练而成，霎是恐怖。。
<br/>　　山峰乍一出现，便将方圆万里的天穹染成了血红之色，非常的惊人。
<br/>　　“此山名为血煞山，山峰内部蕴含浓郁至极的血煞之气，你们这一关的任务便是吸纳血煞之气！”大祭司徐徐说道。
<br/>　　众人闻言有些疑惑，吸收血煞之气？这如何考验？
<br/>　　就在众人疑惑之际，只见大祭司手掌一翻，掌中出现了大量绿豆大小的暗红色种子。
<br/>　　唰唰唰！
<br/>　　下一刻，大祭司手臂一挥，这些种子便飞到了苏莫等人的身前，每人身前一枚种子。
<br/>　　众人心有疑惑，但还是纷纷抓起了身前的种子。
<br/>　　苏莫亦是将种子抓在了手中，随意打量了一番，这种子虽小，其内却是蕴含磅礴的生机。
<br/>　　“这是血阳花的种子，你们只需将吸纳的血煞之气注入种子之内，血阳花的种子便会盛开，血煞之气越多，血阳花便会越大，一个时辰的时间，血阳花超越一尺大小，便算通过考核，当然，血阳花越大便代表考核成绩越好。”大祭司将这一关的规则，简单的告诉了众人。
<br/>　　众人闻言恍然，这一关看似简单，实则是考验众人吸收灵气的天赋，或者说是战魂的天赋。
<br/>　　因为血煞之气，也是一种能量体，属于灵气的范畴。
<br/>　　吸纳的血煞之气越多，血阳花才能开的越大，这是一种天赋的比试。
<br/>　　苏莫闻言先是一怔，随即心中有些惊讶，这巫族的考验，还真是奇特，这种考验之法居然都能想的出来。
<br/>　　只不过，他心中有些无语了，吸纳血煞之气？吸纳灵气的天赋？
<br/>　　别说在巫族之中了，就算在整个古灵星河、甚至是浩瀚无垠的天荒星域，有谁能比的上他？

<br/>　　“大祭司，我等修为不同，如何能保证公平！”
<br/>　　这时，通过第一关考验的巫族天才之中，一名武帝境一重的青年，向大祭司恭敬抱拳，沉声问道。
<br/>　　这是一个关键的问题，战魂吸纳灵气的速度，也与修为有关。
<br/>　　战魂受到修为的滋养，修为越高的人，战魂之力自然约是强大。

<br/>　　一百八十余名巫族天才，闻言纷纷身形一动，向血煞山之上飞去，而后各自钻进了山上的一座座山洞之中。
<br/>　　苏莫亦是毫不犹豫，飞上了血煞山，寻了一处山洞钻了进去。
<br/>　　巫天御在苏莫之后，望着苏莫钻进山洞，他冷笑一声，这一次他一定会得到好成绩，足以碾压巫尘的好成绩。
<br/>　　他的战魂可是飓风，吸纳之力非常的恐怖，吸纳血煞之气的速度，绝对是超强的存在。
<br/>　　随即，巫天御身形一闪，钻进了旁边一座毗邻苏莫的山洞之中。
