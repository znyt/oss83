　　所有人都震惊了，这剑威之凌厉、磅礴，简直无与伦比，完全不是中阶武帝所能展现的地步。
<br/>　　这种程度的剑威，就算是一般武帝境七重的剑客，都不太可能做到。
<br/>　　滕金山亦是心中狂震，被苏莫身上的剑威，狠狠的惊吓了一把。
<br/>　　一种不好的预感，陡然在他心中升起。
<br/>　　就在滕金山想要后退之时，苏莫手中之剑，已经斩了下去。
<br/>　　咻！
<br/>　　耀眼的剑气，璀璨夺目，如同分开天地的神光，裹挟惊天剑威，直击滕金山。
<br/>　　剑气如电，瞬间跨越百里距离，临近了滕金山的眼前。
<br/>　　“这……！”滕金山着实吓得不轻，不过距离太近，他想躲避完全来不及了。
<br/>　　电光火石之间，滕金山再次双臂交叉挡在身前，其全身玄力沸腾，金刚之体的血脉力量也催动到了极致。
<br/>　　耀眼的紫色光芒，从他身上冲天而起，滕金山犹如金铁浇筑的身躯，彻底的变成了一块铁疙瘩。
<br/>　　他全身上下，更是布满了浑厚无比的护体玄力，准备硬抗苏莫的剑气。
<br/>　　下一息。
<br/>　　璀璨的剑气，以绝杀之势，狠狠的斩击在了滕金山的身上。
<br/>　　铛！！
<br/>　　先是一声震耳的金铁交鸣之声，随即，众人便看到剑气瞬间撕裂了滕金山的身体。
<br/>　　啊！
<br/>　　一声凄厉的惨叫声响起，鲜血溅起十数丈高，滕金山犹如金钢浇筑的身躯，直接被剑气一劈两半。
<br/>　　所有人都是心中狂震，瞪大了眼眸！
<br/>　　败了！滕金山败了！
<br/>　　号称防御无敌的金刚之体，也挡不住苏莫一剑的锋芒。
<br/>　　嗖！
<br/>　　肉身被劈开，滕金山的黄金巨猿战魂却并没有死，毫不犹豫，立刻转身奔逃。
<br/>　　“逃得了吗？”苏莫见此冷笑一声，立刻催动了吞噬战魂，他可不会放任对方逃走。
<br/>　　吞噬战魂催动，强大的吞噬之力瞬间笼罩了黄金巨猿战魂，让其立刻倒飞而回。
<br/>　　没有了肉身之力的加持，黄金巨猿战魂，对于吞噬之力没有任何的反抗之力。
<br/>　　“可恶啊！”黄金巨猿的口中发出愤怒的咆哮，正是滕金山的声音，他现在惊恐不已，只想赶紧逃走。
<br/>　　“住手！”
<br/>　　眼见着黄金巨猿战魂，将要飞到苏莫的身前，一声暴喝陡然响起。
<br/>　　是尤天韩的声音！
<br/>　　声音中蕴含不容拒绝之意！
<br/>　　不过，苏莫却是没有理会对方，瞬间将黄金巨猿战魂吞噬了。
<br/>　　只见黄金巨猿战魂，直接冲进了苏莫的身体中，而后消失无踪。
<br/>　　随即，苏莫一挥手，便将正在跌落的滕金山的残躯收了起来，对方金刚之体的血脉，他可不会舍弃。
<br/>　　所有人都是神情怔怔，滕金山死了吗？战魂被苏莫吞噬了？
<br/>　　恐怖！
<br/>　　苏莫的战力已经逆天了！
<br/>　　越一重修为，强势破开滕金山的金刚之体，将对方斩杀，这种战斗力真是无法形容了。
<br/>　　“找死！”森然的喝声响起，只见尤天韩面色铁青，眸中杀机爆射。
<br/>　　其手臂一挥，以掌代刀，一刀凌空斩击而出。
<br/>　　霎时之间，一道恐怖无比的刀气，逆斩而上，以无与伦比的速度，向苏莫斩击而去。
<br/>　　这刀气并没有多么浩大，但锋芒绝世，无可匹敌，拥有惊天动地的恐怖刀威。
<br/>　　尤天韩彻底怒急了，已经控制不住自己心中的杀机，便直接出手了。
<br/>　　“不好！”苏莫全身陡然一寒，心中大惊，他没想到尤天韩居然毫无征兆的向他出手了。
<br/>　　眼见刀气突破空间的束缚，瞬息而至，他就欲拼命闪避。
<br/>　　就在此时，蓦地。
<br/>　　轰！轰！轰！！
<br/>　　一瞬间，从十几座岛屿之上，足足有数十道攻击冲天而起。
<br/>　　这些攻击有拳茫、剑气、刀气、掌印等等，每一击都威力巨大，威势滔天，如同怒海狂涛，以更快的速度袭上天穹。
<br/>　　而后，这些攻击准确的击中了尤天韩的刀气，瞬间将刀气瓦解。
<br/>　　轰隆隆！！
<br/>　　一众攻击在空中碰撞，发生了猛烈的爆炸，毁天灭地的气息、伴随毁灭般的冲击波，横扫八方。
<br/>　　方圆数万里的空间，全部毁灭，下方的海水瞬间蒸发成了虚无。
<br/>　　苏莫心中大惊，来不及多想，身形急速暴退，瞬息间便暴退数万里，堪堪躲过了爆炸的余波，但也是有些狼狈。
<br/>　　“尤天韩，你们太史世家派系，如此的不要脸吗？挑战失败就要破坏规矩？”
<br/>　　一名年轻的内府弟子朗声喊道，此人乃是其它派系的人，正是刚才出手的人之一。
<br/>　　“就是，你们也太下作了吧？”
<br/>　　“哈哈哈！挑战失败，就要狗急跳墙了吗？”
<br/>　　“真是丢尽了太史爵的脸啊！”
<br/>　　其它派系的人纷纷开口，个个面带冷笑。
<br/>　　适才出手的人，大多数都是其它派系的人，他们自然不会坐视苏莫被尤天韩杀死。
<br/>　　当然，也有少量的导师人物出手，比如木璃导师就是其中之一。
