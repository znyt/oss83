　　第205章
<br/>　　三人脚步急行，循着问明的道路，奔向周府。
<br/>　　心脏在腔子里砰砰跳着，这一刻，三人心情激动又忐忑。他们既想早一点见到那飞天蛤蟆，却也想去的路可以更长一些。
<br/>　　整整寻找一年，也不是没有得到过线索，可每一次心怀希望，最后却要承受希望破灭的痛苦。
<br/>　　这一次，悲剧会不会再重演。
<br/>　　“军师，我们快些。”秦蛮恨不得将徐神机扛在肩上。
<br/>　　“是是。”徐神机气喘吁吁：“莫要太大动静，别惊走了大当家。”
<br/>　　“现在还不知道是不是大当家呢。”徐灵儿。
<br/>　　“很可能就是，我有种感觉，大当家就在我们身边。”秦蛮跑了很长的路，依旧是面不改色心不跳。
<br/>　　“小声些，小声些。”徐神机紧张道：“前面就是周府，你们是想让所有人都知道是不是。”
<br/>　　走到街口时，三人同时停住脚步，深吸一口气，一时竟不敢向前迈步。
<br/>　　“糖炒栗子诶。”
<br/>　　“烤瓜子，煮瓜子，炒瓜子，瓜子嘞。”
<br/>　　“客官，虎威武馆了解一下，强身健体呐？”
<br/>　　走入街道，三人都愣住了，看着眼前的一幕，大家感觉某种奇怪的气氛包围了自己。
<br/>　　以周府为中心，周围挤满了人，树上房顶都坐着人，搬着马扎，领着老婆孩子的男人。中间穿插着出售各种小食的商贩。
<br/>　　“呃……我们是不是到了奇怪的世界，不如我们回去睡一觉，醒来后，世界就恢复正常了。”徐灵儿。
<br/>　　“喂，让一让啦，这地方是我占着的。”一个满脸络腮胡子的大汉挤过来，指着地面道：“看见没，看见这棵花没，是我开春种下的。”
<br/>　　“这明明是狗尾巴草？”秦蛮。
<br/>　　“狗尾巴草怎么了，狗尾巴草就不能是我种的。”
<br/>　　“这位大哥……”徐神机陪着笑道：“我们初来乍到，不懂规矩，还想请问一下，大家都在干什么？”
<br/>　　“嚯，你连这都不知道，外地人吧。还能干嘛，当然是来看飞天蛤蟆。”
<br/>　　“这么多人，呃，那飞天蛤蟆不是大盗么？”徐神机咧咧嘴。
<br/>　　“大盗怎么了，大盗就不能有粉丝了，我就不喜欢你们这些外地人，一副没文化的样子。”
<br/>　　“这个，我们的确是第一次来长安，其中到底有什么奥妙，还要请教。”
<br/>　　“哼，既然你们诚心诚意请教，那我就勉为其难的告诉你们。近半年来，长安城最出名的有三个人，一个是那飞天蛤蟆，一个是那满庭芳的一枝桃，另外一人就是帝国明玉公主。”
<br/>　　“这么多人在这里，飞天蛤蟆不就被抓到了。”
<br/>　　“嘿，官兵想要抓蛤蟆大侠，下辈子吧。”
<br/>　　正说着，忽然大汉举起手，高声兴奋道：“快看，快看，飞天蛤蟆出来了。”
<br/>　　顺着大汉手指的方向看去，只见从周府飘出一人，明月皎皎，他若踏月而行，身上一袭白袍，只是脸上带着一张戏子的面具，看不清本来模样。
<br/>　　“看见没，看见没，这就是蛤蟆大侠，天下夜行人都是穿黑，只有他敢穿白，这就是艺高人胆大。”
<br/>　　徐神机惊愕的发现，这满脸胡子的糙汉子，眼神恍惚冒出粉色小星星。太多人发出一声声狂呼，多少少妇挥舞着双手。
<br/>　　“蛤蟆大侠，蛤蟆大侠看这里！”
<br/>　　“喔，看见没有，蛤蟆大侠看我了。”
<br/>　　“混蛋，他明明是在看我。”
<br/>　　“我家小姐赏珍珠一串。”
<br/>　　有人高声将手中一物丢了出去。
<br/>　　“谢谢秦小姐的打赏。”
<br/>　　那人在房顶上翻了个跟头，如此换来更大的呼声。
<br/>　　徐灵儿呆呆道：“我们还是回去睡觉吧，也许醒来后，一切就恢复正常了。”
<br/>　　“看他的身高倒是和大当家有六七分相似。”秦蛮默默道，心中想得却是：要以这幅无耻的样子看，相似度简直有八九分。
<br/>　　“追！”
<br/>　　三人追了出去，此时官兵也到，开始驱散街道上的人群。
<br/>　　秦蛮只见那人蹿房越脊，如履平地，而他并没有这样的轻身本事，只能在街道上狂奔。而徐神机徐灵儿两个，就更不用提了。
<br/>　　“军师，我先行一步。”
<br/>　　“快，快……别让他跑了。”
<br/>　　徐神机气喘吁吁，连话都说不上来。
<br/>　　秦蛮如同一只猎豹般窜了出去，黑夜里快成一道闪电。他心中也明白，今日若让他逃了，再想找到对方就不容易了。
<br/>　　可对方的速度实在太快，因在屋顶狂奔，更是远远甩下秦蛮一大截。
<br/>　　眼看对方已经要逃脱，秦蛮抓起一块瓦片，用自己最大力气丢了出去。
<br/>　　扑通！
<br/>　　瓦片砸在对方小腿上，对方从空中跌落下来。秦蛮大步走过来，拍拍胸口道：“大当家，你说你见到我们跑什么。”
<br/>　　说着，秦蛮上去揭开了对方脸上的面具，待看清对方模样时，表情忽然愣住。
<br/>　　徐灵儿和徐神机父女相携着走过来，徐神机边走边道：“怎么样，是他么，是他么。”
<br/>　　“你们自己看吧。”秦蛮。
<br/>　　看到秦蛮的样子，徐神机已猜出个大概，但还不死心，走过去看了一眼。
<br/>　　只见对方面目猥琐，嘴上两撇胡须，生得手长脚长，像只大猴子。
<br/>　　“说，你为什么要冒充飞天蛤蟆！”徐灵儿刀抵着对方喉咙。
<br/>　　“女侠冷静，女侠冷静。”飞天蛤蟆无奈道：“什么叫冒充，我本来就是啊。”
<br/>　　徐神机将徐灵儿的手挪开，轻轻拍拍她的手背。
<br/>　　秦蛮站在一旁，猛力用拳捶打身边的大树。
<br/>　　徐灵儿歪在徐神机肩膀，轻轻啜泣。
<br/>　　“一年了，整整一年，他或许已经不再人世了。”
<br/>　　徐神机默默叹气，拍着徐灵儿的背心。
<br/>　　没有希望不可怕，可见到希望后，再看着它破灭，方才难以忍受。
<br/>　　飞天蛤蟆歪在地上，侧头看着这一幕：“大哥，你们是干嘛的，咱们素昧平生，你们不用这么卖力抓我吧，是要拿我去换赏银。”
<br/>　　“杀了他！”徐灵儿猛地拔出刀。
<br/>　　“女侠息怒，女侠息怒。”飞天蛤蟆忙道：“你们是不是要找人，我能帮你们，只要你们不把我送给官府。”
<br/>(本章完)
