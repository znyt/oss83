　　三层宝塔之外，依旧不时的有人被淘汰而出。
<br/>　　有的是在第三层被淘汰，而大多数人则是在第二层被淘汰。
<br/>　　因为每个人的实力不同，闯塔的速度自然也不同。
<br/>　　被淘汰之人，不乏在巫族中有名的厉害天才。
<br/>　　看着一名名天才被淘汰，众人感慨连连，这一次的少司选拔，第一关就有如此难度，那接下来的两关将会有多困难！
<br/>　　就在此刻，本来只有巫人王一人的塔顶，突然走出了一人。
<br/>　　此人身穿紫衣，面色冷峻，正是巫玛。
<br/>　　而巫玛刚刚出现在塔顶，巫幽兰的身影随之出现，两人基本上算是同时过关，来到了塔顶之上
<br/>　　“巫玛和巫幽兰过关了！”
<br/>　　“果然不错我所料，他们两人最先过关！”
<br/>　　“真是厉害，以两人过关的速度，便能知道两人的战斗力，远超其他人！”
<br/>　　众人议论纷纷，既羡慕又惊叹，不过，这并没有出乎众人的预料，两人是天赋仅次于巫人王的存在，能如此快速的过关，再正常不过了。
<br/>　　而这两人不但天赋奇高，还是一对恋人，当真是绝配。
<br/>　　就连看台上的大祭司、四祭司，以及一众核心长老，均是暗暗点头，对于巫玛和巫幽兰的天赋，暗赞不已。
<br/>　　这时，塔顶上闭目盘膝的巫人王，缓缓的睁开了眼眸，微微瞥了巫魔和巫幽兰一眼。
<br/>　　“你们两人总算没有让我失望！”巫人王淡淡的说道。
<br/>　　“巫人王，这一次的少司之位，你不会轻易得到！”巫玛和巫幽兰走到了巫人王身边，巫玛沉声说道。
<br/>　　“就凭你们两人？”巫人王淡淡一笑，眸中露出一丝戏虐的笑容。
<br/>　　“不错，就凭我们两人！”巫玛轻轻的点了点头，随即望了身边的巫幽兰一眼，两人相视一笑。
<br/>　　“看来你们有些底气！”
<br/>　　巫人王冷笑一声，摇了摇头道：“希望你们两人的底气，能为这次选拔增加一丝乐趣，不然就太过无聊了！”
<br/>　　巫玛和巫幽兰两人眼眸微眯，巫人王永远都是那么的高傲，不可一世，这一次他们就算无法最终获胜，也要杀一杀对方的锐气。
<br/>　　下方广场上。
<br/>　　所有人都望着塔顶之上，暗暗期待有哪些人能闯过三层宝塔，到达塔顶之上。
<br/>　　“我估计下一个登上塔顶的是巫穹，他排在少司候选榜第四，天赋和潜力仅次于巫玛和巫幽兰！”巫长远和身边的几名好友说道。
<br/>　　巫长远之前也进入了塔中，只不过第一层都没有闯过去，便被淘汰出局，现在只能作为一个旁观者了。
<br/>　　“可能性很大，不过也有可能是巫墨羽或巫罡天，他们两人分别排在第五和第六。”
<br/>　　“嗯，估计就是他们三人中的一个了！”其他人猜测道。
<br/>　　除了巫长远等人，其他众多巫族天才，均是在猜测，接下来谁能率先登上塔顶。
<br/>　　就在众人暗暗猜测之间，一名瘦小的身影，出现在了三层宝塔的塔顶之上。
<br/>　　看到这具瘦小的身影，所有人均是一怔，随即面现诧异之色。
<br/>　　巫尘！
<br/>　　这仅随巫玛和巫幽兰踏上塔顶的人，不是众人猜测的巫穹，也不是巫墨羽和巫罡天，而是在少司候选榜排在七十多名之后的巫尘。
<br/>　　这怎么可能？
<br/>　　所有人都是惊讶不已，巫尘怎么会这么快闯过三层宝塔？难道战斗力比其他所有超级天才都要强大？
<br/>　　巫长远更是双眼一瞪，面现不可思议之色。
<br/>　　他和巫尘有些交情，上次在地灵塔内还聊了很久，当时他完全没将巫尘放在眼中。
<br/>　　可是接下来的时间里，巫尘强势崛起，在黑耀城打败了众多巫族天才，更是杀死了巫魔，震惊了所有人。
<br/>　　巫长远早已震惊不已，但即便如此，他也完全没有想到，巫尘居然有这么强的战斗力，如此快的速度就登上了塔顶。
<br/>　　“天啊！怎么会是巫尘？”
<br/>　　“巫尘不是刚刚崛起的天才吗？有这么强大？”
<br/>　　“他怎么闯过第三层的，难道有特殊的手段？”
<br/>　　整个广场上一片哗然，惊呼声、议论声四起，这个情况出乎了所有人的预料。
<br/>　　就连四祭司和一众核心长老，对于巫尘能如此登上塔顶，都很是意外。
<br/>　　在以前，巫尘从来没有进入过他们的视野，只是族中无名之辈，就在数月之前，其杀死了巫魔之后，才进入他们的关注之中。
<br/>　　只有大祭司暗暗点头，他能察觉到塔内的情况，自然也能知道巫尘是如何过关的!
<br/>　　苏莫通过楼梯，来到塔顶之后，随即扫视了一眼，见塔顶只有巫人王、巫玛和巫幽兰三人，也是微微有些意外。
<br/>　　他已经稍稍放慢了速度，本以为起码有几十人已经先他一步，来到塔顶了。
<br/>　　没想到塔顶居然只有三人，他是第四个来到塔顶的人。
<br/>　　感受到塔下无数道目光的注视，苏莫面不改色，走到塔顶中央盘坐了下来，静静等待第一关的结束。
<br/>　　这时，苏莫感受到了三道凌厉的目光，仿佛要将他穿透一般。
<br/>　　不用想他也知道，这是巫人王、巫玛和巫幽兰的目光，在查探他的虚实。
<br/>　　巫人王三人，也是有些意外，他们三人从未听闻过巫尘之名，对于巫尘更是半点都不了解。
<br/>　　不过，仔细打量了巫尘一番，三人便收回了目光，并未太过在意。
<br/>　　这一次的少司选拔，他们三人才是主角，才有成为少司的资质。
<br/>　　而在巫人王的眼中，此次少司选拔，也就巫玛和巫幽兰有一丝与他竞争的资格，至于其他人，狗屁都不是。
<br/>　　苏莫刚刚来到塔顶不久，仅仅是一个呼吸的时间，少司候选榜排名第四的巫穹，便也来到了塔顶之上。
<br/>　　巫穹个子极高，足有近两米，但瘦弱无比，整个人就像一根竹竿一般，仿佛风一吹就会倒下。
<br/>　　刚刚来到塔顶的巫穹，面上带着自信的微笑，但是当他看到塔顶的苏莫之时，顿时脸色一沉，眸中闪过一丝阴霾之色。
<br/>　　巫玛和巫幽兰能超过他，率先来到塔顶，这没有出乎他的预料。
<br/>　　但这个小小的巫尘，怎么会比他还先到达塔顶？将他都压了下去！
