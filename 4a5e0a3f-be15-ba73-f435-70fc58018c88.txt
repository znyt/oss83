　　“什么？你杀了血剑魔尊？”
<br/>　　古家家主听到古锦的话，有些诧异的看向了楚风眠。
<br/>　　楚风眠的境界，虽然是生死境巅峰，比起血剑魔尊，还要高上一筹。
<br/>　　可是血剑魔尊，却是极为难缠的人物，生死境巅峰的武者，死在他手中的可不止一位了，甚至是一些超越生死的老祖出手，都被他逃走。
<br/>　　这样难缠的一尊魔头，如今就这样的死在了楚风眠的手中？
<br/>　　古家家主目光看向楚风眠的一刻，便是一道灵识，悄然之间的扫视这楚风眠的身躯，似乎是想要看出楚风眠的来历来。
<br/>　　一般的生死境巅峰的武者，肯定是远远无法瞒过这古家家主的灵识，不过楚风眠是何等人物，别说是生死台阶第六重，炼心之境的武者。
<br/>　　就算是那已经接近生死台阶第九重不朽之境的神秘院长，都无法看穿楚风眠的境界，这古家家主自然远远看不透。
<br/>　　“嗯？”
<br/>　　古家家主的灵识扫过，却是几乎看不破楚风眠的一丝一毫，他的眼神中都微微露出了几分震惊。
<br/>　　一名生死境武者，能够瞒过他的灵识的，可是少见，这种实力，深不可测，远远并非像是寻常的生死境巅峰武者，也怪不得楚风眠居然可以杀了血剑魔尊。
<br/>　　古家家主看着楚风眠开口道。
<br/>　　“很不错，年轻人，在中域之中，像你这样的年轻人真的不多了，你杀了血剑魔尊，为我中州除了一害，也救了小女，对我古家有恩，说吧，你要什么好处。”
<br/>　　“父亲，楚前辈这次前来宁州古城，是想要购买一些灵药，这些就让小女去做吧。”
<br/>　　古锦一把接过话来，看向楚风眠开口道。
<br/>　　“楚前辈还请在这宁州古城中小住几日，好好休息休息。”
<br/>　　“好。”
<br/>　　楚风眠点了点头道。
<br/>　　九域天才战，还有一段时间，现在楚风眠也不着急。
<br/>　　楚风眠所要的圣药，也并非是一朝一夕可以寻找得到的，住上几日也无妨。
<br/>　　“楚前辈请。”
<br/>　　古锦带着楚风眠一路来到了一处客栈前。
<br/>　　“二小姐回来了，这次二小姐离开去除魔，我等也是极为担心了，好几次出去寻找。”
<br/>　　突然就从不远处，几道身影突然走了过来，一共是四名年轻男子，都是一身华服，显现出他们尊贵的身份，每个人的实力都不弱。
<br/>　　楚风眠一眼望过去，便是看的出来，这四个人的实力，至少都是要达到了生死境巅峰的范畴。
<br/>　　以如此年纪达到生死境巅峰，就算是在任何势力之中，都足以称得上是小有名气的一方天才了。
<br/>　　其中为首的年轻男子，长相似乎是跟这古锦有着几分相似，也像是古家之人，轻笑这开口道。
<br/>　　“上次二小姐离去，我等没有劝住，也算是我等的责任了，幸好二小姐居然是全身而退归来了。”
<br/>　　“古权，你来做什么？”
<br/>　　古锦看到这一群人的到来，脸色也是微微变化，没有任何的惊喜，而是她的眼神之中，都露出了几分恼火。
<br/>　　“当然是要来迎接二小姐了，而且据说这次是有一位阁下，救了二小姐，我等也前来，看看到底是谁。”
<br/>　　站在那古权背后的一尊武者，冰冷的开口道。
<br/>　　“我等身为宁州古城的监察使，自然要看看，到底是什么人救了二小姐，是不是外面来的奸细。”
<br/>　　这武者一边开口，一边一双眼睛，已经是死死的盯向了楚风眠，似乎是要想要看穿楚风眠的来历，这目光毫不掩饰，充满着敌意，好像是将楚风眠当做奸细一样的来对待。
<br/>　　“这位阁下陌生的很啊，不知道阁下是何来历，来到这宁州古城，是有何事？”
<br/>　　武者看着楚风眠，如同命令一般的开口道，似乎是在审问这楚风眠一样。
<br/>　　“监察使？”
<br/>　　楚风眠目光闪烁，对这个称呼，楚风眠可不陌生。
<br/>　　中域，是以七大宗门为尊之地，在中域之中的各个城池，各个势力之中，都有着七大宗门的弟子，来担当监察使，来监督这些势力有没有敢去反抗七大宗门的心思。
<br/>　　宁州古城，虽然位于中域边缘，可依然没有逃过七大宗门的监察之内。
<br/>　　眼前这武者，一身的气息，楚风眠一眼便是足以认得出，正是一名荒古门弟子。
<br/>　　而在他背后的另外两名监察使，则分别是天剑宗跟古月峡谷的弟子。
<br/>　　这三名武者，便是这三大宗门排到宁州古城来的眼线，来监督宁州古城的一举一动。
<br/>　　不过显然，他们如今跟这古家的弟子纠缠在一起，想要的可不单单是监督职责，而且另有心思，这才来处处针对楚风眠。
<br/>　　“荒铁血，你不要太过分了！这里是宁州古城！楚前辈是我的救命之恩，你还想要盘问他吗？”
<br/>　　楚风眠思考之际，古锦向着那名为荒铁血的检查使，猛然开口道。
<br/>　　“这件事，不用你们来管。”
<br/>　　“二小姐此言差矣，我等身为监察使，有来历不明的武者前来宁州古城，我等都必须要盘问一番才行，这是我们的分内之事。”
<br/>　　那荒古门弟子，荒铁血傲然道。
<br/>　　“更何况最近中域发生如此之多的变化，九魔城中叛变了如此之多的弟子，成为了魔头，这些魔头不少人都混到了中州来，对待这种来历不明的武者，必须要好好盘问盘问才行！”
<br/>　　“不错，小子，说，你到底是什么人？是不是九魔城的魔头？”
<br/>　　那荒铁血身边，天剑宗的监察使，也是看向楚风眠，冰冷的开口道。
<br/>　　“看此子遮遮掩掩的，他多半就是九魔城的摸头！”
<br/>　　“再不说的话，就别怪我们不客气了！先擒下此子，在好好逼问在说！”
<br/>　　古月峡谷的监察使，也是冰冷的开口道。
<br/>　　一股股的灵力，已经是爆发了出来，全部锁定了楚风眠，几乎是在瞬息之间，他们便是足以出手，来攻击楚风眠。
<br/>　　“古权，你不要太过分了，楚前辈可是我们古家的客人！”
<br/>　　古锦看着那站在最前方的古权，冷声道。
<br/>　　她自然知晓，这一切到底是因谁而起。
