　　第384章
<br/>　　程大雷撇撇嘴，见到他们便该猜到是出事了。不然，这些人何苦千里奔波，来这西北边关。
<br/>　　“不着急，你们先洗个澡，换过衣服。看你们的样子，也知道好久没吃饭了。等收拾好了，去我那里吃饭。”
<br/>　　徐神机领着他们下去，互相也都认识，过程中，一路笑骂，另外徐神机让人把琴川关的军服送过来一百余套，等这些人收拾干净后给他们换上。
<br/>　　当初徐神机去长安，也受过这些人的照料，如今大家算是来了徐神机的地盘，徐神机正好投桃报李。
<br/>　　于此同时，程大雷已经令山寨的厨子开始造饭，煎炒烹炸，在将军府摆开宴席。
<br/>　　等徐神机带着人到的时候，只见十几张八仙桌在庭院里排开，饭菜飘香。一众人都换上了蛤蟆寨的黑色制服，人人背后纹着一只蛤蟆，看上去倒是精神许多。
<br/>　　程大雷猜得不错，这些人一路过来，受够了奔波之苦。既要防着官兵，也要担心绿林匪贼抢劫，仔细想来，也不知道多久没好好吃顿饭。
<br/>　　终于赶到程大雷这里，眼见程大雷混得还算不错，大家像回到老家一样，一颗心也落了踏实。
<br/>　　刚刚举杯，众人就开始狼吞虎咽，从河里捞来的鲜鱼，酱鸭酱鸡，草原上打来的野物……酒过三巡，菜过无味，程大雷悄悄带着阿喜、阿苦、飞天蛤蟆离席，到了将军府的小厅。
<br/>　　“说吧，长安城出了什么事？”程大雷在椅子上坐下，手搭在一起。
<br/>　　阿喜、阿苦、白元飞三人噗通一声跪道，阿喜郑重道：“老大，对不起，我把你打下来的家业丢了。”
<br/>　　程大雷撇撇嘴：“快起来吧，别搞这一套了，告诉我究竟发生了什么事。”
<br/>　　阿喜三人这才站起来，同时板着一张脸，却没人开口回答程大雷的问题。
<br/>　　“好啦，打算抻到什么时候？”
<br/>　　阿喜叹了口气，近前道了一句：“老大，长安可能保不住了。”
<br/>　　“呃……什么情况？”程大雷一怔，最近研究着如何推行纸币，对长安那边的事情的确没怎么关心。
<br/>　　“正义教从幽州起事，一路势如破竹，直扑长安城。现在正义教的十万大军已经攻到长安城下，城里的居民正想法设法逃出来，人心惶惶，长安城很可能会被攻破。”
<br/>　　“呃……”
<br/>　　程大雷愣了很大一会儿，最近不仅没关注长安方面的情报，对于幽州正义教也有些疏忽。本以为林少羽站稳了幽州的形势，会选择休养生息，就和自己现在一样，闷头发大财。却是没有想到，林少羽竟然选择了这一步。
<br/>　　想想平日林少羽的为人，却也不是太难想到。
<br/>　　“凭十万人就能打下长安城？”程大雷皱起眉头：“正义教想得太简单了吧。长安城的城防军、禁卫军、鱼龙卫加起来怎么说也有三十万吧。十万乌合之众，能敌得过三十万精兵。”
<br/>　　“根据眼前的形势推测，并不是没有可能。”阿喜。
<br/>　　“怎么说？”
<br/>　　阿喜理了理思路，道：“从正义教在幽州起事开始，大小十几仗，百战百胜，未尝一败。从幽州到京州，沿途的关卡不是没有善战的老将，可是却无一人能挡。现在天下都传说，那正义教的首领是天命之将，上天派他降世，就是为了终结李氏的气运，取而代之。”
<br/>　　程大雷咧咧嘴，林少羽是天命之将，他有几斤几两自己还不清楚，快别扯淡了。
<br/>　　“如今的长安城人心惶惶，大家都在想办法往外逃。明帝已发了勤王令，号召天下诸侯进京勤王，铲除正义教。”
<br/>　　程大雷沉默下来，很久没有说话，关键是这件事给他带来的震撼太大，他也需要点时间理清思路。
<br/>　　林少羽的动作在意料之外，却也在情理之中。天下对李家的怨恨久矣，林少羽打出了正义的旗帜，恰好成了诸人情绪的突破口。得民心者得天下，虽然在兵力对比上，林少羽不占优势，可他未必不能取得胜利。
<br/>　　只看天命是眷顾明帝，还是眷顾林少羽。
<br/>　　林少羽向明帝下了讨伐信，信中历数明帝的十宗罪，包括：枉杀忠臣，鱼肉百姓，投降戎族。主要就是这三条，其他几条例如好色昏庸之类的，大概是为了凑数用的。
<br/>　　用这十宗罪号召大家起兵攻打长安，到时候坐稳了龙庭，封王拜将不在话下。
<br/>　　天下多的是把脑袋拴在裤腰带上干活的匪类，大家刀口舔血，如果能借着这个机会搏个出身，诸人何乐而不为。
<br/>　　同时，明帝也向天下发了勤王令。天下诸侯闻风而动，所以林少羽真正面对的敌人，不仅仅是三十万长安军。
<br/>　　至于接下来的事情如何发展，此刻还看不出来。
<br/>　　程大雷隐隐觉得有些头疼。
<br/>　　“我们就是趁这个时候逃出来的，大家都想着过来投奔老大，路上兄弟们聚在一起，也有许多弟兄没逃出长安城，被堵在了城里。如今过来这边的，就剩下我们这些人了。”阿喜。
<br/>　　“大家还是愿意在老大手下做事，我们也做得开心些。”白元飞道。
<br/>　　说罢之后，阿喜、阿苦和白元飞都偷眼打量着程大雷，观察着他脸上的表情。
<br/>　　大家当年在长安是一场兄弟不假，可毕竟已隔了三年。现在程大雷是琴川之主，要人有人，要粮有粮。相比之下，诸人却是一穷二白，两个膀子顶一颗脑袋来找程大雷吃饭。
<br/>　　心里难免有些没底气，谁知道程大雷会如何招待大家呢。心里会不会有些轻视……人心叵测如水，这都是说不好的。所以三人在面对程大雷时，一直捧着聊，暗暗看他的反应。
<br/>　　程大雷白了三人一眼，道：“你们来了，就留下来，大家一场兄弟，我还能不管你们不成。我已经让人给你们安排好住处，以后具体做什么，看大家的本事。到了这里，你们就谁也不用怕。”
<br/>　　听到这话，三人心里都落了踏实。同时拱手，齐声道：
<br/>　　“愿为老大鞍前马后，死而后已。”
<br/>　　
<br/>(本章完)
