　　巫妖战场中，形势转瞬即变。
<br/>　　十二祖巫聚在一起，摆下十二都天神煞大阵，随着无尽好黑气弥漫，十二祖巫竟然凝聚成一个人。隐隐看去，这是一个万丈高的大汉，竟有几分盘古的气息。
<br/>　　轰隆隆！
<br/>　　都天神煞大阵中升起十二道巨大道则光柱，迅速融入到那大汉身体之中，化为擎天之势，原本有些虚幻的大汉，顿时变得更加凝实了。
<br/>　　“嗡！”
<br/>　　宛若某种天音，一股巨大的气势从巨汉身上散发，弥漫而出，威压洪荒大地，这股气势还在不断的攀升，瞬间便突破准圣限制，进入到混元大罗金仙境界，几乎可与圣人的威势相媲美，绝对非同小可！
<br/>　　“盘古，这怎么可能？”东皇太一惊声大叫，满脸失色。
<br/>　　“什么，这怎么可能？”伏羲也惊叫道。
<br/>　　“什么？”
<br/>　　就连女娲这个圣人一时间也惊呼不已，她怎么也想不到，十二祖巫竟然能够召唤出盘古真身，这怎么可能？
<br/>　　妖族众人不由得为之惊骇，之前常与巫族大战，却没有见过巫族竟有这等底牌，他们从没有想到，十二祖巫竟然能够凝聚成盘古真身，这实在是惊骇到了极点，令人不敢置信。
<br/>　　“大哥，想不到巫族竟有如此大阵，威势堪比圣人，我们是否该动用妖族的底牌了？”
<br/>　　太一立刻来到帝俊身前，面色变得极为难看，声音之中还有掩饰不住的惊骇。
<br/>　　其他妖族也汇聚到帝俊身边，听闻太一此言，不明所以，妖族有何地牌，怎么我们不知道？他们纷纷将目光投向帝俊，看他要如何解决此事。
<br/>　　“哼！”
<br/>　　帝俊冷哼一声，感受到那盘古真身的威势，脸色变得十分难看，不过面上却冷笑道：
<br/>　　“巫族传承于盘古，底蕴果然深厚，竟有这等大阵，但他们又岂知我妖族没有大阵！”
<br/>　　“嗯？”
<br/>　　鲲鹏、伏羲等妖族强者面露疑惑之色，他们真不知妖族有何等厉害大阵，难道还能与巫族的都天神煞大阵抵抗？倒是女娲若有所思，好似早知此事一般。
<br/>　　众人听帝俊有解决之法，顿时大喜，看向帝俊。帝俊冷声道：
<br/>　　“这大阵本来就是为了对付你巫族所创，我本想用不到了，却没想到今天还真要用上了。”
<br/>　　心知对手不好对付，帝俊口中一声怒喝，顿时体内两道光华飞出天灵，正是先天灵宝河图洛书，混沌古朴，祭起虚空，两相交织在一起，结成混元河图阵势。
<br/>　　霎时间，河图之上，光华大涨，一幅幅蕴藏玄机奥妙的图谱，变幻不定。洛书之上，闪现出数十个太古妖文，苍茫古朴，乌光闪烁，配合河图中的幻象，铺开万丈玄光。
<br/>　　高空之上，星空之中，三百六十五道神光倾泻而下，竟是三百六十五道太古星光。刹那之间，混元河图、洛书大阵阵势再变，周遭偌大的虚空之中，妖文变幻，竟演化出一片广阔的星空，亿万星辰闪烁浮现，散发着耀眼摧残的光芒。
<br/>　　然而，这还只是一个开始，但随着那神秘的力量不断浮现，这无尽星空，也开始逐渐扩张开来，化作一方近乎真实的星空世界，向着巫族的盘古真身碾压而来。
<br/>　　“虚空星辰，变幻无穷，
<br/>　　周而复始，无穷无尽，
<br/>　　凝星为阵，演化周天！”
<br/>　　“周天星辰大阵，现！”
<br/>　　随着帝俊一声大叫，挥手之间，周边的那些星辰，一个个的，都绽放出一道道耀眼的光芒，冲霄而起，绵延万丈，划破天地苍穹，星空衍变，生死轮回，似乎皆在其中。瞬息之间，这片星空以肉眼可见的速度成长、扩大，已经不止几千万里大小。
<br/>　　轰隆隆！
<br/>　　惊天巨响之中，一道巨大的光束仿佛凝聚成实质般，瞬息之间便自无边星空深处，猛然喷发而出，刹那之间，直冲九霄，浩荡无比的庞然能量，犹如滔天巨浪一般，狂涌而来，充斥着周边虚空。
<br/>　　以天地寰宇为根基，以日月星辰为阵脚，三百六十五位大罗金仙为挪移主阵，东皇太一手持混元剑，作为杀阵，妖帝帝俊手持河图、洛书操控大阵变化，生出无穷玄妙。
<br/>　　“巫族蛮子们，纵然你们凝聚成盘古真身，也休想与我妖族的周天星辰大阵抗衡！”
<br/>　　帝俊口中冷喝，话音回荡在天地之间，无边的星辰都为之一动，却好似有无穷伟力，震动周天星辰大阵。
<br/>　　“是吗？就凭你们这破阵，也想抵挡盘古神威？哼！兄弟们，让他们知道我们的厉害！”
<br/>　　帝江祖巫一声大喝，满脸不屑，冷哼一声，便操控着盘古真身向着妖族的周天星斗大阵轰去。
<br/>　　黑云翻滚，煞气翻腾，无穷无尽的天地灵气在其中流转，化为遮天蔽日的无穷魔威。
<br/>　　“斧来！”
<br/>　　盘古真身大喝一声，声音传遍洪荒，正在昆仑山观战的元始只觉得自己元神之中的混沌钟与盘古幡竟有想要挣脱自己束缚的冲动。
<br/>　　“哼！既然都被我炼化了，休想再次离去！”
<br/>　　元始冷哼一声，元神一念之间，便将有些震动的混沌钟与盘古幡镇压了。
<br/>　　他继续以神眼观察妖族的周天星辰大阵，自己体内也有三百六十五颗星辰，而且演化出三百六十五方世界，若是能够以这周天星斗大阵演化，各种奥妙显现，自己的战力绝对可以提升至少一倍。
<br/>　　同时，太清宫中的太清老子一样镇压了体内暴动的太极图。
<br/>　　远处的盘古真身不见混沌开天斧汇聚，便直接以天地间无边煞气加持众多法则，凝聚出一柄巨大的开天巨斧。
<br/>　　“叱！”
<br/>　　一声沉喝，惊天动地，赫然见盘古真身举起开天斧影，一击破空，携带着开天辟地之威，直奔妖族周天星辰大阵斩来！
<br/>　　“哗啦！”
<br/>　　终于，周天星辰与都天神煞大阵带着璀璨耀眼的光芒，相互触碰，在这璀璨的星光之中，绽放出一丝耀眼的光辉！
<br/>　　静！
<br/>　　死亡般的寂静！寂静中带着压抑与无边的沉重！
<br/>　　没有想象中的轰鸣爆炸声，妖族那可以堪比圣人的周天星辰大阵，直接被盘古真身一斧划开，大阵仿佛被划开一道口子，不再完美，帝俊、太一以及主持大阵的三百六十五个妖族大罗金仙，纷纷一口逆血喷出，遭受到大阵反噬。
<br/>　　“嘶！”
<br/>　　洪荒天地顿时一阵抽冷气的声音响起。盘古不愧是盘古，就算只是别人凝聚出来的虚影真身，依旧不是一般人能够抵挡的。
<br/>　　所有人集体失声，无论是洪荒各地大神通者，还是女娲、妖族等人，都觉得这不可思议，面容呆滞，久久无语。
