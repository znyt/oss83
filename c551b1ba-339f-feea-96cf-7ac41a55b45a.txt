　　大祭司极为自信，只要苏莫真的去了真圣界，灭杀苏莫没有任何的悬念。
<br/>　　火元圣师闻言，面上终于露出了淡淡的笑容，道：“巫仑大祭司，我们做个约定如何，无论此次真圣界结果如何，苏莫和你巫族的恩怨，都将一笔勾销！”
<br/>　　“哦？”大祭司目光凝视着火元圣师，见对方面上带着若有若无的笑意，仿佛是自信满满，这让他心中惊疑不定。
<br/>　　难道苏莫真有什么超强的底牌？可以无惧任何武帝境九重的超级天才？
<br/>　　大祭司沉默了片刻，虽然有这个可能，但还是不足以让苏莫在武帝境无敌的地步。
<br/>　　修为的巨大差距，是无法弥补的，就算有超强至宝，也发挥不出多强的威力。
<br/>　　“好，我可以答应你，真圣界之行结束以后，我族和苏莫的恩怨一笔勾销！”
<br/>　　少倾之后，大祭司点了点头，他也无需有什么顾忌，就算退一万来说，在真圣界真的没能杀了苏莫，以后再暗中动手便是。
<br/>　　“如此甚好，此事方可圆满！”火元圣师微微颌首，面上露出了轻笑。
<br/>　　苏莫亦是送了口气，今日的危局算是过去了，不过，这次真圣界之行，就危险重重了。
<br/>　　就在此时，大祭司目光如刀锋般落在苏莫的身上，道：“不过，想要达成这个约定，还有一个前提！”
<br/>　　“什么前提？”苏莫开口问道。
<br/>　　“苏莫，老夫后人巫娴，以及族中另外两名天才被你擒获，速速交出来吧！她们若有事，不仅你要死，你们神武学府都要承担严重的后果！”大祭司森然的说道。
<br/>　　“苏莫，可有此事？”杜府主开口问道。
<br/>　　苏莫面色冷然，巫娴三人可是知晓了他不少的秘密，交出三人对他很是不利。
<br/>　　不过，很快他也释然了，他的秘密除了最核心的之外，很多都暴露了，也无需担心了。
<br/>　　唰！
<br/>　　而后，苏莫一挥手，人影闪烁，被他擒拿数年的巫娴三人，出现了大殿之中。
<br/>　　他不得不放人，幸亏他没有将三人杀死，不然今天估计难以善了了！
<br/>　　巫娴三人乍一出现，还有些愣神，不清楚苏莫为何放他们出来，但下一刻，他们就看到了大殿中的大祭司。
<br/>　　“大祭司！”
<br/>　　“大祭司！”
<br/>　　“祖爷爷！”
<br/>　　三声惊呼声响起，声音中冲满了无尽的惊喜与激动。
<br/>　　巫娴三人被苏莫擒拿数年了，现在被放出来，突然看到眼前的大祭司，让三人感觉极不真实，仿佛像是做梦一般。
<br/>　　三人不约而同的揉了揉眼睛，终于确定，这不是在做梦，大祭司真的来了，来救他们了！
<br/>　　“祖爷爷！哇！”巫娴的泪水顿时夺眶而出，她所受的委屈，似乎都要在这一刻发泄出来。
<br/>　　娇生惯养的她，哪里受过如此委屈，如此屈辱！
<br/>　　本来，她都已经绝望了，没想到幸福来得如此之快。
<br/>　　其他两名巫族天才，亦是眼眶泛红，他们以前从未离开天冥星，一直在族中无忧无虑，这次的打击对于他们来说，简直是无法磨灭的伤痛。
<br/>　　“你们没事了！过来吧！”大祭司的脸色亦是稍稍一松，巫娴没事就好，随即他袖袍一挥，将三人卷到了身边。
<br/>　　“苏莫！”
<br/>　　这时，巫娴三人看到了后方的苏莫，顿时恨得咬牙切齿。
<br/>　　“祖爷爷，杀了他，杀了苏莫！”巫娴立刻怒声喊道。
<br/>　　“娴儿放心，他的命还有两个多月！”
<br/>　　大祭司摇了摇头，随即转头看向杜府主喝火元圣师，声音低沉的道：“杜府主、火元圣师，希望两个多月后，苏莫能如约去真圣界，不然我巫族誓不罢休！”
<br/>　　“巫仑大祭司放心，苏莫必定前去！”杜府主朗声说道。
<br/>　　“嗯！告辞了！”大祭司微微点头，随即身形一闪，如同一道幻影一般，裹挟着巫娴三人，离开了大殿，离开了神武学府。
<br/>　　大殿中寂静无声，杜府主瞥了一眼苏莫，又看向火元圣师，轻声问道：“火元导师，苏莫能对抗巫族的众多天才吗？”
<br/>　　事实上，杜府主和大祭司一样，根本不相信苏莫能在真圣界对抗巫族天才。
<br/>　　巫族天才如云，就单单是排进了真圣榜的人，就有数人之多。
<br/>　　而且，好像有一名巫族天才，不仅排在真圣榜前十，修为更是高达武帝境九重，完全不是苏莫所能抗衡的吧！
<br/>　　“不清楚！”火元圣师摇了摇头。
<br/>　　杜府主闻言顿时一怔，他见火元圣师之前平淡的模样，还以为对方对苏莫极有自信。
<br/>　　苏莫也是微微一怔，心中有些郁闷，这火元圣师都对他没有信心，居然还开出此种约定。
<br/>　　当然，苏莫郁闷归郁闷，他心中还是非常感激对方的，若不是对方，今日他凶多吉少。
<br/>　　火元圣师叹了口气，随即转头向苏莫，语重心长的道：“苏莫啊！路老夫为你铺开了，你能不能冲破阻隔，全看你自己的了！”
<br/>　　“前辈放心，武帝境之内，我来者不拒！”苏莫郑重的点了点头，他也没有太过担心。
<br/>　　目前，他正在修炼尊级功法，待属性玄力融合之后，实力的短时间内，将会暴炸式增长。
<br/>　　而且，过去了这么久了，他的修为沉淀已经够了，去真圣界之前，他还能再次提升一重的修为。
<br/>　　武帝境六重的修为，加上尊级中品功法的属性融合，差不多应该够了。
<br/>　　“很好！”
<br/>　　火元圣师微微颌首，心中非常满意，想成为强者，哪能不经过铁血磨炼，绝处逢生！
<br/>　　任何一位大能，都不是仅靠天赋就能崛起的，而是需要一次的冲破阻隔，突破自我。
<br/>　　“苏莫，难得火元导师如此看重你，不要让他老人家失望！”杜府主向苏莫微笑道。
<br/>　　“我不会让火元前辈失望，也不会让府主失望！”苏莫沉声说道。
<br/>　　“嗯！”杜府主颌首，他对于苏莫也是相当看重，如此人杰前途不可限量。
<br/>　　“府主，此事已了，老夫就告退了！”
<br/>　　这时，火元圣师起身，向杜府主抱拳说道。
<br/>　　“嗯，火元导师慢走！”杜府主点头。
<br/>　　而后，火元圣师便带着苏莫，离开了大殿，返回了火元峰。
