　　众人闻听尤天韩之言，顿时惊讶，对方居然派出了林无影。
<br/>　　林无影可是一位特殊的天才，或许战力不如雷灿强大，但却是有特殊的本领，是一位比雷灿还要恐怖的人物。
<br/>　　“嗯！”
<br/>　　这时，便见尤天韩的身后，一名身穿黑袍，身形高瘦、脸庞狭长的青年点了点头。
<br/>　　随即，黑袍青年一步踏出，瞬息千里，眨眼间便来到了海面上，站在了苏莫前方数百里处。
<br/>　　苏莫目光凌厉，打量着前方的林无影，心中有些好奇。
<br/>　　对方已经知道了他的实力，居然还敢上场？
<br/>　　此人难道如此自信，自认为能打败他？
<br/>　　“你，必死无疑！”林无影的目光，仿若是一条毒蛇，盯在苏莫的身上，让他全身皮肤不由得冒出一阵寒意。
<br/>　　“有什么本事都使出来吧！”苏莫淡淡的说道，他有不败的自信。
<br/>　　林无影淡淡的点了点头，随即身影突兀的消失了。
<br/>　　对，就是消失了！
<br/>　　没有任何征兆，完全的消失无踪了！
<br/>　　“嗯？”苏莫见此顿时双眉一挑，眸光精光爆射。
<br/>　　怎么回事？
<br/>　　对方人呢？
<br/>　　这让他立刻收起了轻视之心，顿时释放出自己的灵识，覆盖了方圆数千里。
<br/>　　但是，在他的灵识覆盖的范围内，没有任何对方的踪迹。
<br/>　　仿佛林无影真的消失了！
<br/>　　“有意思！”苏莫嘴角勾起了一丝笑意，看来此人是有特殊的战魂。
<br/>　　这战魂能让对方隐藏身形，不被察觉。
<br/>　　他顿时来了兴趣，相比于得到储物戒等战利品，他更希望得到有特殊能力的战魂。
<br/>　　周围十几座岛屿上，也都彻底安静了下来，仔细观看接下来的战斗，不愿意错过一丝一毫。
<br/>　　学府中的大多数人，都知道林无影的恐怖。
<br/>　　林无影，拥有太古武魂榜排名第八百六十一的‘无影战魂’，非常诡异。
<br/>　　一旦动用战魂，若不是修为超过林无影太多，基本上发现不了对方的存在。
<br/>　　这就恐怖了，简直是偷袭杀人的利器。
<br/>　　据传闻，林无影在加入神武学府之前，是某一个小杀手组织的金牌杀手。
<br/>　　执行了几百次暗杀任务，无一失败，堪称终极杀手。
<br/>　　只要是修为不超过林无影太多的人，就算战力再强，也难逃一死。
<br/>　　后来，林无影修为渐渐强大，杀光了那个杀手组织的所有人，摆脱了杀手组织的控制。
<br/>　　可以说，林无影就是一个行走在黑暗中的毒蛇，一旦被他盯上的人，基本上是死路一条。
<br/>　　现在，所有人都期待了起来，期待战力超凡的苏莫，能不能抗住林无影的袭杀。
<br/>　　时间一分一秒的过去，若不是大海上的浪涛，依旧在不断的涌动，众人都以为时间停止了流动。
<br/>　　没有任何的动静，林无影彻彻底底的消失了，无影无踪。
<br/>　　苏莫身形伫立在海面上，手握长剑，静若磐石，他的眼眸微微眯起，将精神感知力提升到了最强的地步，灵识已经覆盖了方圆万余里。
<br/>　　他知道，林无影必定就隐藏在附近，寻找机会，随时可能给他雷霆一击。
<br/>　　事实上，苏莫丝毫不担心，他的吞噬战魂若是一催动，管你藏在哪里都要被吞噬。
<br/>　　他现在是想要试试，此人这种能力到底有多强大。
<br/>　　时间缓缓流逝，不多时变过去了盏茶的功夫，林无影依旧没有任何动静。
<br/>　　这让不少人都等急了！
<br/>　　有些修为高深之人，或者是内府弟子、学府导师，他们则能发现林无影的踪迹，但却不会说出来。
<br/>　　就在此刻，蓦地，苏莫突然动了，身形猛然一闪，向前暴冲数百丈。
<br/>　　而在他之前立身之处，空间微微的波动了一瞬，一道寒光一闪而过，而后立刻消失无踪。
<br/>　　林无影的身影，依旧没有出现，或者是出现了一瞬间再次的消失了。
<br/>　　苏莫瞥了眼肩膀处，那里的衣裳裂出了一道小口子。
<br/>　　刚才若不是他感知无比明锐，反应也够快，估计已经被一劈两半了！
<br/>　　他的脸色稍稍凝重了三分，这人的确强大，这种能力真是不凡。
<br/>　　若是在平常时候，他猝不及防之时，此人来刺杀他，还真的有很大的可能得手！
<br/>　　这绝对是特殊的战魂！
<br/>　　这战魂，他要了！
<br/>　　嗤！
<br/>　　就在此时，蓦地，苏莫的头顶上空，空间轻微的波动起来，有微不可查的寒光闪过，直击他的难道。
<br/>　　毫不犹豫，苏莫立刻举剑一挡。
<br/>　　铛！
<br/>　　霎时之间，火星四溅，响起剧烈的金铁交鸣之声。
<br/>　　下一息，现场再次恢复了平静，林无影一击即退，再次的隐藏了起来。
<br/>　　众人心中惊讶无比，苏莫居然连续挡住了林无影两次的袭杀，感知力居然如此的敏锐！反应速度如此的快速！
<br/>　　苏莫轻轻的呼了口气，该结涑了！
<br/>　　心念及此，他立刻稍稍催动了吞噬战魂，吞噬之力瞬间席卷八方，笼罩了方圆千里。
<br/>　　嗡嗡~~~
<br/>　　下一刻，苏莫方圆千里内的一切，全部都向他聚拢而去，包括下方无尽的海水。
<br/>　　海水自然进不了苏莫的身，被他体外气势一震，便纷纷蒸发。
<br/>　　林无影的身形再也隐藏不住了，瞬间就暴露了出来。
<br/>　　只见此时，林无影的身上包裹着一股无形的气流，这股气流让他的身形若隐若现。
<br/>　　但此刻，这股气流在吞噬之力下，剧烈的抖动，变得絮乱不堪。
<br/>　　因此，林无影的身形便暴露了出来。
<br/>　　“怎么回事？”
<br/>　　刚刚现身的林无影，口中顿时响起一阵惊呼，他满脸惊慌失措，完全不知道怎么回事！
<br/>　　他的战斗力并没有多强，若是无法隐藏身形的话，根本不是苏莫一招之敌！
<br/>　　感受到强大的吸扯之力，将自己向苏莫的方向拉扯而去，林无影极力控制身形，想要远离苏莫。
<br/>　　但是，这股拉扯之力强大无比，完全将他束缚住了，让他怎么也挣扎不出。
<br/>　　“怎么会这样？”林无影顿时脸色一白，心中充满了慌乱。
<br/>　　“过来吧！”苏莫嘴角勾起一丝笑意，立刻稍稍加大了吞噬之力的强度。
<br/>　　霎时之间，林无影便急速向苏莫飞了过去。
<br/>　　“杀！”
<br/>　　眼见着将要飞到苏莫的身前，林无影挣脱不出，便怒喝一声，手中短刀猛然一挥，向苏莫斩击而去。
