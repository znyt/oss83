
<br/>　　<br/>　　    第二天一早，陈臣是从睡梦中被惊醒的，突然醒来之后人就直挺挺的坐了起来，额头上冒的全都是虚汗。<br/>　　<br/>　　    在被惊醒以前她总感觉身上有东西在压着自己，都快要喘不过气来了，还觉得冥冥之中似乎有人在叫她，一连叫了几次，当她想要答应的时候，却不知道为什么被打断了，一直到现在被惊醒了过来。<br/>　　<br/>　　    陈臣不知道的是，要不是她枕头下面压着那张泡了污秽的符纸，她今天搞不好要昏睡一整天都不一定能醒来呢，就算醒了也是身子发虚得要病上一场。<br/>　　<br/>　　    陈臣浑浑噩噩的洗了个脸刷了牙之后来到楼下，在餐厅吃早餐的陈重看见她后，问道：“小臣过来吃饭了”<br/>　　<br/>　　    “不吃了，没有胃口！”陈臣挥了挥手拎着包就要出门。<br/>　　<br/>　　    陈重看着女儿，放下筷子皱眉问道：“你昨天晚上几点睡的？精神这么差，黑眼圈都出来了？”<br/>　　<br/>　　    陈臣现在的状态非常糟糕，眼眶乌黑精神萎靡，走路的时候就跟耷拉着膀子仿佛负重了几十斤一样。<br/>　　<br/>　　    “没事，可能是没睡好吧？我去公司了，上午实在太困了就补一觉吧”陈臣打了个哈欠，朝着院门外走。<br/>　　<br/>　　    陈重忽然发现，陈臣走路的时候一直都是身子前倾脚尖点地，脚后跟始终都没有落在地上。<br/>　　<br/>　　    “你脚怎么了，伤了嘛？”<br/>　　<br/>　　    “没有啊·····”<br/>　　<br/>　　    陈臣从家里出来后上了车，舔了舔有点干涩开裂的嘴唇，然后把前面的挡板放下来拉开镜子，就只看了一眼，她就被自己的状态给吓了一跳，两个黑眼圈下的脸色非常不好。<br/>　　<br/>　　    陈臣发愣了半天，从包里拿出电话，找到王惊蛰的号码拨了出去，接通之后她就失神的说道：“我感觉不太好，怎么办？”<br/>　　<br/>　　    电话里，王惊蛰叹了口气，说道：“我给你个地址，你过来找我吧·····”<br/>　　<br/>　　    陈臣离家之后没多久，来接陈重的车开进了别墅区，每天早上八点他都会准时走出家门，然后去办公，风雨不误，这是个对生活和自己要求极其自律的人。<br/>　　<br/>　　    车子出了别墅区后，外面的这一条路比较偏，至少要开上十分钟才能上主路，然后进入市区。<br/>　　<br/>　　    陈重的车刚从家中出来，开了没有几分钟，后方两台挂着川牌的越野车就快速开了过来，然后直接横在他这辆车头前，给拦住了。<br/>　　<br/>　　    陈重的司机放下车窗，探出脑袋说道：“有事么，朋友？”<br/>　　<br/>　　    从司机的言语上就能看出，陈重这人的格局有多高了，就他这个身份，司机最正常的反应是破口大骂，但人家尽管看出你是故意拦车的了，说话的时候也是春风细雨的。<br/>　　<br/>　　    前方车里下来一个穿着白西服的青年，满面春风的朝着陈重这辆车走了过来，边走边朝下摆了摆手，示意把车窗放下。<br/>　　<br/>　　    “唰”陈重放下窗户，对方低着腰一手搭在车顶，低着脑袋说道：“不好意思陈先生，这么和你打招呼有点唐突了”<br/>　　<br/>　　    陈重从鼻梁上拿下眼睛，心不在焉的擦了擦，淡淡的说道：“有事？”<br/>　　<br/>　　    “陈先生不愧是川中的传说，真淡定”<br/>　　<br/>　　    陈重笑了，抬头说道：“那还能怎么样？你知道么，如果我被人动一下的话，整个蜀地得有多少人脑袋冒汗啊？”<br/>　　<br/>　　    对方沉吟着，说道：“也有些是不怕死的”<br/>　　<br/>　　    陈重直接伸手指着他说道：“你要是石头缝里蹦出来的，我算你厉害，你不怕死，你一家老小带九族怕不怕？你身边亲朋好友怕不怕，但凡跟你认识的人怕不怕？你死了，别人也好过不了，你怕不怕？但愿你不是从石头缝里蹦出来的······”<br/>　　<br/>　　    “你把我吓住了，陈先生”这青年突然挥起手一巴掌扇在自己脸上。<br/>　　<br/>　　    “啪”<br/>　　<br/>　　    陈重皱了皱眉。<br/>　　<br/>　　    “啪，啪”对方接着又自扇了两巴掌，半边脸瞬间就肿了起来，他擦了下嘴角的血迹，呵呵笑道：“我先给你赔个礼呗，陈先生？”<br/>　　<br/>　　    陈重摇头说道：“有事说事，我公司还有会等着开呢”<br/>　　<br/>　　    “陈先生，是这样的昂”青年从身上拿出纸擦了擦手，咳了一口带血的唾沫说道：“两月前在川南您拿到一块地，过程呢你肯定心里也清楚，我就不说了，我上面的老板让我和您打个招呼，希望您能把那块地让出来，价格我们给您拍到的价格再加上三成，外加再送您两成干股，如何？”<br/>　　<br/>　　    “前年的时候，福布斯的人找到我，说这次榜上会有我的名字，我就问他们，给我的资产定位是多少，福布斯核算部的人说百亿左右，可能上下浮动大概再百分之五吧，我又问他们了，那打算让我排到第几呢？他们说在内地的话，至少前五应该是没问题的，你知道我是怎么回答他们的嘛？”<br/>　　<br/>　　    青年摇了摇头：“您的身家，我也不懂啊”<br/>　　<br/>　　    陈重笑了：“我跟他们说，你要是把我的名字以这种方式放上去的话，知道我底细的人恐怕得要笑掉大牙了，你们这不是捧我，这是让我在给人当笑话看啊，我说撤了吧，别偷摸给我放上去，不然我能告的他们滚出内地市场”<br/>　　<br/>　　    青年叹了口气，说道：“您确实可以”<br/>　　<br/>　　    陈重手指敲了敲车窗，问道：“那你说，我会差钱么？什么两成干股，我会在乎么？我告诉你，我一年扔出去做慈善的钱都不止这个数了，我他么会动心？”<br/>　　<br/>　　    对方顿时沉默了，良久后说道：“您是不同意了？”<br/>　　<br/>　　    “你还有一次威胁我的机会，五分钟！”<br/>　　<br/>　　    “您应该知道，我老板让我过来找您，那就是他肯定有着跟你也差不多的身段，不然档次太低了也没那个胆子是吧？”<br/>　　<br/>　　    “嗯，这一条站得住”<br/>　　<br/>　　    “老板说，和气生财要是不行的话，大家可以掰掰手腕子”<br/>　　<br/>　　    陈重乐了：“文的还是武的啊？”<br/>　　<br/>　　    “他说您拭目以待就行了·······”<br/>　　<br/>　　    青年说完话就走了，陈重的脸色瞬间就阴了下来，司机看了眼前方两台车的车牌，回头问道：“老板，要查一下他们的车么？”<br/>　　<br/>　　    “不用，牌子都没挡，就是让你看见了，也查不出个一二三来，无所谓的事”陈重戴上眼镜，皱眉说道：“哪条过江龙，要来我这里啃骨头了？”
